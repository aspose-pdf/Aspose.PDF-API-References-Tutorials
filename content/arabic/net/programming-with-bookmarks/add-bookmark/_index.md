## مقدمة

هل وجدت نفسك يومًا تتصفح مستندًا طويلاً بتنسيق PDF، وتبحث بشكل يائس عن القسم الذي تحتاجه؟ إذا كان الأمر كذلك، فأنت لست وحدك! قد يكون التنقل عبر المستندات الطويلة أمرًا مزعجًا للغاية. ولكن ماذا لو أخبرتك أن هناك طريقة لجعل ملفات PDF الخاصة بك أكثر سهولة في الاستخدام؟ أدخل الإشارات المرجعية! في هذا البرنامج التعليمي، سنستكشف كيفية إضافة إشارات مرجعية إلى ملف PDF باستخدام Aspose.PDF for .NET. تتيح لك هذه المكتبة القوية التعامل مع مستندات PDF بسهولة، مما يجعل حياتك أبسط كثيرًا. لذا، دعنا نبدأ!

قبل أن نبدأ، هناك بعض الأشياء التي ستحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. فهو بيئة التطوير المتكاملة المثالية لتطوير .NET.
2.  Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك الحصول عليها من[رابط التحميل](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على المتابعة بسلاسة.

## استيراد الحزم

للبدء في إضافة الإشارات المرجعية، ستحتاج إلى استيراد الحزم اللازمة. وإليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. اختر تطبيق وحدة التحكم لتسهيل الأمر.

### إضافة مرجع Aspose.PDF

بمجرد إعداد مشروعك، ستحتاج إلى إضافة مرجع إلى مكتبة Aspose.PDF. يمكنك القيام بذلك من خلال:

- النقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
- تحديد "إدارة حزم NuGet".
- البحث عن "Aspose.PDF" وتثبيته.

### استيراد المساحات المطلوبة

 في الجزء العلوي من`Program.cs` الملف، استيراد المساحات الأساسية الضرورية:

```csharp
using System;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الكود الفعلي لإضافة الإشارات المرجعية!

## الخطوة 1: تحديد دليل المستندات

أولاً، عليك تحديد المسار إلى دليل المستندات. هذا هو المكان الذي سيتم فيه وضع ملف PDF. إليك كيفية القيام بذلك:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملف PDF الخاص بك فيه.

## الخطوة 2: افتح مستند PDF

بعد ذلك، ستحتاج إلى فتح مستند PDF الذي تريد إضافة إشارات مرجعية إليه. استخدم الكود التالي:

```csharp
Document pdfDocument = new Document(dataDir + "AddBookmark.pdf");
```

 يقوم هذا السطر من التعليمات البرمجية بتهيئة سطر جديد`Document` قم بإضافة الكائن إلى ملف PDF الخاص بك.

## الخطوة 3: إنشاء كائن إشارة مرجعية

الآن حان الوقت لإنشاء كائن إشارة مرجعية. هنا يمكنك تحديد عنوان ومظهر الإشارة المرجعية الخاصة بك. وإليك كيفية القيام بذلك:

```csharp
OutlineItemCollection pdfOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfOutline.Title = "Test Outline";
pdfOutline.Italic = true;
pdfOutline.Bold = true;
```

في هذا المثال، نقوم بإنشاء إشارة مرجعية بعنوان "Test Outline" ونجعلها غامقة ومائلة. لا تتردد في تخصيص العنوان كما يحلو لك!

## الخطوة 4: تعيين رقم الصفحة الوجهة

تحتاج كل إشارة مرجعية إلى وجهة. يمكنك تعيين رقم الصفحة التي سترتبط بها الإشارة المرجعية باستخدام الكود التالي:

```csharp
pdfOutline.Action = new GoToAction(pdfDocument.Pages[1]);
```

يحدد هذا السطر إجراء الإشارة المرجعية للانتقال إلى الصفحة الأولى من ملف PDF. يمكنك تغيير رقم الصفحة حسب الحاجة.

## الخطوة 5: إضافة الإشارة المرجعية إلى المستند

الآن بعد أن قمت بإنشاء الإشارة المرجعية الخاصة بك، حان الوقت لإضافتها إلى مجموعة مخططات المستند:

```csharp
pdfDocument.Outlines.Add(pdfOutline);
```

يضيف هذا السطر الإشارة المرجعية التي قمت بإنشائها حديثًا إلى مستند PDF.

## الخطوة 6: احفظ الناتج

أخيرًا، قد ترغب في حفظ مستند PDF المعدّل. إليك كيفية القيام بذلك:

```csharp
dataDir = dataDir + "AddBookmark_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nBookmark added successfully.\nFile saved at " + dataDir);
```

يحفظ هذا الكود ملف PDF مع الإشارة المرجعية المضافة باسم "AddBookmark_out.pdf" في الدليل المحدد.
