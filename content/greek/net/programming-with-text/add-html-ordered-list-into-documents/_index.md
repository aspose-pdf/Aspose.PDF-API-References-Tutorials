Η δημιουργία εγγράφων PDF εν κινήσει μπορεί να ανοίξει έναν κόσμο δυνατοτήτων για τους προγραμματιστές. Σε αυτό το άρθρο, θα εξετάσουμε πώς να προσθέσετε μια λίστα με σειρά HTML σε έγγραφα χρησιμοποιώντας το Aspose.PDF για .NET.

## Βήμα 1: Καθορίστε το Έγγραφο και το Περιεχόμενο HTML σας

Θα ξεκινήσουμε με τη ρύθμιση του εγγράφου PDF και τον ορισμό του περιεχομένου μας HTML που περιλαμβάνει τη λίστα παραγγελίας.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Η διαδρομή προς το έγγραφο εξόδου.
string outFile = dataDir + "AddHTMLOrderedListIntoDocuments_out.pdf";

// Δημιουργία αντικειμένου εγγράφου
Document doc = new Document();
```

Σε αυτό το βήμα, ρυθμίζουμε τις διαδρομές αρχείων όπου θέλουμε να αποθηκεύσουμε το έγγραφο PDF αργότερα.

## Βήμα 2: Δημιουργήστε το τμήμα HTML

 Στη συνέχεια, θα δημιουργήσουμε ένα`HtmlFragment` αντικείμενο που περιέχει το HTML μας. Εδώ θα συμπεριλάβουμε μια ταξινομημένη λίστα μαζί με κάποιο κείμενο.

```csharp
// Δημιουργήστε ένα αντικείμενο HtmlFragment με το αντίστοιχο τμήμα HTML
HtmlFragment htmlFragment = new HtmlFragment("<body style='line-height: 100px;'><ul><li>First</li><li>Second</li><li>Third</li><li>Fourth</li><li>Fifth</li></ul>Text after the list.<br/>Next line<br/>Last line</body>");
```

Εδώ έχουμε δημιουργήσει ένα τμήμα HTML που περιλαμβάνει μια λίστα στοιχείων. Το HTML αποθηκεύεται ως συμβολοσειρά, καθιστώντας εύκολο τον χειρισμό του.

## Βήμα 3: Προσθέστε μια Σελίδα στο Έγγραφο

Τώρα, πρέπει να προσθέσουμε μια σελίδα στο έγγραφο PDF μας. Κάθε PDF πρέπει να έχει σελίδες, και δεν διαφέρουμε!

```csharp
//Προσθήκη σελίδας στη Συλλογή Σελίδων
Page page = doc.Pages.Add();
```

Αυτή η γραμμή κώδικα προσθέτει μια νέα σελίδα στο έγγραφό μας. Να θυμάστε ότι κάθε σελίδα μπορεί να περιέχει διάφορα στοιχεία, όπως κείμενο, εικόνες και το περιεχόμενο HTML μας.

## Βήμα 4: Εισαγάγετε το Τμήμα HTML στη Σελίδα

Εδώ συμβαίνει η μαγεία! Τώρα θα προσθέσουμε το τμήμα HTML που καθορίσαμε προηγουμένως στη σελίδα που μόλις δημιουργήσαμε.

```csharp
// Προσθήκη HtmlFragment στην εσωτερική σελίδα
page.Paragraphs.Add(htmlFragment);
```

Προσθέτοντας το τμήμα HTML στις παραγράφους της σελίδας μας, ουσιαστικά λέμε στο PDF να αποδώσει το HTML μας όπως θα εμφανιζόταν σε ένα πρόγραμμα περιήγησης ιστού.

## Βήμα 5: Αποθηκεύστε το έγγραφο PDF

Με όλο το περιεχόμενό μας στη θέση του, το τελευταίο βήμα είναι να αποθηκεύσετε το έγγραφο στο δίσκο.

```csharp
// Αποθηκεύστε το αρχείο PDF που προκύπτει
doc.Save(outFile);
```

 Εδώ ονομάζουμε το`Save` μέθοδο στο αντικείμενο εγγράφου μας, καθορίζοντας τη διαδρομή του αρχείου εξόδου όπου θα βρίσκεται το νέο μας PDF.
