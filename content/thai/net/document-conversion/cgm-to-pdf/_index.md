## แปลงไฟล์ CGM เป็น PDF

หากต้องการแปลงไฟล์ CGM เป็น PDF ให้ทำตามขั้นตอนเหล่านี้:

```csharp
// พาธไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// สร้างอินสแตนซ์วัตถุ LoadOption โดยใช้ CGMLoadOption
Aspose.Pdf.CgmLoadOptions cgmload = new Aspose.Pdf.CgmLoadOptions();
// สร้างอินสแตนซ์วัตถุเอกสาร
Document doc = new Document(dataDir + "CGMToPDF.CGM", cgmload);
// บันทึกเอกสาร PDF ที่ได้
doc.Save(dataDir + "TECHDRAW_out.pdf");
```

 ในโค้ดด้านบน อย่าลืมเปลี่ยนใหม่`"YOUR DOCUMENTS DIRECTORY"`ด้วยเส้นทางจริงไปยังไดเร็กทอรีที่มีไฟล์ CGM ของคุณที่จะแปลง รหัสนี้จะโหลดไฟล์ CGM โดยใช้นามสกุล`CgmLoadOptions` class จากนั้นสร้างเอกสาร PDF โดยใช้ไฟล์`Document` วัตถุ. ในที่สุด เอกสาร PDF ที่ได้จะถูกบันทึก

### ตัวอย่างซอร์สโค้ดสำหรับ CGM เป็น PDF โดยใช้ Aspose.PDF สำหรับ .NET

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างอินสแตนซ์วัตถุ LoadOption โดยใช้ CGMLoadOption
Aspose.Pdf.CgmLoadOptions cgmload = new Aspose.Pdf.CgmLoadOptions();
// สร้างอินสแตนซ์วัตถุเอกสาร
Document doc = new Document(dataDir + "CGMToPDF.CGM", cgmload);
// บันทึกเอกสาร PDF ที่เป็นผลลัพธ์
doc.Save(dataDir+ "TECHDRAW_out.pdf");
```
