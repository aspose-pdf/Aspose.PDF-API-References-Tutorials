इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके रेखा खींचने के लिए निम्नलिखित C# स्रोत कोड के माध्यम से चरण दर चरण मार्गदर्शन करेंगे।

सुनिश्चित करें कि आपने Aspose.PDF लाइब्रेरी स्थापित कर ली है और शुरू करने से पहले अपना डेवलपमेंट एनवायरनमेंट सेट कर लिया है। साथ ही C# प्रोग्रामिंग का बुनियादी ज्ञान भी रखें।

## चरण 1: दस्तावेज़ निर्देशिका सेटअप

दिए गए सोर्स कोड में, आपको वह डायरेक्टरी निर्दिष्ट करनी होगी जहाँ आप परिणामी PDF फ़ाइल को सहेजना चाहते हैं। "dataDir" वैरिएबल को वांछित डायरेक्टरी में बदलें।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: दस्तावेज़ इंस्टेंस बनाना और पेज जोड़ना

हम डॉक्यूमेंट क्लास का एक उदाहरण बनाते हैं और इस डॉक्यूमेंट में एक पृष्ठ जोड़ते हैं।

```csharp
Document pDoc = new Document();
Page pg = pDoc.Pages.Add();
```

## चरण 3: पेज मार्जिन सेट करना

हमने पृष्ठ के मार्जिन को सभी तरफ 0 पर सेट कर दिया है।

```csharp
pg.PageInfo.Margin.Left = pg.PageInfo.Margin.Right = pg.PageInfo.Margin.Bottom = pg.PageInfo.Margin.Top = 0;
```

## चरण 4: ग्राफ़ ऑब्जेक्ट और पहली पंक्ति बनाना

हम पृष्ठ के बराबर आयामों वाला एक ग्राफ ऑब्जेक्ट बनाते हैं और पृष्ठ के निचले बाएं कोने से ऊपरी दाएं कोने तक जाने वाली पहली रेखा खींचते हैं।

```csharp
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph((float)pg.PageInfo.Width, (float)pg.PageInfo.Height);
Aspose.Pdf.Drawing.Line line = new Aspose.Pdf.Drawing.Line(new float[] { (float)pg.Rect.LLX, 0, (float)pg.PageInfo.Width, (float)pg.Rect. URY });
graph.Shapes.Add(line);
```

## चरण 5: दूसरी रेखा खींचना

हम पृष्ठ के ऊपरी बाएँ कोने से निचले दाएँ कोने तक दूसरी रेखा खींचते हैं।

```csharp
Aspose.Pdf.Drawing.Line line2 = new Aspose.Pdf.Drawing.Line(new float[] { 0, (float)pg.Rect.URY, (float)pg.PageInfo.Width, (float)pg.Rect. LLX });
graph.Shapes.Add(line2);
```

## चरण 6: पृष्ठ पर ग्राफ़ ऑब्जेक्ट जोड़ना

हम पृष्ठ के पैराग्राफ संग्रह में ग्राफ ऑब्जेक्ट जोड़ते हैं।

```csharp
pg.Paragraphs.Add(graph);
```

## चरण 7: परिणामी पीडीएफ फाइल को सेव करना

अंत में, हम परिणामी पीडीएफ फाइल को "DrawingLine_out.pdf" नाम से निर्दिष्ट निर्देशिका में सहेजते हैं।

```csharp
pDoc.Save(dataDir + "DrawingLine_out.pdf");
```

### .NET के लिए Aspose.PDF का उपयोग करके रेखा खींचने के लिए नमूना स्रोत कोड 

```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ उदाहरण बनाएँ
Document pDoc = new Document();
// पीडीएफ दस्तावेज़ के पेज टू पेज संग्रह को जोड़ें
Page pg = pDoc.Pages.Add();
// सभी तरफ पेज मार्जिन को 0 पर सेट करें
pg.PageInfo.Margin.Left = pg.PageInfo.Margin.Right = pg.PageInfo.Margin.Bottom = pg.PageInfo.Margin.Top = 0;
// पृष्ठ आयामों के बराबर चौड़ाई और ऊंचाई के साथ ग्राफ ऑब्जेक्ट बनाएं
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph((float)pg.PageInfo.Width , (float)pg.PageInfo.Height);
// पृष्ठ के निचले-बाएँ कोने से शुरू करके ऊपरी-दाएँ कोने तक पहली पंक्ति ऑब्जेक्ट बनाएँ
Aspose.Pdf.Drawing.Line line = new Aspose.Pdf.Drawing.Line(new float[] { (float)pg.Rect.LLX, 0, (float)pg.PageInfo.Width, (float)pg.Rect.URY });
// ग्राफ़ ऑब्जेक्ट के आकार संग्रह में रेखा जोड़ें
graph.Shapes.Add(line);
// पृष्ठ के ऊपरी-बाएँ कोने से निचले-दाएँ कोने तक रेखा खींचें
Aspose.Pdf.Drawing.Line line2 = new Aspose.Pdf.Drawing.Line(new float[] { 0, (float)pg.Rect.URY, (float)pg.PageInfo.Width, (float)pg.Rect.LLX });
// ग्राफ़ ऑब्जेक्ट के आकार संग्रह में रेखा जोड़ें
graph.Shapes.Add(line2);
// पृष्ठ के पैराग्राफ़ संग्रह में ग्राफ़ ऑब्जेक्ट जोड़ें
pg.Paragraphs.Add(graph);
dataDir = dataDir + "DrawingLine_out.pdf";
// पीडीएफ फाइल सहेजें
pDoc.Save(dataDir);
Console.WriteLine("\nLine drawn successfully across the page.\nFile saved at " + dataDir);            

```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.PDF का उपयोग करके रेखा खींचने का तरीका बताया है। अब आप इस ज्ञान का उपयोग अपनी PDF फ़ाइलों में कस्टम लाइनों के साथ ज्यामितीय आकृतियाँ बनाने के लिए कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: इस ट्यूटोरियल का उद्देश्य क्या है?

उत्तर: इस ट्यूटोरियल का उद्देश्य आपको .NET के लिए Aspose.PDF का उपयोग करके रेखाएँ खींचने की प्रक्रिया के माध्यम से मार्गदर्शन करना है। आप सीखेंगे कि PDF पृष्ठ पर रेखाएँ कैसे बनाएँ और उनकी उपस्थिति को कैसे अनुकूलित करें।

#### प्रश्न: शुरू करने से पहले क्या पूर्व शर्तें आवश्यक हैं?

उत्तर: शुरू करने से पहले, सुनिश्चित करें कि आपने Aspose.PDF लाइब्रेरी स्थापित कर ली है और अपना डेवलपमेंट एनवायरनमेंट सेट कर लिया है। C# प्रोग्रामिंग का बुनियादी ज्ञान भी अनुशंसित है।

#### प्रश्न: मैं पीडीएफ फाइल को सहेजने के लिए निर्देशिका कैसे निर्दिष्ट करूं?

उत्तर: दिए गए स्रोत कोड में "dataDir" वेरिएबल को संशोधित करके वह डायरेक्टरी इंगित करें जहां आप परिणामी PDF फ़ाइल को सहेजना चाहते हैं।

#### प्रश्न: मैं पीडीएफ पेज पर लाइनें कैसे बनाऊं?

उत्तर: ट्यूटोरियल पेज के आयामों के साथ एक ग्राफ ऑब्जेक्ट बनाना और फिर उसमें लाइन ऑब्जेक्ट जोड़ना प्रदर्शित करता है। वांछित रेखाएँ बनाने के लिए लाइन ऑब्जेक्ट के निर्देशांक और गुणों को संशोधित करें।

#### प्रश्न: क्या मैं लाइनों के स्वरूप को अनुकूलित कर सकता हूँ?

उत्तर: हां, आप लाइन ऑब्जेक्ट के गुणों को संशोधित करके लाइनों की उपस्थिति को अनुकूलित कर सकते हैं। इसमें उनके निर्देशांक, रंग, मोटाई और अन्य ग्राफ़िकल विशेषताओं को बदलना शामिल है।

#### प्रश्न: रेखाएँ खींचने के बाद मैं पीडीएफ दस्तावेज़ को कैसे सेव करूँ?

 उत्तर: पृष्ठ पर रेखा ऑब्जेक्ट के साथ ग्राफ ऑब्जेक्ट जोड़ने के बाद, आप परिणामी पीडीएफ दस्तावेज़ को निम्न का उपयोग करके सहेज सकते हैं:`pDoc.Save(dataDir + "DrawingLine_out.pdf");` प्रदान किए गए स्रोत कोड में पंक्ति।

#### प्रश्न: क्या मैं विभिन्न कोणों और दिशाओं वाली रेखाएँ खींच सकता हूँ?

उत्तर: हां, आप ग्राफ के भीतर रेखा ऑब्जेक्ट्स के निर्देशांक और गुणों को समायोजित करके विभिन्न कोणों और अभिविन्यासों के साथ रेखाएं खींच सकते हैं।
