มักจำเป็นต้องกำหนดสิทธิ์การเข้าถึงเฉพาะในไฟล์ PDF ด้วย Aspose.PDF สำหรับ .NET คุณสามารถกำหนดสิทธิ์การเข้าถึงได้อย่างง่ายดายโดยใช้ซอร์สโค้ดต่อไปนี้:

## ขั้นตอนที่ 1: นำเข้าไลบรารีที่จำเป็น

ก่อนที่คุณจะเริ่มต้น คุณต้องนำเข้าไลบรารีที่จำเป็นสำหรับโครงการ C# ของคุณ ต่อไปนี้คือคำสั่งนำเข้าที่จำเป็น:

```csharp
using Aspose.Pdf;
```

## ขั้นตอนที่ 2: ตั้งค่าเส้นทางไปยังโฟลเดอร์เอกสาร

 ในขั้นตอนนี้ คุณต้องระบุเส้นทางไปยังโฟลเดอร์ที่มีไฟล์ PDF ที่คุณต้องการแก้ไข แทนที่`"YOUR DOCUMENTS DIRECTORY"` ในโค้ดต่อไปนี้โดยมีเส้นทางจริงไปยังโฟลเดอร์เอกสารของคุณ:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 3: โหลดไฟล์ PDF ต้นฉบับ

ตอนนี้เราจะโหลดไฟล์ PDF ต้นฉบับโดยใช้โค้ดต่อไปนี้:

```csharp
using (Document document = new Document(dataDir + "input.pdf"))
```

## ขั้นตอนที่ 4: ตั้งค่าสิทธิ์การเข้าถึง

 ในขั้นตอนนี้เราจะสร้างตัวอย่าง`DocumentPrivilege` วัตถุเพื่อกำหนดสิทธิ์การเข้าถึงที่ต้องการ คุณสามารถใช้ข้อจำกัดกับสิทธิ์ทั้งหมดได้โดยใช้`DocumentPrivilege.ForbidAll` ตัวอย่างเช่น หากคุณต้องการอนุญาตเฉพาะการอ่านหน้าจอเท่านั้น คุณสามารถตั้งค่าได้`AllowScreenReaders` ถึง`true`. นี่คือโค้ดที่สอดคล้องกัน:

```csharp
DocumentPrivilege documentPrivilege = DocumentPrivilege.ForbidAll;
documentPrivilege.AllowScreenReaders = true;
```

## ขั้นตอนที่ 5: เข้ารหัสและบันทึกเอกสาร

 ในที่สุดเราก็สามารถเข้ารหัสเอกสาร PDF ด้วยรหัสผ่านผู้ใช้และเจ้าของได้โดยใช้`Encrypt` และระบุอัลกอริทึมการเข้ารหัสที่ต้องการ จากนั้นบันทึกเอกสารที่อัปเดตแล้ว นี่คือรหัสที่เกี่ยวข้อง:

```csharp
document.Encrypt("user", "owner", documentPrivilege, CryptoAlgorithm.AESx128, false);
document.Save(dataDir + "SetPrivileges_out.pdf");
```

### ตัวอย่างโค้ดที่มาสำหรับการกำหนดสิทธิ์โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// โหลดไฟล์ PDF ต้นฉบับ
using (Document document = new Document(dataDir + "input.pdf"))
{
	// สร้างอินสแตนซ์ของวัตถุสิทธิ์เอกสาร
	// ใช้ข้อจำกัดกับสิทธิพิเศษทั้งหมด
	DocumentPrivilege documentPrivilege = DocumentPrivilege.ForbidAll;
	// อนุญาติให้อ่านหน้าจอเท่านั้น
	documentPrivilege.AllowScreenReaders = true;
	// เข้ารหัสไฟล์ด้วยรหัสผ่านผู้ใช้และเจ้าของ
	// จำเป็นต้องตั้งรหัสผ่าน เพื่อให้เมื่อผู้ใช้ดูไฟล์ด้วยรหัสผ่านผู้ใช้
	// เปิดใช้งานตัวเลือกการอ่านหน้าจอเท่านั้น
	document.Encrypt("user", "owner", documentPrivilege, CryptoAlgorithm.AESx128, false);
	// บันทึกเอกสารอัพเดต
	document.Save(dataDir + "SetPrivileges_out.pdf");
}
```
