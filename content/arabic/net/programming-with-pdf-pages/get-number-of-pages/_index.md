## المتطلبات الأساسية
قبل أن تبدأ، تأكد من أن لديك ما يلي:

- المعرفة الأساسية بلغة البرمجة C#
- تم تثبيت Aspose.PDF لـ .NET في بيئة التطوير الخاصة بك

## الخطوة 1: تحديد دليل المستندات
أولاً، عليك تحديد المسار إلى دليل المستندات. هذا هو موقع ملف PDF الذي تريد الحصول على عدد الصفحات الخاصة به. استبدل "دليل المستندات الخاص بك" بالمسار المناسب.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## الخطوة 2: افتح مستند PDF
 ثم يمكنك فتح ملف PDF باستخدام`Document` فئة Aspose.PDF. تأكد من تحديد المسار الصحيح لملف PDF.

```csharp
Document pdfDocument = new Document(dataDir + "GetNumberofPages.pdf");
```

## الخطوة 3: الحصول على عدد الصفحات
 الآن يمكنك الحصول على عدد الصفحات في المستند باستخدام`Count` خاصية الوثيقة`s `مجموعة الصفحات. سيعطيك هذا العدد الإجمالي للصفحات في ملف PDF.

```csharp
System.Console.WriteLine("Number of pages: {0}", pdfDocument.Pages.Count);
```

### عينة من كود المصدر للحصول على عدد الصفحات باستخدام Aspose.PDF لـ .NET 

```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// فتح المستند
Document pdfDocument = new Document(dataDir + "GetNumberofPages.pdf");
// الحصول على عدد الصفحات
System.Console.WriteLine("Page Count : {0}", pdfDocument.Pages.Count);

```
