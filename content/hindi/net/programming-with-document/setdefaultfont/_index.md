यदि आप .NET में PDF दस्तावेज़ों के साथ काम कर रहे हैं, तो आपको ऐसी समस्याओं का सामना करना पड़ सकता है जहाँ PDF में उपयोग किया गया फ़ॉन्ट उस सिस्टम पर उपलब्ध नहीं है जहाँ इसे देखा या प्रिंट किया जा रहा है। इसके परिणामस्वरूप टेक्स्ट गलत तरीके से दिखाई दे सकता है या बिल्कुल भी दिखाई नहीं दे सकता है। Aspose.PDF for .NET आपको दस्तावेज़ के लिए डिफ़ॉल्ट फ़ॉन्ट सेट करने की अनुमति देकर इस समस्या का समाधान प्रदान करता है। इस उदाहरण में, Aspose.PDF for .NET का उपयोग करके डिफ़ॉल्ट फ़ॉन्ट कैसे सेट करें।

## चरण 1: दस्तावेज़ निर्देशिका का पथ सेट करें

हमें उस निर्देशिका का पथ सेट करना होगा जहाँ हमारा PDF दस्तावेज़ स्थित है। हम इस पथ को "dataDir" नामक चर में संग्रहीत करेंगे।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: पीडीएफ दस्तावेज़ लोड करें

 हम एक मौजूदा पीडीएफ दस्तावेज़ लोड करके शुरू करेंगे जिसमें फ़ॉन्ट गायब हैं। इस उदाहरण में, हम मान लेंगे कि पीडीएफ दस्तावेज़ द्वारा निर्दिष्ट निर्देशिका में स्थित है`dataDir` चर।

```csharp
string documentName = dataDir + "input.pdf";
using (System.IO.FileStream fs = new System.IO.FileStream(documentName, System.IO.FileMode.Open))
using (Document document = new Document(fs))
{
    // कोड यहाँ जाता है
}
```

## चरण 3: डिफ़ॉल्ट फ़ॉन्ट सेट करें

 इसके बाद, हम PDF दस्तावेज़ के लिए डिफ़ॉल्ट फ़ॉन्ट सेट करेंगे.`PdfSaveOptions`इस उदाहरण में, हम डिफ़ॉल्ट फ़ॉन्ट को "Arial" पर सेट करेंगे।

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.DefaultFontName = "Arial";
```

## चरण 4: अपडेट किए गए दस्तावेज़ को सहेजें

अंत में, हम अपडेट किए गए दस्तावेज़ को एक नई फ़ाइल में सहेजेंगे। इस उदाहरण में, हम अपडेट किए गए दस्तावेज़ को इनपुट फ़ाइल के समान निर्देशिका में "output_out.pdf" नामक फ़ाइल में सहेजेंगे।

```csharp
document.Save(dataDir + "output_out.pdf", pdfSaveOptions);
```

### .NET के लिए Aspose.PDF का उपयोग करके डिफ़ॉल्ट फ़ॉन्ट सेट करने के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// मौजूदा PDF दस्तावेज़ को गायब फ़ॉन्ट के साथ लोड करें
string documentName = dataDir + "input.pdf";
string newName = "Arial";
using (System.IO.FileStream fs = new System.IO.FileStream(documentName, System.IO.FileMode.Open))
using (Document document = new Document(fs))
{
	PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
	// डिफ़ॉल्ट फ़ॉन्ट नाम निर्दिष्ट करें
	pdfSaveOptions.DefaultFontName = newName;
	document.Save(dataDir + "output_out.pdf", pdfSaveOptions);
}
```

## निष्कर्ष

.NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ों में डिफ़ॉल्ट फ़ॉन्ट सेट करना यह सुनिश्चित करने का एक सरल और प्रभावी तरीका है कि मूल फ़ॉन्ट उपलब्ध न होने पर भी पाठ सही ढंग से प्रदर्शित हो। चरण-दर-चरण मार्गदर्शिका का पालन करके और प्रदान किए गए C# स्रोत कोड का उपयोग करके, डेवलपर्स आसानी से डिफ़ॉल्ट फ़ॉन्ट सेट कर सकते हैं और ऐसे PDF बना सकते हैं जो विभिन्न वातावरणों में एक सुसंगत और विश्वसनीय देखने का अनुभव प्रदान करते हैं। यह सुविधा विशेष रूप से उन परिदृश्यों में उपयोगी है जहाँ PDF को विभिन्न सिस्टम पर देखा या प्रिंट किया जाएगा जिनमें अलग-अलग फ़ॉन्ट सेट इंस्टॉल हो सकते हैं।

### पीडीएफ फाइल में डिफ़ॉल्ट फ़ॉन्ट सेट करने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ों में डिफ़ॉल्ट फ़ॉन्ट सेट करना क्यों महत्वपूर्ण है?

उत्तर: पीडीएफ दस्तावेजों में डिफ़ॉल्ट फ़ॉन्ट सेट करना महत्वपूर्ण है क्योंकि यह सुनिश्चित करता है कि पाठ सही ढंग से प्रदर्शित होगा, भले ही मूल फ़ॉन्ट उस सिस्टम पर उपलब्ध न हों जहाँ पीडीएफ देखा या प्रिंट किया जा रहा है। यह गायब या अस्पष्ट पाठ जैसी समस्याओं को रोकने में मदद करता है, जिससे एक सुसंगत और विश्वसनीय देखने का अनुभव सुनिश्चित होता है।

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF का उपयोग करते हुए किसी भी फ़ॉन्ट को डिफ़ॉल्ट फ़ॉन्ट के रूप में चुन सकता हूँ?

 उत्तर: हां, आप .NET के लिए Aspose.PDF का उपयोग करके सिस्टम पर उपलब्ध किसी भी फ़ॉन्ट को डिफ़ॉल्ट फ़ॉन्ट के रूप में चुन सकते हैं। बस फ़ॉन्ट का नाम निर्दिष्ट करें`DefaultFontName` की संपत्ति`PdfSaveOptions` कक्षा।

#### प्रश्न: यदि निर्दिष्ट डिफ़ॉल्ट फ़ॉन्ट सिस्टम पर उपलब्ध नहीं है तो क्या होगा?

उत्तर: यदि सिस्टम पर निर्दिष्ट डिफ़ॉल्ट फ़ॉन्ट उपलब्ध नहीं है, तो पीडीएफ व्यूअर टेक्स्ट प्रदर्शित करने के लिए फ़ॉलबैक फ़ॉन्ट का उपयोग करेगा। विभिन्न सिस्टम में संगतता सुनिश्चित करने के लिए एरियल या टाइम्स न्यू रोमन जैसे सामान्य रूप से उपलब्ध फ़ॉन्ट चुनना उचित है।
