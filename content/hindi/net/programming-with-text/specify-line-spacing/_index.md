यह ट्यूटोरियल बताता है कि .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में लाइन स्पेसिंग कैसे निर्दिष्ट करें। प्रदान किया गया C# स्रोत कोड चरण दर चरण प्रक्रिया को प्रदर्शित करता है।

## आवश्यक शर्तें

ट्यूटोरियल के साथ आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान।
- .NET लाइब्रेरी के लिए Aspose.PDF स्थापित है। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या अपने प्रोजेक्ट में इसे स्थापित करने के लिए NuGet का उपयोग कर सकते हैं।

## चरण 1: प्रोजेक्ट सेट अप करें

अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाकर आरंभ करें और .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें।

## चरण 2: आवश्यक नामस्थान आयात करें

आवश्यक नामस्थानों को आयात करने के लिए अपनी C# फ़ाइल के आरंभ में निम्नलिखित using निर्देश जोड़ें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System.IO;
```

## चरण 3: दस्तावेज़ निर्देशिका का पथ सेट करें

 का उपयोग करके अपने दस्तावेज़ निर्देशिका का पथ सेट करें`dataDir` चर:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

## चरण 4: इनपुट पीडीएफ फाइल लोड करें

 इनपुट पीडीएफ फाइल को लोड करें`Document` कक्षा:

```csharp
Document doc = new Document();
```

## चरण 5: टेक्स्टफ़ॉर्मेटिंग विकल्प बनाएँ

 एक बनाने के`TextFormattingOptions` ऑब्जेक्ट और लाइन स्पेसिंग मोड को सेट करें`FullSize`:

```csharp
TextFormattingOptions formattingOptions = new TextFormattingOptions();
formattingOptions.LineSpacing = TextFormattingOptions.LineSpacingMode.FullSize;
```

## चरण 6: एक टेक्स्टफ़्रेगमेंट बनाएँ

 एक बनाने के`TextFragment` ऑब्जेक्ट और पाठ सामग्री निर्दिष्ट करें:

```csharp
TextFragment textFragment = new TextFragment("Hello world");
```

## चरण 7: फ़ॉन्ट फ़ाइल लोड करें (वैकल्पिक)

 यदि आप पाठ के लिए कोई विशिष्ट फ़ॉन्ट उपयोग करना चाहते हैं, तो ट्रूटाइप फ़ॉन्ट फ़ाइल को किसी फ़ोल्डर में लोड करें.`FileStream` वस्तु:

```csharp
string fontFile = dataDir + "HPSimplified.TTF";
using (FileStream fontStream = File.OpenRead(fontFile))
{
    textFragment.TextState.Font = FontRepository.OpenFont(fontStream, FontTypes.TTF);
}
```

 प्रतिस्थापित करें`"HPSimplified.TTF"` वास्तविक फ़ॉन्ट फ़ाइल नाम के साथ.

## चरण 8: पाठ की स्थिति और पंक्ति रिक्ति निर्दिष्ट करें

 पाठ खंड के लिए स्थिति निर्धारित करें और असाइन करें`TextFormattingOptions` तक`TextState.FormattingOptions` संपत्ति:

```csharp
textFragment.Position = new Position(100, 600);
textFragment.TextState.FormattingOptions = formattingOptions;
```

## चरण 9: दस्तावेज़ में पाठ जोड़ें

 दस्तावेज़ में पाठ अंश जोड़ें, या तो इसे किसी दस्तावेज़ में जोड़कर या किसी दस्तावेज़ में पाठ अंश जोड़कर।`TextBuilder` या सीधे किसी पेज पर`Paragraphs` संग्रह:

```csharp
var page = doc.Pages.Add();
page.Paragraphs.Add(textFragment);
```

## चरण 10: परिणामी PDF दस्तावेज़ को सहेजें

संशोधित PDF दस्तावेज़ सहेजें:

```csharp
dataDir = dataDir + "SpecifyLineSpacing_out.pdf";
doc.Save(dataDir);
```

 प्रतिस्थापित करना सुनिश्चित करें`"SpecifyLineSpacing_out.pdf"` वांछित आउटपुट फ़ाइल नाम के साथ.

### .NET के लिए Aspose.PDF का उपयोग करके लाइन स्पेसिंग निर्दिष्ट करने के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string fontFile = dataDir + "HPSimplified.TTF";
// इनपुट पीडीएफ फाइल लोड करें
Document doc = new Document();
//LineSpacingMode.FullSize के साथ TextFormattingOptions बनाएँ
TextFormattingOptions formattingOptions = new TextFormattingOptions();
formattingOptions.LineSpacing = TextFormattingOptions.LineSpacingMode.FullSize;
// दस्तावेज़ के पहले पृष्ठ के लिए टेक्स्ट बिल्डर ऑब्जेक्ट बनाएँ
//टेक्स्टबिल्डर टेक्स्टबिल्डर = नया टेक्स्टबिल्डर(doc.Pages[1]);
// नमूना स्ट्रिंग के साथ पाठ खंड बनाएँ
TextFragment textFragment = new TextFragment("Hello world");
if (fontFile != "")
{
	// स्ट्रीम ऑब्जेक्ट में ट्रू टाइप फ़ॉन्ट लोड करें
	using (FileStream fontStream = System.IO.File.OpenRead(fontFile))
	{
		// टेक्स्ट स्ट्रिंग के लिए फ़ॉन्ट नाम सेट करें
		textFragment.TextState.Font = FontRepository.OpenFont(fontStream, FontTypes.TTF);
		//टेक्स्ट फ़्रैगमेंट के लिए स्थिति निर्दिष्ट करें
		textFragment.Position = new Position(100, 600);
		//वर्तमान खंड के TextFormattingOptions को पूर्वनिर्धारित पर सेट करें (जो LineSpacingMode.FullSize को इंगित करता है)
		textFragment.TextState.FormattingOptions = formattingOptions;
		// टेक्स्ट को TextBuilder में जोड़ें ताकि उसे PDF फ़ाइल पर रखा जा सके
		//टेक्स्टबिल्डर.AppendText(टेक्स्टफ़्रैगमेंट);
		var page = doc.Pages.Add();
		page.Paragraphs.Add(textFragment);
	}
	dataDir = dataDir + "SpecifyLineSpacing_out.pdf";
	// परिणामी PDF दस्तावेज़ सहेजें
	doc.Save(dataDir);
}
```

## निष्कर्ष

बधाई हो! आपने सफलतापूर्वक सीख लिया है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में लाइन स्पेसिंग कैसे निर्दिष्ट करें। इस ट्यूटोरियल में प्रोजेक्ट सेट अप करने से लेकर संशोधित दस्तावेज़ को सहेजने तक, चरण-दर-चरण मार्गदर्शिका प्रदान की गई है। अब आप PDF फ़ाइलों में टेक्स्ट की लाइन स्पेसिंग को कस्टमाइज़ करने के लिए इस कोड को अपने स्वयं के C# प्रोजेक्ट में शामिल कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: "पीडीएफ फाइल में लाइन स्पेसिंग निर्दिष्ट करें" ट्यूटोरियल का उद्देश्य क्या है?

उत्तर: "पीडीएफ फाइल में लाइन स्पेसिंग निर्दिष्ट करें" ट्यूटोरियल का उद्देश्य उपयोगकर्ताओं को यह बताना है कि पीडीएफ दस्तावेज़ में टेक्स्ट की लाइन स्पेसिंग को कस्टमाइज़ करने के लिए .NET के लिए Aspose.PDF लाइब्रेरी का उपयोग कैसे करें। ट्यूटोरियल प्रक्रिया को प्रदर्शित करने के लिए चरण-दर-चरण निर्देश और C# कोड नमूने प्रदान करता है।

#### प्रश्न: यह ट्यूटोरियल पीडीएफ दस्तावेज़ में लाइन स्पेसिंग निर्दिष्ट करने में कैसे मदद करता है?

उत्तर: यह ट्यूटोरियल उपयोगकर्ताओं को यह समझने में मदद करता है कि PDF दस्तावेज़ में टेक्स्ट के लिए लाइन स्पेसिंग निर्दिष्ट करने के लिए .NET के लिए Aspose.PDF की क्षमताओं का उपयोग कैसे करें। दिए गए चरणों और कोड उदाहरणों का पालन करके, उपयोगकर्ता अपनी प्राथमिकताओं के अनुसार लाइन स्पेसिंग को समायोजित कर सकते हैं।

#### प्रश्न: इस ट्यूटोरियल का अनुसरण करने के लिए क्या पूर्वापेक्षाएँ आवश्यक हैं?

उत्तर: ट्यूटोरियल शुरू करने से पहले, आपको C# प्रोग्रामिंग भाषा की बुनियादी समझ होनी चाहिए। इसके अतिरिक्त, आपके पास .NET लाइब्रेरी के लिए Aspose.PDF स्थापित होना चाहिए। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या NuGet का उपयोग करके अपने प्रोजेक्ट में इंस्टॉल कर सकते हैं।

#### प्रश्न: मैं इस ट्यूटोरियल का अनुसरण करने के लिए अपना प्रोजेक्ट कैसे सेट करूँ?

उत्तर: आरंभ करने के लिए, अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाएँ और .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें। यह आपको PDF दस्तावेज़ों के साथ काम करने और लाइन स्पेसिंग को अनुकूलित करने के लिए लाइब्रेरी की सुविधाओं का लाभ उठाने में सक्षम बनाता है।

#### प्रश्न: क्या मैं किसी भी प्रकार के पाठ के लिए पंक्ति रिक्ति निर्दिष्ट करने के लिए इस ट्यूटोरियल का उपयोग कर सकता हूँ?

उत्तर: हाँ, यह ट्यूटोरियल .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में किसी भी टेक्स्ट सामग्री के लिए लाइन स्पेसिंग निर्दिष्ट करने के निर्देश प्रदान करता है। आप अपनी ज़रूरतों के अनुसार टेक्स्ट की लाइन स्पेसिंग को समायोजित करने के लिए दिए गए कोड नमूनों का उपयोग कर सकते हैं।

#### प्रश्न: मैं ट्यूटोरियल में लाइन स्पेसिंग मोड कैसे निर्दिष्ट करूं?

 उत्तर: ट्यूटोरियल दर्शाता है कि कैसे बनाएं`TextFormattingOptions` ऑब्जेक्ट और इसे सेट करें`LineSpacing` संपत्ति को`TextFormattingOptions.LineSpacingMode.FullSize`यह मोड पाठ सामग्री के लिए पूर्ण पंक्ति रिक्ति निर्दिष्ट करता है।

#### प्रश्न: मैं पाठ के लिए विशिष्ट फ़ॉन्ट कैसे लोड कर सकता हूँ?

 उत्तर: यदि आप पाठ सामग्री के लिए एक विशिष्ट फ़ॉन्ट का उपयोग करना चाहते हैं, तो ट्यूटोरियल एक ट्रू टाइप फ़ॉन्ट फ़ाइल को लोड करने के तरीके के बारे में मार्गदर्शन प्रदान करता है।`FileStream` ऑब्जेक्ट और इसे फ़ॉन्ट के रूप में सेट करें`TextFragment`यह आपको पाठ के फ़ॉन्ट के साथ-साथ उसकी पंक्ति रिक्ति को भी अनुकूलित करने में सक्षम बनाता है।

#### प्रश्न: मैं पीडीएफ दस्तावेज़ में पाठ की स्थिति को कैसे अनुकूलित करूं?

 उत्तर: पाठ की स्थिति को अनुकूलित करने के लिए, एक बनाएँ`TextFragment` ऑब्जेक्ट और इसे सेट करें`Position`प्रॉपर्टी को वांछित निर्देशांक (X और Y) पर सेट करें। यह आपको यह नियंत्रित करने की अनुमति देता है कि पीडीएफ दस्तावेज़ में टेक्स्ट कहाँ रखा जाए।

#### प्रश्न: क्या मैं इन लाइन स्पेसिंग संशोधनों को मौजूदा पीडीएफ दस्तावेज़ों पर लागू कर सकता हूं?

 उत्तर: हां, आप मौजूदा पीडीएफ दस्तावेजों में टेक्स्ट के लिए लाइन स्पेसिंग को संशोधित कर सकते हैं। ट्यूटोरियल दर्शाता है कि कैसे एक पीडीएफ दस्तावेज़ बनाया जाए।`TextFragment` निर्दिष्ट पंक्ति रिक्ति और स्थिति के साथ, और फिर इसे पृष्ठ के`Paragraphs` संग्रह।
