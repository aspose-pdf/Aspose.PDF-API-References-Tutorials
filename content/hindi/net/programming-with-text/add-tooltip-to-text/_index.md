## परिचय

जब आकर्षक और इंटरैक्टिव PDF बनाने की बात आती है, तो टूलटिप्स अमूल्य हो सकते हैं। क्या आप उन छोटे पॉप-अप बॉक्स को जानते हैं जो आपको किसी चीज़ पर माउस घुमाने पर अतिरिक्त जानकारी देते हैं? वे आपके दस्तावेज़ को अव्यवस्थित किए बिना संदर्भ, विवरण या सलाह के कुछ अंश भी प्रदान कर सकते हैं। इस ट्यूटोरियल में, हम .NET लाइब्रेरी के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में टेक्स्ट में टूलटिप्स जोड़ने का तरीका बताएंगे। चाहे आप एक अनुभवी डेवलपर हों या PDF की दुनिया में अभी-अभी कदम रख रहे हों, आप सही जगह पर हैं! तो चलिए शुरू करते हैं!

## आवश्यक शर्तें

इससे पहले कि हम कोडिंग भाग में प्रवेश करें, आइए यह सुनिश्चित कर लें कि आपके पास सुचारू रूप से आगे बढ़ने के लिए आवश्यक सभी चीजें मौजूद हैं।

### Visual Studio स्थापित
आपके मशीन पर Visual Studio स्थापित होना आवश्यक है, क्योंकि यह .NET अनुप्रयोगों के लिए आपका प्राथमिक विकास वातावरण होगा।

### .NET लाइब्रेरी के लिए Aspose.PDF
 आपको Aspose.PDF लाइब्रेरी की भी आवश्यकता होगी।[यहाँ पर डाउनलोड करो](https://releases.aspose.com/pdf/net/). इसे अपने प्रोजेक्ट संदर्भों में शामिल करना सुनिश्चित करें।

### C# का बुनियादी ज्ञान
C# की पृष्ठभूमि बहुत मददगार होगी क्योंकि हम उसी भाषा में कोडिंग करेंगे। लेकिन चिंता न करें - मैं आपको हर कदम पर मार्गदर्शन करूँगा!

### काम करने के लिए एक पीडीएफ दस्तावेज़
आप एक खाली पीडीएफ दस्तावेज़ से शुरुआत कर सकते हैं, जैसा कि हमने इस उदाहरण में किया है, या यदि आप चाहें तो किसी मौजूदा दस्तावेज़ का उपयोग कर सकते हैं।

अब, चलिए कोडिंग भाग की ओर बढ़ते हैं!

## पैकेज आयात करें 

 हमारे कोडिंग एडवेंचर में पहला कदम आवश्यक पैकेज आयात करना है। अपना विज़ुअल स्टूडियो प्रोजेक्ट खोलें, और अपनी C# फ़ाइल के शीर्ष पर, आप निम्न जोड़ना चाहेंगे`using` निर्देश:

```csharp
using Aspose.Pdf.Forms;
using Aspose.Pdf.Text;
```

ये पैकेज आपको पीडीएफ दस्तावेजों को बनाने और उनमें हेरफेर करने के लिए आवश्यक सभी कक्षाओं और कार्यात्मकताओं तक पहुंच प्रदान करते हैं।

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

सबसे पहले, हमें वह पथ सेट करना होगा जहाँ आप अपने दस्तावेज़ सहेजेंगे। इसे अपने फ़ाइल सिस्टम में एक आरामदायक स्थान खोजने के रूप में सोचें जहाँ आपकी सभी रचनाएँ रहेंगी।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string outputFile = dataDir + "Tooltip_out.pdf";
```

 प्रतिस्थापित करना सुनिश्चित करें`YOUR DOCUMENT DIRECTORY` आपके मशीन पर वास्तविक पथ के साथ.

## चरण 2: एक नमूना पीडीएफ दस्तावेज़ बनाएँ

अब बारी है कुछ टेक्स्ट के साथ एक सरल पीडीएफ बनाने की। यहीं से हम अपनी रचनात्मक प्रक्रिया की शुरुआत करते हैं!

```csharp
//पाठ के साथ नमूना दस्तावेज़ बनाएँ
Document doc = new Document();
doc.Pages.Add().Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a tooltip"));
doc.Pages[1].Paragraphs.Add(new TextFragment("Move the mouse cursor here to display a very long tooltip"));
doc.Save(outputFile);
```

इस चरण में, हम एक दस्तावेज़ बनाते हैं, दो पाठ अंश जोड़ते हैं, और इसे हमारे पहले से निर्दिष्ट पथ पर सहेजते हैं।

## चरण 3: प्रसंस्करण के लिए दस्तावेज़ खोलें

अब जबकि हमने अपना दस्तावेज़ बना लिया है, तो चलिए इसे खोलते हैं ताकि हम उन टूलटिप्स पर काम कर सकें!

```csharp
// पाठ सहित दस्तावेज़ खोलें
Document document = new Document(outputFile);
```

यहां, हम केवल उस दस्तावेज़ को लोड करते हैं जिसे हमने अभी बनाया है।

## चरण 4: टेक्स्ट के टुकड़े ढूंढने के लिए टेक्स्ट एब्जॉर्बर बनाएं

हमें उन टेक्स्ट अंशों को ढूँढ़ना होगा जहाँ हम टूलटिप्स जोड़ना चाहते हैं। यह एक बड़े नक्शे के किसी खास हिस्से को हाइलाइट करने के लिए आवर्धक कांच का उपयोग करने जैसा है! 

```csharp
// नियमित अभिव्यक्ति से मेल खाने वाले सभी वाक्यांशों को खोजने के लिए TextAbsorber ऑब्जेक्ट बनाएं
TextFragmentAbsorber absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a tooltip");
document.Pages.Accept(absorber);
```

## चरण 5: पाठ अंश निकालें

इसके बाद, हम पिछले चरण से प्राप्त पाठ अंशों को निकालते हैं।

```csharp
// निकाले गए पाठ अंश प्राप्त करें
TextFragmentCollection textFragments = absorber.TextFragments;
```

यह स्निपेट हमें उन पाठ अंशों के संदर्भों को बनाए रखने में सक्षम बनाता है जिनमें हमारी रुचि है।

## चरण 6: टुकड़ों के माध्यम से लूप करें और टूलटिप्स जोड़ें

अब आता है मज़ेदार हिस्सा! हम हर टेक्स्ट फ़्रैगमेंट को लूप करेंगे और हर एक में एक टूलटिप जोड़ेंगे। कल्पना करें कि कुछ ख़ास आइटम (टेक्स्ट फ़्रैगमेंट) के चारों ओर छोटे-छोटे उपहार (टूलटिप्स) लपेटे गए हैं।

```csharp
// टुकड़ों के माध्यम से लूप
foreach (TextFragment fragment in textFragments)
{
	// पाठ खंड स्थिति पर अदृश्य बटन बनाएं
	ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
	// AlternateName मान को व्यूअर एप्लिकेशन द्वारा टूलटिप के रूप में प्रदर्शित किया जाएगा
	field.AlternateName = "Tooltip for text.";
	// दस्तावेज़ में बटन फ़ील्ड जोड़ें
	document.Form.Add(field);
}
```

प्रत्येक पुनरावृत्ति में, हम एक बटन फ़ील्ड बनाते हैं जो पाठ खंड की स्थिति के अनुरूप होता है और उसे टूलटिप पाठ निर्दिष्ट करते हैं।

## चरण 7: लंबे टूलटिप्स के लिए दोहराएँ

जैसे हमने एक सरल टूलटिप जोड़ा है, वैसे ही हम लंबे टेक्स्ट के लिए भी ऐसा कर सकते हैं। आइए अपनी रचनात्मकता को आगे बढ़ाएं!

```csharp
// अगला नमूना बहुत लंबे टूलटिप का होगा
absorber = new TextFragmentAbsorber("Move the mouse cursor here to display a very long tooltip");
document.Pages.Accept(absorber);
textFragments = absorber.TextFragments;
foreach (TextFragment fragment in textFragments)
{
	ButtonField field = new ButtonField(fragment.Page, fragment.Rectangle);
	// बहुत लंबा पाठ सेट करें
	field.AlternateName = "Lorem ipsum dolor sit amet, consectetur adipiscing elit," +
							" sed do eiusmod tempor incididunt ut labore et dolore magna" +
							" aliqua. Ut enim ad minim veniam, quis nostrud exercitation" +
							" ullamco laboris nisi ut aliquip ex ea commodo consequat." +
							" Duis aute irure dolor in reprehenderit in voluptate velit" +
							" esse cillum dolore eu fugiat nulla pariatur. Excepteur sint" +
							" occaecat cupidatat non proident, sunt in culpa qui officia" +
							" deserunt mollit anim id est laborum.";
	document.Form.Add(field);
}
```

यहां, हम पहले जैसा ही काम कर रहे हैं, लेकिन अधिक विस्तृत टूलटिप के साथ।

## चरण 8: अपना दस्तावेज़ सहेजें

अंतिम चरण अपने दस्तावेज़ को उन सभी चमकदार नए टूलटिप्स के साथ सहेजना है। 

```csharp
// दस्तावेज़ सहेजें
document.Save(outputFile);
```

और बस, आपका काम हो गया! आपने अपने PDF में टूलटिप्स जोड़ दिए हैं, जिससे यह अधिक उपयोगकर्ता-अनुकूल और इंटरैक्टिव हो गया है।

## निष्कर्ष

यह रहा वह - .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइलों में टेक्स्ट में टूलटिप्स जोड़ने के बारे में एक आसान-से-अनुसरण गाइड। यह तकनीक उपयोगकर्ता अनुभव को महत्वपूर्ण रूप से बढ़ा सकती है, जिससे आपके दस्तावेज़ अधिक जानकारीपूर्ण बन सकते हैं, बिना पाठक को एक बार में बहुत अधिक टेक्स्ट से अभिभूत किए। 

किसी शब्द या वाक्यांश पर बस मँडराते ही पाठक को प्रासंगिक जानकारी मिल जाती है जो बिना किसी अव्यवस्था के मूल्य जोड़ती है। तो, अपनी आस्तीन ऊपर चढ़ाएँ और इसे आज़माएँ! इससे पहले कि आप इसे जानें, आप सभी प्रकार के आकर्षक दस्तावेज़ बना सकते हैं जो अलग दिखते हैं।
