Прежде чем погрузиться в код, нам нужно добавить необходимые библиотеки в наш проект. Вот как это можно быстро сделать:

### Включить директиву Using 

В верхней части файла C# добавьте следующие директивы using:
```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

Завершив необходимую настройку, давайте приступим к написанию кода!

Готовы изменить работу ссылок в PDF-документах? Давайте разберем код пошагово, чтобы сделать настройку целевых ссылок управляемой.

## Шаг 1: Определите каталог документов 

Прежде всего, вам нужно указать, где находятся ваши документы. Это место, где будут храниться ваши входные и выходные файлы. 

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

-  Объяснение: Заменить`"YOUR DOCUMENT DIRECTORY"` с фактическим путем, где находится ваш PDF-файл (`UpdateLinks.pdf`) проживает.

## Шаг 2: Загрузите PDF-файл 

Теперь давайте загрузим PDF-файл, который вы хотите изменить. 

```csharp
Document document = new Document(dataDir + "UpdateLinks.pdf");
```

-  Пояснение: Здесь мы создаем новый`Document` объект. Эта строка считывает PDF-файл из указанного каталога.

## Шаг 3: Доступ к аннотации ссылки 

Далее нам необходимо получить доступ к аннотации ссылки, которую вы хотите изменить. 

```csharp
LinkAnnotation linkAnnot = (LinkAnnotation)document.Pages[1].Annotations[1];
```

- Объяснение: Эта строка извлекает аннотацию ссылки со второй страницы вашего PDF. Аннотации имеют нулевой индекс, поэтому отрегулируйте индекс соответствующим образом на основе структуры вашего PDF.

## Шаг 4: Обновите пункт назначения

Вот часть, в которой вы настраиваете место назначения вашей ссылки:

```csharp
GoToRemoteAction goToR = (GoToRemoteAction)linkAnnot.Action;
// Следующая строка обновляет место назначения, не обновляя файл
goToR.Destination = new XYZExplicitDestination(2, 0, 0, 1.5);
```

-  Объяснение:`GoToRemoteAction` позволяет нам изменять действие ссылки.`XYZExplicitDestination` устанавливает целевую страницу (устанавливается на страницу 2), координаты x и y (обе устанавливаются на 0) и уровень масштабирования (устанавливается на 1,5). Не стесняйтесь настраивать эти параметры в соответствии со своими потребностями!

## Шаг 5: Укажите файл для ссылки 

Теперь, если необходимо, давайте сделаем ссылку на другой файл. 

```csharp
//Следующая строка файла обновления
goToR.File = new FileSpecification(dataDir + "input.pdf");
```

- Объяснение: Эта строка задает целевой файл, который откроет наша ссылка. Убедитесь, что файл существует в указанном каталоге.

## Шаг 6: Сохраните документ. 

Наконец-то пришло время сохранить документ со всеми внесенными вами изменениями. 

```csharp
dataDir = dataDir + "SetTargetLink_out.pdf";
// Сохраните документ с обновленной ссылкой
document.Save(dataDir);
Console.WriteLine("\nTarget link setup successfully.\nFile saved at " + dataDir);
```

-  Пояснение: Выходной файл имеет имя`SetTargetLink_out.pdf`, и документ сохраняется с внесенными изменениями. Консоль выводит сообщение с подтверждением и путем к файлу.
