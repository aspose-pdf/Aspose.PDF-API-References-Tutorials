## परिचय

डिजिटल युग में, PDF दस्तावेज़ों को साझा करने के लिए एक मुख्य साधन बन गए हैं। वे बहुमुखी, सुरक्षित हैं, और उनमें बहुत सारी जानकारी हो सकती है, जिसमें अनुलग्नक भी शामिल हैं। क्या आपने कभी सोचा है कि PDF फ़ाइल से उन सभी छिपे हुए रत्नों को कैसे निकाला जाए? खैर, आप भाग्यशाली हैं! इस ट्यूटोरियल में, हम PDF फ़ाइल में सभी अनुलग्नकों को प्राप्त करने के लिए .NET के लिए Aspose.PDF का उपयोग करने के बारे में जानेंगे। चाहे आप एक अनुभवी डेवलपर हों या अभी शुरुआत कर रहे हों, यह गाइड आपको चरण दर चरण प्रक्रिया से परिचित कराएगा।

## आवश्यक शर्तें

इससे पहले कि हम कोड में आगे बढ़ें, आइए सुनिश्चित करें कि आपके पास आरंभ करने के लिए आवश्यक सभी चीजें मौजूद हैं:

1. विज़ुअल स्टूडियो: सुनिश्चित करें कि आपके मशीन पर विज़ुअल स्टूडियो स्थापित है। यह .NET विकास के लिए सबसे उपयोगी IDE है।
2.  .NET के लिए Aspose.PDF: आपको Aspose.PDF लाइब्रेरी डाउनलोड और इंस्टॉल करनी होगी। आप इसे पा सकते हैं[यहाँ](https://releases.aspose.com/pdf/net/).
3. C# का बुनियादी ज्ञान: C# प्रोग्रामिंग से परिचित होने से आपको कोड स्निपेट को बेहतर ढंग से समझने में मदद मिलेगी।

## पैकेज आयात करें

आरंभ करने के लिए, आपको अपने C# प्रोजेक्ट में आवश्यक पैकेज आयात करने होंगे। इसे करने का तरीका यहां बताया गया है:

### एक नया प्रोजेक्ट बनाएं

Visual Studio खोलें और एक नया C# प्रोजेक्ट बनाएँ। सरलता के लिए कंसोल एप्लिकेशन चुनें।

### Aspose.PDF संदर्भ जोड़ें

1. समाधान एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करें।
2. “NuGet पैकेज प्रबंधित करें” चुनें।
3. “Aspose.PDF” खोजें और नवीनतम संस्करण स्थापित करें।

### नामस्थान आयात करें

अपनी C# फ़ाइल के शीर्ष पर, Aspose.PDF नामस्थान आयात करें

```csharp
using System.IO;
using Aspose.Pdf;
using System;
```

अब जबकि हमने अपना परिवेश तैयार कर लिया है, तो आइए पीडीएफ फाइल से अनुलग्नक निकालने की बारीकियों पर आते हैं।

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

सबसे पहले, आपको अपने दस्तावेज़ निर्देशिका का पथ निर्दिष्ट करना होगा। यहीं पर आपकी पीडीएफ फाइल स्थित होगी।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`YOUR DOCUMENT DIRECTORY` वास्तविक पथ के साथ जहाँ आपकी PDF फ़ाइल संग्रहीत है। यह महत्वपूर्ण है क्योंकि प्रोग्राम को यह जानना आवश्यक है कि फ़ाइल को कहाँ देखना है।

## चरण 2: पीडीएफ दस्तावेज़ खोलें

इसके बाद, हम Aspose.PDF लाइब्रेरी का उपयोग करके PDF दस्तावेज़ खोलेंगे। यहीं से जादू शुरू होता है!

```csharp
Document pdfDocument = new Document(dataDir + "GetAlltheAttachments.pdf");
```

 यहाँ, हम एक नया निर्माण करते हैं`Document` ऑब्जेक्ट और पीडीएफ फाइल का पथ पास करें। सुनिश्चित करें कि फ़ाइल का नाम एक्सटेंशन सहित बिल्कुल मेल खाता है।

## चरण 3: एम्बेडेड फ़ाइल संग्रह तक पहुँचें

अब जब हमने दस्तावेज़ खोल लिया है, तो आइए एम्बेडेड फ़ाइल संग्रह तक पहुँचें। यह वह जगह है जहाँ सभी अनुलग्नक संग्रहीत हैं।

```csharp
EmbeddedFileCollection embeddedFiles = pdfDocument.EmbeddedFiles;
```

इस पंक्ति के साथ, हम सभी एम्बेडेड फ़ाइलों को एक संग्रह में खींच रहे हैं, जिसे हम आसानी से लूप कर सकते हैं।

## चरण 4: एम्बेडेड फ़ाइलों की गणना करें

यह जानना हमेशा अच्छा होता है कि आप कितने अटैचमेंट से निपट रहे हैं। आइए एंबेडेड फ़ाइलों की कुल संख्या प्रिंट करें।

```csharp
Console.WriteLine("Total files : {0}", embeddedFiles.Count);
```

इससे आपको त्वरित जानकारी मिल जाएगी कि आपके पीडीएफ में कितने अनुलग्नक हैं।

## चरण 5: अनुलग्नकों के माध्यम से लूप करें

अब आता है मज़ेदार हिस्सा! हम एम्बेडेड फ़ाइल संग्रह में प्रत्येक फ़ाइल विनिर्देश के माध्यम से लूप करेंगे और विवरण निकालेंगे।

```csharp
int count = 1;

foreach (FileSpecification fileSpecification in embeddedFiles)
{
    Console.WriteLine("Name: {0}", fileSpecification.Name);
    Console.WriteLine("Description: {0}", fileSpecification.Description);
    Console.WriteLine("Mime Type: {0}", fileSpecification.MIMEType);
```

इस लूप में, हम प्रत्येक अटैचमेंट का नाम, विवरण और MIME प्रकार प्रिंट कर रहे हैं। इससे आपको अपने PDF के अंदर क्या है, इसकी स्पष्ट तस्वीर मिलती है।

## चरण 6: अतिरिक्त पैरामीटर की जाँच करें

कुछ अनुलग्नकों में अतिरिक्त पैरामीटर हो सकते हैं। आइए जाँचें कि क्या वे मौजूद हैं और उन्हें प्रिंट करें।

```csharp
if (fileSpecification.Params != null)
{
    Console.WriteLine("CheckSum: {0}", fileSpecification.Params.CheckSum);
    Console.WriteLine("Creation Date: {0}", fileSpecification.Params.CreationDate);
    Console.WriteLine("Modification Date: {0}", fileSpecification.Params.ModDate);
    Console.WriteLine("Size: {0}", fileSpecification.Params.Size);
}
```

यह चरण सुनिश्चित करता है कि आप अनुलग्नकों के बारे में कोई महत्वपूर्ण विवरण न चूकें।

## चरण 7: अनुलग्नकों को निकालें और सहेजें

अंत में, आइए प्रत्येक अनुलग्नक की सामग्री को निकालें और उसे एक फ़ाइल में सेव करें। यहीं पर आपको अपनी मेहनत के नतीजे दिखेंगे!

```csharp
byte[] fileContent = new byte[fileSpecification.Contents.Length];
fileSpecification.Contents.Read(fileContent, 0, fileContent.Length);
FileStream fileStream = new FileStream(dataDir + count + "_out" + ".txt", FileMode.Create);
fileStream.Write(fileContent, 0, fileContent.Length);
fileStream.Close();
count += 1;
```

 इस कोड में, हम प्रत्येक अनुलग्नक की सामग्री को बाइट सरणी में पढ़ते हैं और फिर उसे एक नई फ़ाइल में लिखते हैं। फ़ाइलों को क्रमिक रूप से नाम दिया जाएगा (उदाहरण के लिए,`1_out.txt`, `2_out.txt`, वगैरह।)।

## निष्कर्ष

और अब आपका काम हो गया! आपने .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल से सभी अनुलग्नक सफलतापूर्वक निकाल लिए हैं। यह शक्तिशाली लाइब्रेरी PDF दस्तावेज़ों में हेरफेर करना और उनके छिपे हुए खजाने तक पहुँचना आसान बनाती है। चाहे आप किसी व्यक्तिगत प्रोजेक्ट या पेशेवर एप्लिकेशन पर काम कर रहे हों, अनुलग्नकों को निकालने का तरीका जानना अविश्वसनीय रूप से उपयोगी हो सकता है।
