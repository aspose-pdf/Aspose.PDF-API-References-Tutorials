## ขั้นตอนที่ 1: โหลดเอกสาร PDF ของคุณ

สิ่งแรกที่ต้องทำคือโหลดเอกสาร PDF ที่คุณวางแผนจะเพิ่มคำอธิบายประกอบ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DATA DIRECTORY";
// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

 นี่คือสิ่งที่เกิดขึ้น: คุณกำลังระบุไดเรกทอรีที่เก็บไฟล์ PDF ของคุณ จากนั้นโหลดโดยใช้`Document` คลาสที่จัดทำโดย Aspose.PDF ขั้นตอนนี้มีความสำคัญมาก เนื่องจากหากไม่โหลดเอกสาร คุณจะไม่สามารถเปลี่ยนแปลงอะไรได้เลย

## ขั้นตอนที่ 2: สร้างคำอธิบายประกอบ

### การกำหนดคุณสมบัติของคำอธิบายประกอบ
 ตอนนี้เรามาสร้างคำอธิบายประกอบกัน เราจะใช้`TextAnnotation`ซึ่งเหมาะสำหรับการเพิ่มความคิดเห็นหรือหมายเหตุลงใน PDF ของคุณ

```csharp
// สร้างคำอธิบายประกอบ
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

ในสคริปท์นี้:
-  ที่ตั้งและขนาด :`Rectangle` คลาสกำหนดว่าคำอธิบายประกอบของคุณจะปรากฏที่ใดบนเพจและมีขนาดเท่าใด
- ชื่อเรื่อง เรื่อง และเนื้อหา: คุณสมบัติเหล่านี้ช่วยให้คุณระบุได้ว่าคำอธิบายประกอบของคุณเกี่ยวกับอะไรและจะมีเนื้อหาอะไรบ้าง
-  ไอคอน:`TextIcon.Key` กำหนดไอคอนสำหรับคำอธิบายเพื่อให้ดูน่าสนใจยิ่งขึ้น

## ขั้นตอนที่ 3: ปรับแต่งลักษณะของคำอธิบายประกอบ

ต่อไปเรามาทำให้คำอธิบายประกอบนี้โดดเด่นโดยการเพิ่มเส้นขอบและปรับแต่งรูปลักษณ์ของมัน

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

นี่คือรายละเอียดของสิ่งที่เกิดขึ้น:
-  ชายแดน : เราสร้าง`Border` วัตถุและกำหนดความกว้างเป็น 5 โดยให้คำอธิบายของเรามีโครงร่างที่โดดเด่น
-  รูปแบบเส้นประ:`Dash` คุณสมบัตินี้ช่วยให้คุณสร้างเส้นขอบแบบเส้นประ ซึ่งจะเพิ่มสไตล์เล็กน้อยให้กับคำอธิบายประกอบ

## ขั้นตอนที่ 4: เพิ่มคำอธิบายลงในหน้า PDF

หลังจากสร้างและปรับแต่งคำอธิบายประกอบแล้ว ก็ถึงเวลาที่จะเพิ่มคำอธิบายประกอบลงในหน้า PDF ของคุณ

```csharp
// เพิ่มคำอธิบายลงในคอลเล็กชันคำอธิบายของหน้า
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

 โค้ดนี้จะเพิ่มคำอธิบายลงในหน้าแรกของ PDF ของคุณ`Annotations` คอลเลกชันจะเก็บคำอธิบายประกอบทั้งหมดสำหรับหน้าเฉพาะ และขั้นตอนนี้จะทำให้แน่ใจว่าคำอธิบายประกอบใหม่ของคุณเป็นส่วนหนึ่งของคอลเลกชันนั้น

## ขั้นตอนที่ 5: บันทึกเอกสาร PDF ที่อัปเดต

สุดท้ายนี้ ให้บันทึกเอกสารเพื่อให้คำอธิบายของคุณถูกเพิ่มไว้อย่างถาวร

```csharp
// บันทึกไฟล์เอาท์พุต
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

โดยการบันทึกเอกสารด้วยชื่อใหม่ (`AddAnnotation_out.pdf`) คุณจะรักษาไฟล์ต้นฉบับไว้และสร้างไฟล์ใหม่พร้อมคำอธิบายประกอบ ข้อความคอนโซลจะยืนยันว่าทุกอย่างสำเร็จแล้ว และตอนนี้คุณสามารถค้นหา PDF ที่มีคำอธิบายประกอบในไดเร็กทอรีที่ระบุได้
