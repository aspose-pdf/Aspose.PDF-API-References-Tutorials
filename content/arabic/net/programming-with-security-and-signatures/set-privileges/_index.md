غالبًا ما يكون من الضروري تعيين امتيازات وصول محددة في ملف PDF. باستخدام Aspose.PDF لـ .NET، يمكنك بسهولة تعيين امتيازات الوصول باستخدام كود المصدر التالي:

## الخطوة 1: استيراد المكتبات المطلوبة

قبل أن تبدأ، عليك استيراد المكتبات اللازمة لمشروع C# الخاص بك. فيما يلي تعليمات الاستيراد اللازمة:

```csharp
using Aspose.Pdf;
```

## الخطوة 2: تعيين المسار إلى مجلد المستندات

 في هذه الخطوة، تحتاج إلى تحديد المسار إلى المجلد الذي يحتوي على ملف PDF الذي تريد تحريره. استبدل`"YOUR DOCUMENTS DIRECTORY"` في الكود التالي مع المسار الفعلي لمجلد المستندات الخاص بك:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## الخطوة 3: تحميل ملف PDF المصدر

الآن سوف نقوم بتحميل ملف PDF المصدر باستخدام الكود التالي:

```csharp
using (Document document = new Document(dataDir + "input.pdf"))
```

## الخطوة 4: تعيين امتيازات الوصول

 في هذه الخطوة، سنقوم بإنشاء مثيل`DocumentPrivilege` كائن لتعيين امتيازات الوصول المطلوبة. يمكنك تطبيق القيود على جميع الامتيازات باستخدام`DocumentPrivilege.ForbidAll` على سبيل المثال، إذا كنت تريد السماح بقراءة الشاشة فقط، يمكنك ضبط`AllowScreenReaders` ل`true`. هذا هو الكود المقابل:

```csharp
DocumentPrivilege documentPrivilege = DocumentPrivilege.ForbidAll;
documentPrivilege.AllowScreenReaders = true;
```

## الخطوة 5: تشفير المستند وحفظه

 أخيرًا، يمكننا تشفير مستند PDF باستخدام كلمة مرور المستخدم والمالك باستخدام`Encrypt` وتحديد خوارزمية التشفير المطلوبة. ثم نقوم بحفظ المستند المحدث. وهذا هو الكود المقابل:

```csharp
document.Encrypt("user", "owner", documentPrivilege, CryptoAlgorithm.AESx128, false);
document.Save(dataDir + "SetPrivileges_out.pdf");
```

### عينة من كود المصدر لتعيين الامتيازات باستخدام Aspose.PDF لـ .NET 
```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// تحميل ملف PDF المصدر
using (Document document = new Document(dataDir + "input.pdf"))
{
	// إنشاء كائن امتيازات المستند
	// تطبيق القيود على كافة الامتيازات
	DocumentPrivilege documentPrivilege = DocumentPrivilege.ForbidAll;
	// السماح بقراءة الشاشة فقط
	documentPrivilege.AllowScreenReaders = true;
	// تشفير الملف باستخدام كلمة مرور المستخدم والمالك.
	// يجب تعيين كلمة المرور، بحيث بمجرد أن يتمكن المستخدم من عرض الملف بكلمة مرور المستخدم،
	// تم تمكين خيار قراءة الشاشة فقط
	document.Encrypt("user", "owner", documentPrivilege, CryptoAlgorithm.AESx128, false);
	// حفظ المستند المحدث
	document.Save(dataDir + "SetPrivileges_out.pdf");
}
```

## خاتمة

تهانينا! لديك الآن دليل خطوة بخطوة لتعيين امتيازات الوصول إلى مستند PDF باستخدام Aspose.PDF لـ .NET. يمكنك استخدام هذا الكود لتطبيق قيود معينة وحماية ملفات PDF الخاصة بك حسب الحاجة.

تأكد من مراجعة وثائق Aspose.PDF الرسمية للحصول على مزيد من المعلومات حول ميزات الأمان المتقدمة لمستندات PDF وإدارة امتيازات الوصول.

### الأسئلة الشائعة حول تعيين الامتيازات في ملف PDF

#### س: لماذا أحتاج إلى تعيين امتيازات الوصول في ملف PDF؟

ج: يتيح لك ضبط امتيازات الوصول التحكم في كيفية تفاعل المستخدمين مع مستندات PDF الخاصة بك. يمكنك تقييد إجراءات مثل الطباعة والنسخ والتحرير لتعزيز أمان المستندات.

#### س: كيف يمكنني الاستفادة من إعداد امتيازات الوصول باستخدام Aspose.PDF لـ .NET؟

ج: يوفر Aspose.PDF لـ .NET طريقة مباشرة لتطبيق امتيازات الوصول، مما يتيح لك القدرة على تخصيص أذونات المستخدم وحماية المحتوى الحساس.

#### س: هل يمكنني تطبيق امتيازات مختلفة لمستخدمين مختلفين؟

ج: نعم، يمكنك تعيين امتيازات وصول محددة لمجموعات مستخدمين مختلفة، مما يتيح لك ضبط الوصول إلى المستندات استنادًا إلى أدوار المستخدم.

#### س: ما هي بعض امتيازات الوصول المشتركة التي يمكنني تعيينها؟

أ: تتضمن امتيازات الوصول الشائعة السماح بإجراءات مثل الطباعة أو نسخ النصوص أو الصور أو تعديل المستند أو ملء حقول النموذج أو منعها.

#### س: كيف يساعد إعداد امتياز قراءة الشاشة على تعزيز إمكانية الوصول إلى المستندات؟

أ: إن تمكين امتياز قراءة الشاشة يضمن أن يتمكن المستخدمون من الوصول إلى محتوى ملف PDF باستخدام برامج قراءة الشاشة، مما يعزز إمكانية الوصول للأشخاص الذين يعانون من ضعف البصر.

#### س: هل يمكنني إعداد حماية كلمة المرور مع امتيازات الوصول؟

ج: بالتأكيد، يمكنك تشفير مستند PDF الخاص بك باستخدام كلمات مرور أثناء تطبيق امتيازات الوصول. وهذا يوفر طبقة إضافية من الأمان.

#### س: هل هناك طريقة لإلغاء امتيازات الوصول بعد تطبيقها؟

ج: بمجرد تطبيق امتيازات الوصول وتشفير المستند، سيحتاج المستخدمون إلى كلمة المرور المناسبة للوصول إلى المحتوى. ويمكن تعديل الامتيازات عن طريق تغيير الكود المصدر.

#### س: هل هناك أي اعتبارات تتعلق بالأداء عند تعيين امتيازات الوصول؟

ج: يكون تأثير الأداء ضئيلًا، حيث يتم تطبيق إعدادات امتيازات الوصول أثناء التشفير، وهي عملية سريعة.

#### س: هل يمكنني تطبيق امتيازات الوصول إلى مستند PDF موجود؟

ج: نعم، يمكنك استخدام Aspose.PDF لـ .NET لتطبيق امتيازات الوصول إلى مستندات PDF الجديدة والموجودة.
