## مقدمة

هل سبق لك أن فتحت ملف PDF لتجد أن الإشارات المرجعية كلها مطوية؟ قد يكون الأمر محبطًا، خاصة عندما تحاول التنقل عبر مستند طويل. لحسن الحظ، مع Aspose.PDF for .NET، يمكنك بسهولة توسيع الإشارات المرجعية في ملفات PDF الخاصة بك برمجيًا. سيرشدك هذا الدليل خلال العملية خطوة بخطوة، مما يضمن فهمك لكل جزء من الكود وكيفية عمله. لذا، احصل على مشروبك المفضل، ودعنا نغوص في عالم معالجة PDF!

## المتطلبات الأساسية

قبل أن نبدأ، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. فهو البيئة الأفضل لتطوير .NET.
2.  Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك العثور عليها[هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، تحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة التحكم لتبسيط الأمر.

### إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.PDF" وقم بتثبيت الإصدار الأحدث.

### استيراد مساحة الاسم

في الجزء العلوي من ملف C#، قم باستيراد مساحة اسم Aspose.PDF:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
```

الآن بعد أن قمت بإعداد كل شيء، دعنا ننتقل إلى الكود الفعلي!

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، عليك تحديد المسار إلى دليل المستندات. هذا هو المكان الذي يوجد فيه ملف PDF المدخل وحيث سيتم حفظ ملف الإخراج.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: افتح مستند PDF

 بعد ذلك، ستفتح مستند PDF الذي يحتوي على الإشارات المرجعية التي تريد توسيعها. يتم ذلك باستخدام`Document` الفئة من مكتبة Aspose.PDF.

```csharp
// فتح المستند
Document doc = new Document(dataDir + "input-bookmark.pdf");
```

## الخطوة 3: ضبط وضع عرض الصفحة

الآن، عليك ضبط وضع عرض الصفحة للمستند. يحدد هذا كيفية عرض ملف PDF عند فتحه. في هذه الحالة، نريد استخدام الخطوط العريضة (الإشارات المرجعية).

```csharp
//تعيين وضع عرض الصفحة، أي إظهار الصور المصغرة، أو ملء الشاشة، أو إظهار لوحة المرفقات
doc.PageMode = PageMode.UseOutlines;
```

## الخطوة 4: التنقل عبر عناصر المخطط التفصيلي

وهنا يأتي الجزء الممتع! سوف تتصفح كل عنصر من عناصر المخطط التفصيلي في مجموعة المخططات التفصيلية لملف PDF وتضبط حالة فتحه على "صحيح". وسوف يؤدي هذا إلى توسيع الإشارات المرجعية.

```csharp
// قم بالمرور عبر كل عنصر من عناصر المخطط التفصيلي في مجموعة المخططات التفصيلية لملف PDF
foreach (OutlineItemCollection item in doc.Outlines)
{
    // تعيين حالة مفتوحة لعنصر المخطط التفصيلي
    item.Open = true;
}
```

## الخطوة 5: احفظ المستند الناتج

بعد توسيع الإشارات المرجعية، حان الوقت لحفظ المستند المعدّل. ستحدد اسم ملف جديد لملف PDF الناتج.

```csharp
dataDir = dataDir + "ExpandBookmarks_out.pdf";
// حفظ الناتج
doc.Save(dataDir);
```

## الخطوة 6: رسالة التأكيد

أخيرًا، يمكنك طباعة رسالة تأكيد على وحدة التحكم لإعلامك بأن توسيع الإشارات المرجعية تم بنجاح.

```csharp
Console.WriteLine("\nBookmarks expanded successfully.\nFile saved at " + dataDir);
```
