## การแนะนำ

เอกสาร PDF มีอยู่ทั่วไปในโลกดิจิทัลในปัจจุบัน โดยมักใช้สำหรับทุกอย่างตั้งแต่รายงานทางการไปจนถึงหนังสืออิเล็กทรอนิกส์ แต่การแยกข้อความออกจากเอกสารเหล่านี้บางครั้งอาจรู้สึกเหมือนพยายามหาเข็มในมหาสมุทร โชคดีที่ด้วยความช่วยเหลือของไลบรารีอย่าง Aspose.PDF สำหรับ .NET งานนี้จึงง่ายขึ้นมาก ในคู่มือนี้ เราจะแนะนำวิธีแยกส่วนข้อความเฉพาะจาก PDF โดยใช้ Aspose.PDF โดยแบ่งขั้นตอนออกเป็นขั้นตอนที่ชัดเจนและดำเนินการได้

## ขั้นตอนที่ 1: เริ่มต้นเอกสาร

หากต้องการทำงานกับ PDF คุณต้องโหลดไฟล์ดังกล่าวลงในแอปพลิเคชันของคุณ ดังต่อไปนี้:

```csharp
// โหลดเอกสาร PDF
Document pdfDocument = new Document("Path/To/Your/PDFFile.pdf");
```
 แทนที่`"Path/To/Your/PDFFile.pdf"` ด้วยเส้นทางจริงของ PDF ของคุณ

## ขั้นตอนที่ 2: สร้าง TextFragmentAbsorber

 ตอนนี้มาสำรวจตัวอย่างการใช้งานที่แตกต่างกัน`TextFragmentAbsorber` เพื่อดึงข้อความ

### ตัวอย่างที่ 1: แยกคำที่แน่นอน

 เพื่อค้นหาการจับคู่ที่ตรงกันกับคำเฉพาะ คุณสามารถเริ่มต้นคำด้วย`TextFragmentAbsorber` แบบนี้:

```csharp
// ค้นหาคำที่ตรงกันกับคำว่า "Word"
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"\bWord\b", new TextSearchOptions(true));
```
 รหัสนี้ตั้งค่าตัวดูดซับเพื่อค้นหาคำว่า "Word" อย่างแม่นยำ โดยถือว่าเป็นคำทั้งคำ`\b` หมายถึงขอบเขตของคำ

### ตัวอย่างที่ 2: การค้นหาโดยไม่คำนึงถึงตัวพิมพ์เล็ก/ใหญ่

หากคุณต้องการค้นหาสตริงเฉพาะโดยไม่คำนึงถึงตัวพิมพ์ใหญ่เล็ก (เช่น "บรรทัด") คุณสามารถทำได้ดังนี้:

```csharp
// ค้นหาสตริง "Line" โดยไม่คำนึงถึงตัวพิมพ์เล็ก-ใหญ่
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("(?i)Line", new TextSearchOptions(true));
```
 การ`(?i)` ทำให้การค้นหาไม่คำนึงถึงตัวพิมพ์ใหญ่เล็ก 

### ตัวอย่างที่ 3: แยกข้อความทั้งหมดจาก PDF

เคยจำเป็นต้องแยกวิเคราะห์ข้อความทั้งหมดใน PDF หรือไม่ นี่คือวิธีง่ายๆ ในการดำเนินการ:

```csharp
// วิเคราะห์สตริงทั้งหมดภายในเอกสาร PDF
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"[\S]+");
```
 เรกซ์`[\S]+` จับอักขระที่ไม่ใช่ช่องว่างทั้งหมด 

## ขั้นตอนที่ 3: ยอมรับ Absorber ในเอกสาร

ตอนนี้คุณต้องยอมรับตัวดูดซับในเอกสาร PDF:

```csharp
// ยอมรับตัวดูดซับในเอกสาร
pdfDocument.Pages.Accept(textFragmentAbsorber);
```
บรรทัดนี้จะแจ้งให้เอกสาร PDF ดำเนินการกับตัวดูดซับที่คุณได้เริ่มต้นไว้

## ขั้นตอนที่ 4: ดึงข้อมูลและแสดงผลลัพธ์

ในที่สุด เราจะดึงข้อความที่แยกออกมาและแสดงออกมา คุณสามารถทำได้ดังนี้:

```csharp
// รับชิ้นส่วนข้อความที่แยกออกมา
TextFragmentCollection textFragments = textFragmentAbsorber.TextFragments;

// วนซ้ำผ่านแต่ละส่วนและส่งออก
foreach (TextFragment textFragment in textFragments)
{
    Console.WriteLine(textFragment.Text);
}
```
ลูปนี้จะประมวลผลแต่ละส่วนที่พบและพิมพ์ไปยังคอนโซล
