การเพิ่มคำอธิบายประกอบให้กับเอกสาร PDF เป็นคุณสมบัติอันทรงพลังที่สามารถปรับปรุงกระบวนการทำงานร่วมกันและการตรวจสอบ Aspose.PDF สำหรับ .NET ทำให้การเพิ่มคำอธิบายประกอบลงในเอกสาร PDF โดยใช้ C# โดยทางโปรแกรมเป็นเรื่องง่าย ในคู่มือนี้ เราจะอธิบายวิธีใช้ Aspose.PDF สำหรับ .NET เพื่อเพิ่มคำอธิบายประกอบลงในเอกสาร PDF ทีละขั้นตอน

## ขั้นตอนที่ 1: สร้างโปรเจ็กต์ใหม่และติดตั้ง Aspose.PDF สำหรับ .NET

ก่อนที่เราจะเริ่มเขียนโค้ดเพื่อเพิ่มคำอธิบายประกอบ เราจำเป็นต้องสร้างโปรเจ็กต์ใหม่และติดตั้ง Aspose.PDF สำหรับ .NET เมื่อต้องการติดตั้ง Aspose.PDF สำหรับ .NET ให้ทำตามขั้นตอนเหล่านี้:

1. เปิด Visual Studio และสร้างโครงการ C# ใหม่
2. คลิกขวาที่โครงการใน Solution Explorer และเลือก "จัดการแพ็คเกจ NuGet"
3. ค้นหา "Aspose.PDF" และเลือก "ติดตั้ง"

เมื่อการติดตั้งเสร็จสิ้น เราก็สามารถเริ่มเขียนโค้ดได้เลย

## ขั้นตอนที่ 2: เปิดเอกสาร PDF

ขั้นตอนแรกในการเพิ่มคำอธิบายประกอบคือเปิดเอกสาร PDF เราสามารถใช้รหัสต่อไปนี้เพื่อเปิดเอกสาร:

```csharp
string dataDir = "YOUR DATA DIRECTORY";
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

ในโค้ดนี้ เราระบุเส้นทางไปยังเอกสาร PDF ที่เราต้องการเปิด ตรวจสอบให้แน่ใจว่าได้แทนที่ "ไดเรกทอรีข้อมูลของคุณ" ด้วยเส้นทางจริงไปยังไดเรกทอรีข้อมูลของคุณ

## ขั้นตอนที่ 3: สร้างคำอธิบายประกอบ

 ในการเพิ่มคำอธิบายประกอบ เราจำเป็นต้องสร้างอินสแตนซ์ใหม่ของ`TextAnnotation` ระดับ. เราสามารถใช้โค้ดต่อไปนี้เพื่อสร้างคำอธิบายประกอบข้อความใหม่:

```csharp
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.State = AnnotationState.Accepted;
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

ในโค้ดนี้ เราสร้างคำอธิบายประกอบข้อความใหม่บนหน้าที่สองของเอกสาร PDF นอกจากนี้เรายังตั้งชื่อ หัวเรื่อง สถานะ เนื้อหา เปิด และคุณสมบัติไอคอนของคำอธิบายประกอบ

## ขั้นตอนที่ 4: ปรับแต่งคำอธิบายประกอบ

 เราสามารถปรับแต่งลักษณะที่ปรากฏของคำอธิบายประกอบได้โดยใช้`Border` ระดับ. เราสามารถใช้โค้ดต่อไปนี้เพื่อปรับแต่งเส้นขอบของคำอธิบายประกอบ:

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

 ในโค้ดนี้ เราจะสร้างโค้ดใหม่`Border`object และตั้งค่าคุณสมบัติความกว้างและเส้นประ จากนั้นเราก็ตั้งค่า`Border` คุณสมบัติของคำอธิบายประกอบไปใหม่`Border` วัตถุ. ในที่สุดเราก็ตั้งค่า`Rect` คุณสมบัติของคำอธิบายประกอบเพื่อระบุตำแหน่งและขนาด

## ขั้นตอนที่ 5: เพิ่มคำอธิบายประกอบลงในเอกสาร PDF

เมื่อเราสร้างและปรับแต่งคำอธิบายประกอบแล้ว เราจำเป็นต้องเพิ่มลงในเอกสาร PDF เราสามารถใช้โค้ดต่อไปนี้เพื่อเพิ่มคำอธิบายประกอบให้กับเอกสาร PDF:

```csharp
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

ในโค้ดนี้ เราเพิ่มคำอธิบายประกอบลงในคอลเลกชันคำอธิบายประกอบของหน้าที่สองของเอกสาร PDF

## ขั้นตอนที่ 6: บันทึกไฟล์เอาต์พุต

สุดท้าย เราต้องบันทึกเอกสาร PDF พร้อมคำอธิบายประกอบที่เพิ่มเข้ามา เราสามารถใช้โค้ดต่อไปนี้เพื่อบันทึกไฟล์เอาต์พุต:

```csharp
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
```
### ตัวอย่างซอร์สโค้ดสำหรับการเพิ่มคำอธิบายประกอบโดยใช้ Aspose.PDF สำหรับ .NET


```csharp   
 // เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DATA DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");

// สร้างคำอธิบายประกอบ
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.State = AnnotationState.Accepted;
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;

Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);

// เพิ่มคำอธิบายประกอบในคอลเลกชันคำอธิบายประกอบของเพจ
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
dataDir = dataDir + "AddAnnotation_out.pdf";
// บันทึกไฟล์เอาต์พุต
pdfDocument.Save(dataDir);
```
รหัสนี้สาธิตวิธีการเพิ่มคำอธิบายประกอบข้อความที่มีชื่อ หัวเรื่อง สถานะ เนื้อหา และไอคอนเฉพาะลงในหน้า PDF โดยใช้ Aspose.PDF สำหรับ .NET คุณสามารถแก้ไขโค้ดนี้ตามความต้องการของคุณในการเพิ่มคำอธิบายประกอบลงในเอกสาร PDF ของคุณ เพียงอย่าลืมแทนที่ DATA DIRECTORY ของคุณด้วยเส้นทางไดเร็กทอรีจริงซึ่งมีไฟล์ PDF ของคุณอยู่ และตำแหน่งที่คุณต้องการบันทึกไฟล์เอาต์พุต
