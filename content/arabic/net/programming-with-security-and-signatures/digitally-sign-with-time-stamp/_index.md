في هذا البرنامج التعليمي، سنوضح لك عملية التوقيع الرقمي على ملف PDF بعلامة زمنية باستخدام Aspose.PDF لـ .NET. يضمن التوقيع الرقمي بعلامة زمنية صحة وسلامة المستند، من خلال إضافة بصمة إلكترونية بعلامة زمنية.

## الخطوة 1: المتطلبات الأساسية

قبل أن تبدأ، تأكد من أن لديك المتطلبات الأساسية التالية:

- المعرفة الأساسية بلغة البرمجة C#
- تثبيت Visual Studio على جهازك
- تم تثبيت مكتبة Aspose.PDF لـ .NET

## الخطوة 2: إعداد البيئة

للبدء، اتبع الخطوات التالية لإعداد بيئة التطوير الخاصة بك:

1. افتح Visual Studio وقم بإنشاء مشروع C# جديد.
2. استيراد المساحات المطلوبة إلى ملف الكود الخاص بك:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Forms;
```

## الخطوة 3: التوقيع الرقمي مع الطابع الزمني

الخطوة الأولى هي تنفيذ التوقيع الرقمي مع الطابع الزمني على ملف PDF. يوضح الكود المقدم كيفية تحقيق هذا التوقيع باستخدام Aspose.PDF لـ .NET.

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
string pfxFile = "";
using (Document document = new Document(dataDir + @"DigitallySign.pdf"))
{
     using (PdfFileSignature signature = new PdfFileSignature(document))
     {
         PKCS7 pkcs = new PKCS7(pfxFile, "pfx_password");
         TimestampSettings timestampSettings = new TimestampSettings("https:\\your_timestamp_settings", "user:password");
         pkcs. TimestampSettings = timestampSettings;
         System.Drawing.Rectangle rect = new System.Drawing.Rectangle(100, 100, 200, 100);
         signature.Sign(1, "Reason for signing", "Contact", "Location", true, rect, pkcs);
         signature.Save(dataDir + "DigitallySignWithTimeStamp_out.pdf");
     }
}
```

يقوم هذا الكود بتحميل ملف PDF، وإنشاء توقيع رقمي مع علامة زمنية باستخدام ملف PFX (مفتاح خاص) ومعلمات علامة زمنية محددة. ثم تتم إضافة التوقيع إلى ملف PDF وحفظه باللاحقة "_خارج".

### عينة من كود المصدر للتوقيع الرقمي باستخدام ختم الوقت باستخدام Aspose.PDF لـ .NET 
```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
string pfxFile = "";
using (Document document = new Document(dataDir + @"DigitallySign.pdf"))
{
	using (PdfFileSignature signature = new PdfFileSignature(document))
	{
		PKCS7 pkcs = new PKCS7(pfxFile, "pfx_password");
		TimestampSettings timestampSettings = new TimestampSettings("https:\\your_timestamp_settings", "user:password"); // يمكن حذف المستخدم/كلمة المرور
		pkcs.TimestampSettings = timestampSettings;
		System.Drawing.Rectangle rect = new System.Drawing.Rectangle(100, 100, 200, 100);
		// إنشاء أي من أنواع التوقيع الثلاثة
		signature.Sign(1, "Signature Reason", "Contact", "Location", true, rect, pkcs);
		// حفظ ملف PDF الناتج
		signature.Save(dataDir + "DigitallySignWithTimeStamp_out.pdf");
	}
}
```

## خاتمة

تهانينا! لقد نجحت في تنفيذ توقيع رقمي مع ختم زمني على ملف PDF باستخدام Aspose.PDF for .NET. تناول هذا البرنامج التعليمي العملية خطوة بخطوة من إنشاء التوقيع إلى حفظ ملف PDF المحدث. يمكنك الآن استخدام هذه الميزة لإضافة توقيعات رقمية مع ختم زمني إلى ملفات PDF الخاصة بك.

### الأسئلة الشائعة حول التوقيع الرقمي باستخدام ختم الوقت في ملف PDF

#### س: ما هو الغرض من التوقيع الرقمي باستخدام ختم زمني؟

أ: التوقيع الرقمي باستخدام علامة زمنية يضيف بصمة إلكترونية مع علامة زمنية إلى ملف PDF، مما يضمن صحة المستند وسلامته في نقطة زمنية محددة.

#### س: ما هي المتطلبات الأساسية اللازمة لبدء هذا البرنامج التعليمي؟

ج: قبل أن تبدأ، تأكد من أن لديك فهمًا أساسيًا للغة البرمجة C#، وأنك قمت بتثبيت Visual Studio، وأنك قمت بتثبيت مكتبة Aspose.PDF لـ .NET.

#### س: كيف يمكنني إعداد بيئة التطوير الخاصة بي؟

أ: اتبع الخطوات المقدمة لإعداد بيئة التطوير الخاصة بك، بما في ذلك إنشاء مشروع C# جديد في Visual Studio واستيراد المساحات الأساسية الضرورية.

#### س: كيف يمكنني إضافة توقيع رقمي مع علامة زمنية إلى ملف PDF؟

أ: يوضح رمز العينة المقدم كيفية تحميل ملف PDF وإنشاء توقيع رقمي بعلامة زمنية باستخدام ملف PFX (مفتاح خاص) وإعدادات علامة زمنية محددة، وإضافة التوقيع إلى ملف PDF، وحفظ الملف المحدث.

#### س: ما هو ملف PFX، ولماذا يتم استخدامه في المثال؟

ج: يحتوي ملف PFX (تنسيق التبادل الشخصي) على مفتاح خاص وشهادة. ويُستخدم هنا لتوفير وظيفة تشفير للتوقيعات الرقمية. تأكد من استبدال العنصر النائب بملف PFX وكلمة المرور.

#### س: ما هي TimestampSettings؟

أ: تحدد TimestampSettings إعدادات خادم الطوابع الزمنية المستخدم لإضافة الطوابع الزمنية الإلكترونية إلى التوقيع. وهي تتضمن عنوان URL لخادم الطوابع الزمنية وبيانات اعتماد المستخدم الاختيارية.

#### س: هل يمكنني استخدام خادم طابع زمني غير الخادم الموجود في المثال؟
 ج: نعم، يمكنك استخدام أي خادم متوافق للطوابع الزمنية. استبدل عنوان URL، وإذا لزم الأمر، قم بتوفير بيانات اعتماد المستخدم المناسبة في`TimestampSettings` هدف.

#### س: ما هو الغرض من تحديد مستطيل التوقيع؟

أ: يحدد مستطيل التوقيع موضع وأبعاد ظهور التوقيع الرقمي على صفحة PDF. اضبط هذه القيم لتحديد موضع التوقيع بالشكل المطلوب.

#### س: ماذا يحدث إذا كان خادم الطابع الزمني غير متاح أثناء التوقيع؟

ج: إذا لم يكن خادم الطابع الزمني متاحًا أثناء التوقيع، فقد تفشل العملية أو تستغرق وقتًا أطول. تأكد من أن خادم الطابع الزمني الخاص بك موثوق به ويمكن الوصول إليه.

#### س: كيف يمكنني التحقق من وجود علامة زمنية في ملف PDF الموقع؟

 ج: يمكنك فحص ملف PDF الموقّع باستخدام الكود النموذجي المقدم.`TimestampSettings` يجب أن تكون المعلومات التي استخدمتها أثناء التوقيع متاحة في تفاصيل التوقيع.

#### س: هل التوقيعات الرقمية التي تحمل الطوابع الزمنية ملزمة قانونًا؟

ج: تتمتع التوقيعات الرقمية التي تحمل علامات زمنية بقيمة قانونية في العديد من الولايات القضائية، وغالبًا ما تعتبر أكثر موثوقية من التوقيعات الرقمية البسيطة. استشر خبراء قانونيين في ولايتك القضائية للحصول على لوائح محددة.

#### س: هل يمكنني إضافة توقيعات رقمية متعددة مع الطوابع الزمنية إلى ملف PDF؟

ج: نعم، يمكنك إضافة توقيعات رقمية متعددة مع علامات زمنية إلى ملف PDF عن طريق استدعاء عملية إنشاء التوقيع عدة مرات. سيكون لكل توقيع علامة زمنية خاصة به.
