## مقدمة

في العصر الرقمي الحالي، أصبحت ملفات PDF منتشرة في كل مكان. من التقارير إلى الأدلة، تخدم هذه الملفات العديد من الأغراض في مختلف الصناعات. ومع ذلك، فإن ضمان إمكانية وصول الجميع إلى هذه المستندات، بما في ذلك الأفراد ذوي الإعاقة، أمر بالغ الأهمية. وهنا يأتي دور ملفات PDF التي تحمل علامات. فهي تسهل على برامج قراءة الشاشة وغيرها من التقنيات المساعدة تفسير محتوى المستندات بشكل فعال.

## المتطلبات الأساسية

قبل أن نبدأ في الترميز، من الضروري التأكد من أن كل شيء جاهز. إليك ما ستحتاج إليه:

1. Visual Studio - أي إصدار حديث سيفي بالغرض، ولكن من الأفضل أن يكون لديك الإصدار الأحدث للحصول على الأداء الوظيفي الأمثل.
2.  Aspose.PDF for .NET - يمكنك بسهولة تنزيل أحدث إصدار من[رابط التحميل](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C# - ستكون المعرفة بأساسيات برمجة C# مفيدة أثناء تنفيذ الكود.

## استيراد الحزم

بمجرد إعداد البيئة الخاصة بك، تتضمن الخطوة التالية استيراد الحزم اللازمة لاستخدام Aspose.PDF. يمكنك القيام بذلك عن طريق إضافة مكتبة Aspose.PDF إلى مشروعك.

### تثبيت Aspose.PDF عبر NuGet

1. افتح مشروع Visual Studio الخاص بك: إذا لم تقم بإنشاء مشروع بعد، فاستمر في إنشاء مشروع جديد.
2. الوصول إلى مدير حزم NuGet: انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3. ابحث عن Aspose.PDF: أدخل "Aspose.PDF" في شريط البحث.
4. تثبيت الحزمة: انقر فوق "تثبيت" لإضافة الحزمة إلى مشروعك. بمجرد التثبيت، يمكنك البدء!

### إضافة باستخدام التوجيه

للاستفادة من وظيفة Aspose في الكود الخاص بك، تحتاج إلى إضافة التوجيه using الضروري في الجزء العلوي من ملفك:

```csharp
using Aspose.Pdf.LogicalStructure;
using Aspose.Pdf.Tagged;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

بمجرد إكمال هذه الخطوات، ستكون جاهزًا لبدء عملية إنشاء ملف PDF المميز الخاص بك!

الآن بعد أن قمت بإعداد كل شيء، فلننتقل إلى المهمة الرئيسية: إنشاء مستند PDF مُوسم. فيما يلي دليل خطوة بخطوة حول كيفية القيام بذلك باستخدام Aspose.PDF لـ .NET.

## الخطوة 1: تحديد دليل المستندات

أولاً، ستحتاج إلى تحديد المسار الذي سيتم حفظ ملف PDF فيه. هذا هو المكان الذي ستحدد فيه دليل البيانات الخاص بك.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

الشرح: استبدل "دليل المستندات الخاص بك" بالمسار الفعلي الذي تريد حفظ ملف PDF فيه. من المهم أن يكون لديك إذن الكتابة في هذا الدليل لتجنب أي استثناءات وقت التشغيل.

## الخطوة 2: إنشاء مستند PDF جديد

 بعد ذلك، سنقوم بإنشاء مثيل لـ`Document` فصل.

```csharp
Document document = new Document();
```

الشرح: يقوم هذا السطر بإنشاء مستند PDF جديد. يمكنك اعتبار هذا بمثابة إنشاء لوحة قماشية فارغة حيث يمكنك رسم محتوى ملف PDF الخاص بك.

## الخطوة 3: إنشاء محتوى مميز

الآن حان الوقت للوصول إلى جوهر عملية الوسم. سنقوم باستعادة محتوى المستند المُوسم.

```csharp
ITaggedContent taggedContent = document.TaggedContent;
```

 الشرح:`TaggedContent` يسمح لنا الكائن بالتحكم في البنية المنطقية وإمكانية الوصول إلى ملف PDF من خلال تنظيم المحتوى بشكل هرمي.

## الخطوة 4: تعيين العنوان واللغة

بعد ذلك، يمكنك تعيين عنوان المستند وسمة اللغة. يساعد هذا برامج قراءة الشاشة على معالجة المستند بشكل صحيح.

```csharp
taggedContent.SetTitle("Tagged Pdf Document");
taggedContent.SetLanguage("en-US");
```

التوضيح: هنا نقوم بإعلام أي شخص يقرأ ملف PDF هذا عن عنوانه واللغة المستخدمة فيه، مما يساعد على الفهم، وخاصة لأدوات إمكانية الوصول.

## الخطوة 5: إنشاء عناصر البنية المنطقية

الآن حان الوقت لإنشاء عناصر منظمة داخل ملف PDF الخاص بك.

```csharp
SectElement sect = taggedContent.CreateSectElement();
taggedContent.RootElement.AppendChild(sect);
```

 الشرح: تقدم هذه الخطوة عنصر قسم حيث يمكنك البدء في إضافة المحتوى.`RootElement` هي قاعدتك الرئيسية.

## الخطوة 6: إنشاء عناصر الفقرة

الآن نقوم بإنشاء عناصر فقرة مختلفة وتعيين محتوى النص الخاص بها.

```csharp
ParagraphElement p1 = taggedContent.CreateParagraphElement();
ParagraphElement p2 = taggedContent.CreateParagraphElement();
ParagraphElement p3 = taggedContent.CreateParagraphElement();
ParagraphElement p4 = taggedContent.CreateParagraphElement();
p1.SetText("P1. ");
p2.SetText("P2. ");
p3.SetText("P3. ");
p4.SetText("P4. ");
```

التوضيح: كل فقرة تشبه الجملة في القصة، فهي تحدد المشهد وتعطي معنى لمستندك.

## الخطوة 7: تعيين العلامات للفقرات

من المهم وضع علامة على كل فقرة بشكل مناسب لسهولة الوصول إلى ملف PDF الخاص بك.

```csharp
p1.SetTag("P1");
p2.SetTag("Para");
p3.SetTag("Para");
p4.SetTag("Paragraph");
```

التوضيح: من خلال وضع العلامات، فإننا نمكن التقنيات المساعدة من فهم بنية المحتوى - الأمر يشبه منحهم خريطة طريق للتنقل في المنطقة.

## الخطوة 8: إضافة فقرات إلى القسم

سنقوم الآن بإضافة هذه الفقرات إلى القسم الذي تم إنشاؤه مسبقًا.

```csharp
sect.AppendChild(p1);
sect.AppendChild(p2);
sect.AppendChild(p3);
sect.AppendChild(p4);
```

التوضيح: يقوم هذا الإجراء بتنظيم الفقرات داخل القسم، مما يجعل التدفق منطقيًا وسهل المتابعة.

## الخطوة 9: إنشاء عناصر النطاق

تمامًا مثل الفقرات، تضيف المسافات تفاصيل دقيقة إلى محتوى النص الخاص بنا.

```csharp
SpanElement span1 = taggedContent.CreateSpanElement();
SpanElement span2 = taggedContent.CreateSpanElement();
SpanElement span3 = taggedContent.CreateSpanElement();
SpanElement span4 = taggedContent.CreateSpanElement();
span1.SetText("Span 1.");
span2.SetText("Span 2.");
span3.SetText("Span 3.");
span4.SetText("Span 4.");
```

التوضيح: إن المسافات هي بمثابة ضبط التفاصيل؛ فهي تضيف تفاصيل إلى ما تنقله في كل فقرة.

## الخطوة 10: تعيين العلامات لعناصر النطاق

تمامًا كما فعلنا بالنسبة للفقرات، ينبغي علينا وضع علامات على عناصر الامتداد لدينا للمساعدة في الفهم.

```csharp
span1.SetTag("SPAN");
span2.SetTag("Sp");
span3.SetTag("Sp");
span4.SetTag("TheSpan");
```

التوضيح: يضمن وضع علامات على الفواصل الزمنية بشكل صحيح تمثيل جميع الفروق الدقيقة بشكل دقيق، مما يؤدي إلى إنشاء مستند غني لأي شخص يصادفه.

## الخطوة 11: إضافة المسافات إلى الفقرات

الآن، دعونا نضيف عناصر span إلى الفقرات الخاصة بها.

```csharp
p1.AppendChild(span1);
p2.AppendChild(span2);
p3.AppendChild(span3);
p4.AppendChild(span4);
```

التوضيح: يعمل هذا التكامل على تعزيز فقراتك من خلال إضافة طبقات من المعلومات الإضافية، تمامًا مثل إضافة الحواشي السفلية إلى كتاب لتحقيق فهم أعمق.

## الخطوة 12: احفظ مستند PDF المُوسوم

وأخيرًا، حان الوقت لحفظ تحفتك الفنية!

```csharp
document.Save(dataDir + "CustomTag.pdf");
```

الشرح: يقوم هذا السطر بحفظ ملف PDF المُوسم في الدليل المحدد. وبذلك تكون قد أكملت إنشاء مستند PDF المُوسم!

## خاتمة

إن إنشاء مستندات PDF ذات علامات باستخدام Aspose.PDF لـ .NET لا يتعلق فقط بإضافة المحتوى؛ بل يتعلق أيضًا بصياغة تجربة يمكن الوصول إليها لجميع المستخدمين. من خلال بضع خطوات فقط، يمكنك التأكد من أن مستنداتك تتحدث إلى الجميع، وإزالة الحواجز وتعزيز الفهم.
