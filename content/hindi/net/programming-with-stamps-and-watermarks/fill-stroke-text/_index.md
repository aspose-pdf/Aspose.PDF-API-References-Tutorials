इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में टेक्स्ट को भरने और आउटलाइन करने के तरीके के बारे में चरण दर चरण बताएंगे। हम आपको दिखाएंगे कि PDF फ़ाइल में टेक्स्ट पर भरने और आउटलाइन रंग लगाने के लिए दिए गए C# सोर्स कोड का उपयोग कैसे करें।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- एक स्थापित .NET विकास वातावरण.
- .NET के लिए Aspose.PDF लाइब्रेरी डाउनलोड की गई और आपके प्रोजेक्ट में संदर्भित की गई।

## चरण 2: टेक्स्टस्टेट ऑब्जेक्ट बनाना

पहला कदम उन्नत गुणों को पास करने के लिए TextState ऑब्जेक्ट बनाना है। यहाँ बताया गया है कि कैसे:

```csharp
// उन्नत गुणों को स्थानांतरित करने के लिए TextState ऑब्जेक्ट बनाएँ
TextState ts = new TextState();

// रूपरेखा रंग सेट करें
ts.StrokingColor = Color.Gray;

// पाठ रेंडरिंग मोड को परिभाषित करें
ts.RenderingMode = TextRenderingMode.StrokeText;
```

उपरोक्त कोड एक नया TextState ऑब्जेक्ट बनाता है और आउटलाइन का रंग निर्धारित करता है, साथ ही यह भी निर्धारित करता है कि टेक्स्ट को कैसे प्रस्तुत किया जाए।

## चरण 3: पीडीएफ दस्तावेज़ लोड करना

अब जब TextState ऑब्जेक्ट तैयार हो गया है, तो हम PDF दस्तावेज़ को उस स्थान पर लोड कर सकते हैं जहाँ हम टेक्स्ट फ़िल और आउटलाइन लागू करना चाहते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// पीडीएफ दस्तावेज़ को इनपुट के रूप में लोड करें
Facades.PdfFileStamp fileStamp = new Facades.PdfFileStamp(new Aspose.Pdf.Document(dataDir + "input.pdf"));
```

उपरोक्त कोड Aspose.PDF.Facades लाइब्रेरी से PdfFileStamp क्लास का उपयोग करके मौजूदा PDF दस्तावेज़ को लोड करता है।

## चरण 4: टेक्स्ट में भरण और स्ट्रोक जोड़ें

अब जब पीडीएफ दस्तावेज़ लोड हो गया है, तो हम टेक्स्ट में भरण और रूपरेखा जोड़ सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// परिभाषित पाठ और गुणों के साथ एक स्टाम्प (स्टाम्प) बनाएं
Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
stamp.BindLogo(new Facades.FormattedText("PAID IN FULL", System.Drawing.Color.Gray, "Arial", Facades.EncodingType.Winansi, true, 78));

// TextState ऑब्जेक्ट को बाइंड करें
stamp.BindTextState(ts);

// मूल X, Y सेट करें
stamp.SetOrigin(100, 100);
stamp. Opacity = 5;
stamp.BlendingSpace = Facades.BlendingColorSpace.DeviceRGB;
stamp.Rotation = 45.0F;
stamp. IsBackground = false;

// दस्तावेज़ में स्टाम्प जोड़ें
fileStamp.AddStamp(stamp);
```

उपरोक्त कोड निर्दिष्ट टेक्स्ट और परिभाषित भरण और स्ट्रोक गुणों के साथ एक स्टैम्प बनाता है।

## चरण 5: आउटपुट दस्तावेज़ सहेजें

एक बार टेक्स्ट स्टैम्प जोड़ दिए जाने के बाद, हम संशोधित PDF दस्तावेज़ को सहेज सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// संशोधित दस्तावेज़ सहेजें
fileStamp.Save(dataDir + "output_out.pdf");
fileStamp.Close();
```

उपरोक्त कोड संपादित पीडीएफ दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजता है।

### .NET के लिए Aspose.PDF का उपयोग करके Fill Stroke Text के लिए नमूना स्रोत कोड 
```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// उन्नत गुणों को स्थानांतरित करने के लिए TextState ऑब्जेक्ट बनाएँ
TextState ts = new TextState();

// स्ट्रोक के लिए रंग सेट करें
ts.StrokingColor = Color.Gray;

// टेक्स्ट रेंडरिंग मोड सेट करें
ts.RenderingMode = TextRenderingMode.StrokeText;

// इनपुट PDF दस्तावेज़ लोड करें
Facades.PdfFileStamp fileStamp = new Facades.PdfFileStamp(new Aspose.Pdf.Document(dataDir + "input.pdf"));
Aspose.Pdf.Facades.Stamp stamp = new Aspose.Pdf.Facades.Stamp();
stamp.BindLogo(new Facades.FormattedText("PAID IN FULL", System.Drawing.Color.Gray, "Arial", Facades.EncodingType.Winansi, true, 78));

// टेक्स्टस्टेट बाँधें
stamp.BindTextState(ts);

// X,Y मूल सेट करें
stamp.SetOrigin(100, 100);
stamp.Opacity = 5;
stamp.BlendingSpace = Facades.BlendingColorSpace.DeviceRGB;
stamp.Rotation = 45.0F;
stamp.IsBackground = false;

// स्टाम्प जोड़ें
fileStamp.AddStamp(stamp);
fileStamp.Save(dataDir + "ouput_out.pdf");
fileStamp.Close();

```

## निष्कर्ष

बधाई हो! आपने .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में टेक्स्ट को भरना और आउटलाइन करना सीख लिया है। अब आप अपने PDF दस्तावेज़ों में रंग भरने और आउटलाइन को कस्टमाइज़ करने के लिए इस ज्ञान को लागू कर सकते हैं।

### पीडीएफ फाइल में स्ट्रोक टेक्स्ट भरने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ में पाठ भरने और रूपरेखा बनाने का क्या अर्थ है, और मुझे ऐसा कब करना पड़ सकता है?

उत्तर: PDF दस्तावेज़ में टेक्स्ट भरने और आउटलाइन करने में टेक्स्ट के अक्षरों के अंदरूनी हिस्से (भरण) और टेक्स्ट के चारों ओर की सीमाओं (आउटलाइन) पर रंग लगाना शामिल है। इसका उपयोग टेक्स्ट की दृश्य उपस्थिति को बढ़ाने, जोर देने या PDF के भीतर विशिष्ट सामग्री को हाइलाइट करने के लिए किया जा सकता है।

#### प्रश्न: उपलब्ध कराया गया C# स्रोत कोड PDF फ़ाइल में पाठ भरने और रूपरेखा बनाने का कार्य किस प्रकार पूरा करता है?

 उत्तर: प्रदान किया गया स्रोत कोड दर्शाता है कि कैसे बनाया जाए`TextState` ऑब्जेक्ट उन्नत टेक्स्ट गुण, जैसे आउटलाइन रंग और रेंडरिंग मोड को परिभाषित करने के लिए। फिर यह मौजूदा PDF दस्तावेज़ को लोड करने, निर्दिष्ट भरण और स्ट्रोक गुणों के साथ टेक्स्ट युक्त एक स्टैम्प बनाने और दस्तावेज़ में स्टैम्प जोड़ने के लिए Aspose.PDF.Facades का उपयोग करता है।

####  प्रश्न: इसका उद्देश्य क्या है?`TextState` object in the code?

 उत्तर:`TextState`ऑब्जेक्ट का उपयोग उन्नत टेक्स्ट गुणों को परिभाषित करने के लिए किया जाता है, जिसमें टेक्स्ट आउटलाइन (स्ट्रोक) का रंग और रेंडरिंग मोड शामिल है। यह आपको स्ट्रोक और फिल के संदर्भ में टेक्स्ट को कैसे प्रदर्शित किया जाए, इसे कस्टमाइज़ करने की अनुमति देता है।

#### प्रश्न: क्या मैं एक ही पाठ के विभिन्न भागों पर अलग-अलग भरण और रूपरेखा रंग लागू कर सकता हूँ?

 उत्तर: हां, आप अलग-अलग परिणाम बनाने के लिए कोड को संशोधित कर सकते हैं।`TextState` अलग-अलग भरण और रूपरेखा रंगों वाली वस्तुओं को चुनें और उन्हें अलग-अलग का उपयोग करके पाठ के विशिष्ट भागों पर लागू करें`Stamp` वस्तुएं.

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ में पहले से मौजूद पाठ पर भरण और रूपरेखा रंग लागू कर सकता हूं?

 उत्तर: हां, आप पीडीएफ दस्तावेज़ में मौजूदा टेक्स्ट पर भरण और रूपरेखा रंग लागू करने के लिए उपयुक्त टेक्स्ट ऑब्जेक्ट्स का चयन करके और उन्हें वांछित स्टैम्प के रूप में जोड़कर समान सिद्धांतों का उपयोग कर सकते हैं।`TextState` गुण।

#### प्रश्न: मैं भरे हुए और रेखांकित पाठ की अपारदर्शिता और सम्मिश्रण को कैसे समायोजित कर सकता हूं?

 उत्तर: प्रदान किया गया कोड आपको स्टैम्प की अपारदर्शिता और सम्मिश्रण गुणों को सेट करने की अनुमति देता है`Opacity` और`BlendingSpace`गुण, क्रमशः। आप वांछित दृश्य प्रभाव प्राप्त करने के लिए इन मूल्यों को समायोजित कर सकते हैं।

#### प्रश्न: मैं एक ही पीडीएफ दस्तावेज़ में एकाधिक स्टैम्पों पर अलग-अलग भरण और रूपरेखा रंग कैसे लागू कर सकता हूं?

 उत्तर: आप कई बना सकते हैं`TextState` अलग-अलग भरण और रूपरेखा रंगों वाली वस्तुएं, और फिर अलग-अलग बनाएं`Stamp` प्रत्येक टेक्स्ट सेट के लिए अलग-अलग रंगों वाली वस्तुएँ। इन स्टैम्प को उसी PDF दस्तावेज़ में जोड़ें`PdfFileStamp` कक्षा।

#### प्रश्न: क्या मैं रेखांकित और भरे हुए पाठ के लिए एरियल के अलावा अन्य फ़ॉन्ट का उपयोग कर सकता हूं?

 उत्तर: हां, आप फ़ॉन्ट नाम पैरामीटर को संशोधित करके फ़ॉन्ट बदल सकते हैं।`FormattedText` स्टैम्प बनाते समय कन्स्ट्रक्टर का उपयोग करें। आप अपने सिस्टम पर उपलब्ध किसी भी फ़ॉन्ट का उपयोग कर सकते हैं।

#### प्रश्न: मैं रेखांकित और भरे हुए पाठ के घूर्णन कोण को कैसे संशोधित कर सकता हूं?

 उत्तर: प्रदान किया गया कोड आपको स्टाम्प के घूर्णन कोण को सेट करने की अनुमति देता है`Rotation` प्रॉपर्टी। आप टेक्स्ट के लिए वांछित रोटेशन कोण निर्दिष्ट करने के लिए इस प्रॉपर्टी को समायोजित कर सकते हैं।

#### प्रश्न: मैं पृष्ठ पर रेखांकित और भरे हुए पाठ की स्थिति और आकार को कैसे नियंत्रित कर सकता हूं?

 उत्तर: आप इसका उपयोग कर सकते हैं`SetOrigin` की विधि`Stamp` पृष्ठ पर स्टाम्प की स्थिति के X और Y निर्देशांक सेट करने के लिए ऑब्जेक्ट। इसके अतिरिक्त, आप फ़ॉन्ट आकार को समायोजित कर सकते हैं`FormattedText` कंस्ट्रक्टर का उपयोग पाठ के आकार को नियंत्रित करने के लिए किया जाता है।
