यह ट्यूटोरियल आपको .NET के लिए Aspose.PDF में DOM (डॉक्यूमेंट ऑब्जेक्ट मॉडल) का उपयोग करके HTML सामग्री जोड़ने की प्रक्रिया के बारे में बताएगा। इसके अतिरिक्त, आप HTML सामग्री के लिए शैलियों को अधिलेखित करना सीखेंगे। प्रदान किया गया C# स्रोत कोड आवश्यक चरणों को प्रदर्शित करता है।

## आवश्यकताएं
आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- आपके मशीन पर Visual Studio या कोई अन्य C# कंपाइलर स्थापित होना चाहिए।
- .NET लाइब्रेरी के लिए Aspose.PDF. आप इसे आधिकारिक Aspose वेबसाइट से डाउनलोड कर सकते हैं या इसे इंस्टॉल करने के लिए NuGet जैसे पैकेज मैनेजर का उपयोग कर सकते हैं।

## चरण 1: प्रोजेक्ट सेट अप करें
1. अपने पसंदीदा विकास वातावरण में एक नया C# प्रोजेक्ट बनाएं।
2. .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें।

## चरण 2: आवश्यक नामस्थान आयात करें
उस कोड फ़ाइल में जहाँ आप HTML सामग्री जोड़ना चाहते हैं, फ़ाइल के शीर्ष पर निम्नलिखित using निर्देश जोड़ें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## चरण 3: दस्तावेज़ निर्देशिका और आउटपुट फ़ाइल पथ सेट करें
 कोड में, वह पंक्ति ढूंढें जो कहती है`string dataDir = "YOUR DOCUMENT DIRECTORY";` और प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` उस निर्देशिका का पथ जहां आपके दस्तावेज़ संग्रहीत हैं.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 4: एक नया दस्तावेज़ ऑब्जेक्ट बनाएँ
 एक नया उदाहरण बनाएँ`Document` निम्न कोड पंक्ति जोड़कर ऑब्जेक्ट बनाएँ:

```csharp
Document doc = new Document();
```

## चरण 5: दस्तावेज़ में एक पृष्ठ जोड़ें
 दस्तावेज़ में एक नया पृष्ठ जोड़ने के लिए निम्न का उपयोग करें:`Add` की विधि`Pages` संग्रह। प्रदान किए गए कोड में, नया पृष्ठ चर को सौंपा गया है`page`.

```csharp
Page page = doc.Pages.Add();
```

## चरण 6: HTML सामग्री के साथ एक HtmlFragment बनाएँ
 एक उदाहरण बनाना`HtmlFragment` ऑब्जेक्ट और वांछित HTML सामग्री प्रदान करें। प्रदान किए गए कोड में, HTML सामग्री वैरिएबल को सौंपी गई है`title`आप आवश्यकतानुसार HTML सामग्री को संशोधित कर सकते हैं।

```csharp
HtmlFragment title = new HtmlFragment("<p style='font-family: Verdana'><b><i>Table contains text</i></b></p>");
```

## चरण 7: HTML सामग्री के लिए शैलियों को अधिलेखित करें
 HTML सामग्री की शैलियों को अधिलेखित करने के लिए, आप संशोधित कर सकते हैं`TextState` के गुण`HtmlFragment` दिए गए कोड में, फ़ॉन्ट परिवार को "Arial" में बदल दिया गया है और फ़ॉन्ट आकार 20 पर सेट किया गया है।

```csharp
title. TextState = new TextState("Arial");
title.TextState.FontSize = 20;
```

## चरण 8: मार्जिन जानकारी सेट करें
यदि आवश्यक हो तो HTML फ़्रैगमेंट के निचले और ऊपरी मार्जिन को समायोजित करें। दिए गए कोड में, निचला मार्जिन 10 पर सेट है और ऊपरी मार्जिन 400 पर सेट है।

```csharp
title. Margin. Bottom = 10;
title. Margin. Top = 400;
```

## चरण 9: पृष्ठ पर HtmlFragment जोड़ें
 जोड़ें`HtmlFragment` पृष्ठ के पैराग्राफ संग्रह पर आपत्ति।

```csharp
page.Paragraphs.Add(title);
```

## चरण 10: पीडीएफ दस्तावेज़ सहेजें
 पीडीएफ दस्तावेज़ को सेव करने के लिए निम्न का उपयोग करें:`Save` की विधि`Document` ऑब्जेक्ट. चरण 3 में आपके द्वारा सेट किया गया आउटपुट फ़ाइल पथ निर्दिष्ट करें.

```csharp
dataDir = dataDir + "AddHTMLUsingDOMAndOverwrite_out.pdf";
doc.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके DOM का उपयोग करके HTML जोड़ें और ओवरराइट करें के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें
Document doc = new Document();
// पीडीएफ फाइल के पेज संग्रह में एक पेज जोड़ें
Page page = doc.Pages.Add();
// HTML सामग्री के साथ HtmlFragment को तत्कालित करें
HtmlFragment title = new HtmlFragment("<p style='font-family: Verdana'><b><i>Table contains text</i></b></p>");
//'वर्दाना' से फ़ॉन्ट-फ़ैमिली को 'एरियल' पर रीसेट किया जाएगा
title.TextState = new TextState("Arial");
title.TextState.FontSize = 20;
// निचला मार्जिन जानकारी सेट करें
title.Margin.Bottom = 10;
// शीर्ष मार्जिन जानकारी सेट करें
title.Margin.Top = 400;
// पृष्ठ के पैराग्राफ संग्रह में HTML फ़्रैगमेंट जोड़ें
page.Paragraphs.Add(title);
// पीडीएफ फाइल सहेजें
dataDir = dataDir + "AddHTMLUsingDOMAndOverwrite_out.pdf";
// पीडीएफ फाइल सहेजें
doc.Save(dataDir);
```

## निष्कर्ष
आपने .NET के लिए Aspose.PDF में DOM का उपयोग करके HTML सामग्री को सफलतापूर्वक जोड़ा है और HTML सामग्री के लिए शैलियों को अधिलेखित किया है। परिणामी PDF फ़ाइल अब निर्दिष्ट आउटपुट फ़ाइल पथ पर पाई जा सकती है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: इस ट्यूटोरियल का फोकस क्या है?

उत्तर: यह ट्यूटोरियल आपको .NET के लिए Aspose.PDF में डॉक्यूमेंट ऑब्जेक्ट मॉडल (DOM) का उपयोग करके PDF दस्तावेज़ में HTML सामग्री जोड़ने की प्रक्रिया से परिचित कराने के लिए डिज़ाइन किया गया है। इसके अतिरिक्त, आप HTML सामग्री के लिए शैलियों को अधिलेखित करना सीखेंगे, जिससे आप इसकी उपस्थिति को अनुकूलित कर सकेंगे। ट्यूटोरियल आवश्यक चरणों को प्रदर्शित करने के लिए C# स्रोत कोड स्निपेट प्रदान करता है।

#### प्रश्न: इस ट्यूटोरियल के लिए मुझे कौन से नेमस्पेस आयात करने होंगे?

उत्तर: उस कोड फ़ाइल में जहाँ आप HTML सामग्री जोड़ना चाहते हैं, फ़ाइल के आरंभ में निम्नलिखित नामस्थान आयात करें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

#### प्रश्न: मैं दस्तावेज़ निर्देशिका और आउटपुट फ़ाइल पथ कैसे निर्दिष्ट करूँ?

 उत्तर: कोड में, पंक्ति का पता लगाएं`string dataDir = "YOUR DOCUMENT DIRECTORY";` और प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

#### प्रश्न: मैं दस्तावेज़ ऑब्जेक्ट कैसे बनाऊं?

 उत्तर: चरण 4 में, आप एक नया इन्स्टेन्सिएट करेंगे`Document` निम्न कोड लाइन का उपयोग करके ऑब्जेक्ट बनाएँ:

```csharp
Document doc = new Document();
```

#### प्रश्न: मैं दस्तावेज़ में पृष्ठ कैसे जोड़ूं?

 उत्तर: चरण 5 में, आप दस्तावेज़ में एक नया पृष्ठ जोड़ेंगे`Add` की विधि`Pages` संग्रह:

```csharp
Page page = doc.Pages.Add();
```

#### प्रश्न: मैं DOM का उपयोग करके HTML सामग्री कैसे सेट कर सकता हूं?

 उत्तर: चरण 6 में, आप एक बनाएंगे`HtmlFragment` ऑब्जेक्ट चुनें और उसमें अपनी मनचाही HTML सामग्री असाइन करें। HTML सामग्री वैरिएबल को असाइन की जाती है`title`:

```csharp
HtmlFragment title = new HtmlFragment("<p style='font-family: Verdana'><b><i>Table contains text</i></b></p>");
```

#### प्रश्न: मैं HTML सामग्री की शैलियों को कैसे अधिलेखित कर सकता हूँ?

 उत्तर: चरण 7 में, आप HTML सामग्री की शैलियों को संशोधित करके अधिलेखित करेंगे`TextState` के गुण`HtmlFragment`ऑब्जेक्ट. उदाहरण के लिए, आप फ़ॉन्ट परिवार को "Arial" में बदल सकते हैं और फ़ॉन्ट आकार को 20 पर सेट कर सकते हैं:

```csharp
title.TextState = new TextState("Arial");
title.TextState.FontSize = 20;
```

#### प्रश्न: क्या मैं HTML सामग्री के मार्जिन को समायोजित कर सकता हूँ?

उत्तर: हां, चरण 8 में, आप HTML खंड के निचले और ऊपरी मार्जिन को आवश्यकतानुसार समायोजित कर सकते हैं:

```csharp
title.Margin.Bottom = 10;
title.Margin.Top = 400;
```

#### प्रश्न: मैं PDF दस्तावेज़ में HtmlFragment कैसे जोड़ूं?

 उत्तर: चरण 9 में, आप जोड़ेंगे`HtmlFragment` वस्तु (`title`) को पृष्ठ के पैराग्राफ संग्रह में जोड़ें:

```csharp
page.Paragraphs.Add(title);
```

#### प्रश्न: मैं परिणामी पीडीएफ दस्तावेज़ को कैसे सहेजूँ?

 उत्तर: HTML सामग्री जोड़ने और इसकी शैलियों को अनुकूलित करने के बाद, का उपयोग करें`Save` की विधि`Document` पीडीएफ दस्तावेज़ को सहेजने के लिए ऑब्जेक्ट:

```csharp
dataDir = dataDir + "AddHTMLUsingDOMAndOverwrite_out.pdf";
doc.Save(dataDir);
```

#### प्रश्न: इस ट्यूटोरियल से मुख्य बात क्या है?

उत्तर: इस ट्यूटोरियल का अनुसरण करके, आपने सफलतापूर्वक सीखा है कि Aspose.PDF for .NET में डॉक्यूमेंट ऑब्जेक्ट मॉडल (DOM) का उपयोग करके HTML सामग्री को कैसे शामिल किया जाए। इसके अतिरिक्त, आपने परिणामी PDF दस्तावेज़ में HTML सामग्री की उपस्थिति को अनुकूलित करने के लिए शैलियों को अधिलेखित करने की क्षमता प्राप्त की है।
