इस ट्यूटोरियल में, हम आपको दिखाएंगे कि .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ॉर्म फ़ील्ड निर्देशांक कैसे प्राप्त करें। हम इस प्रक्रिया में आपका मार्गदर्शन करने के लिए C# स्रोत कोड को चरण दर चरण समझाएँगे।

## चरण 1: तैयारी

सुनिश्चित करें कि आपने आवश्यक लाइब्रेरीज़ आयात कर ली हैं और दस्तावेज़ निर्देशिका का पथ सेट कर लिया है:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: आउटपुट दस्तावेज़ लोड करें

आउटपुट पीडीएफ दस्तावेज़ लोड करें:

```csharp
Document doc1 = new Document(dataDir + "input.pdf");
```

## चरण 3: जोड़े गए फ़ील्ड ढूँढ़ें

जोड़े गए फ़ॉर्म फ़ील्ड ढूँढें (इस उदाहरण में, हम "आइटम1", "आइटम2" और "आइटम3" फ़ील्ड का उपयोग कर रहे हैं):

```csharp
RadioButtonField field0 = doc1.Form["Item1"] as RadioButtonField;
RadioButtonField field1 = doc1.Form["Item2"] as RadioButtonField;
RadioButtonField field2 = doc1.Form["Item3"] as RadioButtonField;
```

## चरण 4: प्रत्येक फ़ील्ड के लिए उप-आइटम की स्थिति प्रदर्शित करें

प्रत्येक फ़ील्ड के लिए विकल्पों पर जाएँ और प्रत्येक उप-आइटम के निर्देशांक देखें:

```csharp
foreach(RadioButtonOptionField option in field0)
{
Console.WriteLine(option.Rect);
}
foreach(RadioButtonOptionField option in field1)
{
Console.WriteLine(option.Rect);
}
foreach(RadioButtonOptionField option in field2)
{
Console.WriteLine(option.Rect);
}
```

### .NET के लिए Aspose.PDF का उपयोग करके निर्देशांक प्राप्त करने के लिए नमूना स्रोत कोड 
```csharp
try
{
	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	// आउटपुट दस्तावेज़ लोड करें
	Document doc1 = new Document( dataDir + "input.pdf");
	// जोड़े गए फ़ील्ड खोजें
	RadioButtonField field0 = doc1.Form["Item1"] as RadioButtonField;
	RadioButtonField field1 = doc1.Form["Item2"] as RadioButtonField;
	RadioButtonField field2 = doc1.Form["Item3"] as RadioButtonField;
	// और उनमें से प्रत्येक के लिए उप-आइटम की स्थिति दिखाएं।
	foreach (RadioButtonOptionField option in field0)
	{
		Console.WriteLine(option.Rect);
	}
	foreach (RadioButtonOptionField option in field1)
	{
		Console.WriteLine(option.Rect);
	}
	foreach (RadioButtonOptionField option in field2)
	{
		Console.WriteLine(option.Rect);
	}
}
catch (Exception ex)
{
	Console.WriteLine(ex.Message);
}
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.PDF का उपयोग करके फ़ॉर्म फ़ील्ड निर्देशांक कैसे प्राप्त करें। इन चरणों का पालन करके, आप Aspose.PDF का उपयोग करके अपने PDF दस्तावेज़ों में अपने फ़ॉर्म फ़ील्ड के उप-तत्वों के निर्देशांक आसानी से प्राप्त कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF में किसी भी प्रकार के फॉर्म फ़ील्ड के निर्देशांक प्राप्त करने के लिए इस विधि का उपयोग कर सकता हूं?

उत्तर: हाँ, आप .NET के लिए Aspose.PDF में विभिन्न प्रकार के फ़ॉर्म फ़ील्ड के लिए निर्देशांक प्राप्त करने के लिए इस विधि का उपयोग कर सकते हैं। प्रदान किया गया C# स्रोत कोड दर्शाता है कि रेडियोबटन फ़ील्ड के लिए निर्देशांक कैसे प्राप्त करें, लेकिन आप अन्य फ़ॉर्म फ़ील्ड प्रकारों, जैसे कि TextBox, CheckBox, ListBox, और अधिक के लिए समान दृष्टिकोण को अपना सकते हैं।

#### प्रश्न: मैं फ़ॉर्म फ़ील्ड निर्देशांक को कैसे संशोधित या समायोजित कर सकता हूँ?

उत्तर: फ़ॉर्म फ़ील्ड निर्देशांक PDF दस्तावेज़ के निर्देशांक सिस्टम पर आधारित होते हैं, जहाँ मूल (0,0) पृष्ठ के निचले-बाएँ कोने में स्थित होता है। फ़ॉर्म फ़ील्ड निर्देशांक को संशोधित या समायोजित करने के लिए, आप इसे अपडेट कर सकते हैं`Rect` संबंधित फ़ॉर्म फ़ील्ड या उसके उप-आइटम, जैसे रेडियोबटनऑप्शनफ़ील्ड, की संपत्ति.

#### प्रश्न: क्या मैं प्रोग्रामेटिक रूप से PDF दस्तावेज़ में फॉर्म फ़ील्ड के निर्देशांक जोड़ सकता हूँ?

उत्तर: हाँ, आप PDF दस्तावेज़ में प्रोग्रामेटिक रूप से जोड़े गए फ़ॉर्म फ़ील्ड के निर्देशांक प्राप्त कर सकते हैं। .NET के लिए Aspose.PDF आपको गतिशील रूप से फ़ॉर्म फ़ील्ड जोड़ने की अनुमति देता है, और एक बार जोड़ने के बाद, आप इस ट्यूटोरियल में दिखाए गए दृष्टिकोण का उपयोग करके उनके निर्देशांक प्राप्त कर सकते हैं।

#### प्रश्न: फॉर्म फ़ील्ड निर्देशांक प्राप्त करने का उद्देश्य क्या है?

उत्तर: जब आपको PDF दस्तावेज़ में फ़ॉर्म फ़ील्ड पर विशिष्ट लेआउट-संबंधित संचालन या सत्यापन करने की आवश्यकता होती है, तो फ़ॉर्म फ़ील्ड निर्देशांक प्राप्त करना सहायक हो सकता है। यह आपको फ़ॉर्म फ़ील्ड को उनके निर्देशांक के आधार पर सटीक रूप से स्थिति और संरेखित करने की अनुमति देता है, यह सुनिश्चित करता है कि वे दस्तावेज़ में सही ढंग से दिखाई दें और एक सहज उपयोगकर्ता अनुभव प्रदान करें।

#### प्रश्न: क्या फॉर्म फ़ील्ड निर्देशांक बिंदुओं या किसी अन्य इकाई में व्यक्त किए जाते हैं?

उत्तर: .NET के लिए Aspose.PDF में फ़ॉर्म फ़ील्ड निर्देशांक पॉइंट में व्यक्त किए जाते हैं। एक पॉइंट 1/72 इंच के बराबर होता है, जो इसे PDF फ़ॉर्मेट में माप की एक मानक इकाई बनाता है।
