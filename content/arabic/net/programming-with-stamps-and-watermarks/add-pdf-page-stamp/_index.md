في هذا البرنامج التعليمي، سنوضح لك خطوة بخطوة كيفية إضافة ختم صفحة PDF في ملف PDF باستخدام Aspose.PDF لـ .NET. وسنوضح لك كيفية استخدام كود المصدر C# المقدم لإضافة ختم مخصص إلى صفحة معينة من ملف PDF.

## الخطوة 1: إعداد البيئة

قبل أن تبدأ، تأكد من أن لديك ما يلي:

- بيئة تطوير .NET مثبتة.
- تم تنزيل مكتبة Aspose.PDF لـ .NET والإشارة إليها في مشروعك.

## الخطوة 2: تحميل مستند PDF

الخطوة الأولى هي تحميل مستند PDF الموجود في مشروعك. وإليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// افتح المستند
Document pdfDocument = new Document(dataDir + "PDFPageStamp.pdf");
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي إلى الدليل الذي يوجد به مستند PDF الخاص بك.

## الخطوة 3: إنشاء مخزن الصفحة

الآن بعد أن قمت بتحميل مستند PDF، يمكنك إنشاء ختم الصفحة لإضافته. إليك كيفية القيام بذلك:

```csharp
// إنشاء مخزن الصفحة
PdfPageStamp pageStamp = new PdfPageStamp(pdfDocument.Pages[1]);
```

يقوم الكود أعلاه بإنشاء مخزن صفحة جديد باستخدام الصفحة الأولى من مستند PDF.

## الخطوة 4: تكوين خصائص مخزن الصفحة

قبل إضافة ختم الصفحة إلى مستند PDF، يمكنك تكوين خصائص مختلفة للختم، مثل الخلفية والموضع والدوران وما إلى ذلك. وإليك الطريقة:

```csharp
// تكوين خصائص مخزن الصفحة
pageStamp. Background = true;
pageStamp. XIndent = 100;
pageStamp. YIndent = 100;
pageStamp.Rotate = Rotate.on180;
```

يمكنك تعديل هذه الخصائص وفقًا لاحتياجاتك.

## الخطوة 5: إضافة ختم الصفحة إلى ملف PDF

الآن بعد أن أصبح ختم الصفحة جاهزًا، يمكنك إضافته إلى صفحة معينة من مستند PDF. إليك الطريقة:

```csharp
// إضافة مخزن الصفحات إلى صفحة معينة
pdfDocument.Pages[1].AddStamp(pageStamp);
```

يضيف الكود أعلاه ختم الصفحة إلى الصفحة الأولى من مستند PDF. يمكنك تحديد صفحة أخرى إذا لزم الأمر.

## الخطوة 6: احفظ المستند الناتج

بمجرد إضافة ختم الصفحة، يمكنك حفظ مستند PDF المعدّل. إليك الطريقة:

```csharp
// حفظ المستند الناتج
pdfDocument.Save(dataDir);
```

### عينة من كود المصدر لإضافة ختم PDFPage باستخدام Aspose.PDF لـ .NET 
```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// فتح المستند
Document pdfDocument = new Document(dataDir+ "PDFPageStamp.pdf");

// إنشاء ختم الصفحة
PdfPageStamp pageStamp = new PdfPageStamp(pdfDocument.Pages[1]);
pageStamp.Background = true;
pageStamp.XIndent = 100;
pageStamp.YIndent = 100;
pageStamp.Rotate = Rotation.on180;

// إضافة طابع إلى صفحة معينة
pdfDocument.Pages[1].AddStamp(pageStamp);
dataDir = dataDir + "PDFPageStamp_out.pdf";

// حفظ المستند الناتج
pdfDocument.Save(dataDir);
Console.WriteLine("\nPdf page stamp added successfully.\nFile saved at " + dataDir);

```

يقوم الكود أعلاه بحفظ مستند PDF المحرر في الدليل المحدد.

#### س: هل يمكنني إضافة طوابع صفحات متعددة إلى صفحات مختلفة من نفس مستند PDF؟

#### س: كيف يمكنني تعديل موضع ودوران ختم الصفحة داخل مستند PDF؟

 أ: يمكنك تخصيص موضع وتدوير ختم الصفحة عن طريق تعديل خصائص`PdfPageStamp` الكائن. يوضح الكود المقدم في البرنامج التعليمي كيفية تعيين خصائص مثل`XIndent`, `YIndent` ، و`Rotate` للتحكم في وضع الطوابع واتجاهها.

#### س: هل يمكنني تطبيق هذه الطريقة على مستندات PDF الموجودة لإضافة طوابع الصفحة؟

#### س: كيف أحدد الصفحة التي أريد إضافة ختم الصفحة إليها؟

#### س: هل يمكنني استخدام هذه الطريقة لإضافة طوابع أخرى غير العلامات المائية، مثل الشعارات أو التوقيعات؟
