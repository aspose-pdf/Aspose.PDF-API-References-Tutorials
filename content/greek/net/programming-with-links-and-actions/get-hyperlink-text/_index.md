## Βήμα 1: Ρυθμίστε τη διαδρομή του εγγράφου σας

Στον κώδικά μας, θα πρέπει πρώτα να καθορίσουμε τη διαδρομή όπου βρίσκεται το έγγραφο PDF μας. Αυτό γίνεται χρησιμοποιώντας μια μεταβλητή συμβολοσειράς. 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Φροντίστε να αντικαταστήσετε`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή του αρχείου PDF σας. Για παράδειγμα, θα μπορούσε να μοιάζει κάπως`"C:\\Documents\\"`.

## Βήμα 2: Φορτώστε το έγγραφο PDF

 Το επόμενο βήμα περιλαμβάνει τη φόρτωση του αρχείου PDF, ώστε να μπορέσουμε να ξεκινήσουμε την επεξεργασία του. Θα δημιουργήσουμε ένα παράδειγμα του`Document` τάξη και περάστε τη διαδρομή του αρχείου μας σε αυτήν.

```csharp
Document document = new Document(dataDir + "input.pdf");
```

Σε αυτό το σημείο, εάν όλα έχουν ρυθμιστεί σωστά, το αρχείο PDF σας θα φορτωθεί και θα είναι έτοιμο για αλληλεπίδραση.

## Βήμα 3: Επανάληψη σε κάθε σελίδα

Τα PDF μπορεί να έχουν πολλές σελίδες, επομένως θα περιηγούμαστε σε κάθε σελίδα για να βρούμε σχολιασμούς συνδέσμων. Δείτε πώς μπορείτε να το πετύχετε αυτό:

```csharp
foreach (Page page in document.Pages)
{
    // Εμφάνιση σχολιασμού συνδέσμου
    ShowLinkAnnotations(page);
}
```

 Σε αυτόν τον βρόχο, θα ορίσουμε μια μέθοδο που ονομάζεται`ShowLinkAnnotations` που θα χειριστεί την εξαγωγή υπερσυνδέσμων. 

## Βήμα 4: Καθορίστε τη μέθοδο ShowLinkAnnotations

Εδώ συμβαίνει η μαγεία! Θα δημιουργήσετε μια μέθοδο εξαγωγής του κειμένου υπερσύνδεσης σε κάθε σελίδα. Ακολουθεί μια απλοποιημένη έκδοση αυτής της μεθόδου:

```csharp
private static void ShowLinkAnnotations(Page page)
{
    foreach (Annotation annotation in page.Annotations)
    {
        if (annotation is LinkAnnotation link)
        {
            Console.WriteLine("Link Text: " + link.Title);
            Console.WriteLine("Link URI: " + link.Action.URI);
        }
    }
}
```

-  Ελέγξτε εάν ο σχολιασμός είναι σύνδεσμος: Εδώ, ελέγχουμε αν ο σχολιασμός στη σελίδα είναι α`LinkAnnotation`. Εάν είναι, προχωράμε στην εξαγωγή του τίτλου και του URI του.
-  Εμφάνιση του κειμένου υπερσύνδεσης: Χρησιμοποιώντας`Console.WriteLine`, εκτυπώνουμε το κείμενο του συνδέσμου και το αντίστοιχο URI.
