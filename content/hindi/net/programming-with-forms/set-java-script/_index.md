इस गाइड में, हम चरण दर चरण बताएंगे कि PDF दस्तावेज़ के फ़ॉर्म फ़ील्ड में JavaScript को परिभाषित करने के लिए .NET के लिए Aspose.PDF लाइब्रेरी का उपयोग कैसे करें। हम आपको दिखाएंगे कि टेक्स्ट फ़ील्ड पर विशिष्ट ऑपरेशन करने के लिए JavaScript क्रियाओं को कैसे कॉन्फ़िगर किया जाए।

## आवश्यक शर्तें

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- आपके सिस्टम पर स्थापित .NET विकास वातावरण.
- .NET के लिए Aspose.PDF लाइब्रेरी। आप इसे Aspose आधिकारिक वेबसाइट से डाउनलोड कर सकते हैं।

## चरण 1: दस्तावेज़ निर्देशिका को कॉन्फ़िगर करना

 पहला कदम उस डॉक्यूमेंट डायरेक्टरी को कॉन्फ़िगर करना है जहाँ वह पीडीएफ फाइल स्थित है जिस पर आप काम करना चाहते हैं।`dataDir` निर्देशिका पथ निर्दिष्ट करने के लिए चर.

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

 प्रतिस्थापित करना सुनिश्चित करें`"YOUR DOCUMENTS DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

## चरण 2: इनपुट पीडीएफ फाइल लोड करना

 इस चरण में, हम इनपुट पीडीएफ फाइल को लोड करेंगे`Document` Aspose.PDF का वर्ग.

```csharp
// इनपुट पीडीएफ फाइल लोड करें
Document doc = new Document(dataDir + "SetJavaScript.pdf");
```

सुनिश्चित करें कि इनपुट पीडीएफ फाइल निर्दिष्ट दस्तावेज़ निर्देशिका में मौजूद है।

## चरण 3: टेक्स्टबॉक्स फ़ील्ड तक पहुँचना

 जावास्क्रिप्ट को किसी विशिष्ट टेक्स्ट फ़ील्ड पर लागू करने के लिए, हमें पहले उस फ़ील्ड तक पहुँचना होगा। इस उदाहरण में, हम मानते हैं कि टेक्स्ट फ़ील्ड को "textbox1" कहा जाता है।`doc.Form["textbox1"]` संगत प्राप्त करने की विधि`TextBoxField` वस्तु।

```csharp
TextBoxField field = (TextBoxField)doc.Form["textbox1"];
```

सुनिश्चित करें कि निर्दिष्ट टेक्स्ट फ़ील्ड इनपुट PDF फ़ाइल में मौजूद है।

## चरण 4: जावास्क्रिप्ट क्रियाएँ कॉन्फ़िगर करें

 अब जब हमने टेक्स्ट फ़ील्ड तक पहुँच प्राप्त कर ली है, तो हम इस फ़ील्ड से जुड़ी जावास्क्रिप्ट क्रियाओं को कॉन्फ़िगर कर सकते हैं। इस उदाहरण में, हम दो क्रियाओं का उपयोग करेंगे:`OnModifyCharacter` और`OnFormat` इन क्रियाओं को परिभाषित किया जाएगा`JavascriptAction` वस्तुएं.

```csharp
field.Actions.OnModifyCharacter = new JavascriptAction("AFNumber_Keystroke(2, 1, 1, 0, \"\", true)");
field.Actions.OnFormat = new JavascriptAction("AFNumber_Format(2, 1, 1, 0, \"\", true)");
```

अपनी आवश्यकताओं के अनुसार जावास्क्रिप्ट क्रियाओं को अनुकूलित करना सुनिश्चित करें।

## चरण 5: प्रारंभिक फ़ील्ड मान सेट करना

परिणामी PDF को सहेजने से पहले, हम टेक्स्ट फ़ील्ड के लिए एक प्रारंभिक मान सेट कर सकते हैं। इस उदाहरण में, हम फ़ील्ड के लिए "123" मान सेट करेंगे।

```csharp
field.Value = "123";
```

इस मान को अपनी आवश्यकताओं के अनुसार अनुकूलित करें.

## चरण 6: परिणामी पीडीएफ को सहेजना

 अब जब हमने टेक्स्ट फ़ील्ड और जावास्क्रिप्ट क्रियाएँ सेट कर ली हैं, तो हम परिणामी पीडीएफ को इसका उपयोग करके सहेज सकते हैं`Save` की विधि`Document` कक्षा।

```csharp
dataDir = dataDir + "Restricted_out.pdf";
// परिणामी PDF सहेजें
doc.Save(dataDir);
```

परिणामी PDF के लिए पूर्ण पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।


### .NET के लिए Aspose.PDF का उपयोग करके जावा स्क्रिप्ट सेट करने के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// इनपुट पीडीएफ फाइल लोड करें
Document doc = new Document(dataDir + "SetJavaScript.pdf");
TextBoxField field = (TextBoxField)doc.Form["textbox1"];
// बिंदु के बाद 2 अंक
// कोई विभाजक नहीं
// नकारात्मक शैली = ऋणात्मक
// कोई मुद्रा नहीं
field.Actions.OnModifyCharacter = new JavascriptAction("AFNumber_Keystroke(2, 1, 1, 0, \"\", true)");
field.Actions.OnFormat = new JavascriptAction("AFNumber_Format(2, 1, 1, 0, \"\", true)");
// प्रारंभिक फ़ील्ड मान सेट करें
field.Value = "123";
dataDir = dataDir + "Restricted_out.pdf";
// परिणामी PDF सहेजें
doc.Save(dataDir);
Console.WriteLine("\nJavaScript on form field setup successfully.\nFile saved at " + dataDir);
```

## निष्कर्ष

इस गाइड में, हमने सीखा कि PDF दस्तावेज़ के फ़ॉर्म फ़ील्ड में JavaScript सेट करने के लिए .NET के लिए Aspose.PDF लाइब्रेरी का उपयोग कैसे करें। बताए गए चरणों का पालन करके, आप टेक्स्ट फ़ील्ड पर विभिन्न ऑपरेशन करने के लिए JavaScript क्रियाओं को कस्टमाइज़ कर सकते हैं। PDF फ़ाइलों में हेरफेर करने की संभावनाओं का विस्तार करने के लिए .NET के लिए Aspose.PDF की विशेषताओं को और अधिक जानने के लिए स्वतंत्र महसूस करें।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं अन्य फॉर्म तत्वों, जैसे चेकबॉक्स और रेडियो बटन में जावास्क्रिप्ट जोड़ने के लिए .NET के लिए Aspose.PDF का उपयोग कर सकता हूं?

 उत्तर: हां, .NET के लिए Aspose.PDF आपको चेकबॉक्स, रेडियो बटन और ड्रॉपडाउन सूचियों सहित विभिन्न फ़ॉर्म तत्वों में जावास्क्रिप्ट जोड़ने की अनुमति देता है। आप इसका उपयोग कर सकते हैं`JavascriptAction` विभिन्न फॉर्म तत्वों के लिए जावास्क्रिप्ट क्रियाओं को परिभाषित करने के लिए क्लास।

#### प्रश्न: क्या फॉर्म फ़ील्ड में जावास्क्रिप्ट का उपयोग करके उपयोगकर्ता इनपुट को मान्य करना संभव है?

 उत्तर: हाँ, आप फ़ॉर्म फ़ील्ड में उपयोगकर्ता इनपुट को मान्य करने के लिए जावास्क्रिप्ट का उपयोग कर सकते हैं। जावास्क्रिप्ट क्रियाओं को परिभाषित करके जैसे`OnBlur` या`OnKeystroke` किसी फॉर्म फ़ील्ड के लिए, आप दर्ज किए गए डेटा को सत्यापित कर सकते हैं और यदि आवश्यक हो तो त्रुटि संदेश प्रदर्शित कर सकते हैं।

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF का उपयोग करके जटिल जावास्क्रिप्ट फ़ंक्शन निष्पादित कर सकता हूं?

 उत्तर: हां, आप .NET के लिए Aspose.PDF का उपयोग करके जटिल जावास्क्रिप्ट फ़ंक्शन निष्पादित कर सकते हैं। आपके पास कस्टम जावास्क्रिप्ट फ़ंक्शन परिभाषित करने और उन्हें कॉल करने की सुविधा है।`JavascriptAction`.

#### प्रश्न: क्या .NET के लिए Aspose.PDF इस ट्यूटोरियल में उल्लिखित के अलावा अन्य जावास्क्रिप्ट इवेंट का समर्थन करता है?

 उत्तर: हां, .NET के लिए Aspose.PDF जावास्क्रिप्ट घटनाओं की एक विस्तृत श्रृंखला का समर्थन करता है, जिसमें शामिल हैं`OnMouseEnter`, `OnMouseExit`, `OnMouseDown` , और`OnMouseUp`, दूसरों के बीच। आप इन घटनाओं का उपयोग उपयोगकर्ता इंटरैक्शन के आधार पर जावास्क्रिप्ट क्रियाओं को ट्रिगर करने के लिए कर सकते हैं।

#### प्रश्न: क्या मैं मौजूदा PDF दस्तावेज़ों से जावास्क्रिप्ट कोड निकालने के लिए .NET के लिए Aspose.PDF का उपयोग कर सकता हूं?

 उत्तर: .NET के लिए Aspose.PDF मौजूदा PDF दस्तावेज़ों से जावास्क्रिप्ट कोड निकालने की क्षमता प्रदान करता है। आप इसका उपयोग कर सकते हैं`JavascriptAction` क्लास और अन्य प्रासंगिक विधियों का उपयोग पीडीएफ फॉर्म में जावास्क्रिप्ट क्रियाओं तक पहुंचने और उनका विश्लेषण करने के लिए किया जा सकता है।
