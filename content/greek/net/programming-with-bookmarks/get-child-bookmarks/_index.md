## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Πρώτα πράγματα πρώτα, πρέπει να καθορίσετε τη διαδρομή προς το έγγραφο PDF σας. Εδώ το Aspose θα αναζητήσει το αρχείο.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το αρχείο PDF σας.

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Στη συνέχεια, θα ανοίξουμε το έγγραφο PDF χρησιμοποιώντας το`Document` τάξη που παρέχεται από το Aspose.PDF.

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "GetChildBookmarks.pdf");
```

 Αυτή η γραμμή κώδικα αρχικοποιεί μια νέα`Document` αντικείμενο με το αρχείο PDF σας. Βεβαιωθείτε ότι το όνομα αρχείου ταιριάζει με αυτό που έχετε.

## Βήμα 3: Κάνε βρόχο μέσω σελιδοδεικτών

Τώρα, ας δούμε όλους τους σελιδοδείκτες στο PDF. Εδώ συμβαίνει η μαγεία!

```csharp
// Περιηγηθείτε σε όλους τους σελιδοδείκτες
foreach (OutlineItemCollection outlineItem in pdfDocument.Outlines)
{
    Console.WriteLine(outlineItem.Title);
    Console.WriteLine(outlineItem.Italic);
    Console.WriteLine(outlineItem.Bold);
    Console.WriteLine(outlineItem.Color);
```

 Εδώ, χρησιμοποιούμε ένα`foreach` βρόχο για επανάληψη σε κάθε σελιδοδείκτη στο έγγραφο. Εκτυπώνουμε τον τίτλο, την πλάγια κατάσταση, την έντονη γραφή και το χρώμα κάθε σελιδοδείκτη.

## Βήμα 4: Ελέγξτε για παιδικούς σελιδοδείκτες

Μερικές φορές, οι σελιδοδείκτες έχουν θυγατρικούς σελιδοδείκτες. Ας ελέγξουμε αν υπάρχουν και ας τα περιηγηθούμε επίσης.

```csharp
if (outlineItem.Count > 0)
{
    Console.WriteLine("Child Bookmarks");
    // Υπάρχουν θυγατρικοί σελιδοδείκτες και στη συνέχεια πραγματοποιήστε βρόχο μέσω αυτού επίσης
    foreach (OutlineItemCollection childOutline in outlineItem)
    {
        Console.WriteLine(childOutline.Title);
        Console.WriteLine(childOutline.Italic);
        Console.WriteLine(childOutline.Bold);
        Console.WriteLine(childOutline.Color);
    }
}
```

Σε αυτό το απόσπασμα, ελέγχουμε αν ο τρέχων σελιδοδείκτης έχει θυγατρικούς σελιδοδείκτες. Εάν συμβαίνει, τα περιηγούμαστε και εκτυπώνουμε τα στοιχεία τους.
