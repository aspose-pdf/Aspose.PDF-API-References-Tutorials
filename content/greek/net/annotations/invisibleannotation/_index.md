Οι σχολιασμοί σε αρχείο PDF είναι μια ισχυρή δυνατότητα που σας επιτρέπει να προσθέτετε επιπλέον πληροφορίες ή σημειώσεις σε ένα έγγραφο χωρίς να αλλάξετε το πραγματικό περιεχόμενο. Μπορούν να χρησιμοποιηθούν για να επισημάνουν κείμενο, να τραβήξουν την προσοχή σε συγκεκριμένες περιοχές ενός εγγράφου ή να προσθέσουν σχόλια ή σχόλια.

Υπάρχουν πολλοί διαφορετικοί τύποι σχολιασμών που μπορείτε να χρησιμοποιήσετε σε έγγραφα PDF, όπως:

- Σχολιασμοί κειμένου
- Σχολιασμοί συνδέσμων
- Σχολιασμοί γραμματοσήμων
- Σχολιασμοί ήχου
- Σχολιασμοί συνημμένου αρχείου
- και πολλά άλλα

## Βήμα 1: Δημιουργία ενός αόρατου σχολιασμού σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET

 Για να δημιουργήσουμε έναν αόρατο σχολιασμό σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET, πρέπει πρώτα να δημιουργήσουμε ένα`FreeTextAnnotation` αντικείμενο και καθορίστε τη θέση και το μέγεθος του σχολιασμού.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Άνοιγμα εγγράφου
Document doc = new Document(dataDir + "input.pdf");

FreeTextAnnotation annotation = new FreeTextAnnotation(doc.Pages[1], new Aspose.Pdf.Rectangle(50, 600, 250, 650), new DefaultAppearance("Helvetica", 16, System.Drawing.Color.Red));
```

 Στον παραπάνω κώδικα, δημιουργούμε ένα`FreeTextAnnotation`αντικείμενο και καθορίστε τη θέση του σχολιασμού στη σελίδα 2 του εγγράφου PDF. Καθορίζουμε επίσης τον τύπο γραμματοσειράς, το μέγεθος και το χρώμα για το κείμενο που θα εμφανίζεται στον σχολιασμό.

## Βήμα 2: Προσθήκη χαρακτηριστικών στον αόρατο σχολιασμό

Στη συνέχεια, μπορούμε να προσθέσουμε ορισμένα χαρακτηριστικά στον σχολιασμό, όπως χρώμα περιγράμματος, χρώμα φόντου ή αδιαφάνεια.

```csharp
annotation.Characteristics.Border = System.Drawing.Color.Red;
```

Στον παραπάνω κώδικα, ορίσαμε το χρώμα του περιγράμματος του σχολιασμού σε κόκκινο.

## Βήμα 3: Ρύθμιση των σημαιών σχολιασμού

Αφού δημιουργήσουμε τον σχολιασμό και ορίσουμε τα χαρακτηριστικά του, μπορούμε να καθορίσουμε τις σημαίες σχολιασμού. Σε αυτό το σεμινάριο, θέλουμε ο σχολιασμός να είναι εκτυπώσιμος, αλλά όχι ορατός.

```csharp
annotation.Flags = AnnotationFlags.Print | AnnotationFlags.NoView;
doc.Pages[1].Annotations.Add(annotation);
```

## Βήμα 4: Αποθήκευση του τροποποιημένου εγγράφου PDF

Τέλος, μπορούμε να αποθηκεύσουμε το τροποποιημένο έγγραφο PDF με τον νέο αόρατο σχολιασμό.

```csharp
dataDir = dataDir + "InvisibleAnnotation_out.pdf";
doc.Save(dataDir);
```

## Παράδειγμα πηγαίου κώδικα για τον τρόπο δημιουργίας αόρατου σχολιασμού χρησιμοποιώντας το Aspose.PDF για .NET

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Άνοιγμα εγγράφου
Document doc = new Document(dataDir + "input.pdf");

FreeTextAnnotation annotation = new FreeTextAnnotation(doc.Pages[1], new Aspose.Pdf.Rectangle(50, 600, 250, 650), new DefaultAppearance("Helvetica", 16, System.Drawing.Color.Red));
annotation.Contents = "ABCDEFG";
annotation.Characteristics.Border = System.Drawing.Color.Red;
annotation.Flags = AnnotationFlags.Print | AnnotationFlags.NoView;
doc.Pages[1].Annotations.Add(annotation);

dataDir = dataDir + "InvisibleAnnotation_out.pdf";
// Αποθήκευση αρχείου εξόδου
doc.Save(dataDir);
// ExEnd:InvisibleAnnotation
Console.WriteLine("\nAnnotation nvisible successfully.\nFile saved at " + dataDir);
```
