## การเพิ่ม Aspose.PDF สำหรับ Java ในโครงการของคุณ

เมื่อโปรเจ็กต์ของคุณพร้อมแล้ว ก็ถึงเวลาเพิ่ม Aspose.PDF สำหรับ Java เข้าไป ต่อไปนี้คือวิธีที่คุณสามารถทำได้:

1.  ดาวน์โหลดไลบรารี Aspose.PDF สำหรับ Java จาก[ที่นี่](https://releases.aspose.com/pdf/java/).
2. เพิ่มไฟล์ JAR ที่ดาวน์โหลดไปยัง classpath ของโปรเจ็กต์ของคุณ

## การสร้างแอปพลิเคชัน Java อย่างง่าย

มาสร้างแอปพลิเคชัน Java ง่ายๆ ที่แปลงไฟล์ข้อความเป็นเอกสาร PDF กันดีกว่า

```java
import com.aspose.pdf.Document;
import com.aspose.pdf.TextFragment;

public class TextToPDFConverter {
    public static void main(String[] args) {
        // โหลดไฟล์ข้อความ
        Document pdfDocument = new Document();
        Page page = pdfDocument.getPages().add();
        
        // เพิ่มข้อความลงใน PDF
        TextFragment textFragment = new TextFragment("Hello, Aspose.PDF for Java!");
        page.getParagraphs().add(textFragment);
        
        // บันทึกไฟล์ PDF
        pdfDocument.save("output.pdf");
        
        System.out.println("Text file converted to PDF successfully!");
    }
}
```

ในโค้ดนี้ เราสร้างเอกสาร PDF แบบธรรมดาพร้อมส่วนข้อความและบันทึกเป็น "output.pdf"
