Μια τέτοια βελτίωση που ίσως θέλετε να εφαρμόσετε είναι η προσθήκη ενός περιγράμματος γύρω από το κείμενό σας σε ένα αρχείο PDF. Σε αυτόν τον οδηγό, θα σας καθοδηγήσουμε στα βήματα για να προσθέσετε ένα περίγραμμα κειμένου σε ένα αρχείο PDF χρησιμοποιώντας τη βιβλιοθήκη Aspose.PDF για .NET. Λοιπόν, ας βουτήξουμε αμέσως!

## Εισαγωγή πακέτων

Τώρα που έχουμε ρυθμίσει τα πάντα, ας εισάγουμε τα απαραίτητα πακέτα για να χρησιμοποιήσουμε το Aspose.PDF στο έργο μας. Μπορείτε να το κάνετε αυτό προσθέτοντας τα ακόλουθα χρησιμοποιώντας οδηγίες στην κορυφή του αρχείου C#:

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

```

Αυτοί οι χώροι ονομάτων θα σας επιτρέψουν να εργάζεστε αποτελεσματικά με έγγραφα PDF και θραύσματα κειμένου. 

Τώρα, ας αναλύσουμε τη διαδικασία προσθήκης περιγράμματος κειμένου σε λεπτομερή βήματα. Θα περάσουμε από κάθε βήμα για να καταλάβετε τι ακριβώς συμβαίνει κάτω από το καπό.

## Βήμα 1: Ρύθμιση του εγγράφου

Πρώτα πράγματα πρώτα, πρέπει να δημιουργήσουμε ένα νέο έγγραφο PDF. Εδώ θα γίνει όλη η μαγεία μας.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Δημιουργία νέου αντικειμένου εγγράφου
Document pdfDocument = new Document();
```

 Σε αυτό το βήμα, καθορίζουμε τον κατάλογο στον οποίο θέλουμε να αποθηκεύσουμε το αρχείο PDF μας. Στη συνέχεια δημιουργούμε μια νέα παρουσία του`Document` τάξη, η οποία αντιπροσωπεύει το έγγραφο PDF μας.

## Βήμα 2: Προσθήκη νέας σελίδας

Στη συνέχεια, πρέπει να προσθέσουμε μια σελίδα στο έγγραφό μας. Σκεφτείτε αυτό ως προσθήκη ενός κενού καμβά όπου θα τοποθετήσουμε το κείμενό μας.

```csharp
// Λάβετε συγκεκριμένη σελίδα
Page pdfPage = (Page)pdfDocument.Pages.Add();
```

 Εδώ, καλούμε το`Add()` μέθοδος στο`Pages` συλλογή μας`pdfDocument` αντικείμενο. Αυτό προσθέτει μια νέα σελίδα στο έγγραφο και αποθηκεύουμε μια αναφορά σε αυτό στο`pdfPage` μεταβλητός.

## Βήμα 3: Δημιουργήστε ένα τμήμα κειμένου

Τώρα, ας δημιουργήσουμε το κείμενο που θέλουμε να εμφανίσουμε στο PDF μας. Εδώ ορίζουμε το περιεχόμενο του τμήματος του κειμένου μας.

```csharp
// Δημιουργία τμήματος κειμένου
TextFragment textFragment = new TextFragment("main text");
textFragment.Position = new Position(100, 600);
```

 Σε αυτόν τον κώδικα, δημιουργούμε έναν νέο`TextFragment` αντικείμενο με το κείμενο "κύριο κείμενο". Ορίζουμε επίσης τη θέση του στη σελίδα χρησιμοποιώντας το`Position` τάξη. Οι συντεταγμένες (100, 600) καθορίζουν πού θα τοποθετηθεί το κείμενο στη σελίδα.

## Βήμα 4: Ορισμός ιδιοτήτων κειμένου

Στη συνέχεια, θα προσαρμόσουμε το τμήμα κειμένου μας για να το κάνουμε οπτικά ελκυστικό. Αυτό περιλαμβάνει τη ρύθμιση του μεγέθους της γραμματοσειράς, του τύπου γραμματοσειράς, του χρώματος φόντου και του χρώματος προσκηνίου.

```csharp
// Ορισμός ιδιοτήτων κειμένου
textFragment.TextState.FontSize = 12;
textFragment.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment.TextState.BackgroundColor = Aspose.Pdf.Color.LightGray;
textFragment.TextState.ForegroundColor = Aspose.Pdf.Color.Red;
```

Εδώ, ορίζουμε το μέγεθος της γραμματοσειράς στο 12, χρησιμοποιούμε το "Times New Roman" ως γραμματοσειρά και εφαρμόζουμε ένα ανοιχτό γκρι χρώμα φόντου με κόκκινο κείμενο. Αυτές οι ιδιότητες συμβάλλουν στη βελτίωση της ορατότητας του κειμένου.

## Βήμα 5: Ορίστε το χρώμα χαϊδεύματος για το περίγραμμα

Τώρα, φτάνουμε στο συναρπαστικό μέρος - προσθέτοντας ένα περίγραμμα γύρω από το κείμενό μας!

```csharp
// Ορίστε την ιδιότητα StrokingColor για σχεδίαση περιγράμματος (χαϊδεύοντας) γύρω από το ορθογώνιο κειμένου
textFragment.TextState.StrokingColor = Aspose.Pdf.Color.DarkRed;
```

Σε αυτό το βήμα, καθορίζουμε το χρώμα του περιγράμματος που θέλουμε να σχεδιάσουμε γύρω από το κείμενό μας. Εδώ, επιλέξαμε ένα σκούρο κόκκινο χρώμα.

## Βήμα 6: Ενεργοποιήστε το πλαίσιο του ορθογώνιου κειμένου

 Για να σχεδιάσουμε πραγματικά το περίγραμμα γύρω από το κείμενό μας, πρέπει να ενεργοποιήσουμε το`DrawTextRectangleBorder` ιδιοκτησία.

```csharp
// Ορίστε την τιμή της ιδιότητας DrawTextRectangleBorder σε true
textFragment.TextState.DrawTextRectangleBorder = true;
```

 Ορίζοντας αυτήν την ιδιότητα σε`true`, λέμε στο Aspose.PDF να σχεδιάσει το περίγραμμα γύρω από το ορθογώνιο κειμένου με βάση το καθορισμένο χρώμα χαϊδεύματος.

## Βήμα 7: Προσθέστε το Τμήμα κειμένου στη σελίδα

Τώρα που έχουμε έτοιμο το τμήμα κειμένου μας με όλες τις ιδιότητες που έχουν οριστεί, ήρθε η ώρα να το προσθέσουμε στη σελίδα.

```csharp
TextBuilder tb = new TextBuilder(pdfPage);
tb.AppendText(textFragment);
```

 Εδώ, δημιουργούμε ένα`TextBuilder` αντικείμενο που σχετίζεται με το δικό μας`pdfPage` . Στη συνέχεια χρησιμοποιούμε το`AppendText` μέθοδος προσθήκης μας`textFragment` στη σελίδα. 

## Βήμα 8: Αποθηκεύστε το έγγραφο

Τέλος, πρέπει να αποθηκεύσουμε το έγγραφο PDF μας στον καθορισμένο κατάλογο. Αυτή είναι η στιγμή της αλήθειας!

```csharp
// Αποθηκεύστε το έγγραφο
pdfDocument.Save(dataDir + "PDFWithTextBorder_out.pdf");
```

Σε αυτό το βήμα, καλούμε το`Save` μέθοδος σε μας`pdfDocument` αντικείμενο, παρέχοντας τη διαδρομή όπου θέλουμε να αποθηκεύσουμε το αρχείο. Μόλις εκτελέσετε τον κώδικα, θα πρέπει να βρείτε το PDF που δημιουργήθηκε πρόσφατα με το περίγραμμα κειμένου στον καθορισμένο κατάλογο!
