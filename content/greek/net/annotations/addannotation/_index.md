## Βήμα 1: Φορτώστε το έγγραφο PDF σας

Πρώτα πράγματα πρώτα, πρέπει να φορτώσετε το έγγραφο PDF όπου σκοπεύετε να προσθέσετε τον σχολιασμό.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DATA DIRECTORY";
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

 Να τι συμβαίνει: καθορίζετε τον κατάλογο όπου είναι αποθηκευμένο το αρχείο PDF και, στη συνέχεια, το φορτώνετε χρησιμοποιώντας το`Document` τάξη που παρέχεται από το Aspose.PDF. Αυτό το βήμα είναι κρίσιμο γιατί χωρίς να φορτώσετε το έγγραφο, δεν μπορείτε να κάνετε αλλαγές σε αυτό.

## Βήμα 2: Δημιουργήστε έναν σχολιασμό

### Καθορισμός των ιδιοτήτων σχολιασμού
 Τώρα, ας δημιουργήσουμε τον ίδιο τον σχολιασμό. Θα χρησιμοποιήσουμε α`TextAnnotation`, το οποίο είναι ιδανικό για την προσθήκη σχολίων ή σημειώσεων στο PDF σας.

```csharp
// Δημιουργία σχολιασμού
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

Σε αυτό το απόσπασμα:
-  Τοποθεσία και μέγεθος: Το`Rectangle` Η κλάση ορίζει πού στη σελίδα θα εμφανίζεται ο σχολιασμός σας και τις διαστάσεις του.
- Τίτλος, Θέμα και Περιεχόμενα: Αυτές οι ιδιότητες σάς δίνουν τη δυνατότητα να προσδιορίσετε τι είναι ο σχολιασμός σας και τι θα περιέχει.
-  Εικονίδιο: Το`TextIcon.Key` ορίζει ένα εικονίδιο για τον σχολιασμό, καθιστώντας τον πιο ελκυστικό οπτικά.

## Βήμα 3: Προσαρμόστε την εμφάνιση του σχολιασμού

Στη συνέχεια, ας κάνουμε αυτόν τον σχολιασμό να ξεχωρίζει προσθέτοντας ένα περίγραμμα και τροποποιώντας την εμφάνισή του.

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

Ακολουθεί μια ανάλυση του τι συμβαίνει:
-  Σύνορα: Δημιουργούμε ένα`Border` αντικείμενο και ορίστε το πλάτος του σε 5, δίνοντας στον σχολιασμό μας εμφανές περίγραμμα.
-  Μοτίβο παύλας: Το`Dash` Η ιδιότητα σάς επιτρέπει να δημιουργήσετε ένα διακεκομμένο περίγραμμα, προσθέτοντας λίγο στυλ στον σχολιασμό.

## Βήμα 4: Προσθέστε τον σχολιασμό στη σελίδα PDF

Αφού δημιουργήσετε και προσαρμόσετε τον σχολιασμό, ήρθε η ώρα να τον προσθέσετε στη σελίδα PDF σας.

```csharp
// Προσθέστε σχολιασμούς στη συλλογή σχολιασμών της σελίδας
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

 Αυτός ο κώδικας προσθέτει τον σχολιασμό στην πρώτη σελίδα του PDF σας. Ο`Annotations` Η συλλογή περιέχει όλους τους σχολιασμούς για μια συγκεκριμένη σελίδα και αυτό το βήμα διασφαλίζει ότι ο νέος σας σχολιασμός είναι μέρος αυτής της συλλογής.

## Βήμα 5: Αποθηκεύστε το ενημερωμένο έγγραφο PDF

Τέλος, ας αποθηκεύσουμε το έγγραφο έτσι ώστε ο σχολιασμός σας να προστεθεί μόνιμα.

```csharp
// Αποθήκευση αρχείου εξόδου
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

Αποθηκεύοντας το έγγραφο με νέο όνομα (`AddAnnotation_out.pdf`), διατηρείτε το αρχικό αρχείο και δημιουργείτε ένα νέο με προσθήκη του σχολιασμού. Το μήνυμα της κονσόλας επιβεβαιώνει ότι όλα ήταν επιτυχή και τώρα μπορείτε να βρείτε το σχολιασμένο PDF στον καθορισμένο κατάλογο.
