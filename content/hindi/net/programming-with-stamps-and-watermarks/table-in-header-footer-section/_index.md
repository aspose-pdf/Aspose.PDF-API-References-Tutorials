इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के हेडर या फ़ुटर अनुभाग में टेबल जोड़ने के तरीके के बारे में चरण दर चरण मार्गदर्शन करेंगे। प्रदान किया गया C# स्रोत कोड आपको दिखाता है कि खाली PDF दस्तावेज़ कैसे बनाएँ, एक पृष्ठ कैसे जोड़ें, हेडर अनुभाग को कॉन्फ़िगर करें, एक टेबल बनाएँ, टेबल में पंक्तियाँ और सेल जोड़ें, और अंत में PDF दस्तावेज़ को कैसे सेव करें।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- एक स्थापित .NET विकास वातावरण.
- .NET के लिए Aspose.PDF लाइब्रेरी डाउनलोड की गई और आपके प्रोजेक्ट में संदर्भित की गई।

## चरण 2: पीडीएफ दस्तावेज़ और पृष्ठ बनाना

 पहला कदम इसका एक उदाहरण बनाना है`Document` क्लास में जाकर दस्तावेज़ में एक पेज जोड़ें। यहाँ बताया गया है कि कैसे:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document();

// पीडीएफ दस्तावेज़ में एक पृष्ठ बनाएँ
Aspose.Pdf.Page page = pdfDocument.Pages.Add();
```

"आपकी दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें जहां आप पीडीएफ दस्तावेज़ को सहेजना चाहते हैं।

## चरण 3: हेडर अनुभाग को कॉन्फ़िगर करना

 अब हम पीडीएफ दस्तावेज़ के हेडर अनुभाग को एक उदाहरण बनाकर कॉन्फ़िगर करेंगे`HeaderFooter` कक्षा। यहाँ बताया गया है कि कैसे:

```csharp
// पीडीएफ फाइल के लिए हेडर अनुभाग बनाएं
Aspose.Pdf.HeaderFooter header = new Aspose.Pdf.HeaderFooter();

// पृष्ठ के लिए शीर्ष लेख अनुभाग परिभाषित करें
page. Header = header;

// हेडर अनुभाग का शीर्ष मार्जिन सेट करें
header. Margin. Top = 20;
```

## चरण 4: तालिका बनाना

 अब हम इसका उपयोग करके एक तालिका बनाने जा रहे हैं`Table` क्लास में जाकर इसे हेडिंग सेक्शन के पैराग्राफ कलेक्शन में जोड़ें। यहाँ बताया गया है कि कैसे:

```csharp
// टेबल ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

// शीर्ष लेख अनुभाग के पैराग्राफ़ संग्रह में तालिका जोड़ें
header.Paragraphs.Add(tab1);

// तालिका के स्तंभों की चौड़ाई निर्धारित करें
tab1.ColumnWidths = "60,300";
```

उपरोक्त कोड निर्दिष्ट चौड़ाई के दो स्तंभों वाली एक तालिका बनाता है।

## चरण 5: तालिका में पंक्तियाँ और कक्ष जोड़ें

 अब हम तालिका में पंक्तियाँ और कक्ष जोड़ेंगे`Row` वर्ग और`Cell` कक्षा। यहाँ बताया गया है कि कैसे:

```csharp
// तालिका में एक पंक्ति बनाएं और कक्ष जोड़ें
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("Table in header section");
row1.BackgroundColor = Color.Gray;

// पहली पंक्ति के पहले सेल को मर्ज करें
tab1.Rows[0].Cells[0].ColSpan = 2;
tab1.Rows[0].Cells[0].DefaultCellTextState.ForegroundColor = Color.Cyan;
tab1.Rows[0].Cells[0].DefaultCellTextState.Font = FontRepository.FindFont("Helvetica");

// तालिका में एक और पंक्ति बनाएं और एक छवि वाला कक्ष जोड़ें
Aspose.Pdf.Row row2 = tab1.Rows.Add();
row2.BackgroundColor = Color.White;
Aspose.Pdf.Cell cell2 = row2.Cells.Add();
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose-logo.jpg";
img. FixWidth = 60;
cell2.Paragraphs.Add(img);
row2.Cells.Add("The logo is beautiful!");
row2.Cells[1].DefaultCellTextState.Font = FontRepository.FindFont("Helvetica");
row2.Cells[1].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
row2.Cells[1].Alignment = Aspose.Pdf.HorizontalAlignment.Center;
```

## चरण 6: पीडीएफ दस्तावेज़ को सहेजना

एक बार जब तालिका को हेडर अनुभाग में जोड़ दिया जाता है, तो हम PDF दस्तावेज़ को सहेज सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// पीडीएफ फाइल को सेव करें
pdfDocument.Save(dataDir + "TableInHeaderFooterSection_out.pdf");
```

"आपकी दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें जहां आप पीडीएफ दस्तावेज़ को सहेजना चाहते हैं।

### .NET के लिए Aspose.PDF का उपयोग करके हेडर फ़ुटर अनुभाग में तालिका के लिए नमूना स्रोत कोड 
```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// खाली कन्स्ट्रक्टर को कॉल करके डॉक्यूमेंट इंस्टैंस को इंस्टैंसिएट करें
Aspose.Pdf.Document pdfDocument = new Aspose.Pdf.Document();

// पीडीएफ दस्तावेज़ में एक पृष्ठ बनाएँ
Aspose.Pdf.Page page = pdfDocument.Pages.Add();

//पीडीएफ फाइल का हेडर अनुभाग बनाएं
Aspose.Pdf.HeaderFooter header = new Aspose.Pdf.HeaderFooter();

// पीडीएफ फाइल के लिए विषम हेडर सेट करें
page.Header = header;

// हेडर अनुभाग के लिए शीर्ष मार्जिन सेट करें
header.Margin.Top = 20;

// तालिका ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

// इच्छित अनुभाग के पैराग्राफ संग्रह में तालिका जोड़ें
header.Paragraphs.Add(tab1);

// BorderInfo ऑब्जेक्ट का उपयोग करके डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);

// तालिका की स्तंभ चौड़ाई के साथ सेट करें
tab1.ColumnWidths = "60 300";
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose-logo.jpg";

// तालिका में पंक्तियाँ बनाएँ और फिर पंक्तियों में कक्ष बनाएँ
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("Table in Header Section");
row1.BackgroundColor = Color.Gray;

// पहली पंक्ति के लिए पंक्ति अवधि मान 2 के रूप में सेट करें
tab1.Rows[0].Cells[0].ColSpan = 2;
tab1.Rows[0].Cells[0].DefaultCellTextState.ForegroundColor = Color.Cyan;
tab1.Rows[0].Cells[0].DefaultCellTextState.Font = FontRepository.FindFont("Helvetica");

// तालिका में पंक्तियाँ बनाएँ और फिर पंक्तियों में कक्ष बनाएँ
Aspose.Pdf.Row row2 = tab1.Rows.Add();

// पंक्ति 2 के लिए पृष्ठभूमि रंग सेट करें
row2.BackgroundColor = Color.White;

// वह सेल जोड़ें जिसमें छवि है
Aspose.Pdf.Cell cell2 = row2.Cells.Add();

// छवि की चौड़ाई 60 पर सेट करें
img.FixWidth = 60;

// तालिका सेल में छवि जोड़ें
cell2.Paragraphs.Add(img);
row2.Cells.Add("Logo is looking fine !");
row2.Cells[1].DefaultCellTextState.Font = FontRepository.FindFont("Helvetica");

// पाठ के ऊर्ध्वाधर संरेखण को केंद्र संरेखित के रूप में सेट करें
row2.Cells[1].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
row2.Cells[1].Alignment = Aspose.Pdf.HorizontalAlignment.Center;

// पीडीएफ फाइल को सेव करें
pdfDocument.Save(dataDir + "TableInHeaderFooterSection_out.pdf");

```

## निष्कर्ष

बधाई हो! आपने सीख लिया है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के हेडर या फ़ुटर अनुभाग में तालिका कैसे जोड़ें। अब आप अपने PDF दस्तावेज़ों में अतिरिक्त जानकारी प्रदर्शित करने के लिए तालिकाएँ जोड़कर अपने हेडर और फ़ुटर को कस्टमाइज़ कर सकते हैं।

### शीर्षलेख/पादलेख अनुभाग में तालिका के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ के शीर्षलेख या पादलेख अनुभाग में तालिका जोड़ने का उद्देश्य क्या है?

उत्तर: पीडीएफ दस्तावेज़ के शीर्षलेख या पादलेख अनुभाग में तालिका जोड़ने से आप संरचित और संगठित जानकारी प्रदर्शित कर सकते हैं, जैसे शीर्षक, उपशीर्षक, लोगो या कोई अन्य सामग्री जिसे आप दस्तावेज़ के प्रत्येक पृष्ठ पर लगातार प्रदर्शित करना चाहते हैं।

#### प्रश्न: उपलब्ध कराया गया C# स्रोत कोड किसी PDF दस्तावेज़ के शीर्षलेख या पादलेख अनुभाग में तालिका जोड़ने का कार्य किस प्रकार करता है?

उत्तर: कोड एक खाली पीडीएफ दस्तावेज़ बनाने, एक पृष्ठ जोड़ने, हेडर अनुभाग को कॉन्फ़िगर करने, पंक्तियों और कक्षों के साथ एक तालिका बनाने और अंत में पीडीएफ दस्तावेज़ को सहेजने की प्रक्रिया को प्रदर्शित करता है। परिणाम पीडीएफ दस्तावेज़ के हेडर अनुभाग में प्रदर्शित एक तालिका है।

#### प्रश्न: क्या मैं तालिका कक्षों के स्वरूप को अनुकूलित कर सकता हूँ, जैसे बॉर्डर, पृष्ठभूमि रंग और पाठ शैली?

उत्तर: हां, आप सेल बॉर्डर, पृष्ठभूमि रंग, पाठ शैली, फ़ॉन्ट, फ़ॉन्ट आकार आदि जैसे गुण सेट करके तालिका कोशिकाओं की उपस्थिति को अनुकूलित कर सकते हैं।

#### प्रश्न: पीडीएफ दस्तावेज़ के हेडर अनुभाग में तालिका कैसे जोड़ी जाती है?

उत्तर: कोड तालिका को हेडर अनुभाग के पैराग्राफ संग्रह में जोड़ता है, जो यह सुनिश्चित करता है कि तालिका प्रत्येक पृष्ठ के हेडर में प्रदर्शित हो।

#### प्रश्न: क्या मैं आवश्यकतानुसार तालिका में अधिक पंक्तियाँ और कक्ष जोड़ सकता हूँ?

 उत्तर: बिल्कुल, आप इसका उपयोग करके तालिका में अधिक पंक्तियाँ और कक्ष जोड़ सकते हैं`Rows.Add()` और`Cells.Add()` यह आपको तालिका सामग्री को इच्छानुसार संरचित करने की अनुमति देता है।

#### प्रश्न: क्या तालिका स्तंभों की चौड़ाई समायोजित करना संभव है?
 उत्तर: हां, आप तालिका स्तंभों की चौड़ाई को समायोजित कर सकते हैं`ColumnWidths` संपत्ति। यह आपको तालिका के लेआउट को नियंत्रित करने में सक्षम बनाता है।

#### प्रश्न: मैं तालिका के भीतर कोशिकाओं को एकाधिक स्तंभों या पंक्तियों में कैसे फैला सकता हूँ?
 उत्तर: कोशिकाओं को एकाधिक स्तंभों में फैलाने के लिए, आप इसका उपयोग कर सकते हैं`ColSpan`संबंधित सेल की संपत्ति। इसी तरह, आप उपयोग कर सकते हैं`RowSpan` कोशिकाओं को एकाधिक पंक्तियों में फैलाने के लिए संपत्ति।

#### प्रश्न: यदि मैं पीडीएफ दस्तावेज़ के शीर्षलेख और पादलेख दोनों अनुभागों में तालिका जोड़ना चाहूँ तो क्या होगा?

 उत्तर: आप हेडर और फ़ुटर दोनों अनुभागों के लिए एक समान दृष्टिकोण अपना सकते हैं। बस एक बनाएँ`HeaderFooter` फ़ुटर के लिए उदाहरण बनाएँ, इसे कॉन्फ़िगर करें, और तालिका को इसके पैराग्राफ़ संग्रह में जोड़ें।

#### प्रश्न: क्या मैं तालिका कक्षों में छवियों का उपयोग कर सकता हूँ, और यह कैसे संभव है?

 उत्तर: हां, आप टेबल सेल में इमेज जोड़ सकते हैं। कोड उदाहरण एक सेल में इमेज जोड़कर दिखाता है`Image` ऑब्जेक्ट का चयन करना, उसका फ़ाइल पथ और आयाम निर्धारित करना, और फिर उसे सेल के पैराग्राफ़ में जोड़ना।

#### प्रश्न: मैं यह कैसे सुनिश्चित करूँ कि तालिका PDF दस्तावेज़ के सभी पृष्ठों पर एक समान दिखाई दे?

 उत्तर: जब आप शीर्षलेख या पादलेख अनुभाग में तालिका जोड़ते हैं`HeaderFooter` उदाहरण के लिए, Aspose.PDF यह सुनिश्चित करता है कि तालिका प्रत्येक पृष्ठ पर एकसमान रूप से प्रदर्शित हो तथा एक समान लेआउट प्रदान करे।
