للبدء، ستحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

using System;

## الخطوة 1: إعداد دليل المستندات الخاص بك

قبل أن تتمكن من تحويل ملف PDF، يتعين عليك تحديد الدليل الذي يتم تخزين مستنداتك فيه. هذا هو المكان الذي ستتواجد فيه ملفات PDF المدخلة وملفات EPUB الصادرة.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي على جهازك حيث يوجد ملف PDF الخاص بك.

## الخطوة 2: تحميل مستند PDF

 الآن بعد أن قمت بإعداد دليل المستندات، فإن الخطوة التالية هي تحميل مستند PDF الذي تريد تحويله. يتم ذلك باستخدام`Document` تم توفير الفئة بواسطة Aspose.PDF.

```csharp
// تحميل مستند PDF
Document pdfDocument = new Document(dataDir + "PDFToEPUB.pdf");
```

 تأكد من الاستبدال`"PDFToEPUB.pdf"` مع اسم ملف PDF الخاص بك. يقوم هذا السطر من التعليمات البرمجية بتهيئة ملف جديد`Document` الكائن مع ملف PDF المحدد.

## الخطوة 3: إنشاء خيارات حفظ EPUB

 قبل حفظ المستند بتنسيق EPUB، تحتاج إلى إنشاء مثيل لـ`EpubSaveOptions`تتيح لك هذه الفئة تحديد خيارات مختلفة لإخراج EPUB.

```csharp
// إنشاء خيارات حفظ Epub
EpubSaveOptions options = new EpubSaveOptions();
```

 يؤدي هذا الخط إلى إنشاء مثيل جديد لـ`EpubSaveOptions`، والتي سوف تقوم بتكوينها في الخطوة التالية.

## الخطوة 4: تحديد وضع التعرف على المحتوى

من أهم ميزات EPUB قدرته على تدفق المحتوى. يمكنك تحديد كيفية التعرف على المحتوى أثناء عملية التحويل. في هذا المثال، سنقوم بتعيين وضع التعرف على المحتوى على التدفق.

```csharp
// تحديد تخطيط المحتويات
options.ContentRecognitionMode = EpubSaveOptions.RecognitionMode.Flow;
```

يضمن هذا الإعداد تحويل المحتوى الموجود في ملف PDF الخاص بك بطريقة مناسبة لقارئات EPUB، مما يسمح بتجربة قراءة أفضل.

## الخطوة 5: احفظ مستند EPUB

 أخيرًا، حان الوقت لحفظ المستند المحول. ستستخدم`Save` طريقة`Document` الصف للقيام بذلك.

```csharp
// حفظ مستند ePUB
pdfDocument.Save(dataDir + "PDFToEPUB_out.epub", options);
```

 يحفظ هذا السطر ملف EPUB المحول في نفس الدليل الذي يحتوي على ملف PDF الأصلي. سيتم تسمية ملف الإخراج`PDFToEPUB_out.epub`.
