## Εισαγωγή

Στο σημερινό ψηφιακό τοπίο, η αλληλεπίδραση με έγγραφα PDF είναι απαραίτητη προϋπόθεση τόσο για τις επιχειρήσεις όσο και για τα άτομα. Είτε δημιουργείτε, επεξεργάζεστε ή χειρίζεστε αρχεία PDF, το να έχετε τα σωστά εργαλεία στα χέρια σας κάνει τη διαφορά. Ένα από αυτά τα ισχυρά εργαλεία είναι το Aspose.PDF για .NET, μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να εργάζονται με αρχεία PDF απρόσκοπτα. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στον τρόπο ανάκτησης συντεταγμένων πεδίων φόρμας PDF χρησιμοποιώντας αυτήν τη βιβλιοθήκη. Μέχρι το τέλος αυτού του οδηγού, θα είστε εξοπλισμένοι με τις γνώσεις για να βελτιώσετε τις δεξιότητές σας στο χειρισμό PDF και να προσθέσετε μεγαλύτερη ευελιξία στις εφαρμογές σας.

## Προαπαιτούμενα

Πριν βουτήξουμε, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε για να ακολουθήσετε. Να τι θα χρειαστούμε:

1. Βασική κατανόηση της C#: Η εξοικείωση με τον προγραμματισμό C# είναι απαραίτητη, καθώς θα χρησιμοποιούμε αυτήν τη γλώσσα σε όλο το σεμινάριο.

4. Ένα δείγμα PDF με πεδία φόρμας: Για να δοκιμάσετε τον κώδικα, έχετε έτοιμο ένα δείγμα PDF. Αυτό το έγγραφο θα πρέπει να περιέχει πεδία κουμπιών επιλογής για να δείξει πώς να λάβετε τις συντεταγμένες τους.

Μόλις έχετε αυτές τις προϋποθέσεις, μπορούμε να μεταβούμε απευθείας στον κώδικα!

### Εισαγάγετε τη Βιβλιοθήκη

Στην κορυφή του αρχείου κώδικα, θα χρειαστεί να εισαγάγετε τον χώρο ονομάτων Aspose.PDF. Ακολουθεί το απόσπασμα κώδικα για αυτό:

Τώρα, ας προχωρήσουμε στη διαδικασία ανάκτησης των συντεταγμένων των πεδίων των κουμπιών επιλογής σε ένα PDF. 

## Βήμα 1: Καθορίστε τη διαδρομή προς τα έγγραφά σας

Προτού μπορέσουμε να χειριστούμε οποιοδήποτε PDF, πρέπει να καθορίσουμε πού βρίσκεται. Ξεκινήστε δηλώνοντας μια μεταβλητή για τη διαδρομή προς τον κατάλογο εγγράφων σας. Εδώ θα αποθηκεύσετε το αρχείο εισόδου PDF.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Ενημερώστε το με την πραγματική διαδρομή σας
```

## Βήμα 2: Φορτώστε το έγγραφο PDF

Χρησιμοποιώντας τη διαδρομή που ορίστηκε παραπάνω, θα φορτώσετε τώρα το έγγραφο PDF σε μια παρουσία της κλάσης Document. Αυτό σας επιτρέπει να έχετε πρόσβαση στα περιεχόμενά του, συμπεριλαμβανομένων των πεδίων φόρμας.

```csharp
// Φορτώστε το έγγραφο εξόδου
Document doc1 = new Document(dataDir + "input.pdf");
```

## Βήμα 3: Βρείτε Προστιθέμενα Πεδία

 Στη συνέχεια, ας ανακτήσουμε τα πεδία του κουμπιού επιλογής από το PDF. Για το σκοπό αυτό, θα μεταφέρουμε τα πεδία φόρμας από το έγγραφο`RadioButtonField` τύπους.

```csharp
// Βρείτε τα προστιθέμενα πεδία
RadioButtonField field0 = doc1.Form["Item1"] as RadioButtonField;
RadioButtonField field1 = doc1.Form["Item2"] as RadioButtonField;
RadioButtonField field2 = doc1.Form["Item3"] as RadioButtonField;
```

Βεβαιωθείτε ότι τα "Item1", "Item2" και "Item3" ταιριάζουν με τα ονόματα που ορίζονται στο PDF σας.

## Βήμα 4: Επανάληψη βρόχου και εμφάνιση συντεταγμένων

Τώρα έρχεται το συναρπαστικό μέρος — η λήψη των συντεταγμένων των επιλογών των κουμπιών επιλογής. Κάθε κουμπί επιλογής μπορεί να έχει πολλαπλές επιλογές, επομένως θα πραγματοποιήσουμε αναζήτηση σε αυτές τις επιλογές για να εμφανίσουμε τα ορθογώνιά τους.

```csharp
// Και εμφανίστε θέσεις υποστοιχείων για καθένα από αυτά.
foreach (RadioButtonOptionField option in field0)
{
    Console.WriteLine(option.Rect);
}
```

 Επαναλάβετε αυτόν τον βρόχο για`field1` και`field2` για να διασφαλίσετε ότι λαμβάνονται υπόψη όλες οι επιλογές κουμπιών επιλογής:

```csharp
foreach (RadioButtonOptionField option in field1)
{
    Console.WriteLine(option.Rect);
}

foreach (RadioButtonOptionField option in field2)
{
    Console.WriteLine(option.Rect);
}
```

Τώρα, όταν εκτελείτε αυτόν τον κωδικό, θα εξάγει τις συντεταγμένες κάθε επιλογής κουμπιού επιλογής απευθείας στην κονσόλα.
