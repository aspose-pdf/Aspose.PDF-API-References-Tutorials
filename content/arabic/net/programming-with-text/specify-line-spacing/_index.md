## مقدمة

هل واجهتك مشكلة في التحكم في المسافات بين السطور في ملف PDF؟ ربما كان لديك نص يبدو مكتظًا للغاية أو لا يبدو مصقولاً كما تريد. في هذا البرنامج التعليمي، سنوضح لك كيفية تحديد المسافات بين السطور بسهولة في ملف PDF باستخدام Aspose.PDF لـ .NET. سنستخدم دليلًا بسيطًا خطوة بخطوة لننقلك من ملف PDF فارغ إلى ملف يتضمن المسافات بين السطور المخصصة. هذا مثالي إذا كنت بحاجة إلى الدقة في تخطيط النص الخاص بك للمستندات مثل التقارير أو الفواتير أو الشهادات.

## المتطلبات الأساسية

قبل أن ننتقل إلى الكود، دعنا نتأكد من أنك حصلت على كل ما تحتاجه:

1.  تم تثبيت Aspose.PDF لـ .NET. إذا لم يكن لديك، فاحصل عليه من[صفحة تحميل ملف Aspose.PDF](https://releases.aspose.com/pdf/net/).
2. بيئة تطوير .NET (مثل Visual Studio).
3. ملف الخط TrueType (`.ttf` ) الذي سنستخدمه في المثال. يمكنك استخدام أي خط، ولكن في هذا الدليل، سنستخدم`HPSimplified.TTF` الخط.
4. المعرفة الأساسية بلغة C# ومعالجة PDF.

إذا كنت مستعدًا، فلننتقل إلى استيراد الحزم الضرورية.

## استيراد الحزم

في مشروع C# الخاص بك، ستحتاج إلى استيراد مساحات أسماء Aspose.PDF للعمل مع وظائف PDF. إليك كيفية القيام بذلك:

```csharp
using Aspose.Pdf.Text;
using System.IO;
```

تتيح لك هذه المساحات الاسمية إنشاء مستندات PDF ومعالجتها، بالإضافة إلى العمل مع تنسيق النص وخيارات الخط.

سنقسم هذه العملية إلى خطوات صغيرة الحجم، حتى تتمكن من متابعتها بسهولة. ستركز كل خطوة على جزء أساسي من العملية، بدءًا من إعداد ملف PDF الخاص بك وحتى تحديد المسافة بين السطور.

## الخطوة 1: إعداد مشروعك وتحديد دليل المستندات

أول شيء يتعين علينا القيام به هو تحديد مكان وجود ملفاتنا. يساعد هذا البرنامج على معرفة مكان العثور على الخط ومكان حفظ ملف PDF الناتج.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string fontFile = dataDir + "HPSimplified.TTF";
```

 في هذه الخطوة، سوف تقوم باستبدال`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي إلى المكان الذي تخزن فيه ملفاتك. سيكون هذا هو المكان الذي تضع فيه ملف الخط الخاص بك (`HPSimplified.TTF`) ومكان حفظ ملف PDF.

## الخطوة 2: تحميل مستند PDF

الآن، نحتاج إلى إنشاء مستند PDF جديد. في هذا الدليل، سنبدأ بمستند فارغ، ولكن يمكنك أيضًا تحميل ملف PDF موجود إذا لزم الأمر.

```csharp
Document doc = new Document();
```

يؤدي هذا إلى إنشاء مستند PDF جديد فارغ. الأمر سهل، أليس كذلك؟

## الخطوة 3: تعيين خيارات تنسيق النص

 وهنا يحدث السحر. سنحدد وضع تباعد الأسطر للنص الذي نريد إضافته إلى ملف PDF. يمنحنا Aspose.PDF عدة خيارات، ولكن في هذا الدليل، سنستخدم`LineSpacingMode.FullSize`، مما يضمن احترام مسافة السطور بشكل كامل.

```csharp
TextFormattingOptions formattingOptions = new TextFormattingOptions();
formattingOptions.LineSpacing = TextFormattingOptions.LineSpacingMode.FullSize;
```

 يضبط هذا الكود وضع تباعد الأسطر إلى`FullSize` ، مما يضمن عرض النص بمسافات مناسبة. هناك خيارات أخرى مثل`Proportional` إذا كنت تريد سلوكيات تباعد مختلفة، ولكن في الوقت الحالي، دعنا نلتزم بها`FullSize`.

## الخطوة 4: إنشاء جزء نصي

الآن، سنقوم بإنشاء النص الفعلي الذي سيتم وضعه في ملف PDF. سيحترم هذا النص المسافة بين السطور التي حددناها.

```csharp
TextFragment textFragment = new TextFragment("Hello world");
```

 لقد قمنا بإنشاء جزء نصي بالسلسلة`"Hello world"`يمكنك بالطبع تخصيص هذا النص كما يحلو لك.

## الخطوة 5: تحميل الخط المخصص وتطبيقه

لإبراز النص، سنقوم بتحميل خط TrueType مخصص من ملف. هذه الخطوة اختيارية، ولكنها يمكن أن تضيف لمسة احترافية إلى ملفات PDF الخاصة بك.

```csharp
if (fontFile != "")
{
    using (FileStream fontStream = System.IO.File.OpenRead(fontFile))
    {
        textFragment.TextState.Font = FontRepository.OpenFont(fontStream, FontTypes.TTF);
```

هنا نقوم بتحميل ملف الخط وتطبيقه على جزء النص. إذا كان مسار الملف صحيحًا، فسيتم استخدام الخط. وإلا، فسيتم تطبيق الخط الافتراضي.

## الخطوة 6: تعيين موضع النص وتنسيقه

بعد ذلك، نحتاج إلى تحديد موضع النص في ملف PDF. وسنطبق أيضًا خيارات التنسيق التي أنشأناها سابقًا.

```csharp
textFragment.Position = new Position(100, 600);
textFragment.TextState.FormattingOptions = formattingOptions;
```

 ال`Position` تحدد الطريقة إحداثيات المكان الذي سيظهر فيه النص على الصفحة (في هذه الحالة، 100 وحدة من اليسار و600 وحدة من الأسفل). يتم تطبيق خيارات التنسيق، بما في ذلك وضع تباعد الأسطر، هنا.

## الخطوة 7: إضافة نص إلى صفحة PDF

الآن بعد أن تم تنسيق النص وتحديد موضعه، حان الوقت لإضافته إلى مستند PDF.

```csharp
var page = doc.Pages.Add();
page.Paragraphs.Add(textFragment);
```

يقوم هذا الكود بإنشاء صفحة جديدة في مستند PDF ويضيف إليها جزء النص.

## الخطوة 8: احفظ ملف PDF

لقد وصلنا إلى الخطوة الأخيرة! الآن بعد أن تم إعداد كل شيء، فلنبدأ في حفظ ملف PDF.

```csharp
dataDir = dataDir + "SpecifyLineSpacing_out.pdf";
doc.Save(dataDir);
```

سيؤدي هذا إلى حفظ ملف PDF بمسافة السطور المحددة، وسيكون ملفك جاهزًا!
