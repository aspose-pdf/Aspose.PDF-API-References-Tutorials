บทช่วยสอนนี้จะอธิบายวิธีการเพิ่มข้อความและรูปภาพเป็นย่อหน้าแบบอินไลน์ในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET โค้ดต้นฉบับ C# ที่ให้มาจะสาธิตขั้นตอนต่างๆ ทีละขั้นตอน

## ข้อกำหนดเบื้องต้น

ก่อนที่จะดำเนินการตามบทช่วยสอน โปรดแน่ใจว่าคุณมีสิ่งต่อไปนี้:

- ความรู้พื้นฐานเกี่ยวกับภาษาการเขียนโปรแกรม C#
- ติดตั้งไลบรารี Aspose.PDF สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้จากเว็บไซต์ Aspose หรือใช้ NuGet เพื่อติดตั้งในโปรเจ็กต์ของคุณ

## ขั้นตอนที่ 1: ตั้งค่าโครงการ

เริ่มต้นด้วยการสร้างโครงการ C# ใหม่ในสภาพแวดล้อมการพัฒนาแบบบูรณาการ (IDE) ที่คุณต้องการ และเพิ่มการอ้างอิงไปยังไลบรารี Aspose.PDF สำหรับ .NET

## ขั้นตอนที่ 2: นำเข้าเนมสเปซที่จำเป็น

เพิ่มคำสั่ง using ต่อไปนี้ที่จุดเริ่มต้นของไฟล์ C# ของคุณเพื่อนำเข้าเนมสเปซที่จำเป็น:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using Aspose.Pdf.Drawing;
```

## ขั้นตอนที่ 3: ตั้งค่าเส้นทางไปยังไดเรกทอรีเอกสาร

 ตั้งค่าเส้นทางไปยังไดเรกทอรีเอกสารของคุณโดยใช้`dataDir` ตัวแปร:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไดเร็กทอรีเอกสารของคุณ

## ขั้นตอนที่ 4: สร้างเอกสารและหน้าใหม่

 สร้างใหม่`Document` วัตถุและเพิ่มหน้าลงในคอลเล็กชั่นหน้าของมัน:

```csharp
Document doc = new Document();
Page page = doc.Pages.Add();
```

## ขั้นตอนที่ 5: สร้าง TextFragment และเพิ่มเป็นย่อหน้า

 สร้าง`TextFragment` วัตถุและเพิ่มเข้าในคอลเล็กชั่นย่อหน้าของหน้า:

```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```

## ขั้นตอนที่ 6: เพิ่มรูปภาพเป็นย่อหน้าอินไลน์

 สร้าง`Aspose.Pdf.Image` วัตถุและตั้งเป็นย่อหน้าอินไลน์เพื่อให้ปรากฏหลังย่อหน้าก่อนหน้า:

```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
image.File = dataDir + "aspose-logo.jpg";
image.FixHeight = 30; // ตัวเลือก: ตั้งค่าความสูงของภาพ
image.FixWidth = 100; // ตัวเลือก: ตั้งค่าความกว้างของภาพ
page.Paragraphs.Add(image);
```

 แทนที่`"aspose-logo.jpg"` โดยใช้ชื่อไฟล์ภาพจริง และปรับความสูงและความกว้างของรูปภาพตามต้องการ

## ขั้นตอนที่ 7: เพิ่ม TextFragment อีกอันเป็นย่อหน้าอินไลน์

 เริ่มต้นใหม่อีกครั้ง`TextFragment` วัตถุที่มีเนื้อหาแตกต่างกันและเพิ่มเป็นย่อหน้าอินไลน์:

```csharp
text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
page.Paragraphs.Add(text);
```

## ขั้นตอนที่ 8: บันทึกเอกสาร PDF

บันทึกเอกสาร PDF ที่แก้ไขแล้ว:

```csharp
dataDir = dataDir + "TextAndImageAsParagraph_out.pdf";
doc.Save(dataDir);
```

 อย่าลืมเปลี่ยน`"TextAndImageAsParagraph_out.pdf"` พร้อมชื่อไฟล์เอาท์พุตตามที่ต้องการ

### ตัวอย่างโค้ดต้นฉบับสำหรับข้อความและรูปภาพในรูปแบบย่อหน้าโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// สร้างอินสแตนซ์เอกสาร
Document doc = new Document();
// เพิ่มหน้าเข้าในคอลเล็กชั่นหน้าของอินสแตนซ์เอกสาร
Page page = doc.Pages.Add();
// สร้าง TextFragmnet
TextFragment text = new TextFragment("Hello World.. ");
// เพิ่มข้อความส่วนย่อยลงในคอลเล็กชั่นย่อหน้าของวัตถุหน้า
page.Paragraphs.Add(text);
// สร้างอินสแตนซ์ภาพ
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
// ตั้งค่ารูปภาพเป็นย่อหน้าอินไลน์เพื่อให้ปรากฏทันทีหลังจากนั้น
// วัตถุย่อหน้าก่อนหน้า (TextFragment)
image.IsInLineParagraph = true;
// ระบุเส้นทางไฟล์ภาพ
image.File = dataDir + "aspose-logo.jpg";
// ตั้งค่าความสูงของภาพ (ทางเลือก)
image.FixHeight = 30;
// ตั้งค่าความกว้างของภาพ (ตัวเลือก)
image.FixWidth = 100;
// เพิ่มรูปภาพลงในคอลเล็กชั่นย่อหน้าของวัตถุหน้า
page.Paragraphs.Add(image);
// เริ่มต้นวัตถุ TextFragment ใหม่ด้วยเนื้อหาที่แตกต่างกัน
text = new TextFragment(" Hello Again..");
// ตั้งค่า TextFragment ให้เป็นย่อหน้าอินไลน์
text.IsInLineParagraph = true;
// เพิ่ม TextFragment ที่เพิ่งสร้างใหม่ลงในคอลเล็กชันย่อหน้าของหน้า
page.Paragraphs.Add(text);
dataDir = dataDir + "TextAndImageAsParagraph_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nText and image added successfully as an inline paragraphs.\nFile saved at " + dataDir);
```
