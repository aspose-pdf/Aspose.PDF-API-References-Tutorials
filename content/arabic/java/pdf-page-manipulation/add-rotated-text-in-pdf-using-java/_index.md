## مقدمة

في هذا البرنامج التعليمي الشامل، سوف نتعمق في عملية إضافة نص تم تدويره إلى مستند PDF باستخدام Java. سواء كنت بحاجة إلى تسمية الرسوم البيانية، أو إنشاء علامات مائية، أو إضافة تأثيرات خاصة إلى ملفات PDF الخاصة بك، فسوف يرشدك هذا الدليل خلال الخطوات. سنستخدم Aspose.PDF for Java، وهي مكتبة قوية لمعالجة ملفات PDF، لتوضيح العملية.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من توفر المتطلبات الأساسية التالية:

1. بيئة تطوير Java: تأكد من تثبيت Java على نظامك.

2.  Aspose.PDF لـ Java: قم بتنزيل مكتبة Aspose.PDF وتضمينها في مشروع Java الخاص بك. يمكنك العثور على رابط التحميل[هنا](https://releases.aspose.com/pdf/java/).

## الخطوة 1: إنشاء مستند PDF جديد

لنبدأ بإنشاء مستند PDF جديد باستخدام Aspose.PDF. سيكون هذا المستند بمثابة لوحة قماشية للنص الذي تم تدويره.

```java
// تهيئة مستند PDF
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document();
```

## الخطوة 2: إضافة صفحة

بعد ذلك، أضف صفحة إلى مستند PDF حيث تريد إدراج النص الذي تم تدويره:

```java
//إضافة صفحة جديدة إلى المستند
com.aspose.pdf.Page page = pdfDocument.getPages().add();
```

## الخطوة 3: تحديد النص المُدار

الآن، دعونا نحدد النص الذي تريد إدراجه وتدويره. يمكنك تخصيص النص والخط وزاوية التدوير وفقًا لمتطلباتك:

```java
// تحديد محتوى النص
String text = "Rotated Text Example";

// قم بإنشاء كائن TextFragment
com.aspose.pdf.TextFragment textFragment = new com.aspose.pdf.TextFragment(text);

// ضبط حجم الخط ونمطه
textFragment.getTextState().setFontSize(12);
textFragment.getTextState().setFont(com.aspose.pdf.FontRepository.findFont("Arial"));

// تحديد زاوية الدوران (بالدرجات)
textFragment.setTextRotation(45);
```

في هذا المثال، قمنا بتعيين النص على "مثال نص مستدير"، واخترنا الخط Arial، وضبطنا حجم الخط على 12، وقمنا بتدوير النص بمقدار 45 درجة. اضبط هذه المعلمات لتتوافق مع متطلباتك المحددة.

## الخطوة 4: ضع النص الذي تمت استدارته

حدد الموضع على الصفحة حيث تريد وضع النص الذي تم تدويره:

```java
// ضبط موضع النص
textFragment.setPosition(new com.aspose.pdf.Position(100, 200));
```

هنا، قمنا بوضع النص عند الإحداثيات (100، 200) على الصفحة. قم بتعديل هذه الإحداثيات لوضع النص بدقة في المكان الذي تريده.

## الخطوة 5: إضافة نص تمت استدارته إلى الصفحة

الآن، قم بإضافة النص الذي تم تدويره إلى الصفحة:

```java
// أضف النص الذي تم تدويره إلى الصفحة
page.getParagraphs().add(textFragment);
```

## الخطوة 6: احفظ ملف PDF

أخيرًا، احفظ مستند PDF بالنص الذي تم تدويره:

```java
// احفظ مستند PDF
pdfDocument.save("output.pdf");
```

## خاتمة

في هذا البرنامج التعليمي، اكتشفنا عملية إضافة نص تمت استدارته إلى مستند PDF باستخدام Java وAspose.PDF لـ Java. لقد تعلمت كيفية إنشاء ملف PDF جديد، وتحديد النص الذي تم تدويره باستخدام أنماط مخصصة، ووضعه على الصفحة، وحفظ ملف PDF المعدل.

يمكن أن يكون النص الذي تم تدويره إضافة قيمة إلى ملفات PDF الخاصة بك لأغراض متعددة، مثل تسمية المخططات أو العلامة المائية أو إضافة عناصر إبداعية إلى مستنداتك.

قم بتحسين مستندات PDF الخاصة بك عن طريق دمج النص الذي تم تدويره بسهولة، وذلك بفضل إمكانيات Aspose.PDF لـ Java.

## الأسئلة الشائعة (الأسئلة المتداولة)

### 1. هل يمكنني تدوير النص بزوايا مختلفة في نفس ملف PDF؟
   نعم، يمكنك إضافة مثيلات متعددة للنص الذي تم تدويره بزوايا مختلفة لنفس مستند PDF. ما عليك سوى تكرار العملية الموضحة في هذا البرنامج التعليمي لكل جزء من النص الذي تم تدويره.

### 2. كيف يمكنني تغيير لون النص الذي تم تدويره؟
    لتغيير لون النص، استخدم`textFragment.getTextState().setForegroundColor` الطريقة وحدد اللون بتنسيق RGB. على سبيل المثال، لتعيين لون النص إلى اللون الأحمر، استخدم`textFragment.getTextState().setForegroundColor(com.aspose.pdf.Color.getRed());`.

### 3. هل يعتبر Aspose.PDF for Java مكتبة مجانية؟
   Aspose.PDF for Java هي مكتبة تجارية قوية، ولكنها تقدم نسخة تجريبية مجانية للاختبار والتقييم. اعتمادا على متطلبات مشروعك، يمكنك اختيار خيار الترخيص المناسب.

### 4. هل يمكنني تدوير النص بمقدار 90 درجة لإنشاء نص رأسي؟
   نعم، يمكنك تدوير النص بمقدار 90 درجة لإنشاء نص رأسي. ما عليك سوى ضبط زاوية التدوير على 90 درجة، وسيظهر النص عموديًا على الصفحة.

### 5. هل توجد مكتبات أخرى للعمل مع ملفات PDF في Java؟
   نعم، تتوفر العديد من المكتبات، مثل iText وPDFBox، لمعالجة ملفات PDF في Java. تتمتع كل مكتبة بميزات وإمكانيات فريدة، لذا اختر المكتبة التي تناسب احتياجات مشروعك.
