## Импортные пакеты

Теперь, когда у нас все настроено, давайте импортируем необходимые пакеты для использования Aspose.PDF в нашем проекте. Вы можете сделать это, добавив следующие директивы using в начало вашего файла C#:

```csharp
using Aspose.Pdf.Text;
using System;

```

Эти пространства имен позволят вам эффективно работать с PDF-документами и текстовыми фрагментами. 

Теперь давайте разберем процесс добавления текстовой границы на подробные шаги. Мы пройдем каждый шаг, чтобы вы могли точно понять, что происходит под капотом.

## Шаг 1: Настройте документ

Первым делом нам нужно создать новый PDF-документ. Здесь и будет происходить вся наша магия.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Создать новый объект документа
Document pdfDocument = new Document();
```

 На этом шаге мы указываем каталог, в котором хотим сохранить наш PDF-файл. Затем мы создаем новый экземпляр`Document` класс, представляющий наш PDF-документ.

## Шаг 2: Добавьте новую страницу

Далее нам нужно добавить страницу в наш документ. Думайте об этом как о добавлении чистого холста, на котором мы разместим наш текст.

```csharp
// Получить определенную страницу
Page pdfPage = (Page)pdfDocument.Pages.Add();
```

 Здесь мы называем`Add()` метод на`Pages` коллекция наших`pdfDocument` объект. Это добавляет новую страницу в документ, и мы сохраняем ссылку на нее в`pdfPage` переменная.

## Шаг 3: Создайте текстовый фрагмент

Теперь давайте создадим текст, который мы хотим отобразить в нашем PDF. Здесь мы определяем содержание нашего текстового фрагмента.

```csharp
// Создать фрагмент текста
TextFragment textFragment = new TextFragment("main text");
textFragment.Position = new Position(100, 600);
```

 В этом коде мы создаем новый`TextFragment` объект с текстом "главный текст". Также мы задаем его положение на странице с помощью`Position` класс. Координаты (100, 600) указывают, где на странице будет размещен текст.

## Шаг 4: Задайте свойства текста

Далее мы настроим наш текстовый фрагмент, чтобы сделать его визуально привлекательным. Это включает в себя установку размера шрифта, типа шрифта, цвета фона и цвета переднего плана.

```csharp
// Установить свойства текста
textFragment.TextState.FontSize = 12;
textFragment.TextState.Font = FontRepository.FindFont("TimesNewRoman");
textFragment.TextState.BackgroundColor = Aspose.Pdf.Color.LightGray;
textFragment.TextState.ForegroundColor = Aspose.Pdf.Color.Red;
```

Здесь мы устанавливаем размер шрифта 12, используем шрифт "Times New Roman" и применяем светло-серый цвет фона с красным текстом. Эти свойства помогают улучшить видимость текста.

## Шаг 5: Установите цвет обводки для границы

Теперь мы переходим к самой захватывающей части — добавлению рамки вокруг нашего текста!

```csharp
// Установите свойство StrokingColor для рисования границы (обводки) вокруг текстового прямоугольника
textFragment.TextState.StrokingColor = Aspose.Pdf.Color.DarkRed;
```

На этом шаге мы указываем цвет границы, которую хотим нарисовать вокруг нашего текста. Здесь мы выбрали темно-красный цвет.

## Шаг 6: Включите границу текстового прямоугольника

 Чтобы нарисовать границу вокруг нашего текста, нам нужно включить`DrawTextRectangleBorder` свойство.

```csharp
// Установите для свойства DrawTextRectangleBorder значение true.
textFragment.TextState.DrawTextRectangleBorder = true;
```

 Установив это свойство на`true`, мы говорим Aspose.PDF нарисовать границу вокруг текстового прямоугольника на основе указанного цвета обводки.

## Шаг 7: Добавьте фрагмент текста на страницу.

Теперь, когда наш текстовый фрагмент готов и все свойства заданы, пришло время добавить его на страницу.

```csharp
TextBuilder tb = new TextBuilder(pdfPage);
tb.AppendText(textFragment);
```

 Здесь мы создаем`TextBuilder` объект, который связан с нашим`pdfPage` . Затем мы используем`AppendText` метод добавить наш`textFragment` на страницу. 

## Шаг 8: Сохраните документ.

Наконец, нам нужно сохранить наш PDF-документ в указанную директорию. Вот он, момент истины!

```csharp
// Сохранить документ
pdfDocument.Save(dataDir + "PDFWithTextBorder_out.pdf");
```

На этом этапе мы вызываем`Save` метод на нашем`pdfDocument` объект, предоставляющий путь, по которому мы хотим сохранить файл. После запуска кода вы должны найти ваш недавно созданный PDF с текстовой границей в указанном каталоге!
