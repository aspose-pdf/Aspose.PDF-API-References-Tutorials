## परिचय

इंटरैक्टिव PDF बनाने से उपयोगकर्ता अनुभव में उल्लेखनीय वृद्धि हो सकती है, खासकर जब बात फॉर्म की हो। सबसे आम इंटरैक्टिव तत्वों में से एक रेडियो बटन है, जो उपयोगकर्ताओं को एक सेट से एक विकल्प चुनने की अनुमति देता है। इस ट्यूटोरियल में, हम .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में रेडियो बटन बनाने का तरीका जानेंगे। चाहे आप एक अनुभवी डेवलपर हों या अभी शुरुआत कर रहे हों, यह गाइड आपको प्रक्रिया के माध्यम से चरण-दर-चरण मार्गदर्शन करेगा, यह सुनिश्चित करते हुए कि आप कोड के प्रत्येक भाग और उसके उद्देश्य को समझते हैं।

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

सबसे पहले, आपको वह डायरेक्टरी निर्दिष्ट करनी होगी जहाँ आपका PDF सहेजा जाएगा। यह आपकी फ़ाइलों को व्यवस्थित करने के लिए महत्वपूर्ण है।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` उस वास्तविक पथ के साथ जहां आप अपनी पीडीएफ फाइल को सहेजना चाहते हैं।

## चरण 2: दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें

 इसके बाद, आपको इसका एक उदाहरण बनाना होगा`Document` क्लास. यह क्लास आपके पीडीएफ दस्तावेज़ का प्रतिनिधित्व करता है.

```csharp
Document pdfDocument = new Document();
```

यह पंक्ति एक नया PDF दस्तावेज़ आरंभ करती है जिसके साथ आप काम करेंगे।

## चरण 3: पीडीएफ में एक पेज जोड़ें

प्रत्येक PDF दस्तावेज़ में कई पृष्ठ होते हैं। आपको अपने दस्तावेज़ में कम से कम एक पृष्ठ जोड़ना होगा।

```csharp
pdfDocument.Pages.Add();
```

यह पंक्ति आपके PDF दस्तावेज़ में एक नया पृष्ठ जोड़ती है, जिससे यह सामग्री के लिए तैयार हो जाता है।

## चरण 4: रेडियो बटन फ़ील्ड बनाएँ

 अब, रेडियो बटन फ़ील्ड बनाने का समय आ गया है। आप एक इंस्टैंशिएट करेंगे`RadioButtonField` ऑब्जेक्ट चुनें और वह पृष्ठ संख्या निर्दिष्ट करें जहां उसे रखा जाएगा।

```csharp
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

यहां, हम पीडीएफ के पहले पृष्ठ पर रेडियो बटन जोड़ रहे हैं।

## चरण 5: रेडियो बटन में विकल्प जोड़ें

आप अपने रेडियो बटन में कई विकल्प जोड़ सकते हैं। प्रत्येक विकल्प एक चयन योग्य आइटम होगा।

```csharp
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

 इस उदाहरण में, हम दो विकल्प जोड़ रहे हैं: "टेस्ट" और "टेस्ट1"।`Rectangle` ऑब्जेक्ट प्रत्येक विकल्प की स्थिति और आकार निर्दिष्ट करता है।

## चरण 6: दस्तावेज़ फ़ॉर्म में रेडियो बटन जोड़ें

एक बार जब आप अपना रेडियो बटन और उसके विकल्प परिभाषित कर लेते हैं, तो आपको उसे दस्तावेज़ के प्रारूप में जोड़ना होगा।

```csharp
pdfDocument.Form.Add(radio);
```

यह पंक्ति रेडियो बटन को पीडीएफ फॉर्म में एकीकृत कर देती है, जिससे यह इंटरैक्टिव हो जाता है।

## चरण 7: पीडीएफ दस्तावेज़ सहेजें

अंत में, आपको अपने पीडीएफ दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजना होगा।

```csharp
dataDir = dataDir + "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

यह कोड आपके निर्दिष्ट निर्देशिका में "RadioButton_out.pdf" नाम से दस्तावेज़ को सहेजता है।

## चरण 8: अपवादों को संभालें

अपने कोड के निष्पादन के दौरान आने वाले अपवादों को संभालना हमेशा एक अच्छा अभ्यास है।

```csharp
catch (Exception ex)
{
    Console.WriteLine(ex.Message);
}
```

यह किसी भी त्रुटि को पकड़ लेगा और संदेश प्रदर्शित करेगा, जिससे कुछ गलत होने पर आपको डीबग करने में मदद मिलेगी।
