Aspose.PDF สำหรับ .NET เป็นไลบรารีการจัดการ PDF ที่มีคุณลักษณะมากมายสำหรับการดำเนินการต่างๆ กับเอกสาร PDF คุณลักษณะหนึ่งคือความสามารถในการรับค่าการซูมในไฟล์ PDF ในบทช่วยสอนนี้ เราจะอธิบายวิธีการใช้ Aspose.PDF สำหรับ .NET เพื่อรับค่าการซูมในไฟล์ PDF โดยใช้โค้ดต้นฉบับของ C#


## ขั้นตอนที่ 1: สร้างอินสแตนซ์ของวัตถุเอกสารใหม่

 ขั้นตอนแรกในการรับปัจจัยการซูมของไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET คือการสร้างอินสแตนซ์ใหม่`Document` วัตถุ.`Document` วัตถุแสดงถึงเอกสาร PDF ที่สามารถโหลดจากไฟล์หรือสตรีมได้

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างอินสแตนซ์ของวัตถุเอกสารใหม่
Document doc = new Document(dataDir + "Zoomed_pdf.pdf");
```

 ในโค้ดด้านบนเราสร้าง`Document` วัตถุโดยส่งเส้นทางของไฟล์ PDF ไปยังตัวสร้างของ`Document` คลาส คุณต้องแทนที่ "ไดเรกทอรีเอกสารของคุณ" ด้วยเส้นทางจริงของไดเรกทอรีที่ไฟล์ PDF ของคุณตั้งอยู่

## ขั้นตอนที่ 2: สร้างวัตถุ GoToAction

 ขั้นตอนต่อไปคือการสร้าง`GoToAction` วัตถุ. ก.`GoToAction`วัตถุแสดงถึงการดำเนินการที่ไปยังจุดหมายเฉพาะในเอกสาร PDF ในกรณีของเรา เราต้องการทราบค่าการซูมของไฟล์ PDF ดังนั้นเราจะใช้`OpenAction` ทรัพย์สินของ`Document` วัตถุที่จะได้รับ`GoToAction` วัตถุ.

```csharp
// สร้างวัตถุ GoToAction
GoToAction action = doc.OpenAction as GoToAction;
```

 ในโค้ดด้านบนเราสร้าง`GoToAction` วัตถุโดยการหล่อ`OpenAction` ทรัพย์สินของ`Document` คัดค้าน`GoToAction`.

## ขั้นตอนที่ 3: รับปัจจัยการซูมของไฟล์ PDF

 ขั้นตอนที่สามคือการรับค่าการซูมของไฟล์ PDF เราสามารถรับค่าการซูมของไฟล์ PDF ได้โดยเข้าถึง`Destination` ทรัพย์สินของ`GoToAction` วัตถุแล้วโยนมันไป`XYZExplicitDestination` . การ`XYZExplicitDestination` คลาสแสดงถึงจุดหมายปลายทางในเอกสาร PDF ที่ระบุพิกัดและปัจจัยการซูมที่จะไป

```csharp
// รับปัจจัยการซูมของไฟล์ PDF
System.Console.WriteLine((action.Destination as XYZExplicitDestination).Zoom); // ค่าการซูมเอกสาร;
```

 ในโค้ดด้านบนเราเข้าถึง`Destination` ทรัพย์สินของ`GoToAction` วัตถุแล้วโยนมันไป`XYZExplicitDestination` . หลังจากนั้นเราก็ได้เข้าถึง`Zoom` ทรัพย์สินของ`XYZExplicitDestination` วัตถุเพื่อรับปัจจัยการซูมของไฟล์ PDF

## ขั้นตอนที่ 4: ส่งออกปัจจัยการซูม

 ขั้นตอนสุดท้ายคือการแสดงผลค่าการซูมของไฟล์ PDF เราสามารถใช้`System.Console.WriteLine`

```csharp
// รับปัจจัยการซูมของไฟล์ PDF
System.Console.WriteLine((action.Destination as XYZExplicitDestination).Zoom); // ค่าการซูมเอกสาร;
```

นี่คือตัวอย่างซอร์สโค้ดที่สมบูรณ์สำหรับการรับปัจจัยการซูมโดยใช้ Aspose.PDF สำหรับ .NET:
