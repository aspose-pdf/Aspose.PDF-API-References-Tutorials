## مقدمة

في العصر الرقمي، يعد تحويل مستندات HTML إلى PDF مهمة شائعة. ومع ذلك، قد ترغب أحيانًا في إزالة الارتباطات التشعبية من ملف PDF المحول لأسباب مختلفة، مثل تحسين قابلية القراءة أو منع التنقل غير المرغوب فيه. في هذا البرنامج التعليمي، سنستكشف كيفية تحقيق ذلك باستخدام Aspose.PDF لـ .NET. 

## المتطلبات الأساسية

قبل الغوص في الكود، تأكد من أن لديك المتطلبات الأساسية التالية:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. ستكون هذه بيئة التطوير الخاصة بك.
2.  Aspose.PDF لـ .NET: يجب أن يكون لديك مكتبة Aspose.PDF. يمكنك تنزيلها من[هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم الكود بشكل أفضل.

## استيراد الحزم

للبدء، تحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

1. افتح مشروع Visual Studio الخاص بك.
2. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول وحدد "إدارة حزم NuGet".
3.  بحث عن`Aspose.PDF` وتثبيته.

```csharp
using Aspose.Pdf.Annotations;
using Aspose.Pdf.Text;
using System.IO;
```

الآن بعد أن قمت بإعداد كل شيء، دعنا نقوم بتقسيم عملية إزالة الارتباطات التشعبية من ملف HTML بعد تحويله إلى PDF.

## الخطوة 1: إعداد دليل المستندات

أولاً وقبل كل شيء، عليك تحديد المسار إلى دليل المستندات. هذا هو المكان الذي يوجد فيه ملف HTML الخاص بك وحيث سيتم حفظ ملف PDF الناتج.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملف HTML الخاص بك فيه.

## الخطوة 2: تحميل مستند HTML

 بعد ذلك، ستقوم بتحميل مستند HTML باستخدام`Document` فئة من Aspose.PDF. تتيح لك هذه الفئة العمل مع مستندات PDF بسهولة.

```csharp
Document doc = new Document(dataDir + "SampleHtmlFile.html", new HtmlLoadOptions());
```

 هنا نقوم بتحميل ملف HTML المسمى`SampleHtmlFile.html`تأكد من وجود هذا الملف في الدليل المحدد.

## الخطوة 3: حفظ المستند في تدفق الذاكرة

قبل أن نبدأ في معالجة التعليقات التوضيحية، نحتاج إلى حفظ المستند في مجرى ذاكرة. هذه الخطوة بالغة الأهمية لأنها تحضر المستند لمزيد من المعالجة.

```csharp
doc.Save(new MemoryStream());
```

يحفظ هذا السطر المستند في الذاكرة، مما يسمح لنا بالعمل عليه دون الحاجة إلى كتابته على القرص حتى الآن.

## الخطوة 4: التكرار من خلال التعليقات التوضيحية

الآن، سنستعرض التعليقات التوضيحية في المستند. التعليقات التوضيحية هي عناصر مثل الروابط والتعليقات والتمييزات. نحن مهتمون بشكل خاص بتعليقات الروابط التوضيحية.

```csharp
foreach (Annotation a in doc.Pages[1].Annotations)
{
    if (a.AnnotationType == AnnotationType.Link)
    {
        // معالجة تعليق الرابط
    }
}
```

في هذه الحلقة، نتحقق مما إذا كان نوع التعليق عبارة عن رابط. إذا كان كذلك، ننتقل إلى الخطوات التالية.

## الخطوة 5: إزالة إجراء الارتباط التشعبي

بالنسبة لكل تعليق على الرابط، نحتاج إلى التحقق مما إذا كان يحتوي على إجراء ارتباط تشعبي. إذا كان الأمر كذلك، فسنقوم بإزالة الرابط التشعبي عن طريق تعيين عنوان URI الخاص به إلى سلسلة فارغة.

```csharp
LinkAnnotation la = (LinkAnnotation)a;
if (la.Action is GoToURIAction)
{
    GoToURIAction gta = (GoToURIAction)la.Action;
    gta.URI = "";
```

تضمن مقتطفات التعليمات البرمجية هذه إزالة إجراء الارتباط التشعبي بشكل فعال.

## الخطوة 6: استيعاب أجزاء النص

بعد ذلك، سنقوم باستيعاب أجزاء النص المرتبطة بتعليقات الارتباط. وهذا يسمح لنا بالتلاعب بمظهر النص.

```csharp
TextFragmentAbsorber tfa = new TextFragmentAbsorber();
tfa.TextSearchOptions = new TextSearchOptions(a.Rect);
doc.Pages[a.PageIndex].Accept(tfa);
```

 هنا، نقوم بإنشاء`TextFragmentAbsorber` وضبط خيارات البحث على المستطيل الخاص بالتعليق التوضيحي. وهذا يساعدنا في العثور على النص المرتبط.

## الخطوة 7: تعديل مظهر النص

بمجرد حصولنا على أجزاء النص، يمكننا تعديل مظهرها. في هذه الحالة، سنزيل الخط السفلي ونغير لون النص إلى الأسود.

```csharp
foreach (TextFragment tf in tfa.TextFragments)
{
    tf.TextState.Underline = false;
    tf.TextState.ForegroundColor = Color.Black;
}
```

تعمل هذه الخطوة على تحسين قابلية قراءة النص عن طريق إزالة نمط الارتباط التشعبي.

## الخطوة 8: حذف التعليقات التوضيحية

بعد تعديل النص، يمكننا حذف تعليق الرابط بأمان من المستند.

```csharp
doc.Pages[a.PageIndex].Annotations.Delete(a);
}
```

يؤدي هذا السطر إلى إزالة الارتباط التشعبي من ملف PDF، مما يضمن عدم وجوده في الناتج النهائي.

## الخطوة 9: حفظ المستند المعدل

أخيرًا، نحتاج إلى حفظ المستند المعدّل في ملف PDF جديد. هذه هي الخطوة الأخيرة في عمليتنا.

```csharp
doc.Save(dataDir + "RemoveHyperlinksFromText_out.pdf");
```

 يحفظ هذا السطر المستند مع إزالة الارتباطات التشعبية، مما يؤدي إلى إنشاء ملف PDF جديد باسم`RemoveHyperlinksFromText_out.pdf`.
