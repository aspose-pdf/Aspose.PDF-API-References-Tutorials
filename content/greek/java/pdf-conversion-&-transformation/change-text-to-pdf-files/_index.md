## Δημιουργία απλής εφαρμογής Java

Ας δημιουργήσουμε μια απλή εφαρμογή Java που μετατρέπει ένα αρχείο κειμένου σε έγγραφο PDF.

```java
import com.aspose.pdf.Document;
import com.aspose.pdf.TextFragment;

public class TextToPDFConverter {
    public static void main(String[] args) {
        // Φορτώστε το αρχείο κειμένου
        Document pdfDocument = new Document();
        Page page = pdfDocument.getPages().add();
        
        // Προσθήκη κειμένου στο PDF
        TextFragment textFragment = new TextFragment("Hello, Aspose.PDF for Java!");
        page.getParagraphs().add(textFragment);
        
        // Αποθηκεύστε το αρχείο PDF
        pdfDocument.save("output.pdf");
        
        System.out.println("Text file converted to PDF successfully!");
    }
}
```

Σε αυτόν τον κώδικα, δημιουργούμε ένα απλό έγγραφο PDF με ένα τμήμα κειμένου και το αποθηκεύουμε ως "output.pdf".
