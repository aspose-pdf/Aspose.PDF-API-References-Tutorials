ในบทช่วยสอนนี้ เราจะอธิบายวิธีตั้งค่าคีย์ใบอนุญาตแบบคิดค่าบริการตามปริมาณข้อมูลในไฟล์ PDF ด้วย Aspose.PDF สำหรับ .NET ทีละขั้นตอน ใบอนุญาตแบบคิดค่าบริการตามปริมาณข้อมูลช่วยให้คุณสามารถใช้คุณสมบัติขั้นสูงของ Aspose.PDF ตามการใช้งานจริงของคุณ

### ขั้นตอนที่ 1: การกำหนดค่าคีย์ใบอนุญาต

ในซอร์สโค้ดที่ให้มา คุณต้องระบุคีย์สาธารณะและคีย์ส่วนตัวของใบอนุญาตแบบคิดค่าบริการตามปริมาณข้อมูล แทนที่ "*****" ค่าด้วยคีย์ของคุณเอง คีย์เหล่านี้จะมอบให้กับคุณเมื่อคุณซื้อใบอนุญาตแบบมิเตอร์จาก Aspose

```csharp
Aspose.Pdf.Metered metered = new Aspose.Pdf.Metered();
metered.SetMeteredKey("PUBLIC_KEY", "PRIVATE_KEY");
```

### ขั้นตอนที่ 2: กำลังโหลดเอกสาร

 โหลดเอกสาร PDF จากดิสก์โดยใช้ไฟล์`Document` คลาสของ Aspose.PDF

```csharp
Document doc = new Document(dataDir + "input.pdf");
```

### ขั้นตอนที่ 3: รับจำนวนหน้าเอกสาร

 ใช้`Count` ทรัพย์สินของ`Pages` คอลเลกชันเพื่อให้ได้จำนวนหน้าทั้งหมดในเอกสาร

```csharp
Console.WriteLine(doc.Pages.Count);
```

### ตัวอย่างซอร์สโค้ดสำหรับกำหนดค่าสิทธิ์การใช้งานแบบมิเตอร์โดยใช้ Aspose.PDF สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// ตั้งค่าคีย์สาธารณะและคีย์ส่วนตัวแบบมิเตอร์
Aspose.Pdf.Metered metered = new Aspose.Pdf.Metered();
//เข้าถึงคุณสมบัติ setMeteredKey และส่งคีย์สาธารณะและคีย์ส่วนตัวเป็นพารามิเตอร์
metered.SetMeteredKey("*****", "*****");
// โหลดเอกสารจากดิสก์
Document doc = new Document(dataDir + "input.pdf");
//รับจำนวนหน้าของเอกสาร
Console.WriteLine(doc.Pages.Count);

```
