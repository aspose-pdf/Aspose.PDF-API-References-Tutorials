Aspose.PDF for .NET, C# प्रोग्रामिंग भाषा का उपयोग करके PDF फ़ाइल में जानकारी को हेरफेर करने और निकालने के लिए एक शक्तिशाली लाइब्रेरी है। इस ट्यूटोरियल में, हम Aspose.PDF for .NET का उपयोग करके PDF फ़ाइल से हाइपरलिंक गंतव्यों को निकालने पर ध्यान केंद्रित करेंगे।

## आवश्यक शर्तें

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- एक एकीकृत विकास वातावरण (IDE) जैसे कि विजुअल स्टूडियो.
- .NET के लिए Aspose.PDF लाइब्रेरी आपकी मशीन पर स्थापित है।

## चरण 1: विकास परिवेश की स्थापना

कोड लिखना शुरू करने से पहले, आपको अपने पसंदीदा IDE में एक नया C# प्रोजेक्ट बनाकर अपना विकास वातावरण सेट करना होगा।

## चरण 2: Aspose.PDF संदर्भ आयात करें

.NET के लिए Aspose.PDF का उपयोग करने के लिए, आपको अपने प्रोजेक्ट में उचित संदर्भ जोड़ने होंगे। आवश्यक संदर्भ आयात करने के लिए नीचे दिए गए चरणों का पालन करें:

1. अपने प्रोजेक्ट में, "संदर्भ" पर राइट-क्लिक करें और "संदर्भ जोड़ें" चुनें।
2. "संदर्भ जोड़ें" विंडो में, .NET के लिए Aspose.PDF की DLL फ़ाइलों का पता लगाएं और उनका चयन करें।
3. अपने प्रोजेक्ट में संदर्भ आयात करने के लिए "ओके" पर क्लिक करें।

## चरण 3: पीडीएफ फाइल लोड करना

हाइपरलिंक गंतव्यों को निकालने से पहले, आपको PDF फ़ाइल को अपने एप्लिकेशन में लोड करना होगा। PDF फ़ाइल लोड करने के लिए निम्न कोड का उपयोग करें:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// पीडीएफ फाइल लोड करें
Document document = new Document(dataDir + "input.pdf");
```

अपने दस्तावेज़ निर्देशिका और उस PDF फ़ाइल का सही पथ निर्दिष्ट करना सुनिश्चित करें जिसे आप संसाधित करना चाहते हैं।

## चरण 4: दस्तावेज़ के पृष्ठों पर नेविगेट करना

अब जब पीडीएफ फाइल लोड हो गई है, तो आपको दस्तावेज़ के सभी पृष्ठों को देखना होगा। इससे आप दस्तावेज़ को आसानी से पढ़ सकेंगे।

प्रत्येक पृष्ठ पर मौजूद हाइपरलिंक एनोटेशन को देखें। दस्तावेज़ के पृष्ठों के माध्यम से पुनरावृति करने के लिए निम्न कोड का उपयोग करें:

```csharp
foreach(Aspose.Pdf.Page page in document.Pages)
{
     // किसी विशिष्ट पृष्ठ के लिंक एनोटेशन प्राप्त करें
     AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
     page. Accept(selector);
     // सभी लिंक संग्रहीत करने के लिए एक सूची बनाएं
     IList<Annotation> list = selector. Selected;
     // सूची में प्रत्येक आइटम के माध्यम से लूप करें
     foreach(LinkAnnotation a in list)
     {
         // गंतव्य URL प्रिंट करें
         Console.WriteLine("\nDestination: " + (a.Action as Aspose.Pdf.Annotations.GoToURIAction).URI + "\n");
     }
}
```

यह कोड दस्तावेज़ के प्रत्येक पृष्ठ पर लूप करता है और प्रत्येक पृष्ठ पर मौजूद हाइपरलिंक एनोटेशन का चयन करता है। फिर यह इन एनोटेशन को एक सूची में संग्रहीत करता है और प्रत्येक लिंक के लिए गंतव्य URL प्रिंट करता है।

## चरण 5: हाइपरलिंक गंतव्य प्राप्त करना

अंतिम चरण हाइपरलिंक एनोटेशन से हाइपरलिंक गंतव्यों को निकालना है। निम्न कोड आपको यह करने का तरीका बताता है:

```csharp
foreach(Aspose.Pdf.Page page in document.Pages)
{
     AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
     page. Accept(selector);
     IList<Annotation> list = selector. Selected;
     foreach(LinkAnnotation a in list)
     {
         string destination = (a.Action as Aspose.Pdf.Annotations.GoToURIAction).URI;
         // गंतव्य का उपयोग अपनी इच्छानुसार करें
     }
}
```

इस कोड में, हम लिंक एनोटेशन से प्रत्येक हाइपरलिंक गंतव्य प्राप्त करते हैं और गंतव्य को एक चर में संग्रहीत करते हैं। फिर आप अपने एप्लिकेशन में अपनी इच्छानुसार इस गंतव्य का उपयोग कर सकते हैं।

### .NET के लिए Aspose.PDF का उपयोग करके हाइपरलिंक गंतव्य प्राप्त करने के लिए नमूना स्रोत कोड 
```csharp
try
{
	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	// पीडीएफ फाइल लोड करें
	Document document = new Document(dataDir + "input.pdf");
	// पीडीएफ के सभी पृष्ठों पर जाएँ
	foreach (Aspose.Pdf.Page page in document.Pages)
	{
		// किसी विशेष पृष्ठ से लिंक एनोटेशन प्राप्त करें
		AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
		page.Accept(selector);
		// सभी लिंक वाली सूची बनाएं
		IList<Annotation> list = selector.Selected;
		// सूची के अंदर व्यक्तिगत आइटम के माध्यम से पुनरावृति करें
		foreach (LinkAnnotation a in list)
		{
			// गंतव्य URL प्रिंट करें
			Console.WriteLine("\nDestination: " + (a.Action as Aspose.Pdf.Annotations.GoToURIAction).URI + "\n");
		}
	}
}
catch (Exception ex)
{
	Console.WriteLine(ex.Message);
}
```

### पीडीएफ फाइल में हाइपरलिंक गंतव्य प्राप्त करने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ फाइल में हाइपरलिंक गंतव्य क्या है?

उत्तर: PDF फ़ाइल में हाइपरलिंक गंतव्य एक विशिष्ट स्थान या लक्ष्य होता है, जिस पर हाइपरलिंक इंगित करता है। यह एक URL, उसी दस्तावेज़ के भीतर एक पृष्ठ या एक बाहरी दस्तावेज़ हो सकता है।

#### प्रश्न: हाइपरलिंक गंतव्यों को निकालने से मेरे पीडीएफ दस्तावेज़ विश्लेषण को कैसे लाभ हो सकता है?

उत्तर: हाइपरलिंक गंतव्यों को निकालने से आप उन सभी लक्ष्यों की पहचान और सूची बना सकते हैं, जिनकी ओर हाइपरलिंक PDF दस्तावेज़ में इंगित करते हैं। यह जानकारी सामग्री सत्यापन, लिंक सत्यापन और डेटा विश्लेषण के लिए उपयोगी हो सकती है।

#### प्रश्न: .NET के लिए Aspose.PDF हाइपरलिंक गंतव्यों को निकालने में किस प्रकार सहायता करता है?

उत्तर: .NET के लिए Aspose.PDF हाइपरलिंक गंतव्यों को आसानी से निकालने के लिए शक्तिशाली API प्रदान करता है। यह ट्यूटोरियल चरण-दर-चरण प्रदर्शित करता है कि C# का उपयोग करके हाइपरलिंक गंतव्यों को कैसे निकाला जाए।

#### प्रश्न: क्या मैं कुछ मानदंडों के आधार पर चुनिंदा हाइपरलिंक गंतव्यों को निकाल सकता हूं?

उत्तर: हां, आप पीडीएफ दस्तावेज़ के पृष्ठों के माध्यम से पुनरावृत्ति करके और अपने मानदंडों के आधार पर वांछित हाइपरलिंक एनोटेशन को फ़िल्टर करके चुनिंदा हाइपरलिंक गंतव्यों को निकाल सकते हैं।

#### प्रश्न: क्या पासवर्ड-संरक्षित पीडीएफ दस्तावेजों से हाइपरलिंक गंतव्यों को निकालना संभव है?

उत्तर: Aspose.PDF for .NET पासवर्ड-संरक्षित PDF दस्तावेज़ों से हाइपरलिंक गंतव्यों को निकाल सकता है, बशर्ते आप दस्तावेज़ खोलते समय आवश्यक प्रमाणीकरण क्रेडेंशियल प्रदान करें।

#### प्रश्न: मैं अपने एप्लिकेशन में निकाले गए हाइपरलिंक गंतव्यों का उपयोग कैसे कर सकता हूं?

उत्तर: एक बार जब आप हाइपरलिंक गंतव्यों को निकाल लेते हैं, तो आप उनका उपयोग विभिन्न क्रियाएं करने के लिए कर सकते हैं, जैसे लिंक URL को मान्य करना, रिपोर्ट बनाना, या कस्टम नेविगेशन को लागू करना।

#### प्रश्न: हाइपरलिंक गंतव्यों को निकालते समय क्या कोई सीमाएं हैं?

उत्तर: हाइपरलिंक गंतव्य निष्कर्षण शक्तिशाली है, लेकिन पीडीएफ दस्तावेज़ की संरचना पर विचार करना आवश्यक है। जटिल ग्राफ़िक्स या मल्टीमीडिया सामग्री में एम्बेड किए गए हाइपरलिंक को अतिरिक्त हैंडलिंग की आवश्यकता हो सकती है।

#### प्रश्न: क्या मैं हाइपरलिंक की अन्य विशेषताएं, जैसे लिंक प्रकार या निर्देशांक निकाल सकता हूं?

उत्तर: ट्यूटोरियल हाइपरलिंक गंतव्यों को निकालने पर केंद्रित है। हालाँकि, आप लिंक प्रकार और निर्देशांक निकालने सहित उन्नत सुविधाओं का पता लगाने के लिए आधिकारिक Aspose.PDF दस्तावेज़ देख सकते हैं।
