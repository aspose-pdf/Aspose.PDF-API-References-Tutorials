## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ขั้นแรก คุณต้องระบุเส้นทางไปยังเอกสาร PDF ของคุณ นี่คือจุดที่ Aspose จะค้นหาไฟล์

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงที่ไฟล์ PDF ของคุณตั้งอยู่

## ขั้นตอนที่ 2: เปิดเอกสาร PDF

ต่อไปเราจะเปิดเอกสาร PDF โดยใช้`Document` คลาสที่จัดทำโดย Aspose.PDF

```csharp
// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "GetChildBookmarks.pdf");
```

 บรรทัดโค้ดนี้จะเริ่มต้นสิ่งใหม่`Document` วัตถุกับไฟล์ PDF ของคุณ ตรวจสอบให้แน่ใจว่าชื่อไฟล์ตรงกับที่คุณมี

## ขั้นตอนที่ 3: วนซ้ำผ่านบุ๊กมาร์ก

ตอนนี้มาดูบุ๊กมาร์กทั้งหมดใน PDF กัน นี่คือจุดที่ความมหัศจรรย์เกิดขึ้น!

```csharp
// วนซ้ำผ่านบุ๊กมาร์กทั้งหมด
foreach (OutlineItemCollection outlineItem in pdfDocument.Outlines)
{
    Console.WriteLine(outlineItem.Title);
    Console.WriteLine(outlineItem.Italic);
    Console.WriteLine(outlineItem.Bold);
    Console.WriteLine(outlineItem.Color);
```

 ที่นี่เราใช้`foreach` วนซ้ำเพื่อวนซ้ำผ่านบุ๊กมาร์กแต่ละรายการในเอกสาร เราจะพิมพ์ชื่อเรื่อง สถานะตัวเอียง สถานะตัวหนา และสีของบุ๊กมาร์กแต่ละรายการออกมา

## ขั้นตอนที่ 4: ตรวจสอบบุ๊กมาร์กของเด็ก

บางครั้งบุ๊กมาร์กอาจมีบุ๊กมาร์กย่อย ลองตรวจสอบว่ามีบุ๊กมาร์กย่อยหรือไม่และทำซ้ำด้วย

```csharp
if (outlineItem.Count > 0)
{
    Console.WriteLine("Child Bookmarks");
    // มีบุ๊กมาร์กสำหรับเด็กที่วนซ้ำผ่านบุ๊กมาร์กนั้นด้วย
    foreach (OutlineItemCollection childOutline in outlineItem)
    {
        Console.WriteLine(childOutline.Title);
        Console.WriteLine(childOutline.Italic);
        Console.WriteLine(childOutline.Bold);
        Console.WriteLine(childOutline.Color);
    }
}
```

ในสไนปเป็ตนี้ เราจะตรวจสอบว่าบุ๊กมาร์กปัจจุบันมีบุ๊กมาร์กย่อยหรือไม่ หากมี เราจะวนซ้ำและพิมพ์รายละเอียดของบุ๊กมาร์กเหล่านั้น
