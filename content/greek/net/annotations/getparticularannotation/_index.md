Εάν εργάζεστε με αρχεία PDF στο .NET, μπορεί να συναντήσετε την ανάγκη να λάβετε έναν συγκεκριμένο σχολιασμό σε ένα αρχείο PDF. Σε αυτόν τον οδηγό, θα σας δείξουμε πώς να χρησιμοποιήσετε το Aspose.PDF για .NET για να λάβετε έναν συγκεκριμένο σχολιασμό από ένα έγγραφο PDF χρησιμοποιώντας C#.

Ακολουθήστε αυτά τα απλά βήματα για να λάβετε έναν συγκεκριμένο σχολιασμό από ένα έγγραφο PDF:

## Βήμα 1: Λάβετε συγκεκριμένο σχολιασμό από το έγγραφο PDF

Αρχικά, βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.PDF για .NET και την έχετε αναφέρει στο έργο σας.

Στη συνέχεια, δημιουργήστε μια νέα παρουσία της κλάσης Document και φορτώστε το έγγραφο PDF χρησιμοποιώντας τη διαδρομή προς τον κατάλογο εγγράφων.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document pdfDocument = new Document(dataDir + "GetParticularAnnotation.pdf");
```

## Βήμα 2: Μπορείτε να λάβετε έναν συγκεκριμένο σχολιασμό χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
TextAnnotation textAnnotation = (TextAnnotation)pdfDocument.Pages[1].Annotations[1];
```

Αυτός ο κώδικας ανακτά τον δεύτερο σχολιασμό στη δεύτερη σελίδα του εγγράφου PDF.

## Βήμα 3: Τέλος, μπορείτε να λάβετε τις ιδιότητες του σχολιασμού χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
Console.WriteLine("Title : {0} ", textAnnotation.Title);
Console.WriteLine("Subject : {0} ", textAnnotation.Subject);
Console.WriteLine("Contents : {0} ", textAnnotation.Contents);
```

Αυτός ο κωδικός εμφανίζει τον τίτλο, το θέμα και τα περιεχόμενα του σχολιασμού στην κονσόλα.


### Παράδειγμα πηγαίου κώδικα για λήψη ειδικών σχολίων χρησιμοποιώντας το Aspose.PDF για .NET

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "GetParticularAnnotation.pdf");

// Λάβετε συγκεκριμένο σχολιασμό
TextAnnotation textAnnotation = (TextAnnotation)pdfDocument.Pages[1].Annotations[1];

// Λάβετε ιδιότητες σχολιασμού
Console.WriteLine("Title : {0} ", textAnnotation.Title);
Console.WriteLine("Subject : {0} ", textAnnotation.Subject);
Console.WriteLine("Contents : {0} ", textAnnotation.Contents);
```
