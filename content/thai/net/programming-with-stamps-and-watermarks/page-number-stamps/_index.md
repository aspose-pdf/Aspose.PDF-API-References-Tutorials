ในบทช่วยสอนนี้ เราจะแนะนำคุณทีละขั้นตอนเกี่ยวกับวิธีการเพิ่มตราประทับหมายเลขหน้าในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเปิดเอกสาร PDF ที่มีอยู่ สร้างตราประทับหมายเลขหน้า ตั้งค่าคุณสมบัติ และเพิ่มตราประทับดังกล่าวลงในหน้าเฉพาะในไฟล์ PDF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร PDF ที่มีอยู่

ขั้นตอนแรกคือโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร PDF ที่มีอยู่
Document pdfDocument = new Document(dataDir + "PageNumberStamp.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 3: การสร้างและกำหนดค่าตราประทับหมายเลขหน้า

เมื่อโหลดเอกสาร PDF เสร็จแล้ว เราสามารถสร้างบัฟเฟอร์การนับหน้าและกำหนดค่าตามความต้องการได้ ดังนี้:

```csharp
// สร้างบัฟเฟอร์หมายเลขหน้า
PageNumberStamp pageNumberStamp = new PageNumberStamp();

// กำหนดว่าบัฟเฟอร์อยู่ในพื้นหลังหรือไม่
pageNumberStamp.Background = false;

// รูปแบบของบัฟเฟอร์การนับหน้า
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;

// ระยะขอบล่างของบัฟเฟอร์กำหนดหมายเลขหน้า
pageNumberStamp.BottomMargin = 10;

// การจัดตำแหน่งแนวนอนของบัฟเฟอร์การนับหน้า
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;

// เลขเริ่มต้นของการนับหน้า
pageNumberStamp.StartingNumber = 1;

// ตั้งค่าคุณสมบัติข้อความบัฟเฟอร์หมายเลขหน้า
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

โค้ดด้านบนจะสร้างตราประทับหมายเลขหน้าพร้อมคุณสมบัติเช่น รูปแบบหมายเลขหน้า ระยะขอบด้านล่าง การจัดแนวแนวนอน หมายเลขเริ่มต้น และคุณสมบัติข้อความ

## ขั้นตอนที่ 4: การเพิ่มตราประทับหมายเลขหน้าลงในหน้าเฉพาะ

เมื่อกำหนดค่าการประทับหมายเลขหน้าแล้ว เราจะเพิ่มหมายเลขหน้าดังกล่าวลงในหน้าเฉพาะของเอกสาร PDF ได้ ดังต่อไปนี้:

```csharp
// เพิ่มบัฟเฟอร์หมายเลขหน้าลงในหน้าเฉพาะ
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
```

โค้ดด้านบนจะเพิ่มหมายเลขหน้าลงในหน้าแรกของเอกสาร PDF คุณสามารถเปลี่ยนหมายเลขหน้าได้ตามต้องการ

## ขั้นตอนที่ 5: บันทึกเอกสาร PDF ที่แก้ไขแล้ว

เมื่อเพิ่มหมายเลขหน้าลงในเอกสาร PDF แล้ว เราจะสามารถบันทึกเอกสาร PDF ที่แก้ไขได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสาร PDF ที่แก้ไขแล้ว
pdfDocument.Save(dataDir + "PageNumberStamp_out.pdf");
```

อย่าลืมแทนที่ "ไดเรกทอรีเอกสารของคุณ" ด้วยเส้นทางจริงไปยังไดเรกทอรีที่คุณต้องการบันทึกเอกสาร PDF ที่แก้ไข

### ตัวอย่างโค้ดต้นฉบับสำหรับการประทับหมายเลขหน้าโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "PageNumberStamp.pdf");

// สร้างแสตมป์หมายเลขหน้า
PageNumberStamp pageNumberStamp = new PageNumberStamp();

// ไม่ว่าแสตมป์จะเป็นพื้นหลัง
pageNumberStamp.Background = false;
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;
pageNumberStamp.BottomMargin = 10;
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;
pageNumberStamp.StartingNumber = 1;

// ตั้งค่าคุณสมบัติข้อความ
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;

// เพิ่มตราประทับบนหน้าที่ต้องการ
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
dataDir = dataDir + "PageNumberStamp_out.pdf";

// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
Console.WriteLine("\nPage number stamp added successfully.\nFile saved at " + dataDir);

```

## บทสรุป

ขอแสดงความยินดี! คุณได้เรียนรู้วิธีการเพิ่มหมายเลขหน้าลงในเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET แล้ว ตอนนี้คุณสามารถปรับแต่งเอกสาร PDF ของคุณได้โดยการเพิ่มหมายเลขหน้าที่ชัดเจนและให้ข้อมูล

### คำถามที่พบบ่อยเกี่ยวกับการประทับตราหมายเลขหน้าในไฟล์ PDF

#### ถาม: Page Number Stamp คืออะไร และใช้เพิ่มหมายเลขหน้าลงในไฟล์ PDF ได้อย่างไร

A: Page Number Stamp เป็นฟีเจอร์ใน Aspose.PDF ที่ช่วยให้คุณสามารถเพิ่มหมายเลขหน้าแบบไดนามิกลงในหน้าเฉพาะของเอกสาร PDF ในบทช่วยสอนนี้ ทำได้โดยสร้างอ็อบเจ็กต์ PageNumberStamp กำหนดค่าคุณสมบัติของอ็อบเจ็กต์ และเพิ่มลงในหน้าที่กำหนด

#### ถาม: โค้ดต้นฉบับ C# ที่ให้มาทำการเพิ่มแสตมป์หมายเลขหน้าลงในไฟล์ PDF ได้อย่างไร

A: โค้ดนี้จะแสดงวิธีการโหลดเอกสาร PDF ที่มีอยู่ สร้าง PageNumberStamp ตั้งค่าคุณสมบัติต่างๆ (เช่น รูปแบบ แบบอักษร การจัดตำแหน่ง ฯลฯ) จากนั้นเพิ่มตราประทับลงในหน้าเฉพาะ ตราประทับจะคำนวณจำนวนหน้าทั้งหมดโดยอัตโนมัติและแทรกหมายเลขหน้าที่ถูกต้อง

#### ถาม: ฉันสามารถกำหนดลักษณะของหมายเลขหน้า เช่น สไตล์แบบอักษร สีและขนาด ได้หรือไม่

A: แน่นอน คุณสามารถปรับแต่งลักษณะที่ปรากฏของแสตมป์หมายเลขหน้าได้โดยการปรับคุณสมบัติเช่น แบบอักษร ขนาดแบบอักษร สไตล์แบบอักษร (ตัวหนา ตัวเอียง ฯลฯ) และสีข้อความ

#### ถาม: สามารถเพิ่มตราประทับหมายเลขหน้าให้กับหลาย ๆ หน้าในเอกสาร PDF ได้หรือไม่

A: ใช่ คุณสามารถเพิ่มแสตมป์หมายเลขหน้าให้กับหลาย ๆ หน้าได้โดยการสร้างอ็อบเจ็กต์ PageNumberStamp หลาย ๆ รายการและเพิ่มแต่ละรายการลงในหน้าที่ต้องการ

#### ถาม: ฉันสามารถเลือกได้ว่าจะให้แสตมป์หมายเลขหน้าปรากฏเป็นส่วนหนึ่งของเนื้อหาหน้าหรือเป็นองค์ประกอบพื้นหลังหรือไม่

 A: ใช่ คุณสามารถควบคุมได้ว่าจะให้แสตมป์หมายเลขหน้าปรากฏเป็นส่วนหนึ่งของเนื้อหาหน้าหรือเป็นองค์ประกอบพื้นหลัง โดยการตั้งค่า`Background` คุณสมบัติของ PageNumberStamp

#### ถาม: ฉันจะระบุรูปแบบของหมายเลขหน้ารวมทั้งจำนวนหน้าทั้งหมดได้อย่างไร

 A: รหัสนี้ใช้`Format`คุณสมบัติของ PageNumberStamp เพื่อระบุรูปแบบของหมายเลขหน้า แมโคร "# of" ใช้เพื่อแสดงจำนวนหน้าทั้งหมด

#### ถาม: จะเกิดอะไรขึ้นหากฉันเพิ่มตราประทับหมายเลขหน้าเดียวกันลงในหลาย ๆ หน้า?

A: การเพิ่มอินสแตนซ์ PageNumberStamp เดียวกันลงในหลายหน้าจะแสดงหมายเลขหน้าที่ถูกต้องสำหรับแต่ละหน้า สแตมป์จะปรับหมายเลขหน้าและจำนวนหน้าทั้งหมดโดยอัตโนมัติ

#### ถาม: ฉันสามารถเพิ่มแสตมป์หมายเลขหน้าลงในส่วนหัวหรือส่วนท้ายของเอกสาร PDF ได้หรือไม่

A: แม้ว่า PageNumberStamps จะถูกเพิ่มลงในเนื้อหาของหน้าโดยตรง แต่คุณสามารถใช้ FloatingBox หรือเทคนิคอื่นเพื่อวางตำแหน่งไว้ในส่วนหัวหรือส่วนท้ายได้

#### ถาม: ฉันจะระบุตำแหน่งของแสตมป์หมายเลขหน้าบนหน้าได้อย่างไร

 ก. การ`BottomMargin` และ`HorizontalAlignment` คุณสมบัติของ PageNumberStamp ช่วยให้คุณควบคุมตำแหน่งของสแตมป์ภายในหน้าได้

#### ถาม: จะเกิดอะไรขึ้นหากฉันต้องการเริ่มการนับหน้าด้วยหมายเลขอื่นแทนที่จะเป็น 1?

 A: คุณสามารถตั้งค่าได้`StartingNumber`คุณสมบัติของ PageNumberStamp เพื่อระบุหมายเลขหน้าเริ่มต้น
