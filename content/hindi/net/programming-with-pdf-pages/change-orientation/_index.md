इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के पेज ओरिएंटेशन को बदलने की चरण-दर-चरण प्रक्रिया से अवगत कराएँगे। हम बंडल किए गए C# स्रोत कोड की व्याख्या करेंगे और आपको अपने स्वयं के प्रोजेक्ट में इस सुविधा को समझने और लागू करने में मदद करने के लिए एक व्यापक मार्गदर्शिका प्रदान करेंगे। इस ट्यूटोरियल के अंत में, आप जानेंगे कि .NET के लिए Aspose.PDF का उपयोग करके अपने PDF दस्तावेज़ों के पेज ओरिएंटेशन को कैसे बदला जाए।

## आवश्यक शर्तें
आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान
- आपके विकास परिवेश में .NET के लिए Aspose.PDF स्थापित है

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें
सबसे पहले, आपको अपने दस्तावेज़ निर्देशिका का पथ सेट करना होगा। यह वह स्थान है जहाँ आपकी इनपुट PDF फ़ाइल स्थित है और जहाँ आप अपनी संशोधित आउटपुट PDF फ़ाइल को सहेजना चाहते हैं। "आपकी दस्तावेज़ निर्देशिका" को उचित पथ से बदलें।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: पीडीएफ दस्तावेज़ लोड करें
 फिर आप इनपुट फ़ाइल से पीडीएफ दस्तावेज़ लोड कर सकते हैं`Document` Aspose.PDF की क्लास। PDF फ़ाइल के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

```csharp
Document doc = new Document(dataDir + "input.pdf");
```

## चरण 3: पृष्ठ अभिविन्यास बदलें
अब हम दस्तावेज़ के प्रत्येक पृष्ठ पर जाएँगे और उसका अभिविन्यास बदलेंगे। प्रत्येक पृष्ठ के लिए, हम मीडिया बॉक्स के आयामों को संशोधित करेंगे (`MediaBox`) चौड़ाई और ऊंचाई को स्वैप करके, फिर हम पेज की स्थिति को बनाए रखने के लिए मीडिया बॉक्स के निर्देशांक को समायोजित करते हैं। अंत में, हम पेज रोटेशन को 90 डिग्री पर सेट करते हैं।

```csharp
foreach(Page page in doc.Pages)
{
Aspose.Pdf.Rectangle r = page.MediaBox;
double newHeight = r.Width;
double newWidth = r.Height;
double newLLX = r.LLX;
double newLLY = r.LLY + (r.Height - newHeight);
page.MediaBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
page.CropBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
page. Rotate = Rotate. on90;
}
```

## चरण 4: संशोधित PDF दस्तावेज़ को सहेजें
 अंत में, आप संशोधित पीडीएफ दस्तावेज़ को आउटपुट फ़ाइल में सहेज सकते हैं`Save()` की विधि`Document`क्लास में सही पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।

```csharp
dataDir = dataDir + "ChangeOrientation_out.pdf";
doc.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके अभिविन्यास परिवर्तन के लिए नमूना स्रोत कोड 

```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "input.pdf");
foreach (Page page in doc.Pages)
{
	Aspose.Pdf.Rectangle r = page.MediaBox;
	double newHeight = r.Width;
	double newWidth = r.Height;
	double newLLX = r.LLX;
	// पृष्ठ के बदलते आकार की भरपाई के लिए हमें पृष्ठ को ऊपर ले जाना होगा
	// (पृष्ठ का निचला किनारा 0,0 है और जानकारी आमतौर पर ऊपर से नीचे तक रखी जाती है)
	// पृष्ठ के शीर्ष पर। यही कारण है कि हम अंतर के बीच प्रेमी किनारे ऊपरी स्थानांतरित करते हैं
	// पुरानी और नई ऊंचाई.
	double newLLY = r.LLY + (r.Height - newHeight);
	page.MediaBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
	// कभी-कभी हमें CropBox भी सेट करने की आवश्यकता होती है (यदि यह मूल फ़ाइल में सेट किया गया था)
	page.CropBox = new Aspose.Pdf.Rectangle(newLLX, newLLY, newLLX + newWidth, newLLY + newHeight);
	// पृष्ठ का घूर्णन कोण सेट करना
	page.Rotate = Rotation.on90;
}
dataDir = dataDir + "ChangeOrientation_out.pdf";
// आउटपुट फ़ाइल सहेजें
doc.Save(dataDir);
System.Console.WriteLine("\nPage orientation changed successfully.\nFile saved at " + dataDir);

```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के पेज ओरिएंटेशन को कैसे बदला जाए। ऊपर बताए गए चरणों का पालन करके, आप आसानी से अपने प्रोजेक्ट में इस कार्यक्षमता को लागू कर सकते हैं। PDF फ़ाइलों के साथ काम करने के लिए अन्य उपयोगी सुविधाओं की खोज करने के लिए Aspose.PDF दस्तावेज़ को आगे भी देखें।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ में पृष्ठ अभिविन्यास बदलने का उद्देश्य क्या है?

उत्तर: PDF दस्तावेज़ में पेज ओरिएंटेशन बदलने से आप पेज की सामग्री को 90 डिग्री तक घुमा सकते हैं। यह उन परिदृश्यों में उपयोगी हो सकता है जहाँ मूल सामग्री को किसी भिन्न ओरिएंटेशन में प्रदर्शित या प्रिंट करने की आवश्यकता होती है, जैसे पोर्ट्रेट से लैंडस्केप मोड में स्विच करना या इसके विपरीत।

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ में विशिष्ट पृष्ठों का अभिविन्यास बदल सकता हूँ?

 उत्तर: हां, आप पीडीएफ दस्तावेज़ में विशिष्ट पृष्ठों का ओरिएंटेशन बदल सकते हैं। दिए गए C# स्रोत कोड में,`foreach` लूप का उपयोग दस्तावेज़ के प्रत्येक पृष्ठ पर जाने और उसके अभिविन्यास को बदलने के लिए किया जाता है। यदि आप केवल विशिष्ट पृष्ठों के अभिविन्यास को बदलना चाहते हैं, तो आप उन पृष्ठों को उनके पृष्ठ संख्या या अन्य मानदंडों के आधार पर लक्षित करने के लिए लूप को संशोधित कर सकते हैं।

#### प्रश्न: क्या पृष्ठ अभिविन्यास बदलने से पृष्ठ पर सामग्री का लेआउट प्रभावित होता है?

उत्तर: हां, पेज ओरिएंटेशन बदलने से पेज पर मौजूद कंटेंट का लेआउट प्रभावित होगा। कंटेंट 90 डिग्री घूम जाएगा और पेज की चौड़ाई और ऊंचाई बदल जाएगी। नतीजतन, पेज पर मौजूद कंटेंट का प्लेसमेंट और अलाइनमेंट बदल सकता है।

#### प्रश्न: क्या मैं पृष्ठ को 90 डिग्री के अलावा किसी अन्य कोण पर घुमा सकता हूँ?

 उत्तर: दिए गए C# स्रोत कोड में, पृष्ठ रोटेशन को 90 डिग्री पर सेट किया गया है`page.Rotate = Rotate.on90;` . हालाँकि, यदि आवश्यक हो तो आप रोटेशन कोण को अन्य मानों में बदल सकते हैं। उदाहरण के लिए, आप उपयोग कर सकते हैं`Rotate.on180` पृष्ठ को 180 डिग्री घुमाने के लिए या`Rotate.on270` इसे 270 डिग्री तक घुमाएं।

#### प्रश्न: मैं उस पृष्ठ सामग्री को कैसे संभालूँ जो अभिविन्यास बदलने के बाद ओवरफ्लो हो जाती है?

उत्तर: पेज ओरिएंटेशन बदलते समय, पेज के आयाम बदल सकते हैं, जिसके परिणामस्वरूप कंटेंट ओवरफ्लो हो सकता है। इसे संभालने के लिए, आपको पेज पर कंटेंट के लेआउट और फ़ॉर्मेटिंग को एडजस्ट करने की आवश्यकता हो सकती है। आप .NET के लिए Aspose.PDF द्वारा प्रदान की गई सुविधाओं का उपयोग कर सकते हैं, जैसे कि तत्वों का आकार बदलना, मार्जिन को एडजस्ट करना, या कंटेंट को फिर से व्यवस्थित करना, यह सुनिश्चित करने के लिए कि ओरिएंटेशन परिवर्तन के बाद पेज कंटेंट ठीक से फिट हो।
