การเข้ารหัสไฟล์ PDF เป็นมาตรการรักษาความปลอดภัยที่สำคัญในการปกป้องข้อมูลที่เป็นความลับ ด้วย Aspose.PDF สำหรับ .NET คุณสามารถเข้ารหัสไฟล์ PDF ของคุณได้อย่างง่ายดายโดยใช้ซอร์สโค้ดต่อไปนี้:

## ขั้นตอนที่ 1: นำเข้าไลบรารีที่จำเป็น

ก่อนที่คุณจะเริ่มต้น คุณต้องนำเข้าไลบรารีที่จำเป็นสำหรับโปรเจ็กต์ C# ของคุณ ต่อไปนี้เป็นคำสั่งนำเข้าที่จำเป็น:

```csharp
using Aspose.Pdf;
```

## ขั้นตอนที่ 2: กำหนดเส้นทางไปยังโฟลเดอร์เอกสาร

 ในขั้นตอนนี้ คุณจะต้องระบุเส้นทางไปยังโฟลเดอร์ที่มีไฟล์ PDF ที่จะเข้ารหัส แทนที่`"YOUR DOCUMENTS DIRECTORY"`ในรหัสต่อไปนี้พร้อมเส้นทางจริงไปยังโฟลเดอร์เอกสารของคุณ:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 3: เปิดเอกสาร PDF

ถัดไป คุณต้องเปิดเอกสาร PDF ที่คุณต้องการเข้ารหัส ใช้รหัสต่อไปนี้เพื่อโหลดเอกสาร:

```csharp
Document document = new Document(dataDir + "Encrypt.pdf");
```

## ขั้นตอนที่ 4: เข้ารหัส PDF

ตอนนี้คุณสามารถเข้ารหัส PDF โดยใช้รหัสต่อไปนี้:

```csharp
document. Encrypt("user", "owner", 0, CryptoAlgorithm.RC4x128);
```

ในตัวอย่างนี้ เราใช้อัลกอริธึมการเข้ารหัส RC4x128 กับรหัสผ่าน "ผู้ใช้" และ "เจ้าของ" คุณสามารถเปลี่ยนการตั้งค่าเหล่านี้ได้ตามต้องการ

## ขั้นตอนที่ 5: สำรองข้อมูล PDF ที่เข้ารหัส

สุดท้าย คุณสามารถบันทึก PDF ที่เข้ารหัสไปยังตำแหน่งที่ระบุได้โดยใช้รหัสต่อไปนี้:

```csharp
dataDir = dataDir + "Encrypt_out.pdf";
document. Save(dataDir);
```

อย่าลืมระบุเส้นทางและชื่อไฟล์ที่ต้องการสำหรับ PDF ที่เข้ารหัส

### ตัวอย่างซอร์สโค้ดสำหรับการเข้ารหัสโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// เปิดเอกสาร
Document document = new Document(dataDir+ "Encrypt.pdf");
// เข้ารหัส PDF
document.Encrypt("user", "owner", 0, CryptoAlgorithm.RC4x128);
dataDir = dataDir + "Encrypt_out.pdf";
// บันทึก PDF ที่อัปเดต
document.Save(dataDir);
Console.WriteLine("\nPDF file encrypted successfully.\nFile saved at " + dataDir);
```
