## مقدمة

هل سبق لك أن أردت إنشاء ملفات PDF جذابة بصريًا برمجيًا؟ إذا كان الأمر كذلك، فأنت في المكان المناسب! في هذا البرنامج التعليمي، سنتعمق في عالم Aspose.PDF for .NET، وهي مكتبة قوية تتيح لك التعامل مع مستندات PDF بسهولة. اليوم، سنركز على إنشاء مستطيل مملوء داخل ملف PDF. سواء كنت مطورًا متمرسًا أو مبتدئًا، سيرشدك هذا الدليل خلال كل خطوة بطريقة ودية وجذابة. لذا، ارتدِ قبعة البرمجة الخاصة بك، ولنبدأ!

## المتطلبات الأساسية

قبل أن ننتقل إلى الكود، هناك بعض الأشياء التي تحتاج إلى وضعها في مكانها:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. فهو بيئة تطوير متكاملة رائعة لتطوير .NET.
2.  Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك العثور عليها[هنا](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: إن القليل من المعرفة ببرمجة C# سوف يساعدك على فهم مقتطفات التعليمات البرمجية بشكل أفضل.

## استيراد الحزم

للبدء، تحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

### إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة التحكم لتبسيط الأمر.

### إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.PDF" وقم بتثبيت الإصدار الأحدث.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الكود!

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً وقبل كل شيء، عليك تحديد المسار الذي سيتم حفظ ملف PDF فيه. وهذا أمر بالغ الأهمية لأنه يخبر البرنامج بمكان إنشاء الملف.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي على جهازك حيث تريد حفظ ملف PDF.

## الخطوة 2: إنشاء مثيل مستند

 بعد ذلك، سنقوم بإنشاء مثيل لـ`Document`هذه الفئة تمثل مستند PDF الذي ستعمل عليه.

```csharp
// إنشاء مثيل المستند
Document doc = new Document();
```

يقوم هذا السطر بإنشاء مستند PDF جديد يمكننا التعامل معه.

## الخطوة 3: إضافة صفحة إلى المستند

الآن، دعنا نضيف صفحة إلى مستندنا. يحتاج كل ملف PDF إلى صفحة واحدة على الأقل، أليس كذلك؟

```csharp
// إضافة صفحة إلى مجموعة صفحات ملف PDF
Page page = doc.Pages.Add();
```

يضيف هذا الكود صفحة جديدة إلى المستند، مما يسمح لنا برسم الأشكال عليها.

## الخطوة 4: إنشاء مثيل للرسم البياني

 لرسم الأشكال، نحتاج إلى إنشاء`Graph` على سبيل المثال، فكر في الرسم البياني باعتباره لوحة قماشية يمكنك رسم أشكال مختلفة عليها.

```csharp
// إنشاء مثيل للرسم البياني
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

هنا، نقوم بإنشاء رسم بياني بعرض 100 وارتفاع 400.

## الخطوة 5: إضافة الرسم البياني إلى الصفحة

الآن بعد أن أصبح لدينا الرسم البياني، فلنقم بإضافته إلى الصفحة التي أنشأناها سابقًا.

```csharp
// إضافة كائن الرسم البياني إلى مجموعة فقرات مثيل الصفحة
page.Paragraphs.Add(graph);
```

يربط هذا الخط الرسم البياني بالصفحة، مما يجعله جاهزًا للرسم.

## الخطوة 6: إنشاء مثيل مستطيل

بعد ذلك، سنقوم بإنشاء مستطيل نريد ملؤه باللون.

```csharp
// إنشاء مثيل المستطيل
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
```

في هذا الكود، نقوم بتحديد موضع المستطيل وحجمه. تمثل المعلمات إحداثيات x وy والعرض والارتفاع.

## الخطوة 7: تحديد لون التعبئة

الآن، دعنا نختار لون المستطيل. سنملأه باللون الأحمر في هذا المثال.

```csharp
// تحديد لون التعبئة لكائن الرسم البياني
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
```

يحدد هذا الخط لون تعبئة المستطيل باللون الأحمر. يمكنك اختيار أي لون تريده!

## الخطوة 8: أضف المستطيل إلى الرسم البياني

وبعد أن أصبح المستطيل جاهزًا، حان الوقت لإضافته إلى الرسم البياني.

```csharp
// إضافة كائن مستطيل إلى مجموعة الأشكال الخاصة بكائن الرسم البياني
graph.Shapes.Add(rect);
```

يضيف هذا الكود المستطيل إلى الرسم البياني، مما يجعله جزءًا من رسمنا.

## الخطوة 9: احفظ مستند PDF

وأخيرًا، نحتاج إلى حفظ مستندنا في الدليل المحدد.

```csharp
dataDir = dataDir + "CreateFilledRectangle_out.pdf";
// حفظ ملف PDF
doc.Save(dataDir);
```

 هذا الكود يحفظ ملف PDF باسم`CreateFilledRectangle_out.pdf` في الدليل الذي حددته سابقًا.

## الخطوة 10: رسالة التأكيد

لإعلامنا بأن كل شيء سار بسلاسة، يمكننا طباعة رسالة تأكيد.

```csharp
Console.WriteLine("\nFilled rectangle object created successfully.\nFile saved at " + dataDir);
```

سيؤدي هذا السطر إلى إخراج رسالة في وحدة التحكم، تؤكد أن إنشاء المستطيل المملوء تم بنجاح.
