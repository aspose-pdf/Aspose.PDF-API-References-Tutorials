निम्न लेख .NET के लिए Aspose.PDF लाइब्रेरी का उपयोग करके ऑटो फ़िट टू विंडो कार्यक्षमता प्राप्त करने के लिए प्रदान किए गए C# स्रोत कोड का उपयोग करने के तरीके पर एक चरण-दर-चरण मार्गदर्शिका है। ऑटो फ़िट टू विंडो फ़ंक्शन आपको देखने वाली विंडो के लिए अनुकूलित लेआउट के साथ पीडीएफ फाइलें बनाने की अनुमति देता है। यह सुविधा विशेष रूप से तब उपयोगी होती है जब आप चाहते हैं कि आपका पीडीएफ दस्तावेज़ उपयोगकर्ता द्वारा उपयोग की जाने वाली पीडीएफ रीडर विंडो के आकार में स्वचालित रूप से समायोजित हो जाए।

## चरण 1: वातावरण की स्थापना

शुरू करने से पहले, आपको अपनी मशीन पर .NET के लिए Aspose.PDF लाइब्रेरी इंस्टॉल करनी होगी। साथ ही, अपने प्रोजेक्ट में ज़रूरी नेमस्पेस को भी इंपोर्ट करना न भूलें।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: एक पीडीएफ दस्तावेज़ बनाना

 शुरू करने के लिए, आपको एक बनाना होगा`Document` ऑब्जेक्ट को उसके डिफ़ॉल्ट कन्स्ट्रक्टर को कॉल करके खोलें।

```csharp
Document doc = new Document();
```

 इसके बाद, एक अनुभाग बनाएं`Pdf` वस्तु।

```csharp
Page sec1 = doc.Pages.Add();
```

## चरण 3: दस्तावेज़ में तालिका जोड़ना

 इस चरण में, हम अपने पीडीएफ दस्तावेज़ में एक तालिका जोड़ने जा रहे हैं। सबसे पहले एक तालिका बनाएँ`Table` वस्तु।

```csharp
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

इसके बाद, तालिका को अनुभाग के पैराग्राफ संग्रह में जोड़ें।

```csharp
sec1.Paragraphs.Add(tab1);
```

##  चरण 4: तालिका का स्वरूप अनुकूलित करना

आप सेल बॉर्डर और मार्जिन जैसे गुण सेट करके तालिका के स्वरूप को अनुकूलित कर सकते हैं।

```csharp
tab1. ColumnWidths = "50 50 50";
tab1.ColumnAdjustment = ColumnAdjustment.AutoFitToWindow;

tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
tab1.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 1F);

Aspose.Pdf.MarginInfo margin = new Aspose.Pdf.MarginInfo();
margin. Top = 5f;
margin. Left = 5f;
margin.Right = 5f;
margin. Bottom = 5f;

tab1. DefaultCellPadding = margin;
```

##  चरण 4: तालिका में पंक्तियाँ और कक्ष जोड़ना

अब आइए अपनी टेबल में पंक्तियाँ और सेल जोड़ें। एक पंक्ति बनाकर और उस पंक्ति में सेल जोड़कर शुरू करें।

```csharp
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("col1");
row1.Cells.Add("col2");
row1.Cells.Add("col3");

Aspose.Pdf.Row row2 = tab1.Rows.Add();
row2.Cells.Add("item1");
row2.Cells.Add("item2");
row2.Cells.Add("item3");
```

## चरण 5: दस्तावेज़ को सहेजना

अंत में, आउटपुट फ़ाइल पथ निर्दिष्ट करें और दस्तावेज़ को सहेजें।

```csharp
dataDir = dataDir + "AutoFitToWindow_out.pdf";
doc.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके विंडो में ऑटो फ़िट करने के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// पीडीएफ ऑब्जेक्ट को उसके खाली कन्स्ट्रक्टर को कॉल करके इंस्टैंटिएट करें
Document doc = new Document();
// Pdf ऑब्जेक्ट में अनुभाग बनाएँ
Page sec1 = doc.Pages.Add();

// तालिका ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
// इच्छित अनुभाग के पैराग्राफ संग्रह में तालिका जोड़ें
sec1.Paragraphs.Add(tab1);

// तालिका की स्तंभ चौड़ाई के साथ सेट करें
tab1.ColumnWidths = "50 50 50";
tab1.ColumnAdjustment = ColumnAdjustment.AutoFitToWindow;

// BorderInfo ऑब्जेक्ट का उपयोग करके डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);

// किसी अन्य अनुकूलित BorderInfo ऑब्जेक्ट का उपयोग करके तालिका बॉर्डर सेट करें
tab1.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 1F);
// MarginInfo ऑब्जेक्ट बनाएं और उसका बायां, निचला, दायां और ऊपरी मार्जिन सेट करें
Aspose.Pdf.MarginInfo margin = new Aspose.Pdf.MarginInfo();
margin.Top = 5f;
margin.Left = 5f;
margin.Right = 5f;
margin.Bottom = 5f;

// डिफ़ॉल्ट सेल पैडिंग को MarginInfo ऑब्जेक्ट पर सेट करें
tab1.DefaultCellPadding = margin;

//तालिका में पंक्तियाँ बनाएँ और फिर पंक्तियों में कक्ष बनाएँ
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("col1");
row1.Cells.Add("col2");
row1.Cells.Add("col3");
Aspose.Pdf.Row row2 = tab1.Rows.Add();
row2.Cells.Add("item1");
row2.Cells.Add("item2");
row2.Cells.Add("item3");

dataDir = dataDir + "AutoFitToWindow_out.pdf";
// तालिका ऑब्जेक्ट युक्त अद्यतन दस्तावेज़ सहेजें
doc.Save(dataDir);
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि ऑटो फ़िट टू विंडो फ़ीचर के साथ PDF फ़ाइल बनाने के लिए .NET के लिए Aspose.PDF का उपयोग कैसे करें। यह सुविधा तब बेहद उपयोगी होती है जब आप चाहते हैं कि आपका PDF दस्तावेज़ स्वचालित रूप से देखने वाली विंडो के आकार में समायोजित हो जाए। .NET के लिए Aspose.PDF PDF फ़ाइलों को बनाने और उनमें हेरफेर करने के लिए कई अन्य शक्तिशाली सुविधाएँ प्रदान करता है। मैं आपको इसकी सभी क्षमताओं की खोज करने के लिए इस लाइब्रेरी को और अधिक एक्सप्लोर करने के लिए प्रोत्साहित करता हूँ।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ निर्माण में ऑटो फिट टू विंडो सुविधा का उद्देश्य क्या है?

उत्तर: पीडीएफ़ जेनरेशन में ऑटो फ़िट टू विंडो सुविधा यह सुनिश्चित करती है कि पीडीएफ़ दस्तावेज़ का लेआउट उपयोगकर्ता द्वारा उपयोग की जाने वाली पीडीएफ़ रीडर विंडो के आकार के अनुसार स्वचालित रूप से समायोजित हो जाता है। यह बेहतर देखने की अनुमति देता है और यह सुनिश्चित करता है कि सामग्री उपलब्ध देखने के क्षेत्र में पूरी तरह से फिट हो।

#### प्रश्न: क्या मैं तालिका के स्वरूप को अनुकूलित कर सकता हूँ, जैसे फ़ॉन्ट का आकार और रंग?

उत्तर: हाँ, आप .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में तालिका की उपस्थिति को अनुकूलित कर सकते हैं। प्रदान किया गया कोड स्निपेट दर्शाता है कि सेल बॉर्डर, मार्जिन और कॉलम की चौड़ाई जैसे गुण कैसे सेट करें। आप तालिका और उसकी सामग्री के फ़ॉन्ट आकार, रंग और अन्य स्टाइलिंग पहलुओं को और भी अनुकूलित कर सकते हैं।

#### प्रश्न: मैं अपने C# प्रोजेक्ट में .NET के लिए Aspose.PDF को कैसे एकीकृत करूं?

उत्तर: अपने C# प्रोजेक्ट में .NET के लिए Aspose.PDF का उपयोग करने के लिए, आपको सबसे पहले अपनी मशीन पर .NET के लिए Aspose.PDF लाइब्रेरी इंस्टॉल करनी होगी। फिर, आप अपने C# प्रोजेक्ट में लाइब्रेरी का संदर्भ जोड़ सकते हैं। अंत में, .NET के लिए Aspose.PDF द्वारा प्रदान की गई कक्षाओं और विधियों तक पहुँचने के लिए आवश्यक नामस्थान आयात करें।

#### प्रश्न: क्या .NET के लिए Aspose.PDF .NET कोर अनुप्रयोगों के साथ संगत है?

उत्तर: हां, .NET के लिए Aspose.PDF .NET Core एप्लीकेशन के साथ संगत है। यह .NET फ्रेमवर्क, .NET Core और .NET 5.0+ सहित विभिन्न .NET प्लेटफ़ॉर्म का समर्थन करता है।

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ में और अधिक तालिकाएं जोड़ सकता हूं?

उत्तर: हां, आप कोड स्निपेट में दिखाए गए समान चरणों का पालन करके पीडीएफ दस्तावेज़ में कई तालिकाएँ जोड़ सकते हैं। बस नए इंस्टेंस बनाएँ`Aspose.Pdf.Table` क्लास में से किसी एक को चुनें और उन्हें पीडीएफ दस्तावेज़ के विभिन्न अनुभागों या पृष्ठों में जोड़ें।
