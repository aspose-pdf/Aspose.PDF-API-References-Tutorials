इस ट्यूटोरियल में, हम सीखेंगे कि एक्सेल वर्कशीट से डेटा कैसे एक्सपोर्ट करें और Aspose.PDF for .NET लाइब्रेरी का उपयोग करके PDF दस्तावेज़ में टेबल कैसे बनाएँ। हम सोर्स कोड को चरण दर चरण देखेंगे और प्रत्येक अनुभाग को विस्तार से समझाएँगे। इस ट्यूटोरियल के अंत तक, आप एक्सेल वर्कशीट से डेटा युक्त टेबल वाली PDF फ़ाइलें जेनरेट करने में सक्षम हो जाएँगे। चलिए शुरू करते हैं!

## आवश्यकताएं
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान
- आपकी मशीन पर Visual Studio स्थापित है
- आपके प्रोजेक्ट में Aspose.PDF for .NET लाइब्रेरी जोड़ी गई

## चरण 1: वातावरण की स्थापना
आरंभ करने के लिए, Visual Studio में एक नया C# प्रोजेक्ट बनाएँ। समाधान एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करके, "Manage NuGet Packages" का चयन करके और "Aspose.PDF" खोजकर Aspose.PDF for .NET लाइब्रेरी में संदर्भ जोड़ें। पैकेज स्थापित करें और आप जाने के लिए तैयार हैं।

## चरण 2: एक्सेल वर्कशीट लोड करना
हमारे कोड के पहले चरण में, हम Excel दस्तावेज़ वाली निर्देशिका का पथ परिभाषित करते हैं। "आपकी दस्तावेज़ निर्देशिका" को उस वास्तविक निर्देशिका पथ से बदलें जहाँ आपकी Excel फ़ाइल स्थित है।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

Aspose.Cells.Workbook workbook = new Aspose.Cells.Workbook(new FileStream(dataDir + "newBook1.xlsx", FileMode.Open));
```

यहाँ, हम Excel कार्यपुस्तिका लोड करने के लिए Aspose.Cells लाइब्रेरी का उपयोग करते हैं। "newBook1.xlsx" को अपनी Excel फ़ाइल के नाम से बदलना सुनिश्चित करें।

## चरण 3: वर्कशीट तक पहुँचना
 इसके बाद, हमें एक्सेल फ़ाइल में पहली वर्कशीट तक पहुँचने की ज़रूरत है। हम इसका उपयोग करते हैं`Worksheets` का संग्रह`Workbook` वस्तु।

```csharp
// एक्सेल फ़ाइल में पहली वर्कशीट तक पहुँचना
Aspose.Cells.Worksheet worksheet = workbook.Worksheets[0];
```

 यदि आपकी एक्सेल फ़ाइल में एकाधिक वर्कशीट हैं, तो आप इंडेक्स मान बदल सकते हैं`[0]` किसी भिन्न कार्यपत्रक तक पहुंचने के लिए.

## चरण 4: डेटा को डेटाटेबल में निर्यात करना
 अब, हम एक्सेल वर्कशीट की सामग्री को एक में निर्यात करेंगे`DataTable` ऑब्जेक्ट। हम निर्यात करने के लिए कोशिकाओं की श्रेणी निर्दिष्ट करते हैं`ExportDataTable` तरीका।

```csharp
// पहले सेल से शुरू करके 7 पंक्तियों और 2 कॉलमों की सामग्री को DataTable में निर्यात करना
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, worksheet.Cells.MaxRow + 1, worksheet.Cells.MaxColumn + 1, true);
```

इस उदाहरण में, हम वर्कशीट में पहले सेल (0, 0) से लेकर आखिरी सेल तक सभी पंक्तियों और स्तंभों को निर्यात करते हैं। अपनी आवश्यकताओं के आधार पर उपयुक्त श्रेणी निर्धारित करें।

## चरण 5: एक पीडीएफ दस्तावेज़ बनाना
अब, हम Aspose.PDF लाइब्रेरी का उपयोग करके एक नया PDF दस्तावेज़ बनाएंगे।

```csharp
// दस्तावेज़ इंस्टैंस को इंस्टैंसिएट करें
Aspose.Pdf.Document pdf1 = new Aspose.Pdf.Document();
```

इससे एक खाली पीडीएफ दस्तावेज़ बनता है जिसमें हम सामग्री जोड़ सकते हैं।

## चरण 6: पृष्ठ और तालिका जोड़ना
डेटा को तालिका प्रारूप में प्रदर्शित करने के लिए, हमें पीडीएफ दस्तावेज़ में एक पृष्ठ और एक तालिका जोड़ने की आवश्यकता है।

```csharp
// दस्तावेज़ इंस्टैंस में एक पृष्ठ बनाएँ
Aspose.Pdf.Page sec1 = pdf1.Pages.Add();

// एक टेबल ऑब्जेक्ट बनाएँ
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

// अनुभाग के पैराग्राफ़ संग्रह में टेबल ऑब्जेक्ट जोड़ें
sec1.Paragraphs.Add(tab1);
```

यहाँ, हम एक नया पेज और एक टेबल ऑब्जेक्ट बनाते हैं। फिर हम टेबल को पेज के पैराग्राफ़ संग्रह में जोड़ते हैं।

## चरण 7: तालिका गुण सेट करना
डेटा आयात करने से पहले, हमें तालिका के कुछ गुण, जैसे कॉलम की चौड़ाई और डिफ़ॉल्ट सेल बॉर्डर, सेट करने की आवश्यकता होती है।

```csharp
// तालिका की स्तंभ चौड़ाई निर्धारित करें
tab1.ColumnWidths = "40 100 100";

// BorderInfo ऑब्जेक्ट का उपयोग करके तालिका की डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
```

इस उदाहरण में, हमने कॉलम की चौड़ाई 40, 100 और 100 इकाइयों पर सेट की है। अपने डेटा के आधार पर मान समायोजित करें। हमने प्रत्येक सेल के सभी तरफ बॉर्डर दिखाने के लिए डिफ़ॉल्ट सेल बॉर्डर भी सेट किया है।

## चरण 8: तालिका में डेटा आयात करना
 अब, हम यहाँ से डेटा आयात करेंगे।`DataTable` ऑब्जेक्ट को तालिका में डालने के लिए`ImportDataTable` तरीका।

```csharp
// ऊपर बनाए गए DataTable से Table ऑब्जेक्ट में डेटा आयात करें
tab1.ImportDataTable(dataTable, true, 0, 0, dataTable.Rows.Count + 1, dataTable.Columns.Count);
```

 यहाँ, हम आयात करने के लिए पंक्तियों और स्तंभों की श्रेणी निर्दिष्ट करते हैं। इस उदाहरण में, हम सभी पंक्तियों और स्तंभों को आयात करते हैं`dataTable` वस्तु।

## चरण 9: तालिका को प्रारूपित करना
तालिका की दिखावट को बेहतर बनाने के लिए, हम विशिष्ट कक्षों या पंक्तियों पर फ़ॉर्मेटिंग लागू कर सकते हैं। इस चरण में, हम तालिका की पहली पंक्ति और वैकल्पिक पंक्तियों को फ़ॉर्मेट करेंगे।

```csharp
// तालिका से पहली पंक्ति प्राप्त करें
Aspose.Pdf.Row row1 = tab1.Rows[0];

// पहली पंक्ति को प्रारूपित करें
foreach(Aspose.Pdf.Cell curCell in row1.Cells)
{
     // पहली पंक्ति में कोशिकाओं का पृष्ठभूमि रंग सेट करें
     curCell.BackgroundColor = Color.Blue;// पहली पंक्ति में कोशिकाओं के लिए चेहरा सेट करें
     curCell.DefaultCellTextState.Font = Aspose.Pdf.Text.FontRepository.FindFont("Helvetica-Oblique");
    
     // पहली पंक्ति में कक्षों का फ़ॉन्ट रंग सेट करें
     curCell.DefaultCellTextState.ForegroundColor = Color.Yellow;
    
     // पहली पंक्ति में कक्षों के लिए पाठ संरेखण सेट करें
     curCell.DefaultCellTextState.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Center;
}

// वैकल्पिक पंक्ति प्रारूप
for (int All_Rows = 1; All_Rows <= dataTable.Rows.Count; All_Rows++)
{
     foreach(Aspose.Pdf.Cell curCell in tab1.Rows[All_Rows].Cells)
     {
         // वैकल्पिक पंक्तियों में कक्षों का पृष्ठभूमि रंग सेट करें
         curCell.BackgroundColor = Color.Gray;
        
         // वैकल्पिक पंक्तियों में कक्षों का पाठ रंग सेट करें
         curCell.DefaultCellTextState.ForegroundColor = Color.White;
     }
}
```

यहाँ, हम पहली पंक्ति में सभी कक्षों के माध्यम से पुनरावृति करते हैं और उनकी पृष्ठभूमि का रंग, फ़ॉन्ट का चेहरा, फ़ॉन्ट का रंग और पाठ संरेखण सेट करते हैं। फिर, हम वैकल्पिक पंक्तियों में सभी कक्षों के माध्यम से पुनरावृति करते हैं और उनकी पृष्ठभूमि और पाठ का रंग सेट करते हैं।

## चरण 10: पीडीएफ दस्तावेज़ को सहेजना
अंत में, हम पीडीएफ दस्तावेज़ को निर्दिष्ट स्थान पर सेव कर देते हैं।

```csharp
// पीडीएफ सहेजें
pdf1.Save(dataDir + @"Exceldata_toPdf_table.pdf");
```

आउटपुट PDF फ़ाइल के लिए "आपकी दस्तावेज़ निर्देशिका" को वांछित निर्देशिका पथ और फ़ाइल नाम से प्रतिस्थापित करना सुनिश्चित करें।

### .NET के लिए Aspose.PDF का उपयोग करके Excel वर्कशीट डेटा को तालिका में निर्यात करने के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

Aspose.Cells.Workbook workbook = new Aspose.Cells.Workbook(new FileStream(dataDir + "newBook1.xlsx", FileMode.Open));
// एक्सेल फ़ाइल में पहली वर्कशीट तक पहुँचना
Aspose.Cells.Worksheet worksheet = workbook.Worksheets[0];
// पहले सेल से शुरू करके 7 पंक्तियों और 2 कॉलमों की सामग्री को DataTable में निर्यात करना
DataTable dataTable = worksheet.Cells.ExportDataTable(0, 0, worksheet.Cells.MaxRow + 1, worksheet.Cells.MaxColumn + 1, true);

// एक दस्तावेज़ को इंस्टेंशिएट करें
Aspose.Pdf.Document pdf1 = new Aspose.Pdf.Document();
// दस्तावेज़ इंस्टैंस में एक पृष्ठ बनाएँ
Aspose.Pdf.Page sec1 = pdf1.Pages.Add();

// एक टेबल ऑब्जेक्ट बनाएँ
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

// अनुभाग के पैराग्राफ़ संग्रह में टेबल ऑब्जेक्ट जोड़ें
sec1.Paragraphs.Add(tab1);

// तालिका की कॉलम चौड़ाई सेट करें। हमें कॉलमकाउंट मैन्युअल रूप से निर्दिष्ट करने की आवश्यकता है।
// चूंकि वर्तमान एक्सेल वर्कशीट में तीन कॉलम हैं, इसलिए हम समान गिनती निर्दिष्ट कर रहे हैं
tab1.ColumnWidths = "40 100 100";

// BorderInfo ऑब्जेक्ट का उपयोग करके तालिका की डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);

// ऊपर बनाए गए DataTable से Table ऑब्जेक्ट में डेटा आयात करें
tab1.ImportDataTable(dataTable, true, 0, 0, dataTable.Rows.Count + 1, dataTable.Columns.Count);
// तालिका से पहली पंक्ति प्राप्त करें
Aspose.Pdf.Row row1 = tab1.Rows[0];

// पहली पंक्ति के सभी कक्षों में पुनरावृत्ति करें और उनकी पृष्ठभूमि का रंग नीला सेट करें
foreach (Aspose.Pdf.Cell curCell in row1.Cells)
{
	// तालिका की पहली पंक्ति में सभी कक्षों की पृष्ठभूमि सेट करें।
	curCell.BackgroundColor = Color.Blue;
	// तालिका में पहली पंक्ति की कोशिकाओं के लिए फ़ॉन्ट फेस सेट करें।
	curCell.DefaultCellTextState.Font = Aspose.Pdf.Text.FontRepository.FindFont("Helvetica-Oblique");
	// तालिका की पहली पंक्ति में सभी कक्षों का फ़ॉन्ट रंग सेट करें।
	curCell.DefaultCellTextState.ForegroundColor = Color.Yellow;
	// पहली पंक्ति के कक्षों के लिए पाठ संरेखण को केंद्र के रूप में सेट करें।
	curCell.DefaultCellTextState.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Center;
}

for (int All_Rows = 1; All_Rows <= dataTable.Rows.Count; All_Rows++)
{
	// पहली पंक्ति के सभी कक्षों में पुनरावृत्ति करें और उनकी पृष्ठभूमि का रंग नीला सेट करें
	foreach (Aspose.Pdf.Cell curCell in tab1.Rows[All_Rows].Cells)
	{
		// पहली पंक्ति को छोड़कर सभी कक्षों का पृष्ठभूमि रंग सेट करें।
		curCell.BackgroundColor = Color.Gray;
		// पहली पंक्ति को छोड़कर सभी कक्षों का पाठ रंग सेट करें।
		curCell.DefaultCellTextState.ForegroundColor = Color.White;
	}
}

// पीडीएफ को सुरक्षित रखें
pdf1.Save(dataDir + @"Exceldata_toPdf_table.pdf");
```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा कि .NET लाइब्रेरी के लिए Aspose.PDF का उपयोग करके Excel वर्कशीट से PDF टेबल में डेटा कैसे निर्यात किया जाए। हमने Excel वर्कशीट लोड करने, PDF दस्तावेज़ बनाने, टेबल जोड़ने, डेटा आयात करने और टेबल को फ़ॉर्मेट करने की चरण-दर-चरण प्रक्रिया को कवर किया। अब आप प्रोग्रामेटिक रूप से Excel डेटा वाली टेबल के साथ PDF फ़ाइलें बना सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: एक्सेल वर्कशीट डेटा को पीडीएफ तालिका में निर्यात करने का उद्देश्य क्या है?

उत्तर: एक्सेल वर्कशीट डेटा को पीडीएफ टेबल में एक्सपोर्ट करने से आप डेटा को संरचित और संगठित प्रारूप में प्रस्तुत कर सकते हैं। यह आपको एक्सेल वर्कशीट से डेटा वाली टेबल के साथ पीडीएफ फाइल बनाने में सक्षम बनाता है, जिससे पोर्टेबल डॉक्यूमेंट फॉर्मेट में जानकारी को साझा करना और संरक्षित करना आसान हो जाता है।

#### प्रश्न: क्या मैं पीडीएफ तालिका के स्वरूप को अनुकूलित कर सकता हूं?

उत्तर: हाँ, आप .NET के लिए Aspose.PDF द्वारा प्रदान किए गए विभिन्न गुणों का उपयोग करके PDF तालिका की उपस्थिति को अनुकूलित कर सकते हैं। प्रदान किए गए C# स्रोत कोड में, आप अपनी विशिष्ट आवश्यकताओं के अनुरूप कॉलम की चौड़ाई, सेल बॉर्डर, टेक्स्ट संरेखण, फ़ॉन्ट शैली और बहुत कुछ संशोधित कर सकते हैं।

#### प्रश्न: मैं एकाधिक वर्कशीट वाली एक्सेल फ़ाइलों को कैसे संभालूँ?

 उत्तर: दिए गए C# कोड में, हमने इंडेक्स का उपयोग करके एक्सेल फ़ाइल में पहली वर्कशीट तक पहुँच बनाई`[0]` यदि आपकी एक्सेल फ़ाइल में एकाधिक वर्कशीट हैं, तो आप इंडेक्स मान को तदनुसार बदलकर उन तक पहुँच सकते हैं, जैसे`[1]` दूसरे वर्कशीट के लिए या`[2]` तीसरे कार्यपत्रक के लिए.

#### प्रश्न: क्या मैं पीडीएफ तालिका में विशिष्ट पंक्तियों या कक्षों पर भिन्न स्वरूपण लागू कर सकता हूँ?

उत्तर: हाँ, आप PDF तालिका में विशिष्ट पंक्तियों या कक्षों पर भिन्न स्वरूपण लागू कर सकते हैं। उपलब्ध C# स्रोत कोड में, हमने प्रदर्शित किया कि पहली पंक्ति और वैकल्पिक पंक्तियों को उनके पृष्ठभूमि रंग, फ़ॉन्ट शैली और फ़ॉन्ट रंग को बदलकर अलग-अलग तरीके से कैसे स्वरूपित किया जाए। आप आवश्यकतानुसार किसी भी विशिष्ट पंक्ति या कक्ष पर समान स्वरूपण तकनीक लागू कर सकते हैं।

#### प्रश्न: क्या .NET के लिए Aspose.PDF एकमात्र लाइब्रेरी है जो Excel डेटा को PDF तालिका में निर्यात करने की अनुमति देती है?

उत्तर: .NET के लिए Aspose.PDF .NET अनुप्रयोगों में PDF दस्तावेज़ों के साथ काम करने के लिए एक शक्तिशाली लाइब्रेरी है। जबकि अन्य लाइब्रेरी उपलब्ध हो सकती हैं, .NET के लिए Aspose.PDF Excel डेटा से तालिकाओं के साथ PDF फ़ाइलों को बनाने, हेरफेर करने और निर्यात करने के लिए कई प्रकार की सुविधाएँ और क्षमताएँ प्रदान करता है, जो इसे ऐसे कार्यों के लिए एक लोकप्रिय विकल्प बनाता है।
