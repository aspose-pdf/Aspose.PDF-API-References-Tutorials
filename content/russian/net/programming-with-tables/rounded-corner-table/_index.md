В этом уроке мы шаг за шагом проведем вас по созданию таблицы с закругленными углами в документе PDF с помощью Aspose.PDF для .NET. Мы объясним предоставленный исходный код C# и покажем, как его реализовать.

## Шаг 1: Создание таблицы
Сначала мы создадим таблицу, используя следующий код:

```csharp
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

## Шаг 2: Настройка стиля с закругленными углами
Далее настроим стиль скругленных углов для таблицы:

```csharp
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
```

## Шаг 3: Настройка границ таблицы
Чтобы придать таблице закругленную угловую границу, нам необходимо создать объект BorderInfo и настроить его с помощью соответствующих параметров:

```csharp
// Создайте объект GraphInfo, чтобы задать цвет границы.
GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red;

// Создайте пустой объект BorderInfo
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);

// Установите радиус скругления границы на 15.
bInfo.RoundedBorderRadius = 15;

// Применить информацию о границе к таблице
tab1.Border = bInfo;
```

### Пример исходного кода для таблицы с закругленными углами с использованием Aspose.PDF для .NET

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red;
// Создайте пустой объект BorderInfo
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);
// Установите границу более круглой формы с радиусом скругления 15.
bInfo.RoundedBorderRadius = 15;
// Установите стиль угла стола как «Круглый».
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
// Установить информацию о границе таблицы
tab1.Border = bInfo;
```

### Часто задаваемые вопросы о таблице с закругленными углами в документе PDF

#### В: Могу ли я настроить радиус скругления углов стола?

A: Да, вы можете настроить радиус скругления углов таблицы, изменив значение`bInfo.RoundedBorderRadius` свойство в предоставленном исходном коде C#. Просто установите желаемое значение радиуса (в точках), чтобы получить желаемый вид скругленного угла.

#### В: Можно ли применить скругленные углы к отдельным ячейкам таблицы?

A: Нет, стиль скругленных углов применяется ко всей таблице в целом. Aspose.PDF для .NET в настоящее время не предоставляет встроенной поддержки для применения скругленных углов к отдельным ячейкам в таблице.

#### В: Можно ли изменить цвет закругленной границы угла?

 A: Да, вы можете изменить цвет закругленной угловой границы, изменив значение`graph.Color` свойство в исходном коде C#. Просто укажите желаемый цвет, например`Aspose.Pdf.Color.Red` или любое другое допустимое представление цвета.

#### В: Можно ли применять разные стили углов (например, квадратные и закругленные) к разным таблицам в одном и том же PDF-документе?

A: Да, можно применять разные стили углов к разным таблицам в одном документе PDF. Вы можете создать несколько таблиц и настроить их стили углов индивидуально в соответствии с вашими требованиями.

#### В: Можно ли настроить толщину закругленной угловой границы?

 A: Да, вы можете настроить толщину закругленной угловой границы, изменив`BorderInfo` свойства объекта в исходном коде C#. Например, вы можете задать`bInfo.Width` свойство для регулировки толщины границы.
