## การแนะนำ

เมื่อต้องสร้างเอกสาร PDF ที่ดูเป็นมืออาชีพ ตารางมักจะมีบทบาทสำคัญในการนำเสนอข้อมูลในลักษณะที่ชัดเจนและเป็นระเบียบ ในคู่มือนี้ เราจะสำรวจวิธีการจัดแนวข้อความภายในแถวตารางในเอกสาร PDF โดยใช้ไลบรารี Aspose.PDF สำหรับ .NET ไม่ว่าคุณจะกำลังสร้างรายงาน ใบแจ้งหนี้ หรือเอกสารใดๆ ที่ต้องการการนำเสนอข้อมูลแบบมีโครงสร้าง การเชี่ยวชาญการสร้างตารางสามารถปรับปรุงผลลัพธ์ของคุณได้อย่างมาก

## แพ็คเกจนำเข้า

หากต้องการเริ่มใช้ Aspose.PDF ก่อนอื่นคุณต้องนำเข้าเนมสเปซที่จำเป็นในไฟล์ C# ของคุณ วิธีตั้งค่ามีดังนี้:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

นี่จะนำเข้าคลาสที่จำเป็นซึ่งจะช่วยให้คุณสร้างและจัดการเอกสารและตาราง PDF ได้

เมื่อทุกอย่างพร้อมแล้ว มาดูขั้นตอนการสร้างเอกสาร PDF ที่มีตารางพร้อมข้อความที่จัดวางอย่างถูกต้องกันทีละขั้นตอน

## ขั้นตอนที่ 1: เริ่มต้นเอกสาร PDF

ก่อนที่จะเพิ่มเนื้อหาใดๆ เราต้องสร้างอินสแตนซ์ใหม่ของเอกสาร PDF

```csharp
// กำหนดไดเรกทอรีที่จะบันทึกเอกสาร
var dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างเอกสาร PDF
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```
 ที่นี่เราตั้งค่าไดเรกทอรีที่จะบันทึก PDF และสร้างอินสแตนซ์ของ`Document` คลาส อินสแตนซ์นี้ทำหน้าที่เป็นผืนผ้าใบสำหรับสร้าง PDF ของเรา

## ขั้นตอนที่ 2: ตั้งโต๊ะ

ต่อไปเราจะต้องสร้างอินสแตนซ์ใหม่ของตารางซึ่งจะเก็บข้อมูลของเราไว้

```csharp
//เริ่มต้นอินสแตนซ์ใหม่ของตาราง
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
```
 การ`Table` คลาสช่วยให้เราสร้างอ็อบเจ็กต์ตารางใหม่ได้ ทำให้เราเพิ่มแถวและคอลัมน์ได้อย่างง่ายดาย

## ขั้นตอนที่ 3: กำหนดค่าขอบตาราง

เพื่อเพิ่มความสวยงามของตาราง เราสามารถกำหนดขอบให้กับตารางทั้งหมดและเซลล์ได้

```csharp
// ตั้งค่าสีเส้นขอบตารางเป็นสีเทาอ่อน
table.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, .5f, Aspose.Pdf.Color.FromRgb(Color.LightGray));

// กำหนดเส้นขอบให้กับเซลล์ตาราง
table.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, .5f, Aspose.Pdf.Color.FromRgb(Color.LightGray));
```
เส้นขอบทำให้ตารางมีโครงสร้าง ทำให้อ่านง่ายขึ้น ในที่นี้ เราใช้สีเทาอ่อนสำหรับทั้งตารางและเซลล์แต่ละเซลล์

## ขั้นตอนที่ 4: เพิ่มแถวลงในตาราง

ต่อไปเราจะสร้างลูปเพื่อเพิ่มแถวในตาราง สำหรับตัวอย่างนี้ เราจะเพิ่มแถว 10 แถว

```csharp
// สร้างลูปเพื่อเพิ่ม 10 แถว
for (int row_count = 0; row_count < 10; row_count++)
{
    // เพิ่มแถวเข้าตาราง
    Aspose.Pdf.Row row = table.Rows.Add();
    row.VerticalAlignment = VerticalAlignment.Center;
    
    // เพิ่มเซลล์ลงในแถว
    row.Cells.Add("Column (" + row_count + ", 1)" + DateTime.Now.Ticks);
    row.Cells.Add("Column (" + row_count + ", 2)");
    row.Cells.Add("Column (" + row_count + ", 3)");
}
```
 ในลูปนี้ เราจะเพิ่มแถวทั้งหมด 10 แถว และสำหรับแต่ละแถว เราจะสร้างเซลล์ 3 เซลล์ เราใช้`DateTime.Now.Ticks` เพื่อเพิ่มค่าประทับเวลาลงในเซลล์แรกของแต่ละแถว ทำให้เนื้อหาเป็นแบบไดนามิกและไม่ซ้ำใคร`VerticalAlignment` ถูกตั้งเป็น`Center`เพื่อให้แน่ใจว่าข้อความในแต่ละเซลล์อยู่กึ่งกลางแนวตั้ง

## ขั้นตอนที่ 5: เพิ่มตารางลงในเอกสาร

เมื่อเพิ่มข้อมูลในตารางของเราเสร็จแล้ว ก็ถึงเวลาที่จะเพิ่มลงในเอกสาร PDF

```csharp
Page tocPage = doc.Pages.Add();
// เพิ่มวัตถุตารางลงในหน้าแรกของเอกสารอินพุต
tocPage.Paragraphs.Add(table);
```
เราสร้างหน้าใหม่ในเอกสาร PDF และเพิ่มตารางเป็นย่อหน้าในหน้านี้ การดำเนินการนี้จะเชื่อมโยงทุกอย่างเข้าด้วยกันเป็นเอกสารเดียว

## ขั้นตอนที่ 6: บันทึกเอกสาร

สุดท้ายเราจะต้องบันทึกการเปลี่ยนแปลงลงในเอกสารของเรา

```csharp
// บันทึกเอกสารอัปเดตที่มีวัตถุตาราง
doc.Save(dataDir + "43620_ByWords_out.pdf");
```
บรรทัดนี้จะเขียนเอกสารไปยังเส้นทางไฟล์ที่ระบุบนดิสก์ของคุณ ทำให้ตารางและเนื้อหาสมบูรณ์
