## Импортные пакеты

Для начала вам нужно импортировать необходимые пакеты в ваш проект C#. Вот как это можно сделать:

1. Откройте свой проект в Visual Studio.
2. Щелкните правой кнопкой мыши свой проект в обозревателе решений и выберите «Управление пакетами NuGet».
3. Найдите «Aspose.PDF» и установите пакет.

```csharp
using System;
using System.IO;
using Aspose.Pdf;
```

После установки пакета вы можете приступить к написанию кода.

## Шаг 1: Настройте каталог документов

Прежде чем вы сможете конвертировать ваш PDF, вам нужно указать каталог, в котором хранятся ваши документы. Это то место, где будут находиться ваши входные файлы PDF и выходные файлы EPUB.

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Заменять`"YOUR DOCUMENT DIRECTORY"` фактический путь на вашем компьютере, где находится ваш PDF-файл.

## Шаг 2: Загрузите PDF-документ

 Теперь, когда у вас настроен каталог документов, следующим шагом будет загрузка PDF-документа, который вы хотите преобразовать. Это делается с помощью`Document` класс предоставлен Aspose.PDF.

```csharp
// Загрузить PDF-документ
Document pdfDocument = new Document(dataDir + "PDFToEPUB.pdf");
```

 Обязательно замените`"PDFToEPUB.pdf"` с именем вашего PDF-файла. Эта строка кода инициализирует новый`Document` объект с указанным PDF-файлом.

## Шаг 3: Создание параметров сохранения EPUB

 Перед сохранением документа в формате EPUB необходимо создать экземпляр`EpubSaveOptions`. Этот класс позволяет указать различные параметры для вывода EPUB.

```csharp
// Параметры сохранения Instantiate Epub
EpubSaveOptions options = new EpubSaveOptions();
```

 Эта строка создает новый экземпляр`EpubSaveOptions`, который вы настроите на следующем шаге.

## Шаг 4: Укажите режим распознавания контента

Одной из ключевых особенностей EPUB является его способность передавать содержимое. Вы можете указать, как содержимое должно распознаваться в процессе конвертации. Для этого примера мы установим режим распознавания содержимого на flow.

```csharp
// Укажите макет для содержимого
options.ContentRecognitionMode = EpubSaveOptions.RecognitionMode.Flow;
```

Эта настройка гарантирует, что содержимое вашего PDF-файла будет преобразовано таким образом, чтобы его было удобно читать в формате EPUB, что обеспечивает более удобный процесс чтения.

## Шаг 5: Сохраните документ EPUB

 Наконец, пришло время сохранить ваш преобразованный документ. Вы будете использовать`Save` Метод`Document` класс, чтобы сделать это.

```csharp
// Сохраните документ ePUB
pdfDocument.Save(dataDir + "PDFToEPUB_out.epub", options);
```

 Эта строка сохраняет преобразованный файл EPUB в том же каталоге, что и ваш исходный PDF. Выходной файл будет иметь имя`PDFToEPUB_out.epub`.
