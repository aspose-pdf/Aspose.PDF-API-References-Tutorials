ในบทช่วยสอนนี้ เราจะแนะนำคุณทีละขั้นตอนในการตั้งค่ารูปภาพเป็นพื้นหลังของหน้าโดยใช้ Aspose.PDF สำหรับ .NET เราจะอธิบายโค้ดต้นฉบับของ C# ที่ให้มา และให้คำแนะนำที่ครอบคลุมเพื่อช่วยให้คุณเข้าใจและนำฟีเจอร์นี้ไปใช้ในโครงการของคุณเอง เมื่อจบบทช่วยสอนนี้ คุณจะทราบวิธีการเพิ่มรูปภาพเป็นพื้นหลังของหน้าในเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET

## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- ความรู้พื้นฐานเกี่ยวกับภาษาการเขียนโปรแกรม C#
- Aspose.PDF สำหรับ .NET ติดตั้งอยู่ในสภาพแวดล้อมการพัฒนาของคุณ

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร
ขั้นแรก คุณต้องกำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือตำแหน่งที่คุณต้องการบันทึกเอกสาร PDF ที่แก้ไขแล้ว แทนที่ "YOUR DOCUMENTS DIRECTORY" ด้วยเส้นทางที่เหมาะสม

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: สร้างเอกสารใหม่
 จากนั้นคุณสามารถสร้างวัตถุเอกสารใหม่โดยใช้`Document` ระดับ.

```csharp
Document doc = new Document();
```

## ขั้นตอนที่ 3: เพิ่มหน้าใหม่ลงในเอกสาร
 ตอนนี้คุณสามารถเพิ่มหน้าใหม่ลงในวัตถุเอกสารได้โดยใช้`Add()` วิธีการของ`Pages` ระดับ.

```csharp
Page page = doc.Pages.Add();
```

## ขั้นตอนที่ 4: สร้างวัตถุสิ่งประดิษฐ์พื้นหลัง
จากนั้นคุณสามารถสร้างวัตถุ BackgroundArtifact ใหม่เพื่อตั้งค่ารูปภาพพื้นหลังได้

```csharp
BackgroundArtifact background = new BackgroundArtifact();
background.BackgroundImage = File.OpenRead(dataDir + "aspose-total-for-net.jpg");
```

## ขั้นตอนที่ 5: เพิ่มพื้นหลังให้กับหน้า
จากนั้นคุณสามารถเพิ่มวัตถุ BackgroundArtifact ลงในคอลเล็กชันสิ่งประดิษฐ์ของหน้าได้โดยใช้`Artifacts` ทรัพย์สินของ`Page` ระดับ.

```csharp
page. Artifacts. Add(background);
```

## ขั้นตอนที่ 6: บันทึกเอกสาร PDF
 สุดท้ายคุณสามารถบันทึกเอกสาร PDF ลงในไฟล์โดยใช้`Save()` วิธีการของ`Document`คลาส โปรดระบุเส้นทางและชื่อไฟล์ให้ถูกต้อง

```csharp
doc.Save(dataDir + "ImageAsBackground_out.pdf");
```

### ตัวอย่างโค้ดต้นฉบับสำหรับรูปภาพเป็นพื้นหลังโดยใช้ Aspose.PDF สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// สร้างวัตถุเอกสารใหม่
Document doc = new Document();
// เพิ่มหน้าใหม่ลงในวัตถุเอกสาร
Page page = doc.Pages.Add();
// สร้างวัตถุสิ่งประดิษฐ์พื้นหลัง
BackgroundArtifact background = new BackgroundArtifact();
// ระบุรูปภาพสำหรับวัตถุ backgroundartifact
background.BackgroundImage = File.OpenRead(dataDir + "aspose-total-for-net.jpg");
// เพิ่มสิ่งประดิษฐ์พื้นหลังลงในคอลเล็กชันสิ่งประดิษฐ์ของหน้า
page.Artifacts.Add(background);
dataDir = dataDir + "ImageAsBackground_out.pdf";
// บันทึกเอกสาร
doc.Save(dataDir);
System.Console.WriteLine("\nImage as page background added successfully.\nFile saved at " + dataDir);

```
