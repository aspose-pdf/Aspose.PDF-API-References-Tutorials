ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับขั้นตอนการเพิ่มรูปภาพลงในเซลล์ตารางโดยใช้ Aspose.PDF สำหรับ .NET โค้ดต้นฉบับ C# ที่ให้มาจะสาธิตวิธีการบรรลุฟังก์ชันนี้ หากปฏิบัติตามขั้นตอนที่ระบุไว้ด้านล่าง คุณจะสามารถรวมรูปภาพลงในเซลล์ตารางได้อย่างมีประสิทธิภาพ

ก่อนที่จะเจาะลึกโค้ด ให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.PDF สำหรับ .NET และอ้างอิงในโปรเจ็กต์ของคุณแล้ว

## ขั้นตอนที่ 1: การตั้งค่าเอกสาร

 ในการเริ่มต้น เราต้องสร้างอินสแตนซ์ใหม่ของ`Document` คลาสจากเนมสเปซ Aspose.Pdf คลาสนี้แสดงถึงเอกสาร PDF

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างอินสแตนซ์ของวัตถุเอกสาร
Document pdfDocument = new Document();
```

## ขั้นตอนที่ 2: การสร้างหน้า

ต่อไปเราต้องเพิ่มหน้าลงในเอกสาร PDF โดยหน้าดังกล่าวจะทำหน้าที่เป็นตัวบรรจุตารางและองค์ประกอบอื่นๆ

```csharp
// สร้างหน้าในเอกสาร pdf
Page sec1 = pdfDocument.Pages.Add();
```

## ขั้นตอนที่ 3: การเพิ่มตาราง

 ในขั้นตอนนี้เราจะสร้างตารางโดยสร้างอินสแตนซ์`Table` คลาสจากเนมสเปซ Aspose.Pdf

```csharp
// สร้างอินสแตนซ์ของวัตถุตาราง
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

## ขั้นตอนที่ 4: ตั้งค่าเส้นขอบเซลล์เริ่มต้น

 เพื่อให้แน่ใจว่ามีความสอดคล้องกัน เราสามารถตั้งค่าเส้นขอบเซลล์เริ่มต้นโดยใช้`DefaultCellBorder` คุณสมบัติของตาราง`BorderInfo` วัตถุ.

```csharp
// ตั้งค่าเส้นขอบเซลล์เริ่มต้นโดยใช้ BorderInfo object
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
```

## ขั้นตอนที่ 5: การตั้งค่าความกว้างของคอลัมน์

 เพื่อกำหนดความกว้างของแต่ละคอลัมน์ในตาราง เราสามารถตั้งค่าได้`ColumnWidths` คุณสมบัติ ระบุความกว้างเป็นสตริงโดยมีค่าคั่นด้วยช่องว่าง

```csharp
// ตั้งค่าด้วยความกว้างของคอลัมน์ของตาราง
tab1.ColumnWidths = "100 100 120";
```

## ขั้นตอนที่ 6: การเพิ่มรูปภาพลงในเซลล์ตาราง

ตอนนี้มาถึงส่วนที่น่าตื่นเต้น นั่นคือการเพิ่มรูปภาพลงในเซลล์ตาราง โดยทำตามขั้นตอนย่อยต่อไปนี้:

## ขั้นตอนที่ 6.1: การสร้างวัตถุรูปภาพ

 สร้างอินสแตนซ์ของ`Image` คลาสจากเนมสเปซ Aspose.Pdf ตั้งค่า`File` คุณสมบัติไปยังเส้นทางของไฟล์รูปภาพที่คุณต้องการเพิ่ม

```csharp
// สร้างวัตถุภาพ
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose.jpg";
```

## ขั้นตอนที่ 6.2: การสร้างแถวและเซลล์

ในการเพิ่มรูปภาพลงในตาราง เราต้องสร้างแถวและเซลล์ที่จำเป็นก่อน

```csharp
// สร้างแถวในตาราง
Aspose.Pdf.Row row1 = tab1.Rows.Add();

// เพิ่มเซลล์ข้อความลงในแถว
row1.Cells.Add("Sample text in cell");

// เพิ่มเซลล์ที่เก็บรูปภาพ
Aspose.Pdf.Cell cell2 = row1.Cells.Add();
```

## ขั้นตอนที่ 6.3: การเพิ่มรูปภาพลงในเซลล์ตาราง

ในที่สุดเราก็สามารถเพิ่มรูปภาพลงในเซลล์ตารางได้โดยการเพิ่มเป็นย่อหน้าภายในเซลล์ได้

```csharp
//เพิ่มรูปภาพลงในเซลล์ตาราง
cell2.Paragraphs.Add(img);
```

## ขั้นตอนที่ 6.4: การเพิ่มเซลล์เพิ่มเติม

หลังจากเพิ่มเซลล์รูปภาพแล้ว เราสามารถเพิ่มเซลล์เพิ่มเติมลงในแถวได้หากจำเป็น

```csharp
// เพิ่มเซลล์อื่นลงในแถว
row1.Cells.Add("Previous cell with image");

// ปรับการจัดตำแหน่งแนวตั้งของเซลล์ที่สาม
row1.Cells[2].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
```

## ขั้นตอนที่ 7: การบันทึกเอกสาร

 ในที่สุดเราสามารถบันทึกเอกสารที่แก้ไขแล้วไปยังตำแหน่งที่ระบุโดยใช้`Save` วิธี.

```csharp
//บันทึกเอกสาร
pdfDocument.Save(dataDir + "AddImageInTableCell_out.pdf");
```

ขอแสดงความยินดี! คุณได้เรียนรู้วิธีการเพิ่มรูปภาพลงในเซลล์ตารางโดยใช้ Aspose.PDF สำหรับ .NET สำเร็จแล้ว โปรดอย่าลังเลที่จะสำรวจตัวเลือกการปรับแต่งเพิ่มเติมและรวมฟังก์ชันนี้เข้ากับโครงการของคุณ

### ตัวอย่างโค้ดต้นฉบับสำหรับเพิ่มรูปภาพลงในเซลล์ตารางโดยใช้ Aspose.PDF สำหรับ .NET

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// สร้างอินสแตนซ์ของวัตถุเอกสาร
Document pdfDocument = new Document();
// สร้างหน้าในเอกสาร pdf
Page sec1 = pdfDocument.Pages.Add();
// สร้างอินสแตนซ์ของวัตถุตาราง
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
// เพิ่มตารางในคอลเลกชันย่อหน้าของหน้าที่ต้องการ
sec1.Paragraphs.Add(tab1);
// ตั้งค่าเส้นขอบเซลล์เริ่มต้นโดยใช้ BorderInfo object
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
// ตั้งค่าด้วยความกว้างของคอลัมน์ของตาราง
tab1.ColumnWidths = "100 100 120";
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose.jpg";
//สร้างแถวในตารางแล้วสร้างเซลล์ในแถว
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("Sample text in cell");
// เพิ่มเซลล์ที่เก็บรูปภาพ
Aspose.Pdf.Cell cell2 = row1.Cells.Add();
//เพิ่มรูปภาพลงในเซลล์ตาราง
cell2.Paragraphs.Add(img);
row1.Cells.Add("Previous cell with image");
row1.Cells[2].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
//บันทึกเอกสาร
pdfDocument.Save(dataDir + "AddImageInTableCell_out.pdf");
```
