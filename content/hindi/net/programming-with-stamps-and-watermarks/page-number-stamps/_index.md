इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में पेज नंबर स्टैम्प जोड़ने के तरीके के बारे में चरण दर चरण मार्गदर्शन करेंगे। हम मौजूदा PDF दस्तावेज़ को खोलने, पेज नंबर स्टैम्प बनाने, उसके गुण सेट करने और उसे PDF फ़ाइल में किसी विशिष्ट पृष्ठ पर जोड़ने के लिए दिए गए C# स्रोत कोड का उपयोग करेंगे।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- एक स्थापित .NET विकास वातावरण.
- .NET के लिए Aspose.PDF लाइब्रेरी डाउनलोड की गई और आपके प्रोजेक्ट में संदर्भित की गई।

## चरण 2: मौजूदा PDF दस्तावेज़ लोड करना

पहला कदम मौजूदा PDF दस्तावेज़ को अपने प्रोजेक्ट में लोड करना है। यहाँ बताया गया है कि कैसे:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// मौजूदा PDF दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir + "PageNumberStamp.pdf");
```

"आपके दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें जहां आपका PDF दस्तावेज़ स्थित है।

## चरण 3: पेज नंबरिंग स्टैम्प बनाना और कॉन्फ़िगर करना

अब जब पीडीएफ दस्तावेज़ लोड हो गया है, तो हम एक पेज नंबरिंग बफर बना सकते हैं और इसे अपनी ज़रूरतों के हिसाब से कॉन्फ़िगर कर सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// पृष्ठ संख्या बफ़र बनाएँ
PageNumberStamp pageNumberStamp = new PageNumberStamp();

// परिभाषित करें कि बफर पृष्ठभूमि में है या नहीं
pageNumberStamp.Background = false;

// पृष्ठ क्रमांकन बफर का प्रारूप
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;

// पृष्ठ क्रमांकन बफर का निचला मार्जिन
pageNumberStamp.BottomMargin = 10;

// पृष्ठ क्रमांकन बफर का क्षैतिज संरेखण
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;

// पृष्ठ क्रमांकन की आरंभ संख्या
pageNumberStamp.StartingNumber = 1;

// पृष्ठ संख्या बफ़र पाठ गुण सेट करें
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

उपरोक्त कोड पृष्ठ संख्या प्रारूप, निचला मार्जिन, क्षैतिज संरेखण, प्रारंभिक संख्या और पाठ गुण जैसे गुणों के साथ एक पृष्ठ संख्या स्टाम्प बनाता है।

## चरण 4: किसी विशिष्ट पृष्ठ पर पृष्ठ संख्या स्टाम्प जोड़ना

एक बार पेज नंबर स्टैम्प कॉन्फ़िगर हो जाने के बाद, हम इसे PDF दस्तावेज़ के किसी विशिष्ट पेज पर जोड़ सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// किसी विशिष्ट पृष्ठ पर पृष्ठ संख्या बफ़र जोड़ें
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
```

उपरोक्त कोड पीडीएफ दस्तावेज़ के पहले पृष्ठ पर पृष्ठ संख्या स्टैम्प जोड़ता है। आप आवश्यकतानुसार पृष्ठ संख्या बदल सकते हैं।

## चरण 5: संशोधित PDF दस्तावेज़ को सहेजना

एक बार जब पेज नंबर स्टैम्प को PDF दस्तावेज़ में जोड़ दिया जाता है, तो हम संशोधित PDF दस्तावेज़ को सहेज सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// संशोधित PDF दस्तावेज़ सहेजें
pdfDocument.Save(dataDir + "PageNumberStamp_out.pdf");
```

"आपकी दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें जहां आप संपादित पीडीएफ दस्तावेज़ को सहेजना चाहते हैं।

### .NET के लिए Aspose.PDF का उपयोग करके पेज नंबर स्टैम्प के लिए नमूना स्रोत कोड 
```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir+ "PageNumberStamp.pdf");

// पेज नंबर स्टाम्प बनाएं
PageNumberStamp pageNumberStamp = new PageNumberStamp();

// क्या स्टाम्प पृष्ठभूमि है
pageNumberStamp.Background = false;
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;
pageNumberStamp.BottomMargin = 10;
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;
pageNumberStamp.StartingNumber = 1;

// पाठ गुण सेट करें
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;

// किसी विशेष पृष्ठ पर स्टाम्प जोड़ें
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
dataDir = dataDir + "PageNumberStamp_out.pdf";

// आउटपुट दस्तावेज़ सहेजें
pdfDocument.Save(dataDir);
Console.WriteLine("\nPage number stamp added successfully.\nFile saved at " + dataDir);

```

## निष्कर्ष

बधाई हो! आपने सीख लिया है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में पेज नंबर स्टैम्प कैसे जोड़ें। अब आप स्पष्ट और जानकारीपूर्ण पेज नंबर जोड़कर अपने PDF दस्तावेज़ों को निजीकृत कर सकते हैं।

### पीडीएफ फाइल में पेज नंबर स्टैम्प के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पेज नंबर स्टैम्प क्या है और इसका उपयोग पीडीएफ फाइल में पेज नंबर जोड़ने के लिए कैसे किया जाता है?

उत्तर: पेज नंबर स्टैम्प Aspose.PDF में एक ऐसी सुविधा है जो आपको PDF दस्तावेज़ के विशिष्ट पृष्ठों पर गतिशील पृष्ठ संख्याएँ जोड़ने की अनुमति देती है। इस ट्यूटोरियल में, इसे PageNumberStamp ऑब्जेक्ट बनाकर, उसके गुणों को कॉन्फ़िगर करके, और उसे निर्दिष्ट पृष्ठ पर जोड़कर प्राप्त किया जाता है।

#### प्रश्न: उपलब्ध कराया गया C# स्रोत कोड, PDF फ़ाइल में पृष्ठ संख्या स्टैम्प जोड़ने का कार्य किस प्रकार पूरा करता है?

उत्तर: कोड दर्शाता है कि मौजूदा PDF दस्तावेज़ को कैसे लोड किया जाए, पेजनंबरस्टैम्प कैसे बनाया जाए, विभिन्न गुण (जैसे कि प्रारूप, फ़ॉन्ट, संरेखण, आदि) सेट करें और फिर किसी विशिष्ट पृष्ठ पर स्टैम्प जोड़ें। स्टैम्प स्वचालित रूप से कुल पृष्ठ संख्या की गणना करता है और सही पृष्ठ संख्याएँ सम्मिलित करता है।

#### प्रश्न: क्या मैं पृष्ठ संख्या के स्वरूप को अनुकूलित कर सकता हूँ, जैसे फ़ॉन्ट शैली, रंग और आकार?

उत्तर: बिल्कुल, आप फ़ॉन्ट, फ़ॉन्ट आकार, फ़ॉन्ट शैली (बोल्ड, इटैलिक, आदि) और पाठ रंग जैसे गुणों को समायोजित करके पृष्ठ संख्या स्टाम्प के स्वरूप को अनुकूलित कर सकते हैं।

#### प्रश्न: क्या किसी पीडीएफ दस्तावेज़ के एकाधिक पृष्ठों पर पृष्ठ संख्या स्टाम्प जोड़ना संभव है?

उत्तर: हां, आप अनेक PageNumberStamp ऑब्जेक्ट बनाकर तथा प्रत्येक को इच्छित पृष्ठों में जोड़कर अनेक पृष्ठों में पृष्ठ संख्या स्टैम्प जोड़ सकते हैं।

#### प्रश्न: क्या मैं चुन सकता हूं कि पृष्ठ संख्या स्टाम्प पृष्ठ की सामग्री के भाग के रूप में प्रदर्शित हो या पृष्ठभूमि तत्व के रूप में?

 उत्तर: हां, आप यह नियंत्रित कर सकते हैं कि पृष्ठ संख्या स्टाम्प पृष्ठ की सामग्री के भाग के रूप में दिखाई दे या पृष्ठभूमि तत्व के रूप में।`Background` पेजनम्बरस्टाम्प की संपत्ति.

#### प्रश्न: मैं कुल पृष्ठ संख्या सहित पृष्ठ संख्या का प्रारूप कैसे निर्दिष्ट करूं?

 उत्तर: कोड का उपयोग करता है`Format`पेज नंबर का प्रारूप निर्दिष्ट करने के लिए PageNumberStamp की संपत्ति। मैक्रो "# of" का उपयोग कुल पृष्ठ संख्या को दर्शाने के लिए किया जाता है।

#### प्रश्न: यदि मैं एक ही पृष्ठ संख्या वाली स्टाम्प को अनेक पृष्ठों पर लगाऊं तो क्या होगा?

उत्तर: एक ही PageNumberStamp इंस्टेंस को कई पेजों में जोड़ने से प्रत्येक पेज के लिए सही पेज नंबर प्रदर्शित होंगे। स्टैम्प स्वचालित रूप से पेज नंबर और कुल पेज संख्या को समायोजित करता है।

#### प्रश्न: क्या मैं किसी पीडीएफ दस्तावेज़ के शीर्षलेख या पादलेख अनुभाग में पृष्ठ संख्या स्टाम्प जोड़ सकता हूँ?

उत्तर: जबकि पेजनंबरस्टैम्प को आमतौर पर सीधे पेज की सामग्री में जोड़ा जाता है, आप उन्हें हेडर या फुटर अनुभागों में रखने के लिए फ्लोटिंगबॉक्स या अन्य तकनीकों का उपयोग कर सकते हैं।

#### प्रश्न: मैं पृष्ठ पर पृष्ठ संख्या स्टाम्प की स्थिति कैसे निर्दिष्ट करूँ?

 उत्तर:`BottomMargin` और`HorizontalAlignment` पेजनंबरस्टाम्प के गुण आपको पृष्ठ के भीतर स्टाम्प की स्थिति को नियंत्रित करने की अनुमति देते हैं।

#### प्रश्न: यदि मैं पृष्ठ क्रमांकन 1 के बजाय किसी अन्य संख्या से प्रारंभ करना चाहूं तो क्या होगा?

 उत्तर: आप सेट कर सकते हैं`StartingNumber`प्रारंभिक पृष्ठ संख्या निर्दिष्ट करने के लिए PageNumberStamp की संपत्ति का उपयोग करें।
