इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF के साथ फ़्लोटिंगबॉक्स का उपयोग करके PDF दस्तावेज़ के हेडर और फ़ुटर में पेज नंबर जोड़ने के तरीके के बारे में चरण दर चरण मार्गदर्शन करेंगे। हम PDF दस्तावेज़ बनाने, पेज जोड़ने, फ़्लोटिंगबॉक्स बनाने, उसकी स्थिति सेट करने और उसमें पेज नंबर जोड़ने के लिए दिए गए C# स्रोत कोड का उपयोग करेंगे, फिर संशोधित PDF दस्तावेज़ को सहेजेंगे।

## चरण 1: वातावरण की स्थापना

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- एक स्थापित .NET विकास वातावरण.
- .NET के लिए Aspose.PDF लाइब्रेरी डाउनलोड की गई और आपके प्रोजेक्ट में संदर्भित की गई।

## चरण 2: पीडीएफ दस्तावेज़ बनाना और एक पृष्ठ जोड़ना

पहला कदम पीडीएफ दस्तावेज़ का एक उदाहरण बनाना और उसमें एक पृष्ठ जोड़ना है। यहाँ बताया गया है कि कैसे:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// पीडीएफ दस्तावेज़ को तत्काल बनाना
Aspose.Pdf.Document pdf = new Aspose.Pdf.Document();

// PDF दस्तावेज़ में पृष्ठ जोड़ें
Aspose.Pdf.Page page = pdf.Pages.Add();
```

"आपकी दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से प्रतिस्थापित करना सुनिश्चित करें जहां आप पीडीएफ दस्तावेज़ को सहेजना चाहते हैं।

## चरण 3: फ्लोटिंगबॉक्स बनाना और पृष्ठ संख्या जोड़ना

अब जब पेज को PDF दस्तावेज़ में जोड़ दिया गया है, तो हम एक फ़्लोटिंगबॉक्स बना सकते हैं, इसकी स्थिति निर्धारित कर सकते हैं, और इसमें पेज नंबर जोड़ सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// 140 की चौड़ाई और 80 की ऊंचाई वाला एक फ्लोटिंगबॉक्स बनाएं
Aspose.Pdf.FloatingBox box1 = new Aspose.Pdf.FloatingBox(140, 80);

// पैराग्राफ़ की बाईं स्थिति सेट करें
box1. Left = 2;

// पैराग्राफ़ का शीर्ष स्थान निर्धारित करें
box1. Top = 10;

// फ़्लोटिंगबॉक्स में पृष्ठ संख्या जोड़ें
box1.Paragraphs.Add(new Aspose.Pdf.Text.TextFragment("Page: ($p/ $P )"));

// पेज पर फ़्लोटिंगबॉक्स जोड़ें
page.Paragraphs.Add(box1);
```

ऊपर दिया गया कोड 140 की चौड़ाई और 80 की ऊँचाई वाला एक फ़्लोटिंगबॉक्स बनाता है। इसके बाद, हम बाएँ और शीर्ष मान निर्दिष्ट करके इसकी स्थिति निर्धारित करते हैं। अंत में, हम फ़्लोटिंगबॉक्स में एक टेक्स्टफ़्रेगमेंट का उपयोग करके पेज नंबर जोड़ते हैं जिसमें सिंटैक्स "($p/ $P )" होता है जिसे वर्तमान पेज नंबर और कुल पेजों की संख्या से बदल दिया जाएगा।

## चरण 4: संशोधित PDF दस्तावेज़ को सहेजना

एक बार जब फ्लोटिंगबॉक्स का उपयोग करके हेडर या फ़ुटर में पेज नंबर जोड़ दिया जाता है, तो हम संशोधित पीडीएफ दस्तावेज़ को सहेज सकते हैं। यहाँ बताया गया है कि कैसे:

```csharp
// संशोधित PDF दस्तावेज़ सहेजें
pdf.Save(dataDir + "PageNumberinHeaderFooterUsingFloatingBox_out.pdf");
```

उपरोक्त कोड संपादित पीडीएफ दस्तावेज़ को निर्दिष्ट निर्देशिका में सहेजता है।

### .NET के लिए Aspose.PDF का उपयोग करके फ्लोटिंग बॉक्स का उपयोग करके हेडर फ़ुटर में पेज नंबर के लिए नमूना स्रोत कोड 
```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ उदाहरण को तत्कालित करें
Aspose.Pdf.Document pdf = new Aspose.Pdf.Document();

// पीडीएफ दस्तावेज़ में एक पृष्ठ जोड़ें
Aspose.Pdf.Page page = pdf.Pages.Add();

// फ़्लोटिंगबॉक्स क्लास का एक नया उदाहरण आरंभ करता है
Aspose.Pdf.FloatingBox box1 = new Aspose.Pdf.FloatingBox(140, 80);

// फ़्लोट मान जो पैराग्राफ़ की बाईं स्थिति को इंगित करता है
box1.Left = 2;

// फ्लोट मान जो पैराग्राफ़ की शीर्ष स्थिति को इंगित करता है
box1.Top = 10;

// फ्लोटिंगबॉक्स के पैराग्राफ संग्रह में मैक्रोज़ जोड़ें
box1.Paragraphs.Add(new Aspose.Pdf.Text.TextFragment("Page: ($p/ $P )"));

// पृष्ठ पर फ़्लोटिंगबॉक्स जोड़ें
page.Paragraphs.Add(box1);

// दस्तावेज़ सहेजें
pdf.Save(dataDir + "PageNumberinHeaderFooterUsingFloatingBox_out.pdf");

```

## निष्कर्ष

बधाई हो! आपने सीखा है कि .NET के लिए Aspose.PDF के साथ फ़्लोटिंगबॉक्स का उपयोग करके PDF दस्तावेज़ के हेडर और फ़ुटर में पेज नंबर कैसे जोड़ें। अब आप पेज नंबर जैसी गतिशील जानकारी जोड़कर अपने हेडर और फ़ुटर को कस्टमाइज़ कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: फ्लोटिंगबॉक्स क्या है, और इसका उपयोग पीडीएफ दस्तावेज़ के हेडर या फ़ूटर में पृष्ठ संख्या जोड़ने के लिए कैसे किया जाता है?

उत्तर: फ़्लोटिंगबॉक्स Aspose.PDF में एक बहुमुखी लेआउट तत्व है जो टेक्स्ट और छवियों सहित विभिन्न सामग्री को होल्ड कर सकता है। इस ट्यूटोरियल में, इसका उपयोग पेज नंबर के लिए एक कंटेनर बनाने के लिए किया जाता है, जिससे आप हेडर या फ़ुटर में वर्तमान पेज नंबर और कुल पेज काउंट को गतिशील रूप से सम्मिलित कर सकते हैं।

#### प्रश्न: प्रदान किया गया C# स्रोत कोड फ्लोटिंगबॉक्स का उपयोग करके पृष्ठ संख्या जोड़ने का कार्य कैसे करता है?

उत्तर: कोड स्निपेट यह दर्शाता है कि PDF दस्तावेज़ कैसे बनाएँ, पेज कैसे जोड़ें, फ़्लोटिंगबॉक्स कैसे बनाएँ, पेज के भीतर उसकी स्थिति कैसे सेट करें, और TextFragment का उपयोग करके पेज नंबर कैसे डालें। TextFragment में सिंटैक्स "($p/ $P )" को वर्तमान पेज नंबर और कुल पेज काउंट से बदल दिया जाता है।

#### प्रश्न: क्या मैं फ्लोटिंगबॉक्स का उपयोग करके जोड़े गए पृष्ठ संख्या के स्वरूप और स्वरूपण को अनुकूलित कर सकता हूं?

उत्तर: हाँ, आप फ़्लोटिंगबॉक्स के भीतर टेक्स्टफ़्रेगमेंट के गुणों को संशोधित करके पृष्ठ संख्या की उपस्थिति को अनुकूलित कर सकते हैं। आप फ़ॉन्ट आकार, रंग, शैली, संरेखण और अन्य स्वरूपण विकल्प बदल सकते हैं।

#### प्रश्न: क्या इसी प्रकार के दृष्टिकोण का उपयोग करके हेडर या फुटर में दिनांक और समय जैसे विभिन्न गतिशील तत्वों को जोड़ना संभव है?

उत्तर: बिल्कुल, आप फ़्लोटिंगबॉक्स के भीतर टेक्स्टफ़्रेगमेंट सामग्री को संशोधित करके दिनांक, समय, दस्तावेज़ मेटाडेटा या कस्टम टेक्स्ट जैसे विभिन्न गतिशील तत्व जोड़ सकते हैं। आप पेज नंबर के लिए "($p/ $P )" या वर्तमान तिथि के लिए "($date)" जैसे मैक्रोज़ का उपयोग कर सकते हैं।

#### प्रश्न: मैं हेडर या फ़ूटर अनुभाग में फ़्लोटिंगबॉक्स की स्थिति कैसे निर्दिष्ट करूँ?
 उत्तर: प्रदान किया गया कोड फ्लोटिंगबॉक्स की स्थिति निर्धारित करता है`Left` और`Top` आप इन मानों को समायोजित कर सकते हैं ताकि फ्लोटिंगबॉक्स को हेडर या फ़ुटर अनुभाग में इच्छानुसार स्थान दिया जा सके।

#### प्रश्न: क्या मैं हेडर या फ़ूटर में पृष्ठ संख्या के लिए भिन्न फ़ॉन्ट या शैली का उपयोग कर सकता हूँ?

उत्तर: हां, आप फ्लोटिंगबॉक्स के भीतर टेक्स्टफ़्रेगमेंट गुणों को संशोधित करके पृष्ठ संख्या पाठ के फ़ॉन्ट, शैली और अन्य स्वरूपण गुणों को अनुकूलित कर सकते हैं।

#### प्रश्न: यदि फ्लोटिंगबॉक्स में सामग्री उसके आयाम से अधिक हो जाए तो क्या होगा?

उत्तर: यदि फ़्लोटिंगबॉक्स के भीतर की सामग्री उसके आयामों से अधिक है, तो उसे काटा जा सकता है या लेआउट संबंधी समस्याएँ उत्पन्न हो सकती हैं। सुनिश्चित करें कि फ़्लोटिंगबॉक्स के आयाम सामग्री को समायोजित करने के लिए उपयुक्त हैं, और यदि आवश्यक हो तो पेज लेआउट को समायोजित करने पर विचार करें।

#### प्रश्न: क्या एक ही पृष्ठ के शीर्षलेख या पादलेख में भिन्न सामग्री वाले अनेक फ़्लोटिंगबॉक्स जोड़ना संभव है?

उत्तर: हां, आप अलग-अलग फ्लोटिंगबॉक्स इंस्टैंस बनाकर और उन्हें पृष्ठ के पैराग्राफ संग्रह में जोड़कर एक ही पृष्ठ के शीर्षलेख या पादलेख में भिन्न सामग्री वाले अनेक फ्लोटिंगबॉक्स जोड़ सकते हैं।

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ के अन्य अनुभागों, जैसे मुख्य भाग या मार्जिन में सामग्री जोड़ने के लिए फ्लोटिंगबॉक्स पद्धति का उपयोग कर सकता हूं?

उत्तर: जबकि फ्लोटिंगबॉक्स का उपयोग सामान्यतः हेडर और फुटर के लिए किया जाता है, आप उनका उपयोग पीडीएफ दस्तावेज़ के अन्य अनुभागों, जैसे मुख्य भाग या मार्जिन में सामग्री जोड़ने के लिए भी कर सकते हैं, उन्हें पृष्ठ के भीतर तदनुसार स्थान देकर।
