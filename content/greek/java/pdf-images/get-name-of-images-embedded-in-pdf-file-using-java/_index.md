## Φόρτωση αρχείου PDF

 Σε αυτό το βήμα, θα γράψουμε κώδικα Java για να φορτώσουμε ένα αρχείο PDF στην εφαρμογή μας. Θα πρέπει να αντικαταστήσετε`"input.pdf"` με τη διαδρομή προς το αρχείο PDF σας.

```java
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document("input.pdf");
```

## Εξαγωγή ενσωματωμένων εικόνων

Για να εξαγάγουμε ενσωματωμένες εικόνες, πρέπει να έχουμε πρόσβαση στους πόρους του εγγράφου PDF. Δείτε πώς μπορείτε να το κάνετε:

```java
com.aspose.pdf.Page page = pdfDocument.getPages().get_Item(1); // Αντικαταστήστε με τον επιθυμητό αριθμό σελίδας
com.aspose.pdf.XImageCollection images = page.getResources().getImages();
```

## Γράψιμο κώδικα Java για εξαγωγή εικόνας

Τώρα, ας περιηγηθούμε στις ενσωματωμένες εικόνες και ας εξαγάγουμε τα ονόματά τους:

```java
for (com.aspose.pdf.XImage image : images) {
    System.out.println("Image Name: " + image.getName());
}
```
