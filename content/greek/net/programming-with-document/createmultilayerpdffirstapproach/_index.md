Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε στη διαδικασία δημιουργίας ενός αρχείου PDF πολλαπλών επιπέδων χρησιμοποιώντας την πρώτη προσέγγιση με το Aspose.PDF για .NET. Αυτή η προσέγγιση σάς επιτρέπει να προσθέσετε πολλαπλά επίπεδα στο αρχείο PDF σας. Ακολουθήστε τον παρακάτω βήμα προς βήμα οδηγό:

## Βήμα 1: Αρχικοποιήστε το έγγραφο PDF

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Aspose.Pdf.Document pdf = new Aspose.Pdf.Document();
```

## Βήμα 2: Προσθέστε μια νέα σελίδα στο έγγραφο

```csharp
Aspose.Pdf.Page sec1 = pdf.Pages.Add();
```

## Βήμα 3: Προσθέστε ένα τμήμα κειμένου στη σελίδα

```csharp
Aspose.Pdf.Text.TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);
```

## Βήμα 4: Προσαρμόστε το τμήμα κειμένου

```csharp
t1.Text = "paragraph 3 segment 1";
t1.TextState.ForegroundColor = Color.Red;
t1.TextState.FontSize = 12;
```

## Βήμα 5: Προσθέστε μια εικόνα στη σελίδα

```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = dataDir + "test_image.png";
```

## Βήμα 6: Προσθέστε ένα αιωρούμενο πλαίσιο στη σελίδα

```csharp
Aspose.Pdf.FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
sec1.Paragraphs.Add(box1);

box1.Left = -4;
box1.Top = -4;
box1.Paragraphs.Add(image1);
```

## Βήμα 7: Αποθηκεύστε το έγγραφο PDF που προκύπτει

```csharp
pdf.Save(dataDir + "CreateMultiLayerPdf_out.pdf");
```

Συγχαρητήρια! Έχετε δημιουργήσει επιτυχώς ένα έγγραφο PDF πολλαπλών επιπέδων χρησιμοποιώντας την πρώτη προσέγγιση με το Aspose.PDF για .NET.

### Παράδειγμα πηγαίου κώδικα για Δημιουργία Πρώτης Προσέγγισης PDF πολλαπλών επιπέδων χρησιμοποιώντας Aspose.PDF για .NET:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

Aspose.Pdf.Document pdf = new Aspose.Pdf.Document();
Aspose.Pdf.Page sec1 = pdf.Pages.Add();
Aspose.Pdf.Text.TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
sec1.Paragraphs.Add(t1);

t1.Text = "paragraph 3 segment 1";
t1.TextState.ForegroundColor = Color.Red;
t1.TextState.FontSize = 12;

Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = dataDir + "test_image.png";

Aspose.Pdf.FloatingBox box1 = new Aspose.Pdf.FloatingBox(117, 21);
sec1.Paragraphs.Add(box1);

box1.Left = -4;
box1.Top = -4;
box1.Paragraphs.Add(image1);

pdf.Save(dataDir + "CreateMultiLayerPdf_out.pdf");
```
