ในบทช่วยสอนนี้ เราจะแนะนำคุณทีละขั้นตอนเกี่ยวกับวิธีการเพิ่มรูปภาพในส่วนท้ายของเอกสาร PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเปิดเอกสาร PDF ที่มีอยู่ สร้างบัฟเฟอร์รูปภาพ ตั้งค่าคุณสมบัติ และเพิ่มลงในทุกหน้าของเอกสาร PDF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 2: โหลดเอกสาร PDF ที่มีอยู่

ขั้นตอนแรกคือโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร PDF ที่มีอยู่
Document pdfDocument = new Document(dataDir + "ImageInFooter.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 3: การสร้างและเพิ่มรูปภาพในส่วนท้าย

เมื่อโหลดเอกสาร PDF เสร็จแล้ว เราสามารถสร้างแสตมป์ภาพและเพิ่มลงในทุกหน้าของเอกสารได้ ดังต่อไปนี้:

```csharp
// สร้างเฟรมบัฟเฟอร์
ImageStamp imageStamp = new ImageStamp(dataDir + "aspose-logo.jpg");

// ตั้งค่าคุณสมบัติบัฟเฟอร์ภาพ
imageStamp.BottomMargin = 10;
imageStamp.HorizontalAlignment = HorizontalAlignment.Center;
imageStamp.VerticalAlignment = VerticalAlignment.Bottom;

//เพิ่มบัฟเฟอร์รูปภาพให้กับทุกหน้า
foreach(Page page in pdfDocument.Pages)
{
     page.AddStamp(imageStamp);
}
```

โค้ดด้านบนจะสร้างบัฟเฟอร์รูปภาพจากไฟล์ "aspose-logo.jpg" และตั้งค่าคุณสมบัติ เช่น ขอบล่าง การจัดแนวแนวนอนและแนวตั้ง จากนั้นบัฟเฟอร์รูปภาพจะถูกเพิ่มลงในทุกหน้าของเอกสาร PDF

## ขั้นตอนที่ 4: บันทึกเอกสาร PDF ที่แก้ไขแล้ว

เมื่อเพิ่มรูปภาพลงในส่วนท้ายแล้ว เราจะสามารถบันทึกเอกสาร PDF ที่แก้ไขแล้วได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสาร PDF ที่แก้ไขแล้ว
pdfDocument.Save(dataDir + "ImageInFooter_out.pdf");
```

โค้ดด้านบนจะบันทึกเอกสาร PDF ที่แก้ไขแล้วไปยังไดเร็กทอรีที่ระบุ

### ตัวอย่างโค้ดต้นฉบับสำหรับ Image In Footer โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "ImageInFooter.pdf");

// สร้างส่วนท้าย
ImageStamp imageStamp = new ImageStamp(dataDir+ "aspose-logo.jpg");

// ตั้งค่าคุณสมบัติของแสตมป์
imageStamp.BottomMargin = 10;
imageStamp.HorizontalAlignment = HorizontalAlignment.Center;
imageStamp.VerticalAlignment = VerticalAlignment.Bottom;

// เพิ่มส่วนท้ายในทุกหน้า
foreach (Page page in pdfDocument.Pages)
{
	page.AddStamp(imageStamp);
}
dataDir = dataDir + "ImageInFooter_out.pdf";

// บันทึกไฟล์ PDF อัพเดต
pdfDocument.Save(dataDir);
Console.WriteLine("\nImage in footer added successfully.\nFile saved at " + dataDir);
```
