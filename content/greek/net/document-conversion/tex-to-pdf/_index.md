## Βήμα 1: Φόρτωση του αρχείου TeX
 Το πρώτο βήμα είναι να φορτώσετε το αρχείο TeX σε ένα`Document` αντικείμενο χρησιμοποιώντας την επιλογή φόρτωσης TeX (`LatexLoadOptions`). Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
// Διαδρομή στον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Αντικείμενο επιλογής Instantiate Latex Load
LatexLoadOptions Latexoptions = new LatexLoadOptions();

// Δημιουργία αντικειμένου εγγράφου
Aspose.Pdf.Document doc = new Aspose.Pdf.Document(dataDir + "samplefile.tex", Latexoptions);
```

 Φροντίστε να αντικαταστήσετε`"YOUR DOCUMENTS DIRECTORY"` με τον πραγματικό κατάλογο όπου βρίσκεται το αρχείο TeX σας.

## Βήμα 2: Μετατροπή σε PDF
 Το δεύτερο βήμα είναι να μετατρέψετε το έγγραφο TeX σε έγγραφο PDF χρησιμοποιώντας το`Save` μέθοδος του`Document` αντικείμενο. Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
// Αποθηκεύστε την έξοδο σε αρχείο PDF
doc.Save(dataDir + "TeXToPDF_out.pdf");
```

Βεβαιωθείτε ότι έχετε καθορίσει την επιθυμητή διαδρομή και το όνομα αρχείου για το αρχείο PDF που προκύπτει.

### Παράδειγμα πηγαίου κώδικα για TeX σε PDF χρησιμοποιώντας Aspose.PDF για .NET

```csharp
try
{
	
	// Η διαδρομή προς τον κατάλογο εγγράφων.

	// Αποθηκεύστε την έξοδο σε αρχείο PDF

catch (Exception ex)
```
