## Шаг 1: Настройте информацию о документе и странице

Первым делом нам нужно создать новый документ и настроить параметры его страницы. Вот как это сделать:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
PageInfo pageInfo = doc.PageInfo;
Aspose.Pdf.MarginInfo marginInfo = pageInfo.Margin;

marginInfo.Left = 37;
marginInfo.Right = 37;
marginInfo.Top = 37;
marginInfo.Bottom = 37;

pageInfo.IsLandscape = true;
```

Объяснение: 
- Начнем с определения места сохранения нашего документа (`dataDir`). 
-  Затем мы создаем новый экземпляр`Document` сорт. 
- Мы настраиваем поля страницы так, чтобы вокруг таблицы оставалось немного свободного пространства.
- Наконец, мы устанавливаем альбомную ориентацию документа, что помогает при отображении более широких таблиц.

## Шаг 2: Создание первой таблицы

Далее давайте создадим нашу первую таблицу и заполним ее некоторыми образцами данных:

```csharp
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
table.ColumnWidths = "50 100"; // Определить ширину столбцов
```

 Пояснение: Здесь мы создаем экземпляр`Table` class и задайте ширину столбцов. Первый столбец будет шириной 50 единиц, а второй столбец будет шириной 100 единиц.

## Шаг 3: Заполните таблицу строками

Теперь давайте добавим строки в нашу таблицу в цикле:

```csharp
Page curPage = doc.Pages.Add(); // Добавление новой страницы
for (int i = 1; i <= 120; i++)
{
    Aspose.Pdf.Row row = table.Rows.Add();
    row.FixedRowHeight = 15; // Установите фиксированную высоту для строк
    
    Aspose.Pdf.Cell cell1 = row.Cells.Add();
    cell1.Paragraphs.Add(new TextFragment("Content 1"));
    
    Aspose.Pdf.Cell cell2 = row.Cells.Add();
    cell2.Paragraphs.Add(new TextFragment("HHHHH"));
}
```

Объяснение: 
- Здесь мы создаем новую страницу для добавления нашей таблицы.
-  Мы используем`for` цикл для добавления 120 строк в нашу таблицу. Каждая строка имеет фиксированную высоту в 15 единиц.
- Внутри каждой строки мы добавляем две ячейки и заполняем их текстом.

## Шаг 4: Добавьте первую таблицу на страницу

После заполнения таблицы мы добавим ее на текущую страницу:

```csharp
Aspose.Pdf.Paragraphs paragraphs = curPage.Paragraphs;
paragraphs.Add(table);
```

Пояснение: Этот шаг просто добавляет созданную нами таблицу к абзацам текущей страницы, делая таблицу видимой в документе PDF.

## Шаг 5: Создайте вторую таблицу

Теперь давайте создадим вторую таблицу с другим содержимым и добавим ее на новую страницу:

```csharp
Aspose.Pdf.Table table1 = new Aspose.Pdf.Table();
table1.ColumnWidths = "100 100";
for (int i = 1; i <= 10; i++)
{
    Aspose.Pdf.Row row = table1.Rows.Add();
    Aspose.Pdf.Cell cell1 = row.Cells.Add();
    cell1.Paragraphs.Add(new TextFragment("LAAAAAAA"));
    
    Aspose.Pdf.Cell cell2 = row.Cells.Add();
    cell2.Paragraphs.Add(new TextFragment("LAAGGGGGG"));
}
table1.IsInNewPage = true; // Настройка отображения второй таблицы на новой странице
paragraphs.Add(table1);
```

Объяснение: 
- Этот фрагмент кода создает новую таблицу с двумя столбцами, каждый шириной 100 единиц.
-  А`for` цикл добавляет 10 строк с образцом содержимого.
-  Установив`table1.IsInNewPage` значение true, мы гарантируем, что эта таблица появится на новой странице, сохраняя порядок и порядок.

## Шаг 6: Сохраните документ.

Теперь, когда наши таблицы готовы, давайте сохраним наш документ:

```csharp
dataDir = dataDir + "IsNewPageProperty_Test_out.pdf";
doc.Save(dataDir);
```

 Объяснение: Мы указываем имя файла и сохраняем документ в указанном каталоге. При запуске этого кода создается PDF-файл с названием`IsNewPageProperty_Test_out.pdf` будет создан в указанном вами месте.

## Шаг 7: Подтверждающее сообщение

Наконец, чтобы дать пользователю знать, что все прошло гладко, мы можем добавить дружелюбное сообщение в консоль:

```csharp
Console.WriteLine("\nTable rendered successfully on a page.\nFile saved at " + dataDir);
```

Пояснение: Это простой способ подтвердить, что операция прошла успешно, и где пользователь может найти свой новый PDF-файл.
