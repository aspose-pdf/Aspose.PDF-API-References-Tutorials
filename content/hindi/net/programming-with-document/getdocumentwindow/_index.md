.NET के लिए Aspose.PDF एक शक्तिशाली PDF मैनिपुलेशन लाइब्रेरी है जो डेवलपर्स को उनके .NET अनुप्रयोगों में PDF फ़ाइलें बनाने, संपादित करने और परिवर्तित करने की अनुमति देती है। इस लाइब्रेरी द्वारा दी जाने वाली सुविधाओं में से एक दस्तावेज़ की विंडो प्रॉपर्टी के बारे में जानकारी प्राप्त करने की क्षमता है। यह ट्यूटोरियल आपको उपयोग करने के चरणों के माध्यम से मार्गदर्शन करेगा`GetDocumentWindow` .NET के लिए Aspose.PDF की सुविधा, PDF दस्तावेज़ के विंडो गुणों के बारे में जानकारी प्राप्त करने के लिए।

## चरण 1: .NET के लिए Aspose.PDF स्थापित करें

 अपने .NET अनुप्रयोगों में .NET के लिए Aspose.PDF का उपयोग करने के लिए, आपको पहले लाइब्रेरी स्थापित करनी होगी। आप लाइब्रेरी का नवीनतम संस्करण यहाँ से डाउनलोड कर सकते हैं।[.NET के लिए Aspose.PDF डाउनलोड पृष्ठ](https://releases.aspose.com/pdf/net).

लाइब्रेरी डाउनलोड करने के बाद, ज़िप फ़ाइल की सामग्री को अपने कंप्यूटर पर किसी फ़ोल्डर में निकालें। फिर आपको अपने .NET प्रोजेक्ट में Aspose.PDF for .NET DLL का संदर्भ जोड़ना होगा।

## चरण 2: पीडीएफ दस्तावेज़ लोड करें

 एक बार जब आपने .NET के लिए Aspose.PDF स्थापित कर लिया है और अपने .NET प्रोजेक्ट में DLL का संदर्भ जोड़ दिया है, तो आप इसका उपयोग शुरू कर सकते हैं।`GetDocumentWindow`पीडीएफ दस्तावेज़ के विंडो गुणों के बारे में जानकारी प्राप्त करने की सुविधा।

इस सुविधा का उपयोग करने का पहला चरण उस PDF दस्तावेज़ को लोड करना है जिसके बारे में आप जानकारी प्राप्त करना चाहते हैं। ऐसा करने के लिए, आप निम्न कोड का उपयोग कर सकते हैं:

```csharp
// पीडीएफ दस्तावेज़ का पथ
string dataDir = "YOUR DOCUMENT DIRECTORY";

// पीडीएफ दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir + "GetDocumentWindow.pdf");
```

 उपरोक्त कोड में, प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` उस निर्देशिका का पथ जहाँ आपका PDF दस्तावेज़ स्थित है। यह कोड PDF दस्तावेज़ को एक निर्देशिका में लोड करेगा जहाँ आपका PDF दस्तावेज़ स्थित है।`Document` ऑब्जेक्ट, जिसका उपयोग आप दस्तावेज़ की विंडो गुणों के बारे में जानकारी प्राप्त करने के लिए कर सकते हैं।

## चरण 3: दस्तावेज़ के विंडो गुण पुनर्प्राप्त करें

किसी PDF दस्तावेज़ के विंडो गुणों के बारे में जानकारी प्राप्त करने के लिए, आप निम्नलिखित कोड का उपयोग कर सकते हैं:

```csharp
// दस्तावेज़ के विंडो गुण पुनर्प्राप्त करें
Console.WriteLine("CenterWindow : {0}", pdfDocument.CenterWindow);
Console.WriteLine("Direction : {0}", pdfDocument.Direction);
Console.WriteLine("DisplayDocTitle : {0}", pdfDocument.DisplayDocTitle);
Console.WriteLine("FitWindow : {0}", pdfDocument.FitWindow);
Console.WriteLine("HideMenuBar : {0}", pdfDocument.HideMenubar);
Console.WriteLine("HideToolBar : {0}", pdfDocument.HideToolBar);
Console.WriteLine("HideWindowUI : {0}", pdfDocument.HideWindowUI);
Console.WriteLine("NonFullScreenPageMode : {0}", pdfDocument.NonFullScreenPageMode);
Console.WriteLine("PageLayout : {0}", pdfDocument.PageLayout);
Console.WriteLine("pageMode : {0}", pdfDocument.PageMode);
```

उपरोक्त कोड में, प्रत्येक पंक्ति PDF दस्तावेज़ की एक अलग विंडो प्रॉपर्टी प्राप्त करती है और इसे कंसोल पर आउटपुट करती है। आप इस कोड को केवल उन प्रॉपर्टी को प्राप्त करने के लिए कस्टमाइज़ कर सकते हैं जिनमें आपकी रुचि है।

### .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल की दस्तावेज़ विंडो प्राप्त करने के लिए उदाहरण स्रोत कोड 

 यहां पीडीएफ दस्तावेज़ के विंडो गुणों को पुनर्प्राप्त करने के लिए पूर्ण स्रोत कोड दिया गया है`GetDocumentWindow` .NET के लिए Aspose.PDF की विशेषता:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir + "GetDocumentWindow.pdf");

// विभिन्न दस्तावेज़ गुण प्राप्त करें
// दस्तावेज़ की विंडो की स्थिति - डिफ़ॉल्ट: गलत
Console.WriteLine("CenterWindow : {0}", pdfDocument.CenterWindow);

// प्रमुख पठन क्रम; पृष्ठ की स्थिति निर्धारित करता है
// जब साथ-साथ प्रदर्शित किया जाता है - डिफ़ॉल्ट: L2R
Console.WriteLine("Direction : {0}", pdfDocument.Direction);

// क्या विंडो के शीर्षक बार को दस्तावेज़ का शीर्षक प्रदर्शित करना चाहिए
// यदि गलत है, तो शीर्षक बार पीडीएफ फ़ाइल नाम प्रदर्शित करता है - डिफ़ॉल्ट: गलत
Console.WriteLine("DisplayDocTitle : {0}", pdfDocument.DisplayDocTitle);

// क्या दस्तावेज़ की विंडो का आकार बदलना है ताकि वह आकार में फ़िट हो जाए
// पहला प्रदर्शित पृष्ठ - डिफ़ॉल्ट: गलत
Console.WriteLine("FitWindow : {0}", pdfDocument.FitWindow);

// व्यूअर एप्लिकेशन का मेनू बार छिपाना है या नहीं - डिफ़ॉल्ट: गलत
Console.WriteLine("HideMenuBar : {0}", pdfDocument.HideMenubar);

// व्यूअर एप्लिकेशन का टूल बार छिपाना है या नहीं - डिफ़ॉल्ट: गलत
Console.WriteLine("HideToolBar : {0}", pdfDocument.HideToolBar);

// स्क्रॉल बार जैसे UI तत्वों को छिपाना है या नहीं
// और केवल पृष्ठ सामग्री प्रदर्शित करना - डिफ़ॉल्ट: गलत
Console.WriteLine("HideWindowUI : {0}", pdfDocument.HideWindowUI);

//दस्तावेज़ का पृष्ठ मोड। पूर्ण-स्क्रीन मोड से बाहर निकलने पर दस्तावेज़ कैसे प्रदर्शित करें।
Console.WriteLine("NonFullScreenPageMode : {0}", pdfDocument.NonFullScreenPageMode);

// पेज लेआउट यानि एकल पृष्ठ, एक कॉलम
Console.WriteLine("PageLayout : {0}", pdfDocument.PageLayout);

// दस्तावेज़ को खोलने पर कैसा दिखना चाहिए
// यानी थंबनेल दिखाएं, पूर्ण स्क्रीन दिखाएं, अटैचमेंट पैनल दिखाएं
Console.WriteLine("pageMode : {0}", pdfDocument.PageMode);
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा है कि PDF दस्तावेज़ की विंडो प्रॉपर्टी के बारे में जानकारी प्राप्त करने के लिए .NET के लिए Aspose.PDF का उपयोग कैसे करें। PDF दस्तावेज़ लोड करके और उसकी विंडो प्रॉपर्टी तक पहुँच कर, आप इस बारे में जानकारी एकत्र कर सकते हैं कि व्यूअर एप्लिकेशन में खोले जाने पर दस्तावेज़ को कैसे प्रदर्शित किया जाना चाहिए। .NET के लिए Aspose.PDF प्रोग्रामेटिक रूप से PDF फ़ाइलों के साथ काम करने के लिए सुविधाओं का एक शक्तिशाली सेट प्रदान करता है, जो इसे .NET अनुप्रयोगों में PDF हेरफेर के लिए एक मूल्यवान उपकरण बनाता है।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ के विंडो गुणों को पुनः प्राप्त करने का उद्देश्य क्या है?

उत्तर: PDF दस्तावेज़ की विंडो प्रॉपर्टी प्राप्त करने से आप इस बारे में जानकारी एकत्र कर सकते हैं कि व्यूअर एप्लिकेशन में खोले जाने पर PDF दस्तावेज़ को किस तरह प्रदर्शित किया जाना चाहिए। ये प्रॉपर्टी विंडो की स्थिति, डिस्प्ले मोड और UI तत्वों की दृश्यता जैसे विभिन्न पहलुओं को नियंत्रित करती हैं।

#### प्रश्न: मैं अपने .NET प्रोजेक्ट में .NET के लिए Aspose.PDF कैसे स्थापित कर सकता हूं?

 उत्तर: .NET के लिए Aspose.PDF को स्थापित करने के लिए, आपको लाइब्रेरी को डाउनलोड करना होगा।[.NET के लिए Aspose.PDF डाउनलोड पृष्ठ](https://releases.aspose.com/pdf/net)डाउनलोड करने के बाद, ZIP फ़ाइल की सामग्री निकालें और अपने .NET प्रोजेक्ट में Aspose.PDF for .NET DLL का संदर्भ जोड़ें।

#### प्रश्न: क्या मैं कोड को केवल विशिष्ट विंडो गुणधर्मों को प्राप्त करने के लिए अनुकूलित कर सकता हूँ?

उत्तर: हां, आप उन पंक्तियों को टिप्पणी करके विशिष्ट विंडो गुण प्राप्त करने के लिए कोड को कस्टमाइज़ कर सकते हैं जिनकी आपको आवश्यकता नहीं है। कोड में प्रत्येक पंक्ति एक विशिष्ट विंडो गुण से मेल खाती है, इसलिए आप अपनी आवश्यकताओं के आधार पर गुणों को शामिल या बहिष्कृत कर सकते हैं।

#### प्रश्न: .NET के लिए Aspose.PDF का उपयोग करके मैं किस प्रकार के विंडो गुण प्राप्त कर सकता हूं?

उत्तर: .NET के लिए Aspose.PDF का उपयोग करके, आप PDF दस्तावेज़ के विभिन्न विंडो गुणों को पुनः प्राप्त कर सकते हैं, जिसमें विंडो को केन्द्रित करना, पृष्ठ लेआउट सेट करना, टूलबार और मेनू बार के प्रदर्शन को नियंत्रित करना आदि शामिल है।
