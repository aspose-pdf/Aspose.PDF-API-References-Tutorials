Aspose.PDF для .NET — это мощная библиотека, которая позволяет разработчикам работать с файлами PDF программно. Одной из функций, которую она предоставляет, является возможность получить все шрифты, используемые в файле PDF. Это может быть полезно, если вам нужно программно анализировать или манипулировать шрифтами в файле PDF.

В этом уроке мы обсудим, как использовать Aspose.PDF для .NET, чтобы получить все шрифты, используемые в документе PDF. Мы предоставим пошаговое руководство о том, как это сделать, вместе с примером исходного кода.

## Шаг 1: Создайте новое консольное приложение C#
Для начала создайте новое консольное приложение C# в Visual Studio. Вы можете назвать его как угодно. После создания проекта вам нужно добавить ссылку на библиотеку Aspose.PDF for .NET.

## Шаг 2: Импорт пространства имен Aspose.PDF
Добавьте следующую строку кода в начало файла C#, чтобы импортировать пространство имен Aspose.PDF:

```csharp
using Aspose.Pdf;
```

## Шаг 3: Загрузите PDF-документ
Загрузите PDF-документ, из которого вы хотите получить шрифты:

```csharp
Document doc = new Document(dataDir + "input.pdf");
```

## Шаг 4: Получите все шрифты
Получите все шрифты, используемые в документе PDF:

```csharp
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();
```

## Шаг 5: Распечатайте все шрифты
Распечатать все шрифты, используемые в PDF-документе:

```csharp
foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```

### Пример исходного кода для получения всех шрифтов с помощью Aspose.PDF для .NET
```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "input.pdf");
Aspose.Pdf.Text.Font[] fonts = doc.FontUtilities.GetAllFonts();
foreach (Aspose.Pdf.Text.Font font in fonts)
{
    Console.WriteLine(font.FontName);
}
```
