## Βήμα 1: Άνοιγμα του εγγράφου προέλευσης PDF
Σε αυτό το βήμα, θα ανοίξουμε το αρχείο προέλευσης PDF χρησιμοποιώντας το Aspose.PDF για .NET. Ακολουθήστε τον παρακάτω κώδικα:

```csharp

string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Ανοίξτε το έγγραφο προέλευσης PDF
Document pdfDocument = new Document(dataDir + "PDFToDOC.pdf");
```

 Φροντίστε να αντικαταστήσετε`"YOUR DOCUMENTS DIRECTORY"` με τον πραγματικό κατάλογο όπου βρίσκεται το αρχείο PDF σας.

## Βήμα 2: Μετατροπή PDF σε μορφή DOC
Αφού ανοίξουμε το αρχείο PDF, μπορούμε να προχωρήσουμε στη μετατροπή σε μορφή DOC. Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp

```

 Ο παραπάνω κώδικας μετατρέπει το αρχείο PDF σε μορφή DOC και το αποθηκεύει ως όνομα αρχείου`"PDFToDOC_out.doc"`.

 Αντικαθιστώ`"YOUR DOCUMENTS DIRECTORY"` με τον επιθυμητό κατάλογο όπου θέλετε να αποθηκεύσετε το αρχείο DOC εξόδου.
