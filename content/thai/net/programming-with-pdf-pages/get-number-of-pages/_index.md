## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร

คุณจะต้องระบุตำแหน่งที่ตั้งของไฟล์ PDF ของคุณ ในขั้นตอนนี้ คุณสามารถกำหนดเส้นทางไปยังไดเร็กทอรีที่เก็บไฟล์ PDF ของคุณได้

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```
 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงไปยังโฟลเดอร์ที่มีไฟล์ PDF ของคุณ นี่คือจุดที่ไลบรารี Aspose จะค้นหาไฟล์ที่คุณต้องการวิเคราะห์ เหมือนกับการมอบแผนที่ไปยังสมบัติให้กับไลบรารีของคุณ!

## ขั้นตอนที่ 2: สร้างอินสแตนซ์ของเอกสาร PDF

 ตอนนี้เราได้ตั้งค่าไดเร็กทอรีแล้ว เราต้องสร้างอินสแตนซ์ของ`Document` คลาสที่จะเก็บข้อมูล PDF ของเรา

```csharp
Document pdfDocument = new Document(dataDir + "GetNumberOfPages.pdf");
```
 เส้นนี้จะสร้างสิ่งใหม่`Document` วัตถุตามไฟล์ PDF ที่คุณระบุ โปรดจำไว้ว่าไฟล์ PDF ของคุณควรตรงกับชื่อที่คุณกำหนดไว้ที่นี่

## ขั้นตอนที่ 3: ดึงจำนวนหน้า

มาถึงช่วงเวลาอันมหัศจรรย์แล้ว! มาลองนับจำนวนหน้าในเอกสาร PDF กัน

```csharp
int pageCount = pdfDocument.Pages.Count;
```
 การใช้`Pages` ทรัพย์สินของ`Document`ตัวอย่างเช่น เราสามารถเข้าถึงจำนวนหน้าที่บรรจุอยู่ในนั้นได้ ง่ายพอๆ กับการเปิดกระป๋องโซดา—ไม่ยุ่งยากเลย!

## ขั้นตอนที่ 4: แสดงจำนวนหน้า

สุดท้ายนี้ เราต้องการดูผลลัพธ์จากการทำงานหนักของเรา มาพิมพ์จำนวนหน้าทั้งหมดออกสู่คอนโซลกัน

```csharp
System.Console.WriteLine("Page Count : {0}", pageCount);
```
โค้ดบรรทัดนี้จะแสดงผลจำนวนหน้าไปยังคอนโซล เหมือนกับการวิ่งมาราธอนเพื่อชัยชนะ—ฉลองความสำเร็จของคุณ!
