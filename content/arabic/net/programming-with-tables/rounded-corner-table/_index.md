في هذا البرنامج التعليمي، سنرشدك خطوة بخطوة لإنشاء جدول بزوايا مستديرة في مستند PDF باستخدام Aspose.PDF لـ .NET. وسنشرح الكود المصدري C# المقدم ونوضح لك كيفية تنفيذه.

## الخطوة 1: إنشاء الجدول
أولاً سنقوم بإنشاء الجدول باستخدام الكود التالي:

```csharp
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

## الخطوة 2: إعداد نمط الزاوية المستديرة
بعد ذلك، سنقوم بتكوين نمط الزاوية المستديرة للجدول:

```csharp
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
```

## الخطوة 3: إعداد حدود الجدول
لإعطاء الجدول حدودًا ذات زاوية مستديرة، نحتاج إلى إنشاء كائن BorderInfo وتكوينه بالمعلمات المناسبة:

```csharp
// إنشاء كائن GraphInfo لتعيين لون الحدود
GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red;

// إنشاء كائن BorderInfo فارغ
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);

// اضبط نصف قطر الحدود المستديرة على 15
bInfo.RoundedBorderRadius = 15;

// تطبيق معلومات الحدود على الجدول
tab1.Border = bInfo;
```

### مثال على كود المصدر لجدول الزوايا المستديرة باستخدام Aspose.PDF لـ .NET

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();

GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red;
// إنشاء كائن BorderInfo فارغ
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);
// قم بتعيين الحدود لتكون حدودًا أكثر استدارة حيث يبلغ نصف قطر الدائرة 15
bInfo.RoundedBorderRadius = 15;
// قم بضبط نمط زاوية الجدول ليكون دائريًا.
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
// تعيين معلومات حدود الجدول
tab1.Border = bInfo;
```

## خاتمة
تهانينا! لقد تعلمت الآن كيفية إنشاء جدول ذي زوايا مستديرة في مستند PDF باستخدام Aspose.PDF لـ .NET. يوضح لك هذا الدليل خطوة بخطوة كيفية إعداد نمط الزوايا المستديرة وحدود الجدول. يمكنك الآن تطبيق هذه المعرفة على مشاريعك الخاصة.

### الأسئلة الشائعة حول الطاولة ذات الزاوية المستديرة في مستند PDF

#### س: هل يمكنني تخصيص نصف قطر الزوايا المستديرة للجدول؟

ج: نعم، يمكنك تخصيص نصف قطر الزوايا المستديرة للجدول عن طريق تعديل قيمة`bInfo.RoundedBorderRadius` الخاصية في كود المصدر C# المقدم. ما عليك سوى تعيين قيمة نصف القطر المطلوبة (بالنقاط) لتحقيق مظهر الزاوية المستديرة المطلوب.

#### س: هل يمكنني تطبيق الزوايا المستديرة على خلايا فردية داخل الجدول؟

ج: لا، يتم تطبيق نمط الزوايا المستديرة على الجدول بأكمله ككل. لا يوفر Aspose.PDF لـ .NET حاليًا دعمًا مدمجًا لتطبيق الزوايا المستديرة على الخلايا الفردية داخل الجدول.

#### س: هل يمكنني تغيير لون حدود الزاوية المستديرة؟

 ج: نعم، يمكنك تغيير لون حدود الزاوية المستديرة عن طريق تعديل قيمة`graph.Color` الخاصية في كود المصدر C#. ما عليك سوى توفير اللون المطلوب، مثل`Aspose.Pdf.Color.Red` أو أي تمثيل لوني صالح آخر.

#### س: هل من الممكن تطبيق أنماط زوايا مختلفة (على سبيل المثال، مربعة ومدورة) على جداول مختلفة ضمن نفس مستند PDF؟

ج: نعم، من الممكن تطبيق أنماط زوايا مختلفة على جداول مختلفة ضمن نفس مستند PDF. يمكنك إنشاء جداول متعددة وتكوين أنماط زواياها بشكل فردي بناءً على متطلباتك.

#### س: هل يمكنني تعديل سمك حافة الزاوية المستديرة؟

 ج: نعم، يمكنك تعديل سمك حدود الزاوية المستديرة عن طريق تعديل`BorderInfo` خصائص الكائن في الكود المصدري للغة C#. على سبيل المثال، يمكنك تعيين`bInfo.Width` خاصية لضبط سمك الحدود.
