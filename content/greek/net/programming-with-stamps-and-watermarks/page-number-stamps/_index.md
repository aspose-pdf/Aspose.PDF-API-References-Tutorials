## Βήμα 1: Καθορίστε τον Κατάλογο εγγράφων

Για να ξεκινήσετε, πρέπει να καθορίσετε τον κατάλογο όπου βρίσκεται το αρχείο PDF σας. Αυτό είναι το σημείο εκκίνησης του έργου σας.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Ενημερώστε αυτήν τη διαδρομή
```

 Επεξήγηση: Αντικατάσταση`"YOUR DOCUMENT DIRECTORY"` με τη διαδρομή που οδηγεί στον κατάλογο που περιέχει το αρχείο PDF σας. Αυτό είναι κρίσιμο, καθώς λέει στον κώδικά σας πού να βρείτε το αρχείο που θέλετε να χειριστείτε.

## Βήμα 2: Ανοίξτε το Έγγραφο

Στη συνέχεια, θα ανοίξουμε το υπάρχον έγγραφο PDF στο οποίο θέλουμε να προσθέσουμε τις σφραγίδες αριθμού σελίδας.

```csharp
Document pdfDocument = new Document(dataDir + "PageNumberStamp.pdf");
```

 Επεξήγηση: Εδώ, χρησιμοποιούμε το`Document` τάξη που παρέχεται από το Aspose.PDF για να ανοίξετε το συγκεκριμένο αρχείο PDF μας. Βεβαιωθείτε ότι το όνομα του αρχείου ταιριάζει με το πραγματικό αρχείο που έχετε στον κατάλογό σας.

## Βήμα 3: Δημιουργήστε μια σφραγίδα αριθμού σελίδας

Τώρα έρχεται το διασκεδαστικό μέρος! Ας δημιουργήσουμε μια σφραγίδα αριθμού σελίδας για προσθήκη στο PDF μας.

```csharp
PageNumberStamp pageNumberStamp = new PageNumberStamp();
```

 Εξήγηση: Το`PageNumberStamp`class θα μας επιτρέψει να δημιουργήσουμε μια σφραγίδα που θα εμφανίζει τον τρέχοντα αριθμό σελίδας σε σχέση με τον συνολικό αριθμό σελίδων στο έγγραφο.

## Βήμα 4: Διαμόρφωση της σφραγίδας

Τώρα, θα πρέπει να διαμορφώσετε τις ρυθμίσεις σφραγίδας σας. Εδώ σχεδιάζετε πώς φαίνεται και συμπεριφέρεται η σφραγίδα.

```csharp
pageNumberStamp.Background = false;
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;
pageNumberStamp.BottomMargin = 10;
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;
pageNumberStamp.StartingNumber = 1;
```

Εξήγηση:
- `Background = false`: Αυτό σημαίνει ότι η σφραγίδα θα εμφανιστεί στο προσκήνιο.
- `Format`: Εδώ, ορίζετε τη μορφή ώστε να εμφανίζεται η "Σελίδα X of Y", όπου λαμβάνετε δυναμικά τις συνολικές σελίδες του εγγράφου.
- `BottomMargin`: Προσαρμόζει την απόσταση από το κάτω μέρος της σελίδας.
- `HorizontalAlignment`: Κεντράρει τη σφραγίδα οριζόντια.
- `StartingNumber`: Ορίζει ποιος θα είναι ο αριθμός αρχικής σελίδας, συνήθως από 1.

## Βήμα 5: Ορισμός ιδιοτήτων κειμένου

Στη συνέχεια, μπορείτε να προσαρμόσετε τον τρόπο εμφάνισης του κειμένου στη σφραγίδα.

```csharp
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

Επεξήγηση: Αυτά τα χαρακτηριστικά διαμορφώνουν τον τύπο γραμματοσειράς, το μέγεθος γραμματοσειράς, το στυλ (τόσο έντονη όσο και πλάγια γραφή) και το χρώμα του κειμένου εντός της σφραγίδας για να το κάνουν οπτικά ελκυστικό.

## Βήμα 6: Προσθέστε τη σφραγίδα σε μια συγκεκριμένη σελίδα

Με τη σφραγίδα σας διαμορφωμένη, ήρθε η ώρα να την προσθέσετε σε μια συγκεκριμένη σελίδα στο έγγραφό σας.

```csharp
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
```

 Επεξήγηση: Αυτή η γραμμή προσθέτει τη σφραγίδα στην πρώτη σελίδα του PDF. Μπορείτε να προσαρμόσετε το`Pages[1]` ευρετήριο για άλλες σελίδες όπως απαιτείται.

## Βήμα 7: Αποθηκεύστε το Έγγραφο Εξόδου

Τέλος, αποθηκεύστε το τροποποιημένο έγγραφο PDF ώστε οι αλλαγές σας να είναι μόνιμες.

```csharp
dataDir = dataDir + "PageNumberStamp_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nPage number stamp added successfully.\nFile saved at " + dataDir);
```

Επεξήγηση: Καθορίζετε τη διαδρομή του αρχείου εξόδου και αποθηκεύετε το έγγραφο. Η κονσόλα θα σας ενημερώσει ότι η σφραγίδα προστέθηκε με επιτυχία και πού είναι αποθηκευμένο το αρχείο.
