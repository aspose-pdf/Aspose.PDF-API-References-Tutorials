## استيراد الحزم

للبدء في استخدام Aspose.PDF لـ .NET، تحتاج إلى استيراد الحزم اللازمة إلى مشروعك. إليك كيفية القيام بذلك:

### الخطوة 1: إنشاء مشروع جديد

افتح Visual Studio وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة التحكم لتبسيط الأمر.

### الخطوة 2: إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول.
2. حدد "إدارة حزم NuGet".
3. ابحث عن "Aspose.PDF" وقم بتثبيت الإصدار الأحدث.

### الخطوة 3: استيراد مساحة الاسم

في الجزء العلوي من ملف C#، قم باستيراد مساحة اسم Aspose.PDF:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا ننتقل إلى الجزء الرئيسي من برنامجنا التعليمي: ضبط تنسيق التعليق النصي الحر.

## الخطوة 1: تحديد دليل المستندات

أولاً وقبل كل شيء، عليك تحديد المسار إلى دليل المستندات. هذا هو المكان الذي سيتم فيه وضع ملف PDF. وإليك كيفية القيام بذلك:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يتم تخزين ملف PDF فيه. هذه الخطوة بالغة الأهمية لأنها تخبر برنامجك بمكان العثور على مستند PDF الذي تريد العمل عليه.

## الخطوة 2: افتح مستند PDF

 بعد ذلك، ستحتاج إلى فتح مستند PDF الذي ستعلق عليه. يتم ذلك باستخدام`Document` الفئة من مكتبة Aspose.PDF:

```csharp
// فتح المستند
Document pdfDocument = new Document(dataDir + "SetFreeTextAnnotationFormatting.pdf");
```

 يقوم هذا السطر من التعليمات البرمجية بتهيئة سطر جديد`Document` يقوم بتحميل الكائن وتحميل ملف PDF المحدد. تأكد من أن اسم الملف يتطابق مع الاسم الموجود في الدليل.

## الخطوة 3: إنشاء كائن DefaultAppearance

 الآن، دعونا ننشئ`DefaultAppearance` الكائن. سيحدد هذا الكائن مظهر التعليق النصي المجاني الخاص بك، مثل الخط والحجم واللون:

```csharp
// إنشاء كائن DefaultAppearance
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 28, System.Drawing.Color.Red);
```

في هذا المثال، نستخدم الخط Arial، ونضبط حجم الخط على 28، ونختار اللون الأحمر. لا تتردد في تخصيص هذه القيم لتناسب احتياجاتك!

## الخطوة 4: إنشاء التعليق النصي الحر

بعد تحديد المظهر، حان الوقت لإنشاء التعليق النصي الحر الفعلي. هنا يمكنك تحديد المكان الذي سيظهر فيه التعليق على ملف PDF:

```csharp
// إنشاء التعليقات التوضيحية
FreeTextAnnotation freetext = new FreeTextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600), default_appearance);
```

 في هذا السطر، نقوم بإنشاء سطر جديد`FreeTextAnnotation` في الصفحة الأولى من ملف PDF. يحدد المستطيل موضع وحجم التعليق التوضيحي. يمكنك ضبط الإحداثيات (200، 400، 400، 600) لوضع التعليق التوضيحي بالضبط حيث تريد.

## الخطوة 5: تحديد محتويات التعليق التوضيحي

الآن بعد أن قمنا بإنشاء التعليق التوضيحي الخاص بنا، دعنا نضيف بعض النص إليه:

```csharp
// تحديد محتويات التعليق التوضيحي
freetext.Contents = "Free Text";
```

 يمكنك استبدال`"Free Text"`مع أي رسالة تريد عرضها في التعليق التوضيحي. هذا هو النص الذي سيكون مرئيًا لأي شخص يشاهد ملف PDF.

## الخطوة 6: إضافة التعليقات التوضيحية إلى الصفحة

بعد ذلك، نحتاج إلى إضافة التعليقات التوضيحية إلى مجموعة التعليقات التوضيحية الخاصة بالصفحة:

```csharp
// إضافة تعليق توضيحي إلى مجموعة التعليقات التوضيحية للصفحة
pdfDocument.Pages[1].Annotations.Add(freetext);
```

يضمن هذا السطر من التعليمات البرمجية إضافة التعليقات التوضيحية التي أنشأتها حديثًا إلى مستند PDF. بدون هذه الخطوة، لن تظهر التعليقات التوضيحية في الناتج النهائي.

## الخطوة 7: حفظ المستند المحدث

أخيرًا، حان الوقت لحفظ التغييرات. ستحتاج إلى تحديد اسم ملف جديد للمستند المحدث:

```csharp
dataDir = dataDir + "SetFreeTextAnnotationFormatting_out.pdf";
// حفظ المستند المحدث
pdfDocument.Save(dataDir);
```

يحفظ هذا الكود ملف PDF المعدّل باسم جديد، مما يضمن بقاء المستند الأصلي دون تغيير. يمكنك الآن فتح ملف PDF الجديد لمشاهدة التعليقات التوضيحية النصية المجانية أثناء العمل!
