
คำสั่งนี้จะดึงและติดตั้ง Aspose.PDF ลงในโปรเจ็กต์ของคุณ ช่วยให้คุณพร้อมสำหรับความสำเร็จ

ตอนนี้ มาดูขั้นตอนหลักๆ ที่เกี่ยวข้องกับการระบุรหัสผ่านที่ถูกต้องสำหรับไฟล์ PDF กัน เราจะอธิบายขั้นตอนการใช้งานตัวอย่างทีละขั้นตอนเพื่อความชัดเจน

## ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์

 ก่อนอื่น คุณจะต้องระบุเส้นทางไปยังไฟล์ PDF ที่คุณกำลังทำงานด้วย อย่าลืมแทนที่`"YOUR DOCUMENTS DIRECTORY"` ด้วยเส้นทางจริงที่ไฟล์ PDF ของคุณอยู่

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: โหลดไฟล์ PDF ต้นฉบับ

 ถัดไปใช้`PdfFileInfo` ในการโหลดไฟล์ PDF ต้นฉบับของคุณ:

```csharp
PdfFileInfo info = new PdfFileInfo();
info.BindPdf(dataDir + "IsPasswordProtected.pdf");
```

 ขั้นตอนนี้จะผูกไฟล์ PDF เข้ากับ`info` วัตถุที่ให้เราสามารถเข้าถึงคุณสมบัติของมันได้

## ขั้นตอนที่ 3: ตรวจสอบว่า PDF ได้รับการเข้ารหัสหรือไม่

ตอนนี้ถึงเวลาตรวจสอบว่าเอกสาร PDF ได้รับการปกป้องด้วยรหัสผ่านจริงหรือไม่:

```csharp
Console.WriteLine("File is password protected " + info.IsEncrypted);
```

 โดยการตรวจสอบ`IsEncrypted` ทรัพย์สิน คุณสามารถตรวจสอบสถานะการล็อคเอกสารได้ หากเป็น`true`ถ้าอย่างนั้นคุณจะต้องถอดรหัส!

## ขั้นตอนที่ 4: เตรียมรายการรหัสผ่านที่เป็นไปได้

ในการค้นหารหัสผ่าน ให้เตรียมอาร์เรย์สตริงที่ประกอบด้วยรหัสผ่านที่เป็นไปได้ที่คุณต้องการทดสอบ:

```csharp
String[] passwords = new String[5] { "test", "test1", "test2", "test3", "sample" };
```

คุณสามารถปรับเปลี่ยนอาร์เรย์นี้ตามความต้องการของคุณหรือรหัสผ่านที่น่าจะเป็นไปได้มากที่สุด

## ขั้นตอนที่ 5: พยายามเปิด PDF ด้วยรหัสผ่านแต่ละอัน

ตอนนี้เราจะวนซ้ำผ่านรหัสผ่านแต่ละรายการเพื่อพยายามเปิดไฟล์ PDF 

```csharp
for (int passwordcount = 0; passwordcount < passwords.Length; passwordcount++)
{
    try
    {
        Document doc = new Document(dataDir + "IsPasswordProtected.pdf", passwords[passwordcount]);
        if (doc.Pages.Count > 0)
            Console.WriteLine("Number of Page in document are = " + doc.Pages.Count);
    }
    catch (InvalidPasswordException)
    {
        Console.WriteLine("Password = " + passwords[passwordcount] + "  is not correct");
    }
}
```
