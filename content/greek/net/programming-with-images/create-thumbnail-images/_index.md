## Εισαγωγή πακέτων

Για να χρησιμοποιήσετε τις λειτουργίες Aspose.PDF, πρέπει να συμπεριλάβετε τους σχετικούς χώρους ονομάτων στο επάνω μέρος του αρχείου C#. Δείτε πώς το κάνετε:

```csharp
using Aspose.Pdf.Devices;
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text;
```

Η συμπερίληψη αυτών των χώρων ονομάτων διασφαλίζει ότι έχετε πρόσβαση σε όλες τις απαραίτητες κλάσεις και μεθόδους στο Aspose για τις λειτουργίες που θα εκτελούμε.

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Το πρώτο βήμα στη διαδικασία μας είναι να καθορίσουμε τη διαδρομή προς τον κατάλογο των εγγράφων σας όπου αποθηκεύονται όλα τα αρχεία PDF σας. Πρέπει να πείτε στο πρόγραμμα πού να αναζητήσει αυτά τα PDF. 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Αντικαταστήστε με την πραγματική διαδρομή καταλόγου σας
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με τη διαδρομή όπου βρίσκονται τα αρχεία PDF σας. Αυτό το βήμα είναι κρίσιμο γιατί χωρίς τον σωστό κατάλογο, το πρόγραμμά σας δεν θα βρει τα PDF που χρειάζεται να επεξεργαστεί.

## Βήμα 2: Ανάκτηση ονομάτων αρχείων PDF

Στη συνέχεια, θα θέλετε να λάβετε τα ονόματα όλων των αρχείων PDF στον κατάλογό σας. Αυτό το βήμα βοηθά στην επανάληψη κάθε αρχείου αργότερα. 

```csharp
string[] fileEntries = Directory.GetFiles(dataDir, "*.pdf");
```

 Εδώ, χρησιμοποιούμε το`Directory.GetFiles` μέθοδο φιλτραρίσματος και ανάκτησης μόνο αρχείων PDF. Ο`*.pdf` μπαλαντέρ διασφαλίζει ότι συλλαμβάνουμε κάθε PDF στον καθορισμένο κατάλογο. 

## Βήμα 3: Επανάληψη μέσω κάθε αρχείου PDF

Τώρα θα κάνουμε βρόχο σε κάθε αρχείο που μόλις ανακτήσαμε. Για κάθε PDF, θα το ανοίξουμε και θα δημιουργήσουμε μικρογραφίες για τις σελίδες του. 

```csharp
for (int counter = 0; counter < fileEntries.Length; counter++)
{
    Document pdfDocument = new Document(fileEntries[counter]);
}
```

 Σε αυτόν τον βρόχο,`counter` παρακολουθεί ποιο αρχείο δουλεύουμε. Ο`Document` class χρησιμοποιείται για το άνοιγμα κάθε αρχείου PDF. Θα χειρίζεστε κάθε PDF ένα κάθε φορά για να δημιουργείτε μικρογραφίες από τις σελίδες του.

## Βήμα 4: Δημιουργήστε μικρογραφίες για κάθε σελίδα

Για κάθε σελίδα στο PDF, θα δημιουργήσουμε μια μικρογραφία εικόνας. Ας αναλύσουμε αυτό το μέρος βήμα προς βήμα.

### Βήμα 4.1: Αρχικοποιήστε το FileStream για κάθε μικρογραφία

Μέσα στο βρόχο μας, θα χρειαστεί να δημιουργήσουμε μια ροή όπου θα αποθηκευτεί η μικρογραφία εικόνας.

```csharp
using (FileStream imageStream = new FileStream(dataDir + "\\Thumbanils" + counter.ToString() + "_" + pageCount + ".jpg", FileMode.Create))
{
```

 Εδώ, δημιουργούμε ένα νέο αρχείο JPG για κάθε μικρογραφία χρησιμοποιώντας`FileStream`Το όνομα αρχείου περιλαμβάνει τον μετρητή, ώστε κάθε μικρογραφία να έχει ένα μοναδικό όνομα.

### Βήμα 4.2: Καθορίστε την Ανάλυση

Στη συνέχεια, πρέπει να ορίσουμε την ανάλυση για τις μικρογραφίες μας. Οι υψηλότερες αναλύσεις αποδίδουν πιο καθαρές εικόνες, αλλά μπορούν επίσης να αυξήσουν το μέγεθος του αρχείου.

```csharp
Resolution resolution = new Resolution(300);
```

Η ανάλυση 300 DPI (κουκκίδες ανά ίντσα) είναι στάνταρ για ποιοτικές εικόνες. Μη διστάσετε να προσαρμόσετε αυτήν την τιμή με βάση τις ανάγκες σας.

### Βήμα 4.3: Ρύθμιση JpegDevice

 Τώρα, θα ρυθμίσουμε το`JpegDevice` που θα χρησιμοποιηθεί για τη μετατροπή των σελίδων PDF σε εικόνες.

```csharp
JpegDevice jpegDevice = new JpegDevice(45, 59, resolution, 100);
```

Εδώ, καθορίζουμε τις διαστάσεις των μικρογραφιών και την ποιότητα. Σε αυτήν την περίπτωση, έχουμε ορίσει διαστάσεις σε 45x59 pixel, αλλά μπορούμε να προσαρμόσουμε αυτές τις τιμές ανάλογα με το τι χρειάζεται για την εφαρμογή σας.

### Βήμα 4.4: Επεξεργαστείτε κάθε σελίδα

Έχοντας τα πάντα στη θέση τους, μπορούμε τώρα να επεξεργαστούμε κάθε σελίδα του PDF και να αποθηκεύσουμε τη μικρογραφία που δημιουργήθηκε στη ροή μας.

```csharp
jpegDevice.Process(pdfDocument.Pages[pageCount], imageStream);
```

 Αυτή η γραμμή παίρνει τη συγκεκριμένη σελίδα από το PDF και την επεξεργάζεται σε μορφή JPEG, τροφοδοτώντας την απευθείας στο`imageStream`όπου θα αποθηκεύσουμε τη μικρογραφία.

### Βήμα 4.5: Κλείστε τη ροή

Τέλος, μετά την επεξεργασία κάθε σελίδας, πρέπει να κλείσουμε τη ροή για να ελευθερώσουμε πόρους.

```csharp
imageStream.Close();
```

Το κλείσιμο της ροής είναι απαραίτητο για την αποφυγή διαρροών μνήμης και τη διασφάλιση ότι όλες οι αλλαγές έχουν εγγραφεί σωστά στο δίσκο.
