## مقدمة

في هذا الدليل التفصيلي، سنستكشف كيفية استخراج أسماء الصور المضمنة في ملف PDF باستخدام Java ومكتبة Aspose.PDF لـ Java. يمكن أن تحتوي الصور المضمنة على معلومات قيمة، وقد تكون القدرة على الوصول إلى أسمائها برمجيًا مفيدة جدًا.

## إضافة Aspose.PDF لـ Java إلى مشروعك

Aspose.PDF for Java هي مكتبة قوية للعمل مع مستندات PDF. لاستخدامه في مشروعك، يمكنك إضافته باعتباره تبعية. إليك مقتطف تبعية Maven:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>21.10</version>
</dependency>
```

## تحميل ملف PDF

 في هذه الخطوة، سنقوم بكتابة كود Java لتحميل ملف PDF في تطبيقنا. يجب عليك استبدال`"input.pdf"` مع المسار إلى ملف PDF الخاص بك.

```java
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document("input.pdf");
```

## استخراج الصور المضمنة

لاستخراج الصور المضمنة، نحتاج إلى الوصول إلى موارد مستند PDF. وإليك كيف يمكنك القيام بذلك:

```java
com.aspose.pdf.Page page = pdfDocument.getPages().get_Item(1); // استبدله برقم الصفحة المطلوبة
com.aspose.pdf.XImageCollection images = page.getResources().getImages();
```

## كتابة كود جافا لاستخراج الصور

الآن، دعونا نستعرض الصور المضمنة ونستخرج أسمائها:

```java
for (com.aspose.pdf.XImage image : images) {
    System.out.println("Image Name: " + image.getName());
}
```
