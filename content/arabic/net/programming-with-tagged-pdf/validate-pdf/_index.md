## الخطوة 2: تحضير مستند PDF

ضع ملف PDF الذي تريد التحقق من صحته في الدليل المحدد. تأكد من ضبط مسار الملف في الكود المصدر باستخدام دليل المستندات الخاص بك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// مسار ملف الإدخال PDF
string inputFileName = dataDir + "StructureElements.pdf";

// مسار ملف إخراج تقرير التحقق
string outputLogName = dataDir + "ua-20.xml";
```

تأكد من أن ملف PDF الذي تريد التحقق منه محدد بشكل صحيح في الكود المصدر.

## الخطوة 3: التحقق من صحة ملف PDF

في هذه الخطوة، سوف نستخدم Aspose.PDF لـ .NET للتحقق من صحة مستند PDF المحدد وإنشاء تقرير التحقق.

```csharp
// افتح مستند PDF
using (var document = new Aspose.Pdf.Document(inputFileName))
{
// التحقق من صحة مستند PDF
bool isValid = document.Validate(outputLogName, Aspose.Pdf.PdfFormat.PDF_UA_1);
}
```

لقد فتحنا مستند PDF وقمنا بالتحقق من تنسيقه باستخدام Aspose.PDF لـ .NET. سيتم تخزين نتيجة التحقق في ملف التقرير المحدد.

### عينة من كود المصدر للتحقق من صحة ملف PDF باستخدام Aspose.PDF لـ .NET 
```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string inputFileName = dataDir + "StructureElements.pdf";
string outputLogName = dataDir + "ua-20.xml";

using (var document = new Aspose.Pdf.Document(inputFileName))
{
	bool isValid = document.Validate(outputLogName, Aspose.Pdf.PdfFormat.PDF_UA_1);
}

```
