## Βήμα 1: Αρχικοποιήστε το έγγραφο

Για να εργαστείτε με οποιοδήποτε PDF, πρέπει να το φορτώσετε στην εφαρμογή σας. Δείτε πώς:

```csharp
// Φορτώστε το έγγραφο PDF
Document pdfDocument = new Document("Path/To/Your/PDFFile.pdf");
```
 Αντικαθιστώ`"Path/To/Your/PDFFile.pdf"` με την πραγματική διαδρομή του PDF σας.

## Βήμα 2: Δημιουργήστε ένα TextFragmentAbsorber

 Τώρα, ας εξερευνήσουμε διαφορετικά παραδείγματα χρήσης`TextFragmentAbsorber` για ανάκτηση κειμένου.

### Παράδειγμα 1: Εξαγωγή μιας ακριβούς λέξης

 Για να βρείτε μια ακριβή αντιστοίχιση για μια συγκεκριμένη λέξη, μπορείτε να αρχικοποιήσετε το`TextFragmentAbsorber` τοιουτοτροπώς:

```csharp
// Αναζήτηση για μια ακριβή αντιστοίχιση του "Word"
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"\bWord\b", new TextSearchOptions(true));
```
 Αυτός ο κωδικός ρυθμίζει τον απορροφητή να βρίσκει τη λέξη "Λέξη" με ακρίβεια, αντιμετωπίζοντάς την ως ολόκληρη λέξη. Ο`\b` υποδηλώνει όρια λέξεων.

### Παράδειγμα 2: Αναζήτηση χωρίς διάκριση πεζών-κεφαλαίων

Εάν θέλετε να βρείτε μια συγκεκριμένη συμβολοσειρά ανεξάρτητα από την περίπτωση (όπως "γραμμή"), δείτε πώς μπορείτε να το κάνετε αυτό:

```csharp
// Αναζητήστε τη συμβολοσειρά "Γραμμή" με τρόπο χωρίς διάκριση πεζών-κεφαλαίων
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber("(?i)Line", new TextSearchOptions(true));
```
 Ο`(?i)` κάνει την αναζήτηση να αγνοεί πεζά. 

### Παράδειγμα 3: Εξαγωγή όλου του κειμένου από το PDF

Χρειάστηκε ποτέ να αναλύσετε όλο το κείμενο σε ένα PDF; Εδώ είναι ένας απλός τρόπος για να το κάνετε:

```csharp
// Αναλύστε όλες τις συμβολοσειρές μέσα στο έγγραφο PDF
TextFragmentAbsorber textFragmentAbsorber = new TextFragmentAbsorber(@"[\S]+");
```
 Το regex`[\S]+` καταγράφει όλους τους χαρακτήρες χωρίς κενά. 

## Βήμα 3: Αποδεχτείτε τον απορροφητή στο έγγραφο

Τώρα, πρέπει να αποδεχτείτε τον απορροφητή στο έγγραφο PDF:

```csharp
// Αποδεχτείτε τον απορροφητή στο έγγραφο
pdfDocument.Pages.Accept(textFragmentAbsorber);
```
Αυτή η γραμμή λέει στο έγγραφο PDF να επεξεργαστεί τον απορροφητή που αρχικοποιήσατε.

## Βήμα 4: Ανάκτηση και εμφάνιση των αποτελεσμάτων

Τέλος, ας ανακτήσουμε το εξαγόμενο κείμενο και ας το εμφανίσουμε. Δείτε πώς μπορείτε να το κάνετε:

```csharp
// Λάβετε τα εξαγόμενα τμήματα κειμένου
TextFragmentCollection textFragments = textFragmentAbsorber.TextFragments;

// Πραγματοποιήστε βρόχο σε κάθε τμήμα και εξάγετε το
foreach (TextFragment textFragment in textFragments)
{
    Console.WriteLine(textFragment.Text);
}
```
Αυτός ο βρόχος επεξεργάζεται κάθε τμήμα που βρέθηκε και το εκτυπώνει στην κονσόλα.
