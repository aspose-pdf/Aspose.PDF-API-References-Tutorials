## Создание простого PDF-документа

Начнем с создания простого PDF-документа и добавления в него текста. Вот пример кода, который поможет вам начать:

```java
// Инициализировать новый PDF-документ
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document();

// Создать страницу в PDF-документе
com.aspose.pdf.Page page = pdfDocument.getPages().add();

// Создайте объект TextFragment.
com.aspose.pdf.TextFragment textFragment = new com.aspose.pdf.TextFragment("Hello, Aspose.PDF!");

//Установите свойства текста (шрифт, размер, цвет и т. д.)
textFragment.getTextState().setFont(com.aspose.pdf.FontRepository.findFont("Arial"));
textFragment.getTextState().setFontSize(12);
textFragment.getTextState().setForegroundColor(com.aspose.pdf.Color.getBlack());

// Добавьте текст на страницу
page.getParagraphs().add(textFragment);

// Сохраните PDF-документ
pdfDocument.save("output.pdf");
```

Этот код создает новый PDF-документ, добавляет к нему страницу и вставляет текст «Hello, Aspose.PDF!» на странице. Полученный PDF-файл будет называться «output.pdf».

## Манипулирование элементами текстовой структуры

Теперь, когда вы создали простой PDF-документ, давайте рассмотрим, как манипулировать элементами текстовой структуры внутри него. Элементы текстовой структуры необходимы для обеспечения доступности и вспомогательных технологий. Вы можете указать тип структуры и добавить альтернативный текст для пользователей с нарушениями зрения. Вот пример того, как создать структурированный текстовый элемент:

```java
// Создать PDF-документ с тегами
pdfDocument.setTagged(true);

// Включить тегирование для страницы
page.setTagged(true);

// Создать элемент структуры (абзац)
com.aspose.pdf.TaggedPdfElement taggedPdfElement = new com.aspose.pdf.TaggedPdfElement(pdfDocument);

// Задайте тип структуры (в данном случае это абзац)
taggedPdfElement.setStructureType("P");

// Добавьте альтернативный текст для доступности
taggedPdfElement.setAlternativeText("This is a structured paragraph.");

// Добавьте элемент структуры на страницу
page.getParagraphs().add(taggedPdfElement);
```

В этом коде мы создали PDF-документ с тегами и добавили абзац с тегами с указанным типом структуры и альтернативным текстом.

## Расширенное управление текстом

Aspose.PDF для Java предоставляет широкие возможности для расширенных манипуляций с текстом в документах PDF. Вы можете работать с таблицами, шрифтами, стилями и многим другим. Вот пример создания таблицы с форматированным текстом:

```java
// Создать таблицу
com.aspose.pdf.Table table = new com.aspose.pdf.Table();
page.getParagraphs().add(table);

// Добавляем строки и ячейки в таблицу
com.aspose.pdf.Row row = table.getRows().add();
com.aspose.pdf.Cell cell = row.getCells().add();
cell.getParagraphs().add(new com.aspose.pdf.TextFragment("Name"));
cell = row.getCells().add();
cell.getParagraphs().add(new com.aspose.pdf.TextFragment("Age"));

row = table.getRows().add();
cell = row.getCells().add();
cell.getParagraphs().add(new com.aspose.pdf.TextFragment("John"));
cell = row.getCells().add();
cell.getParagraphs().add(new com.aspose.pdf.TextFragment("30"));

// Применить форматирование к таблице
table.setDefaultCellBorder(new com.aspose.pdf.BorderInfo(com.aspose.pdf.BorderSide.All, 1f));
table.setColumnWidths("100 100");
```

Этот фрагмент кода создает простую таблицу с двумя столбцами и добавляет ее в документ PDF.
