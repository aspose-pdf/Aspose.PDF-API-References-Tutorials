## परिचय

.NET में PDF के साथ काम करते समय, Aspose.PDF लाइब्रेरी PDF दस्तावेज़ बनाने, हेरफेर करने और बदलने के लिए एक असाधारण उपकरण है। Aspose.PDF द्वारा प्रदान की जाने वाली उन्नत सुविधाओं में से एक PDF फ़ाइल में टेबल सेल के अंदर HTML सामग्री शामिल करने की क्षमता है। यह ट्यूटोरियल आपको .NET के लिए Aspose.PDF का उपयोग करके इसे प्राप्त करने के तरीके के बारे में मार्गदर्शन करेगा। इस गाइड के अंत तक, आप कोशिकाओं में एम्बेडेड HTML सामग्री के साथ गतिशील रूप से टेबल बनाने में सक्षम होंगे।

```csharp
using System;
using System.Data;
```

अब, आइए कार्य को विस्तृत चरणों में विभाजित करें, जहां हम प्रक्रिया के प्रत्येक भाग को स्पष्ट और संक्षिप्त रूप से समझाएंगे।

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

पहला कदम आपके दस्तावेज़ निर्देशिका का पथ निर्धारित करना है। यह वह जगह है जहाँ पीडीएफ को बनाने और उसमें बदलाव करने के बाद उसे सहेजा जाएगा।

```csharp
// दस्तावेज़ निर्देशिका का पथ निर्धारित करें.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करना सुनिश्चित करें`"YOUR DOCUMENT DIRECTORY"`वास्तविक पथ के साथ जहाँ आप अपनी पीडीएफ फाइल को सहेजना चाहते हैं। यह आवश्यक है ताकि जब दस्तावेज़ तैयार हो जाए, तो आप इसे आसानी से ढूँढ सकें।

## चरण 2: HTML सामग्री के साथ डेटाटेबल बनाएं और भरें

 अब, हम एक बनाते हैं`DataTable` हमारे पीडीएफ में तालिका के अंदर प्रदर्शित होने वाले डेटा को रखने के लिए।`DataTable` HTML सामग्री संग्रहीत करेगा, जैसे कि`<li>` टैग, जिन्हें हम कोशिकाओं के भीतर एम्बेड करना चाहते हैं।

```csharp
// डेटाटेबल बनाएं और कॉलम जोड़ें
DataTable dt = new DataTable("Employee");
dt.Columns.Add("data", System.Type.GetType("System.String"));
```

 एक बार`DataTable` बनाया जाता है, तो आपको इसे HTML सामग्री से भरना होगा जिसे आप तालिका में दिखाना चाहते हैं। इस मामले में, हम पतों के साथ HTML सूची आइटम जोड़ रहे हैं।

```csharp
// HTML सामग्री के साथ पंक्तियाँ जोड़ें
DataRow dr = dt.NewRow();
dr[0] = "<li>Department of Emergency Medicine: 3400 Spruce Street Ground Silverstein Bldg Philadelphia PA 19104-4206</li>";
dt.Rows.Add(dr);
dr = dt.NewRow();
dr[0] = "<li>Penn Observation Medicine Service: 3400 Spruce Street Ground Floor Donner Philadelphia PA 19104-4206</li>";
dt.Rows.Add(dr);
dr = dt.NewRow();
dr[0] = "<li>UPHS/Presbyterian - Dept. of Emergency Medicine: 51 N. 39th Street . Philadelphia PA 19104-2640</li>";
dt.Rows.Add(dr);
```

यह चरण सुनिश्चित करता है कि तालिका कक्षों में HTML-स्वरूपित सामग्री होगी, जो PDF दस्तावेज़ के अंदर उचित रूप से प्रस्तुत की जाएगी।

## चरण 3: एक नया पीडीएफ दस्तावेज़ बनाएँ

एक बार जब हमारे पास हमारा डेटा आ जाता है, तो अगला कदम एक नया पीडीएफ दस्तावेज़ आरंभ करना है। यह दस्तावेज़ कैनवास के रूप में काम करेगा जहाँ हम अपनी तालिका जोड़ेंगे।

```csharp
// एक नया PDF दस्तावेज़ आरंभ करें
Document doc = new Document();
doc.Pages.Add();
```

यह सरल कोड स्निपेट एक रिक्त पीडीएफ दस्तावेज़ बनाता है और उसमें एक नया पृष्ठ जोड़ता है, जिसमें बाद में तालिका शामिल होगी।

## चरण 4: टेबल सेट करें

अब, हम PDF दस्तावेज़ के अंदर टेबल बनाएंगे और उसे सेट अप करेंगे। यह टेबल अपने कॉलम की चौड़ाई और बॉर्डर सेटिंग को परिभाषित करेगी।

```csharp
// तालिका का एक नया उदाहरण आरंभ करें
Aspose.Pdf.Table tableProvider = new Aspose.Pdf.Table();
// तालिका की स्तंभ चौड़ाई निर्धारित करें
tableProvider.ColumnWidths = "400 50";
// टेबल बॉर्डर का रंग हल्का ग्रे पर सेट करें
tableProvider.Border = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.5F, Aspose.Pdf.Color.FromRgb(System.Drawing.Color.LightGray));
// अलग-अलग तालिका कक्षों के लिए बॉर्डर सेट करें
tableProvider.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.5F, Aspose.Pdf.Color.FromRgb(System.Drawing.Color.LightGray));
```

इस चरण में, आपने सफलतापूर्वक एक टेबल बना ली है और टेबल और उसके सेल दोनों के लिए कस्टम कॉलम की चौड़ाई और बॉर्डर सेट कर लिए हैं। कॉलम की चौड़ाई टेबल के अंदर डेटा के उचित संरेखण को सुनिश्चित करती है।

## चरण 5: पैडिंग परिभाषित करें और डेटा आयात करें

 तालिका के दृश्य सौंदर्य को बढ़ाने के लिए, हम कोशिकाओं के लिए पैडिंग परिभाषित करेंगे। फिर, हम आयात करते हैं`DataTable` HTML सामग्री को PDF तालिका में परिवर्तित करें।

```csharp
// तालिका कक्षों के लिए पैडिंग सेट करें
Aspose.Pdf.MarginInfo margin = new Aspose.Pdf.MarginInfo();
margin.Top = 2.5F;
margin.Left = 2.5F;
margin.Bottom = 1.0F;
tableProvider.DefaultCellPadding = margin;

// डेटाटेबल को पीडीएफ टेबल में आयात करें
tableProvider.ImportDataTable(dt, false, 0, 0, 3, 1, true);
```

मार्जिन सेट करके, हम टेबल सेल को कुछ जगह देते हैं, जिससे सामग्री अधिक आकर्षक दिखती है।`ImportDataTable` विधि खींचती है`DataTable` हमने पहले जो बनाया था, उसमें यह सुनिश्चित किया गया था कि HTML सामग्री कोशिकाओं में अंतर्निहित है।

## चरण 6: तालिका को पीडीएफ में जोड़ें और सहेजें

अंत में, हम तालिका को पीडीएफ दस्तावेज़ के पहले पृष्ठ पर जोड़ते हैं और फ़ाइल को सेव करते हैं।

```csharp
// पीडीएफ दस्तावेज़ के पहले पृष्ठ पर तालिका जोड़ें
doc.Pages[1].Paragraphs.Add(tableProvider);

// पीडीएफ दस्तावेज़ सहेजें
doc.Save(dataDir + "HTMLInsideTableCell_out.pdf");
```

इस चरण में, HTML सामग्री वाली तालिका को PDF के प्रथम पृष्ठ पर रखा जाता है, तथा फ़ाइल को निर्दिष्ट निर्देशिका में सहेजा जाता है।

## निष्कर्ष

ऊपर दिए गए चरणों का पालन करके, आपने .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में टेबल सेल के अंदर HTML टैग को सफलतापूर्वक एम्बेड किया है। यह ट्यूटोरियल दर्शाता है कि आप अपने .NET अनुप्रयोगों में गतिशील और आकर्षक PDF दस्तावेज़ बनाने के लिए Aspose.PDF की शक्तिशाली सुविधाओं का लाभ कैसे उठा सकते हैं। चाहे आप HTML सामग्री के साथ चालान, रिपोर्ट या विस्तृत तालिकाएँ बना रहे हों, यह विधि आपकी PDF हेरफेर आवश्यकताओं के लिए एक ठोस आधार प्रदान करती है।

## अक्सर पूछे जाने वाले प्रश्न

### क्या Aspose.PDF तालिका कक्षों के अंदर जटिल HTML सामग्री को संभाल सकता है?  
हां, Aspose.PDF तालिका कक्षों के अंदर सूचियों, छवियों और लिंक सहित HTML टैग्स की एक विस्तृत श्रृंखला को संसाधित और प्रस्तुत कर सकता है।

### मैं तालिका में स्तंभों का आकार कैसे समायोजित कर सकता हूँ?  
 आप कॉलम की चौड़ाई को नियंत्रित कर सकते हैं`ColumnWidths` प्रत्येक कॉलम के लिए चौड़ाई निर्दिष्ट करके संपत्ति का चयन करें।

### क्या तालिका कक्षों के अंदर पाठ को प्रारूपित करना संभव है?  
 बिलकुल! आप HTML टैग का उपयोग कर सकते हैं जैसे`<b>`, `<i>` , और`<u>` तालिका कक्षों के अंदर पाठ को प्रारूपित करने के लिए सामग्री के भीतर।

### यदि मेरी HTML सामग्री तालिका सेल के लिए बहुत बड़ी है तो क्या होगा?  
यदि सामग्री सेल से बाहर निकल जाती है, तो तालिका स्वचालित रूप से समायोजित हो जाएगी, लेकिन आप सामग्री को प्रदर्शित करने के तरीके को नियंत्रित करने के लिए सेल आकार और शब्द रैपिंग विकल्पों को अनुकूलित कर सकते हैं।

### क्या मैं एक PDF दस्तावेज़ में एक से अधिक तालिकाएँ जोड़ सकता हूँ?  
हां, आप पीडीएफ दस्तावेज़ में एकाधिक तालिकाएं जोड़ सकते हैं, बस तालिकाएं जोड़ने के चरणों को दोहराकर, पीडीएफ के प्रत्येक नए पृष्ठ या अनुभाग पर।
