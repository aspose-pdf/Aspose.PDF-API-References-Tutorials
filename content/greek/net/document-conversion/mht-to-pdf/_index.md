## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Πρώτα πράγματα πρώτα, πρέπει να καθορίσετε τη διαδρομή προς τον κατάλογο των εγγράφων σας. Εδώ βρίσκεται το αρχείο MHT και όπου θα αποθηκευτεί το PDF που έχει μετατραπεί.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στο μηχάνημά σας. Αυτό θα μπορούσε να είναι κάτι σαν`@"C:\Documents\"`.

## Βήμα 2: Φόρτωση επιλογών MHT

 Στη συνέχεια, θα χρειαστεί να δημιουργήσετε ένα παράδειγμα του`MhtLoadOptions`. Αυτή η κλάση σάς επιτρέπει να καθορίσετε επιλογές για τη φόρτωση αρχείων MHT.

```csharp
MhtLoadOptions options = new MhtLoadOptions();
```

Αυτό το βήμα είναι κρίσιμο γιατί προετοιμάζει τη βιβλιοθήκη για να χειριστεί σωστά το αρχείο MHT.

## Βήμα 3: Φορτώστε το έγγραφο MHT

 Τώρα είναι ώρα να φορτώσετε το έγγραφο MHT στη βιβλιοθήκη Aspose.PDF. Αυτό γίνεται χρησιμοποιώντας το`Document` τάξη.

```csharp
// Φόρτωση εγγράφου
Document document = new Document(dataDir + "test.mht", options);
```

 Φροντίστε να αντικαταστήσετε`"test.mht"` με το όνομα του αρχείου MHT σας. Αυτή η γραμμή κώδικα διαβάζει το αρχείο MHT και το προετοιμάζει για μετατροπή.

## Βήμα 4: Αποθηκεύστε το Έγγραφο ως PDF

Τέλος, μπορείτε να αποθηκεύσετε το φορτωμένο έγγραφο ως PDF. Εδώ συμβαίνει η μαγεία!

```csharp
// Αποθηκεύστε την έξοδο ως έγγραφο PDF
document.Save(dataDir + "MHTToPDF_out.pdf");
```

Αυτή η γραμμή αποθηκεύει το PDF που έχει μετατραπεί στον ίδιο κατάλογο με το αρχείο MHT. Μπορείτε να αλλάξετε το όνομα του αρχείου εξόδου όπως απαιτείται.
