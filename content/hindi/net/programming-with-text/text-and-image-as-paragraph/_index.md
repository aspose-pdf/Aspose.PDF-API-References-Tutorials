यह ट्यूटोरियल बताता है कि .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में इनलाइन पैराग्राफ़ के रूप में टेक्स्ट और इमेज कैसे जोड़ें। प्रदान किया गया C# स्रोत कोड चरण दर चरण प्रक्रिया को प्रदर्शित करता है।

## आवश्यक शर्तें

ट्यूटोरियल के साथ आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान।
- .NET लाइब्रेरी के लिए Aspose.PDF स्थापित है। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या अपने प्रोजेक्ट में इसे स्थापित करने के लिए NuGet का उपयोग कर सकते हैं।

## चरण 1: प्रोजेक्ट सेट अप करें

अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाकर आरंभ करें और .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें।

## चरण 2: आवश्यक नामस्थान आयात करें

आवश्यक नामस्थानों को आयात करने के लिए अपनी C# फ़ाइल के आरंभ में निम्नलिखित using निर्देश जोड़ें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using Aspose.Pdf.Drawing;
```

## चरण 3: दस्तावेज़ निर्देशिका का पथ सेट करें

 का उपयोग करके अपने दस्तावेज़ निर्देशिका का पथ सेट करें`dataDir` चर:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

## चरण 4: नया दस्तावेज़ और पेज बनाएँ

 एक नया बनाएँ`Document` ऑब्जेक्ट और उसके पृष्ठ संग्रह में एक पृष्ठ जोड़ें:

```csharp
Document doc = new Document();
Page page = doc.Pages.Add();
```

## चरण 5: एक टेक्स्टफ़्रेगमेंट बनाएं और उसे पैराग्राफ़ के रूप में जोड़ें

 एक बनाने के`TextFragment` ऑब्जेक्ट को चुनें और उसे पृष्ठ के पैराग्राफ़ संग्रह में जोड़ें:

```csharp
TextFragment text = new TextFragment("Hello World.. ");
page.Paragraphs.Add(text);
```

## चरण 6: इनलाइन पैराग्राफ़ के रूप में छवि जोड़ें

 एक बनाएं`Aspose.Pdf.Image` ऑब्जेक्ट चुनें और इसे इनलाइन पैराग्राफ के रूप में सेट करें ताकि यह पिछले पैराग्राफ के ठीक बाद दिखाई दे:

```csharp
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
image.IsInLineParagraph = true;
image.File = dataDir + "aspose-logo.jpg";
image.FixHeight = 30; // वैकल्पिक: छवि की ऊंचाई सेट करें
image.FixWidth = 100; // वैकल्पिक: छवि की चौड़ाई सेट करें
page.Paragraphs.Add(image);
```

 प्रतिस्थापित करें`"aspose-logo.jpg"` वास्तविक छवि फ़ाइल नाम के साथ और वैकल्पिक छवि ऊंचाई और चौड़ाई को इच्छानुसार समायोजित करें।

## चरण 7: इनलाइन पैराग्राफ़ के रूप में एक और टेक्स्टफ़्रेगमेंट जोड़ें

 पुनः आरंभ करें`TextFragment` भिन्न सामग्री वाला ऑब्जेक्ट चुनें और उसे इनलाइन पैराग्राफ के रूप में जोड़ें:

```csharp
text = new TextFragment(" Hello Again..");
text.IsInLineParagraph = true;
page.Paragraphs.Add(text);
```

## चरण 8: पीडीएफ दस्तावेज़ सहेजें

संशोधित PDF दस्तावेज़ सहेजें:

```csharp
dataDir = dataDir + "TextAndImageAsParagraph_out.pdf";
doc.Save(dataDir);
```

 प्रतिस्थापित करना सुनिश्चित करें`"TextAndImageAsParagraph_out.pdf"` वांछित आउटपुट फ़ाइल नाम के साथ.

### .NET के लिए Aspose.PDF का उपयोग करके पैराग्राफ के रूप में पाठ और छवि के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ उदाहरण को तत्कालित करें
Document doc = new Document();
// दस्तावेज़ इंस्टेंस के पेज टू पेज संग्रह को जोड़ें
Page page = doc.Pages.Add();
// टेक्स्टफ्रैग्मनेट बनाएं
TextFragment text = new TextFragment("Hello World.. ");
// पेज ऑब्जेक्ट के पैराग्राफ़ संग्रह में टेक्स्ट अंश जोड़ें
page.Paragraphs.Add(text);
// एक छवि उदाहरण बनाएँ
Aspose.Pdf.Image image = new Aspose.Pdf.Image();
// छवि को इनलाइन पैराग्राफ के रूप में सेट करें ताकि वह ठीक बाद में दिखाई दे
// पिछला पैराग्राफ़ ऑब्जेक्ट (TextFragment)
image.IsInLineParagraph = true;
// छवि फ़ाइल पथ निर्दिष्ट करें
image.File = dataDir + "aspose-logo.jpg";
// छवि की ऊंचाई निर्धारित करें (वैकल्पिक)
image.FixHeight = 30;
// छवि की चौड़ाई सेट करें (वैकल्पिक)
image.FixWidth = 100;
// पृष्ठ ऑब्जेक्ट के पैराग्राफ़ संग्रह में छवि जोड़ें
page.Paragraphs.Add(image);
// TextFragment ऑब्जेक्ट को भिन्न सामग्री के साथ पुनः आरंभ करें
text = new TextFragment(" Hello Again..");
// टेक्स्टफ़्रेगमेंट को इनलाइन पैराग्राफ़ के रूप में सेट करें
text.IsInLineParagraph = true;
// पृष्ठ के पैराग्राफ़ संग्रह में नव निर्मित टेक्स्टफ़्रेगमेंट जोड़ें
page.Paragraphs.Add(text);
dataDir = dataDir + "TextAndImageAsParagraph_out.pdf";
doc.Save(dataDir);
Console.WriteLine("\nText and image added successfully as an inline paragraphs.\nFile saved at " + dataDir);
```

## निष्कर्ष

बधाई हो! आपने सफलतापूर्वक सीख लिया है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में इनलाइन पैराग्राफ़ के रूप में टेक्स्ट और इमेज कैसे जोड़ें। इस ट्यूटोरियल में प्रोजेक्ट सेट अप करने से लेकर संशोधित दस्तावेज़ को सहेजने तक, चरण-दर-चरण मार्गदर्शिका प्रदान की गई है। अब आप PDF फ़ाइलों में टेक्स्ट और इमेज के लेआउट को कस्टमाइज़ करने के लिए इस कोड को अपने स्वयं के C# प्रोजेक्ट में शामिल कर सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: "पीडीएफ फाइल में पैराग्राफ के रूप में टेक्स्ट और छवि" ट्यूटोरियल का उद्देश्य क्या है?

उत्तर: "PDF फ़ाइल में पैराग्राफ़ के रूप में टेक्स्ट और इमेज" ट्यूटोरियल का उद्देश्य उपयोगकर्ताओं को यह बताना है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में टेक्स्ट और इमेज दोनों को इनलाइन पैराग्राफ़ के रूप में कैसे जोड़ा जाए। ट्यूटोरियल प्रक्रिया को प्रदर्शित करने के लिए चरण-दर-चरण निर्देश और C# कोड नमूने प्रदान करता है।

#### प्रश्न: यह ट्यूटोरियल इनलाइन पैराग्राफ के रूप में टेक्स्ट और चित्र जोड़ने में कैसे मदद करता है?

उत्तर: यह ट्यूटोरियल उपयोगकर्ताओं को यह समझने में मदद करता है कि .NET के लिए Aspose.PDF का उपयोग कैसे करें ताकि PDF दस्तावेज़ में इनलाइन पैराग्राफ़ के रूप में टेक्स्ट और इमेज दोनों को शामिल किया जा सके। दिए गए चरणों और कोड उदाहरणों का पालन करके, उपयोगकर्ता कस्टम लेआउट के साथ PDF फ़ाइलें बना सकते हैं जो टेक्स्ट और इमेज को मिलाते हैं।

#### प्रश्न: इस ट्यूटोरियल का अनुसरण करने के लिए क्या पूर्वापेक्षाएँ आवश्यक हैं?

उत्तर: ट्यूटोरियल शुरू करने से पहले, आपको C# प्रोग्रामिंग भाषा की बुनियादी समझ होनी चाहिए। इसके अतिरिक्त, आपके पास .NET लाइब्रेरी के लिए Aspose.PDF स्थापित होना चाहिए। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या NuGet का उपयोग करके अपने प्रोजेक्ट में इंस्टॉल कर सकते हैं।

#### प्रश्न: मैं इस ट्यूटोरियल का अनुसरण करने के लिए अपना प्रोजेक्ट कैसे सेट करूँ?

उत्तर: आरंभ करने के लिए, अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाएँ और .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें। यह आपको PDF दस्तावेज़ों, टेक्स्ट अंशों और छवियों के साथ काम करने के लिए लाइब्रेरी की सुविधाओं का उपयोग करने की अनुमति देता है।

#### प्रश्न: क्या मैं इस ट्यूटोरियल का उपयोग पीडीएफ में एकाधिक टेक्स्ट और छवि पैराग्राफ जोड़ने के लिए कर सकता हूं?

उत्तर: हाँ, आप दिए गए कोड सैंपल का उपयोग करके एक ही PDF दस्तावेज़ में टेक्स्ट और इमेज पैराग्राफ़ दोनों के कई उदाहरण जोड़ सकते हैं। यह ट्यूटोरियल इनलाइन पैराग्राफ़ बनाने का तरीका बताता है, जिससे टेक्स्ट और इमेज के अलग-अलग संयोजनों को शामिल करना आसान हो जाता है।

#### प्रश्न: मैं पाठ पैराग्राफ और छवियों की सामग्री और स्वरूप कैसे निर्दिष्ट करूं?

 उत्तर: ट्यूटोरियल दर्शाता है कि कैसे बनाएं`TextFragment`पाठ पैराग्राफ का प्रतिनिधित्व करने के लिए ऑब्जेक्ट्स और`Aspose.Pdf.Image` छवियों का प्रतिनिधित्व करने के लिए ऑब्जेक्ट। आप प्रदान किए गए कोड नमूनों का उपयोग करके पाठ और छवियों दोनों की सामग्री, आयाम और उपस्थिति को अनुकूलित कर सकते हैं।

#### प्रश्न: क्या मैं इनलाइन पैराग्राफ़ के लेआउट को समायोजित कर सकता हूँ?

 उत्तर: हां, आप इनलाइन पैराग्राफ़ के लेआउट को पेज के भीतर उनकी स्थिति, आयाम और क्रम को नियंत्रित करके समायोजित कर सकते हैं। ट्यूटोरियल दिखाता है कि इनलाइन विशेषताएँ कैसे सेट करें, जैसे`IsInLineParagraph`, पाठ और छवि पैराग्राफ के लेआउट को नियंत्रित करने के लिए।

#### प्रश्न: मैं संशोधित पीडीएफ दस्तावेज़ को कैसे सहेजूँ?

 उत्तर: संशोधित पीडीएफ दस्तावेज़ को सहेजने के लिए, आप इसका उपयोग कर सकते हैं`Save` की विधि`Document` यह ट्यूटोरियल कोड नमूने प्रदान करता है जो दर्शाता है कि परिणामी पीडीएफ दस्तावेज़ को कैसे सहेजना है।
