في هذا البرنامج التعليمي، سنتعلم كيفية إضافة نص في رأس ملف PDF باستخدام Aspose.PDF لـ .NET. اتبع الخطوات التالية:

## الخطوة 1: إعداد المشروع

تأكد من أنك قمت بتثبيت Aspose.PDF لـ .NET وقمت بإنشاء مشروع C#.

## الخطوة 2: استيراد المساحات الاسمية

أضف المساحات الأسماء التالية إلى ملف المصدر C# الخاص بك:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## الخطوة 3: فتح المستند

افتح مستند PDF الموجود باستخدام المسار المقدم:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document pdfDocument = new Document(dataDir + "TextinHeader.pdf");
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي إلى دليل المستندات الخاص بك.

## الخطوة 4: إنشاء نص الرأس

قم بإنشاء طابع نصي جديد بالنص الذي تريد إضافته في العنوان:

```csharp
TextStamp textStamp = new TextStamp("Header text");
```

يمكنك تخصيص النص عن طريق تغيير خصائصه مثل الهامش العلوي والمحاذاة الأفقية والمحاذاة الرأسية.

## الخطوة 5: إضافة نص رأس الصفحة إلى جميع الصفحات

قم بالانتقال إلى جميع صفحات مستند PDF وأضف ختم النص في العنوان:

```csharp
foreach(Page page in pdfDocument.Pages)
{
     page.AddStamp(textStamp);
}
```

## الخطوة 6: حفظ مستند PDF

بمجرد إضافة نص الرأس إلى جميع الصفحات، احفظ مستند PDF المحدث:

```csharp
pdfDocument.Save(dataDir + "TextinHeader_out.pdf");
Console.WriteLine("\nText in header added successfully.\nFile saved at: " + dataDir);
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي للدليل الذي تريد حفظ مستند PDF فيه.

### عينة من كود المصدر لـ Textin Header باستخدام Aspose.PDF لـ .NET 
```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// فتح المستند
Document pdfDocument = new Document(dataDir+ "TextinHeader.pdf");

// إنشاء رأس الصفحة
TextStamp textStamp = new TextStamp("Header Text");

// تعيين خصائص الطوابع
textStamp.TopMargin = 10;
textStamp.HorizontalAlignment = HorizontalAlignment.Center;
textStamp.VerticalAlignment = VerticalAlignment.Top;

// إضافة رأس الصفحة إلى جميع الصفحات
foreach (Page page in pdfDocument.Pages)
{
	page.AddStamp(textStamp);
}

// حفظ المستند المحدث
pdfDocument.Save(dataDir+ "TextinHeader_out.pdf");
Console.WriteLine("\nText in header added successfully.\nFile saved at " + dataDir);

```

## خاتمة

تهانينا! لقد تعلمت كيفية إضافة نص في رأس مستند PDF باستخدام Aspose.PDF لـ .NET. يمكنك الآن تخصيص رؤوس المستندات عن طريق إضافة نص إضافي إلى مستندات PDF الخاصة بك.

### الأسئلة الشائعة حول النص الموجود في رأس ملف PDF

#### س: ما هو الغرض من إضافة نص في رأس مستند PDF؟

أ: إضافة نص في رأس مستند PDF يسمح لك بتضمين معلومات مهمة، مثل العناوين أو أسماء المستندات أو التواريخ أو أي نص آخر تريد ظهوره بشكل متسق في أعلى كل صفحة.

#### س: كيف يقوم كود المصدر C# المقدم بإضافة نص في رأس مستند PDF؟

ج: يوضح الكود عملية فتح مستند PDF موجود، وإنشاء ختم نصي بنص الرأس المطلوب، وتخصيص خصائص النص، وإضافة ختم النص إلى جميع الصفحات، وأخيرًا حفظ مستند PDF المحدث بنص الرأس المضاف.

#### س: هل يمكنني تعديل مظهر نص الرأس، مثل الخط والحجم واللون والمحاذاة؟

ج: نعم، يمكنك تخصيص مظهر نص الرأس عن طريق تعديل خصائص`TextStamp` يتضمن مثال التعليمات البرمجية ضبط خصائص مثل الهامش العلوي والمحاذاة الأفقية والمحاذاة الرأسية. يمكنك أيضًا ضبط الخط والحجم واللون وغير ذلك من الخصائص المتعلقة بالنص.

#### س: هل من الممكن إضافة نص مختلف إلى رأس كل صفحة؟

 ج: نعم، يمكنك إضافة نص مختلف إلى رأس كل صفحة عن طريق إنشاء نص منفصل`TextStamp` الكائنات ذات محتوى نصي أو خصائص مختلفة ثم إضافتها إلى صفحات محددة حسب الحاجة.

#### س: كيف يمكنني التأكد من ظهور نص الرأس بشكل متسق في كل صفحة من مستند PDF؟

أ: من خلال استخدام حلقة تتكرر عبر جميع صفحات مستند PDF وإضافة نفس طابع النص إلى كل صفحة، يمكنك التأكد من ظهور نص الرأس بشكل متسق في كل صفحة.

#### س: هل يمكنني إضافة أسطر متعددة من النص أو تنسيق نص الرأس باستخدام فواصل الأسطر؟

 ج: نعم، يمكنك إضافة أسطر متعددة من النص إلى العنوان عن طريق تضمين فواصل الأسطر في سلسلة النص. على سبيل المثال، يمكنك استخدام تسلسل الإفلات`\n` للإشارة إلى كسر السطر في النص.

#### س: ماذا يحدث إذا أردت إضافة محتوى مختلف إلى رأس وتذييل نفس مستند PDF؟

ج: لإضافة محتوى مختلف إلى أقسام الرأس والتذييل، عليك اتباع خطوات مماثلة لكلا القسمين. يوضح الكود إضافة نص إلى الرأس؛ ويمكنك استخدام نهج مماثل لإضافة نص إلى التذييل.

#### س: هل من الممكن إضافة صور أو عناصر أخرى إلى جانب نص العنوان باستخدام هذا النهج؟

ج: على الرغم من أن الكود المقدم يوضح على وجه التحديد إضافة نص إلى الرأس، يمكنك توسيع النهج لإضافة عناصر أخرى مثل الصور أو الخطوط أو الأشكال أو أي محتوى آخر إلى قسم الرأس باستخدام مكتبة Aspose.PDF.
