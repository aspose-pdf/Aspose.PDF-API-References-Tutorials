## Εισαγωγή

Στη σημερινή ψηφιακή εποχή, τα PDF είναι μια κοινή μορφή για κοινή χρήση και παράδοση εγγράφων. Είτε είστε προγραμματιστής, είτε δημιουργός περιεχομένου ή απλά κάποιος που θέλει να βελτιώσει τα αρχεία PDF του, το να ξέρεις πώς να χειρίζεσαι αρχεία PDF μέσω προγραμματισμού μπορεί να αλλάξει το παιχνίδι. Ένα εξαιρετικό χαρακτηριστικό που μπορεί να θέλετε να χρησιμοποιήσετε είναι η δυνατότητα προσθήκης σφραγίδων κειμένου στα αρχεία PDF σας. Η προσθήκη μιας σφραγίδας κειμένου μπορεί να δώσει στα έγγραφά σας μια επαγγελματική πινελιά ή να μεταδώσει σημαντικές πληροφορίες, όπως "Δείγμα", "Εμπιστευτικό" ή ακόμα και ένα υδατογράφημα.

## Εισαγωγή πακέτων

Τώρα, ας εισάγουμε τα απαραίτητα πακέτα. Αυτό το βήμα είναι κρίσιμο, καθώς καθιστά διαθέσιμες τις κλάσεις και τις μεθόδους από τη βιβλιοθήκη Aspose στο έργο σας.

### Εισαγωγή συναρμολόγησης Aspose.PDF

Για να ξεκινήσετε, πρέπει να εισαγάγετε τον χώρο ονομάτων Aspose.PDF. Στην κορυφή του αρχείου C#, προσθέστε τα ακόλουθα χρησιμοποιώντας την οδηγία:

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System;
```

Αυτό θα σας επιτρέψει να αποκτήσετε πρόσβαση σε κλάσεις που είναι απαραίτητες για τη δημιουργία και τον χειρισμό εγγράφων PDF.

Τώρα, ας μπούμε στο κρέας του φροντιστηρίου. Θα αναλύσουμε τη διαδικασία σε σαφή και συνοπτικά βήματα. Κάθε βήμα θα σας καθοδηγήσει στον κώδικα για να προσθέσετε μια σφραγίδα κειμένου σε ένα αρχείο PDF.

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων

Πρώτα, πρέπει να δημιουργήσετε τον κατάλογο όπου είναι αποθηκευμένο το έγγραφο PDF σας. Αυτό σημαίνει ότι ο κώδικάς σας πρέπει να γνωρίζει πού να βρει το αρχείο PDF που θέλετε να επεξεργαστείτε.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Επεξήγηση: Αντικατάσταση`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου το αρχείο PDF σας (`AddTextStamp.pdf`) αποθηκεύεται. Αυτή η διαδρομή χρησιμοποιείται αργότερα για το άνοιγμα και την αποθήκευση του τροποποιημένου PDF.

## Βήμα 2: Ανοίξτε το έγγραφο PDF

 Στη συνέχεια, θα ανοίξουμε το έγγραφο PDF χρησιμοποιώντας το`Document` κλάση από τον χώρο ονομάτων Aspose.PDF.

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "AddTextStamp.pdf");
```

 Επεξήγηση: Εδώ, δημιουργούμε ένα παράδειγμα του`Document` τάξη και περνώντας τη διαδρομή προς το αρχείο PDF μας. Αυτό θα φορτώσει το PDF ώστε να μπορούμε να το χειριστούμε.

## Βήμα 3: Δημιουργήστε μια σφραγίδα κειμένου

Τώρα, θα δημιουργήσουμε μια σφραγίδα κειμένου που θα εφαρμόσουμε αργότερα στο έγγραφο PDF μας.

```csharp
// Δημιουργία σφραγίδας κειμένου
TextStamp textStamp = new TextStamp("Sample Stamp");
```

 Εξήγηση: Το`TextStamp` το αντικείμενο δημιουργείται με το κείμενο που θέλετε να εμφανίσετε. Σε αυτήν την περίπτωση, χρησιμοποιούμε το "Sample Stamp" ως κείμενο για τη σφραγίδα μας.

## Βήμα 4: Ορίστε τις ιδιότητες σφραγίδας

Για να προσαρμόσουμε τη σφραγίδα σας, μπορούμε να ορίσουμε διάφορες ιδιότητες, όπως χρώμα φόντου, θέση και περιστροφή. Ας το κάνουμε τώρα:

```csharp
// Ορίστε εάν η σφραγίδα είναι φόντο
textStamp.Background = true;

// Σύνολο προέλευσης
textStamp.XIndent = 100;
textStamp.YIndent = 100;

// Περιστροφή σφραγίδας
textStamp.Rotate = Rotation.on90;
```

Εξήγηση:
- Φόντο: Ρύθμιση αυτού σε`true` σημαίνει ότι η σφραγίδα θα εμφανίζεται πίσω από το περιεχόμενο του PDF.
- XIndent & YIndent: Αυτές οι ιδιότητες καθορίζουν τη θέση της σφραγίδας στη σελίδα. Σε αυτό το παράδειγμα, η σφραγίδα θα τοποθετηθεί 100 μονάδες από την αριστερή και την επάνω άκρη της σελίδας.
- Περιστροφή: Περιστρέφει τη σφραγίδα κατά 90 μοίρες. Μπορείτε να επιλέξετε διαφορετικές επιλογές περιστροφής με βάση τις απαιτήσεις σχεδιασμού σας.

## Βήμα 5: Προσαρμόστε τις ιδιότητες κειμένου

Στη συνέχεια, ας γίνουμε δημιουργικοί προσαρμόζοντας την εμφάνιση του κειμένου στη σφραγίδα μας:

```csharp
// Ορισμός ιδιοτήτων κειμένου
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(Color.Aqua);
```

Εξήγηση:
- Γραμματοσειρά: Χρησιμοποιούμε τη γραμματοσειρά Arial και την κάνουμε έντονη και πλάγια γραφή.
- FontSize: Έχει οριστεί σε 14 σημεία.
- ForegroundColor: Χρησιμοποιώντας το RGB για να ορίσετε το χρώμα του κειμένου σε Aqua. Μη διστάσετε να αλλάξετε το χρώμα για να ταιριάζει στις ανάγκες σας στο branding ή στο design!

## Βήμα 6: Προσθήκη σφραγίδας στη σελίδα PDF

Τώρα ήρθε η ώρα να προσθέσετε τη σφραγίδα σε μια συγκεκριμένη σελίδα του εγγράφου PDF.

```csharp
// Προσθήκη σφραγίδας σε συγκεκριμένη σελίδα
pdfDocument.Pages[1].AddStamp(textStamp);
```

Επεξήγηση: Σε αυτό το παράδειγμα, η σφραγίδα προστίθεται στην πρώτη σελίδα του PDF (οι σελίδες έχουν ευρετήριο 1). Προσαρμόστε τον αριθμό σελίδας όπως απαιτείται για το έγγραφό σας.

## Βήμα 7: Αποθηκεύστε το τροποποιημένο PDF

Τέλος, ας αποθηκεύσουμε το έγγραφο με τη σφραγίδα κειμένου που προστέθηκε πρόσφατα.

```csharp
dataDir = dataDir + "AddTextStamp_out.pdf";

// Αποθήκευση εγγράφου εξόδου
pdfDocument.Save(dataDir);
Console.WriteLine("\nText stamp added successfully.\nFile saved at " + dataDir);
```

Επεξήγηση: Ορίζουμε μια νέα διαδρομή για το αρχείο εξόδου και στη συνέχεια αποθηκεύουμε το τροποποιημένο έγγραφο. Μετά την αποθήκευση, η διαδρομή εκτυπώνεται στην κονσόλα, επιβεβαιώνοντας την επιτυχή λειτουργία.
