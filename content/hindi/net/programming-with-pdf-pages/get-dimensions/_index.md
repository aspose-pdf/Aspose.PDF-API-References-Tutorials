इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में पेज के आयाम प्राप्त करने की चरण-दर-चरण प्रक्रिया से अवगत कराएँगे। हम बंडल किए गए C# स्रोत कोड की व्याख्या करेंगे और आपको अपने स्वयं के प्रोजेक्ट में इस सुविधा को समझने और लागू करने में मदद करने के लिए एक व्यापक मार्गदर्शिका प्रदान करेंगे। इस ट्यूटोरियल के अंत में, आप जानेंगे कि .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में पेज के आयाम कैसे प्राप्त करें।

## आवश्यक शर्तें
आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान
- आपके विकास परिवेश में .NET के लिए Aspose.PDF स्थापित है

## चरण 1: दस्तावेज़ निर्देशिका निर्धारित करें
सबसे पहले, आपको अपने दस्तावेज़ निर्देशिका का पथ सेट करना होगा। यह वह स्थान है जहाँ आपकी PDF फ़ाइल स्थित है। "आपके दस्तावेज़ निर्देशिका" को उचित पथ से बदलें।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: पीडीएफ दस्तावेज़ खोलें
 फिर आप पीडीएफ फाइल को खोल सकते हैं`Document` Aspose.PDF की क्लास। PDF फ़ाइल के लिए सही पथ निर्दिष्ट करना सुनिश्चित करें।

```csharp
Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
```

## चरण 3: एक रिक्त पृष्ठ जोड़ें (यदि आवश्यक हो)
 यदि PDF दस्तावेज़ में पहले से ही पृष्ठ हैं, तो आप अनुक्रमणिका का उपयोग करके किसी मौजूदा पृष्ठ पर जा सकते हैं`1` (पहले पेज का इंडेक्स 1 है)। अन्यथा, आप दस्तावेज़ में एक नया पेज जोड़ सकते हैं।

```csharp
Page page = pdfDocument.Pages.Count > 0? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
```

## चरण 4: पृष्ठ आयाम प्राप्त करें
 अब आप इसका उपयोग करके पृष्ठ आयाम प्राप्त कर सकते हैं`GetPageRect()` की विधि`Page` ऑब्जेक्ट. यह विधि एक लौटाती है`Rectangle` पृष्ठ के आयाम वाली वस्तु। आप चौड़ाई और ऊंचाई तक पहुँचने के लिए इसका उपयोग कर सकते हैं`Width` और`Height` गुण।

```csharp
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
```

## चरण 5: पृष्ठ घुमाएँ
 यदि आप पृष्ठ को घुमाना चाहते हैं, तो आप इसका उपयोग कर सकते हैं`Rotate` की संपत्ति`Page`इस उदाहरण में, पृष्ठ 90 डिग्री घुमाया गया है।

```csharp
page. Rotate = Rotate. on90;
```

## चरण 6: पृष्ठ आयाम पुनः प्राप्त करें
 पृष्ठ रोटेशन के बाद, आप इसका उपयोग करके पृष्ठ आयाम पुनः प्राप्त कर सकते हैं`GetPageRect()` तरीका।

```csharp
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
```

### .NET के लिए Aspose.PDF का उपयोग करके आयाम प्राप्त करने के लिए नमूना स्रोत कोड 

```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
// पीडीएफ दस्तावेज़ में एक रिक्त पृष्ठ जोड़ता है
Page page = pdfDocument.Pages.Count > 0 ? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
// पृष्ठ की ऊंचाई और चौड़ाई की जानकारी प्राप्त करें
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
// पृष्ठ को 90 डिग्री कोण पर घुमाएँ
page.Rotate = Rotation.on90;
// पृष्ठ की ऊंचाई और चौड़ाई की जानकारी प्राप्त करें
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);

```

## निष्कर्ष
इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में पृष्ठ के आयाम कैसे प्राप्त करें। दिए गए चरणों का पालन करके, आप आसानी से पृष्ठ आयाम निकाल सकते हैं और अन्य PDF हेरफेर ऑपरेशन कर सकते हैं। .NET के लिए Aspose.PDF PDF फ़ाइलों के साथ काम करने के लिए बहुत लचीलापन प्रदान करता है और आपको शक्तिशाली और अनुकूलित समाधान विकसित करने की अनुमति देता है।

इस लाइब्रेरी द्वारा प्रदान की गई सभी सुविधाओं की खोज के लिए Aspose.PDF के दस्तावेज़ीकरण का और अधिक अन्वेषण करने के लिए स्वतंत्र महसूस करें।

### पीडीएफ पृष्ठ आयाम प्राप्त करने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: मैं पीडीएफ फाइल में किसी विशिष्ट पृष्ठ का आयाम कैसे प्राप्त कर सकता हूं?

उत्तर: पीडीएफ फाइल में किसी विशिष्ट पृष्ठ के आयाम प्राप्त करने के लिए, आप इसका उपयोग कर सकते हैं`GetPageRect()` की विधि`Page` .NET के लिए Aspose.PDF में ऑब्जेक्ट। यह विधि एक रिटर्न देती है`Rectangle` पृष्ठ के आयाम (चौड़ाई और ऊंचाई) युक्त ऑब्जेक्ट.

####  प्रश्न: इसका क्या मतलब है?`GetPageRect(true)` method do in the provided C# source code?

 उत्तर:`GetPageRect(true)` दिए गए C# स्रोत कोड में विधि किसी भी घुमाव को लागू करने के बाद पृष्ठ के आयाम लौटाती है। यदि पृष्ठ घुमाया जाता है, तो विधि घुमाए गए पृष्ठ के आयाम लौटाएगी, जो मूल आयामों से भिन्न हो सकते हैं।

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में सभी पृष्ठों के आयाम प्राप्त कर सकता हूँ?

 उत्तर: हां, आप पीडीएफ दस्तावेज़ में सभी पृष्ठों के आयाम प्राप्त कर सकते हैं।`Pages` का संग्रह`Document` वस्तु और का उपयोग कर`GetPageRect(true)` प्रत्येक पृष्ठ के लिए विधि.

#### प्रश्न: मैं किसी पृष्ठ (पोर्ट्रेट या लैंडस्केप) का उसके आयामों के आधार पर अभिविन्यास कैसे निर्धारित कर सकता हूँ?

उत्तर: किसी पेज के आयामों के आधार पर उसका ओरिएंटेशन निर्धारित करने के लिए, आप पेज की चौड़ाई और ऊंचाई की तुलना कर सकते हैं। यदि चौड़ाई ऊंचाई से अधिक है, तो पेज लैंडस्केप ओरिएंटेशन में है, और यदि ऊंचाई चौड़ाई से अधिक है, तो पेज पोर्ट्रेट ओरिएंटेशन में है।

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF का उपयोग करके किसी पृष्ठ के आयाम को संशोधित कर सकता हूँ?

 उत्तर: हां, आप .NET के लिए Aspose.PDF में किसी पृष्ठ के आयाम को संशोधित कर सकते हैं।`Rectangle` पृष्ठ आयामों का प्रतिनिधित्व करने वाली ऑब्जेक्ट में, आप अपनी आवश्यकताओं के अनुसार चौड़ाई और ऊंचाई समायोजित कर सकते हैं और फिर पृष्ठ पर परिवर्तन लागू कर सकते हैं।
