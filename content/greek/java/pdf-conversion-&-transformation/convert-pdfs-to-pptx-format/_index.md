## Φόρτωση εγγράφου PDF

Αρχικά, θα χρειαστεί να φορτώσετε το έγγραφο PDF που θέλετε να μετατρέψετε. Ακολουθεί ένα απόσπασμα κώδικα για να το κάνετε αυτό:

```java
// Φορτώστε το έγγραφο PDF
Document pdfDocument = new Document("sample.pdf");
```

## Μετατροπή PDF σε PPTX

Αφού φορτώσετε το έγγραφο PDF, μπορείτε να προχωρήσετε στη μετατροπή σε μορφή PPTX:

```java
// Δημιουργία μετατροπέα PPTX
PptxSaveOptions pptxSaveOptions = new PptxSaveOptions();
pdfDocument.save("output.pptx", pptxSaveOptions);
```

## Προσαρμογή της μετατροπής PPTX

Μπορείτε να προσαρμόσετε τη μετατροπή PPTX καθορίζοντας διάφορες επιλογές, όπως το μέγεθος της διαφάνειας, την ποιότητα και άλλα. Ακολουθεί ένα παράδειγμα για το πώς να το κάνετε:

```java
// Προσαρμόστε τη μετατροπή PPTX
pptxSaveOptions.setSlideSize(new Size(720, 540));

```
