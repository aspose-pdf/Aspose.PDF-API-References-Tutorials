Η ανάκτηση αριθμών σελίδων που σχετίζονται με σελιδοδείκτες σε αρχείο PDF μπορεί να είναι χρήσιμη για την πλοήγηση. Με το Aspose.PDF για .NET, μπορείτε εύκολα να λάβετε τον αριθμό σελίδας των σελιδοδεικτών ακολουθώντας τον ακόλουθο πηγαίο κώδικα:

## Βήμα 1: Εισαγάγετε τις απαιτούμενες βιβλιοθήκες

Πριν ξεκινήσετε, πρέπει να εισαγάγετε τις απαραίτητες βιβλιοθήκες για το έργο σας C#. Ακολουθεί η απαραίτητη οδηγία εισαγωγής:

```csharp
using Aspose.Pdf.Facades;
```

## Βήμα 2: Ορίστε τη διαδρομή στο φάκελο εγγράφων

 Σε αυτό το βήμα, πρέπει να καθορίσετε τη διαδρομή προς το φάκελο που περιέχει το αρχείο PDF από το οποίο θέλετε να εξαγάγετε τους αριθμούς σελίδων σελιδοδεικτών. Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"`στον ακόλουθο κώδικα με την πραγματική διαδρομή προς το φάκελο των εγγράφων σας:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 3: Δημιουργήστε το πρόγραμμα επεξεργασίας σελιδοδεικτών

 Τώρα θα δημιουργήσουμε ένα`PdfBookmarkEditor` αντικείμενο χειρισμού των σελιδοδεικτών του εγγράφου. Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
```

## Βήμα 4: Ανοίξτε το Αρχείο PDF

 Σε αυτό το βήμα, ανοίγουμε το αρχείο PDF χρησιμοποιώντας το`BindPdf` μέθοδο του προγράμματος επεξεργασίας σελιδοδεικτών. Εδώ είναι ο αντίστοιχος κωδικός:

```csharp
bookmarkEditor.BindPdf(dataDir + "GetBookmarks.pdf");
```

## Βήμα 5: Εξαγωγή σελιδοδεικτών

 Τώρα θα εξαγάγουμε τους σελιδοδείκτες από το έγγραφο χρησιμοποιώντας το`ExtractBookmarks` μέθοδο του προγράμματος επεξεργασίας σελιδοδεικτών. Εδώ είναι ο αντίστοιχος κωδικός:

```csharp
Bookmarks bookmarks = bookmarkEditor.ExtractBookmarks();
```

## Βήμα 6: Περιηγηθείτε σε σελιδοδείκτες και λάβετε αριθμούς σελίδων

 Τέλος, πραγματοποιούμε βρόχο στους εξαγόμενους σελιδοδείκτες και παίρνουμε τους αριθμούς σελίδων που σχετίζονται με κάθε σελιδοδείκτη χρησιμοποιώντας α`foreach` βρόχος. Εδώ είναι ο αντίστοιχος κωδικός:

```csharp
foreach (Bookmark bookmark in bookmarks)
{
     string strLevelSeprator = string.Empty;
     for (int i = 1; i < bookmark.Level; i++)
     {
         strLevelSeprator += "----";
     }
     Console.WriteLine("{0}Title: {1}", strLevelSeprator, bookmark.Title);
     Console.WriteLine("{0}Page number: {1}", strLevelSeprator, bookmark.PageNumber);
     Console.WriteLine("{0}Page Action: {1}", strLevelSeprator, bookmark.Action);
}
```

### Δείγμα πηγαίου κώδικα για Λήψη αριθμού σελίδας σελιδοδείκτη χρησιμοποιώντας το Aspose.PDF για .NET 
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Δημιουργία PdfBookmarkEditor
PdfBookmarkEditor bookmarkEditor = new PdfBookmarkEditor();
// Ανοίξτε το αρχείο PDF
bookmarkEditor.BindPdf(dataDir + "GetBookmarks.pdf");
// Εξαγωγή σελιδοδεικτών
Aspose.Pdf.Facades.Bookmarks bookmarks = bookmarkEditor.ExtractBookmarks();
foreach (Aspose.Pdf.Facades.Bookmark bookmark in bookmarks)
{
	string strLevelSeprator = string.Empty;
	for (int i = 1; i < bookmark.Level; i++)
	{
		strLevelSeprator += "----";
	}
	Console.WriteLine("{0}Title: {1}", strLevelSeprator, bookmark.Title);
	Console.WriteLine("{0}Page Number: {1}", strLevelSeprator, bookmark.PageNumber);
	Console.WriteLine("{0}Page Action: {1}", strLevelSeprator, bookmark.Action);
}
```
