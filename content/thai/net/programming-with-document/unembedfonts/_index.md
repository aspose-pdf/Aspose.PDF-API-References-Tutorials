## การแนะนำ

ในยุคดิจิทัล PDF มีอยู่ทั่วไป ไม่ว่าคุณจะแชร์รายงาน งานนำเสนอ หรืออีบุ๊ก Portable Document Format (PDF) เป็นตัวเลือกแรกๆ ในการรักษาความสมบูรณ์ของเอกสารของคุณ อย่างไรก็ตาม เมื่อเราสร้างและแชร์ PDF มากขึ้น ขนาดไฟล์อาจเพิ่มขึ้น ทำให้ส่งหรือจัดเก็บได้ยาก นี่คือจุดที่ Aspose.PDF สำหรับ .NET เข้ามามีบทบาท โดยนำเสนอเครื่องมืออันทรงพลังเพื่อเพิ่มประสิทธิภาพไฟล์ PDF ของคุณ ในบทช่วยสอนนี้ เราจะเจาะลึกถึงวิธีการแยกแบบอักษรและเพิ่มประสิทธิภาพไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET

ก่อนที่เราจะเข้าสู่รายละเอียด เรามาตรวจสอบก่อนว่าคุณมีทุกสิ่งที่จำเป็นสำหรับการเริ่มต้น:

1. Visual Studio: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Visual Studio ไว้ในเครื่องของคุณแล้ว ซึ่งเป็น IDE ที่เราจะใช้เขียนและรันโค้ด .NET
2.  Aspose.PDF สำหรับ .NET: คุณจะต้องดาวน์โหลดและติดตั้งไลบรารี Aspose.PDF คุณสามารถดาวน์โหลดได้จาก[ลิงค์ดาวน์โหลด](https://releases.aspose.com/pdf/net/).
3. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับการเขียนโปรแกรม C# จะช่วยให้คุณเข้าใจชิ้นส่วนโค้ดที่เราจะใช้
4.  ไฟล์ PDF: เตรียมไฟล์ PDF ที่คุณต้องการเพิ่มประสิทธิภาพ คุณสามารถใช้ PDF ใดก็ได้ แต่สำหรับการสาธิต เราจะเรียกว่า`OptimizeDocument.pdf`.

## แพ็คเกจนำเข้า

ในการเริ่มต้น คุณต้องนำเข้าแพ็คเกจที่จำเป็นลงในโปรเจ็กต์ C# ของคุณ โดยคุณสามารถทำได้ดังนี้:

1. เปิดโปรเจ็กต์ของคุณใน Visual Studio
2. เพิ่มการอ้างอิงถึง Aspose.PDF: คลิกขวาที่โครงการของคุณใน Solution Explorer เลือก "จัดการแพ็คเกจ NuGet" และค้นหา`Aspose.PDF`. ติดตั้งแพคเกจ

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

ตอนนี้เราได้ตั้งค่าทุกอย่างเรียบร้อยแล้ว มาแบ่งกระบวนการเพิ่มประสิทธิภาพออกเป็นขั้นตอนที่จัดการได้

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ขั้นแรก คุณต้องกำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือที่ที่คุณจะเก็บไฟล์ PDF ไว้ วิธีดำเนินการมีดังนี้

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงที่ไฟล์ PDF ของคุณตั้งอยู่ ซึ่งเป็นสิ่งสำคัญเนื่องจากโปรแกรมจำเป็นต้องทราบว่าจะค้นหา PDF ที่คุณต้องการเพิ่มประสิทธิภาพได้ที่ใด

## ขั้นตอนที่ 2: เปิดเอกสาร PDF

ตอนนี้เราได้ตั้งค่าไดเรกทอรีเรียบร้อยแล้ว ถึงเวลาเปิดเอกสาร PDF ที่เราต้องการเพิ่มประสิทธิภาพ นี่คือโค้ดสำหรับดำเนินการดังกล่าว:

```csharp
// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "OptimizeDocument.pdf");
```

 บรรทัดโค้ดนี้จะสร้างสิ่งใหม่`Document` อ็อบเจ็กต์ที่แสดงถึงไฟล์ PDF ของคุณ ตรวจสอบให้แน่ใจว่าชื่อไฟล์ตรงกับชื่อที่คุณมีในไดเร็กทอรี

## ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการเพิ่มประสิทธิภาพ

ต่อไปเราต้องระบุตัวเลือกการเพิ่มประสิทธิภาพ ในกรณีนี้ เราต้องการยกเลิกการฝังแบบอักษร วิธีตั้งค่ามีดังนี้:

```csharp
// ตั้งค่าตัวเลือก UnembedFonts
var optimizeOptions = new Pdf.Optimization.OptimizationOptions
{
    UnembedFonts = true
};
```

 โดยการตั้งค่า`UnembedFonts` ถึง`true`เรากำลังสั่งให้ Aspose.PDF เพิ่มประสิทธิภาพ PDF โดยยกเลิกการฝังฟอนต์ การทำเช่นนี้จะช่วยลดขนาดไฟล์ได้อย่างมาก โดยเฉพาะอย่างยิ่งหาก PDF มีฟอนต์ฝังอยู่จำนวนมาก

## ขั้นตอนที่ 4: เพิ่มประสิทธิภาพเอกสาร PDF

เมื่อตั้งค่าตัวเลือกเรียบร้อยแล้ว ก็ถึงเวลาปรับแต่งเอกสาร PDF ได้แล้ว นี่คือโค้ดสำหรับดำเนินการดังกล่าว:

```csharp
Console.WriteLine("Start");
// เพิ่มประสิทธิภาพเอกสาร PDF โดยใช้ OptimizationOptions
pdfDocument.OptimizeResources(optimizeOptions);
```

โค้ดตัวอย่างนี้เรียกใช้`OptimizeResources` วิธีการบน`pdfDocument` วัตถุ โดยใช้ตัวเลือกการเพิ่มประสิทธิภาพที่เรากำหนดไว้ก่อนหน้านี้ คุณจะเห็นข้อความในคอนโซลที่ระบุว่ากระบวนการเพิ่มประสิทธิภาพได้เริ่มต้นขึ้นแล้ว

## ขั้นตอนที่ 5: บันทึกเอกสารที่อัปเดต

หลังจากปรับแต่ง PDF แล้ว เราจำเป็นต้องบันทึกเอกสารที่อัปเดต วิธีดำเนินการมีดังนี้

```csharp
// บันทึกเอกสารอัพเดต
pdfDocument.Save(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Finished");
```

 รหัสนี้จะบันทึก PDF ที่ได้รับการเพิ่มประสิทธิภาพเป็น`OptimizeDocument_out.pdf` ในไดเร็กทอรีเดียวกัน คุณสามารถเลือกชื่ออื่นได้หากต้องการ แต่การคงชื่อที่คล้ายกันจะช่วยให้ระบุเวอร์ชันต้นฉบับและเวอร์ชันที่ปรับให้เหมาะสมได้

## ขั้นตอนที่ 6: เปรียบเทียบขนาดไฟล์

สุดท้ายนี้ การตรวจสอบว่าคุณประหยัดพื้นที่ไปได้เท่าไรนั้นถือเป็นเรื่องที่ดีเสมอ ต่อไปนี้คือวิธีเปรียบเทียบขนาดไฟล์ต้นฉบับและขนาดไฟล์ที่ปรับให้เหมาะสม:

```csharp
var fi1 = new System.IO.FileInfo(dataDir + "OptimizeDocument.pdf");
var fi2 = new System.IO.FileInfo(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Original file size: {0}. Reduced file size: {1}", fi1.Length, fi2.Length);
```

โค้ดนี้จะดึงข้อมูลขนาดไฟล์ของทั้งไฟล์ PDF ต้นฉบับและไฟล์ PDF ที่ปรับให้เหมาะสมแล้ว แล้วพิมพ์ไฟล์ดังกล่าวไปยังคอนโซล นับเป็นช่วงเวลาอันน่าพอใจที่ได้เห็นว่าคุณลดขนาดไฟล์ได้มากเพียงใด!
