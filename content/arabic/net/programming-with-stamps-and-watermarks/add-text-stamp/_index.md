في هذا البرنامج التعليمي، سنوضح لك خطوة بخطوة كيفية إضافة ختم نصي في ملف PDF باستخدام Aspose.PDF لـ .NET. وسنوضح لك كيفية استخدام كود المصدر C# المقدم لإضافة ختم نصي مخصص إلى صفحة معينة من ملف PDF.

## الخطوة 1: إعداد البيئة

قبل أن تبدأ، تأكد من أن لديك ما يلي:

- بيئة تطوير .NET مثبتة.
- تم تنزيل مكتبة Aspose.PDF لـ .NET والإشارة إليها في مشروعك.

## الخطوة 2: تحميل مستند PDF

الخطوة الأولى هي تحميل مستند PDF الموجود في مشروعك. وإليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// افتح المستند
Document pdfDocument = new Document(dataDir + "AddTextStamp.pdf");
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي إلى الدليل الذي يوجد به مستند PDF الخاص بك.

## الخطوة 3: إنشاء مخزن النص

الآن بعد أن قمت بتحميل مستند PDF، يمكنك إنشاء ختم النص لإضافته. وإليك كيفية القيام بذلك:

```csharp
// إنشاء مخزن النص
TextStamp textStamp = new TextStamp("Example Stamp");
```

يقوم الكود أعلاه بإنشاء مخزن نص جديد يحتوي على النص المحدد.

## الخطوة 4: تكوين خصائص ختم النص

قبل إضافة ختم النص إلى مستند PDF، يمكنك تكوين خصائص مختلفة للختم، مثل الخلفية والموضع والدوران والخط والحجم وما إلى ذلك. وإليك الطريقة:

```csharp
// تكوين خصائص مخزن النص
textStamp. Background = true;
textStamp. XIndent = 100;
textStamp. YIndent = 100;
textStamp.Rotate = Rotate.on90;
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);
```

يمكنك تعديل هذه الخصائص وفقًا لاحتياجاتك.

## الخطوة 5: إضافة ختم النص إلى ملف PDF

الآن بعد أن أصبح ختم النص جاهزًا، يمكنك إضافته إلى صفحة معينة من مستند PDF. إليك الطريقة:

```csharp
//إضافة مخزن نصي إلى صفحة معينة
pdfDocument.Pages[1].AddStamp(textStamp);
```

يضيف الكود أعلاه ختم النص إلى الصفحة الأولى من مستند PDF. يمكنك تحديد صفحة أخرى إذا لزم الأمر.

## الخطوة 6: احفظ المستند الناتج

بمجرد إضافة ختم النص، يمكنك حفظ مستند PDF المحرر. إليك الطريقة:

```csharp
// حفظ المستند الناتج
pdfDocument.Save(dataDir);
```

يقوم الكود أعلاه بحفظ مستند PDF المعدل في الدليل المحدد.

### عينة من كود المصدر لإضافة ختم نص باستخدام Aspose.PDF لـ .NET 
```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// فتح المستند
Document pdfDocument = new Document(dataDir+ "AddTextStamp.pdf");

// إنشاء ختم نصي
TextStamp textStamp = new TextStamp("Sample Stamp");

// تعيين ما إذا كان الختم هو الخلفية
textStamp.Background = true;

// مجموعة الأصل
textStamp.XIndent = 100;
textStamp.YIndent = 100;

// تدوير الطوابع
textStamp.Rotate = Rotation.on90;

// تعيين خصائص النص
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold;
textStamp.TextState.FontStyle = FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);

// إضافة طابع إلى صفحة معينة
pdfDocument.Pages[1].AddStamp(textStamp);
dataDir = dataDir + "AddTextStamp_out.pdf";

// حفظ المستند الناتج
pdfDocument.Save(dataDir);
Console.WriteLine("\nText stamp added successfully.\nFile saved at " + dataDir);            

```

#### س: ما هو الغرض من إضافة طابع نصي في ملف PDF باستخدام Aspose.PDF لـ .NET؟

ج: تتيح لك إضافة ختم نصي وضع نص مخصص على صفحة معينة من مستند PDF. تعد هذه الميزة مفيدة لإضافة تسميات أو تعليقات أو علامات مائية أو أي معلومات نصية أخرى لتحسين محتوى المستند وتوفير سياق إضافي.

#### س: هل يمكنني تخصيص مظهر ختم النص، مثل الخط والحجم واللون والدوران؟

 ج: نعم، يمكنك تخصيص مظهر ختم النص بالكامل. يوضح كود المصدر C# المقدم كيفية تعيين خصائص مختلفة لختم النص.`TextStamp` الكائن، بما في ذلك الخط وحجم الخط ونمط الخط ولون النص ولون الخلفية والدوران.

#### س: هل من الممكن إضافة عدة طوابع نصية إلى صفحات مختلفة من نفس مستند PDF؟

ج: بالتأكيد، يمكنك إضافة عدة طوابع نصية إلى صفحات مختلفة من نفس مستند PDF. يتيح لك الكود المقدم في البرنامج التعليمي تحديد الصفحة المستهدفة لإضافة ختم النص، مما يجعله متعدد الاستخدامات لصفحات مختلفة داخل المستند.

#### س: كيف يمكنني تحديد موضع ختم النص داخل مستند PDF؟

 أ: يمكنك تخصيص موضع ختم النص عن طريق تعديل`XIndent` و`YIndent` خصائص`TextStamp` الكائن. تحدد هذه الخصائص إحداثيات الزاوية العلوية اليسرى للطابع بالنسبة إلى أصل الصفحة.

#### س: هل يمكنني تطبيق هذه الطريقة على مستندات PDF الموجودة لإضافة طوابع نصية؟

ج: نعم، يمكنك تطبيق هذه الطريقة على مستندات PDF الموجودة لإضافة ختم نصي. يوضح الكود المقدم في البرنامج التعليمي كيفية تحميل مستند PDF موجود وإضافة ختم نصي إلى صفحة معينة.

#### س: هل يمكنني إضافة كل من الألوان الخلفية والأمامية إلى ختم النص؟

 ج: نعم، يمكنك إضافة ألوان الخلفية والأمامية إلى ختم النص. من خلال ضبط`Background` الممتلكات ل`true` يمكنك توفير خلفية ملونة لختم النص. بالإضافة إلى ذلك، يمكنك ضبط`TextState.ForegroundColor` الخاصية لتحديد لون النص نفسه.

#### س: كيف يمكنني التأكد من أن ختم النص لا يحجب المحتوى الأساسي لمستند PDF؟

أ: عند إضافة ختم نصي، انتبه إلى موضعه للتأكد من أنه لا يعيق المعلومات المهمة أو يؤثر سلبًا على قابلية قراءة المستند. يمكنك ضبط`XIndent` و`YIndent` خصائص لتحديد موضع ختم النص بشكل مناسب.

#### س: هل يمكنني استخدام هذه الطريقة لإضافة طوابع أخرى غير النص، مثل الصور أو الشعارات؟

ج: يركز هذا البرنامج التعليمي المحدد على إضافة طوابع نصية، ولكن يمكنك أيضًا إضافة أنواع أخرى من الطوابع، مثل الصور أو الشعارات، باستخدام Aspose.PDF لـ .NET. تتضمن العملية إنشاء كائن الطوابع المناسب وتكوين خصائصه.

#### س: كيف يمكنني أتمتة عملية إضافة طوابع النص إلى مستندات PDF المتعددة؟

ج: يمكنك أتمتة عملية إضافة طوابع نصية إلى مستندات PDF متعددة عن طريق إنشاء نص برمجي أو برنامج يتكرر عبر قائمة المستندات ويطبق نفس عملية ختم النص على كل منها.
