في هذا البرنامج التعليمي، سنوضح لك كيفية تعيين حجم الصورة في ملف PDF باستخدام Aspose.PDF لـ .NET. اتبع الخطوات التالية لإجراء هذه العملية بسهولة.

## المتطلبات الأساسية

قبل أن تبدأ، تأكد من أن لديك ما يلي:

- تم تثبيت وتكوين Visual Studio أو أي بيئة تطوير أخرى.
- المعرفة الأساسية للغة البرمجة C#.
- تم تثبيت مكتبة Aspose.PDF لـ .NET. يمكنك تنزيلها من موقع Aspose الرسمي.

## الخطوة 1: إنشاء مستند PDF

للبدء، استخدم الكود التالي لإنشاء مستند PDF جديد:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// إنشاء كائن مستند
Document doc = new Document();

// إضافة صفحة إلى مجموعة صفحات ملف PDF
Aspose.Pdf.Page page = doc.Pages.Add();
```

## الخطوة 2: الصورة المضافة

بعد ذلك، سنضيف صورة إلى صفحة مستند PDF. استخدم الكود التالي:

```csharp
// إنشاء مثيل للصورة
Aspose.Pdf.Image img = new Aspose.Pdf.Image();

// ضبط عرض وارتفاع الصورة بالنقاط
img. FixWidth = 100;
img. FixHeight = 100;

//تعيين نوع الصورة إلى غير معروف (غير معروف)
img.FileType = Aspose.Pdf.ImageFileType.Unknown;

// المسار إلى ملف مصدر الصورة
img.File = dataDir + "aspose-logo.jpg";

// أضف الصورة إلى مجموعة فقرات الصفحة
page.Paragraphs.Add(img);
```

تأكد من توفير المسار الصحيح لملف مصدر الصورة.

## الخطوة 3: ضبط خصائص الصفحة

أخيرًا، سنقوم بتعيين خصائص الصفحة، بما في ذلك عرضها وارتفاعها. استخدم الكود التالي:

```csharp
// تعيين خصائص الصفحة
page.PageInfo.Width = 800;
page.PageInfo.Height = 800;
```

### عينة من كود المصدر لتعيين حجم الصورة باستخدام Aspose.PDF لـ .NET 
```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// إنشاء كائن مستند
Document doc = new Document();
// إضافة صفحة إلى مجموعة صفحات ملف PDF
Aspose.Pdf.Page page = doc.Pages.Add();
// إنشاء مثيل للصورة
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
// تعيين عرض الصورة وارتفاعها بالنقاط
img.FixWidth = 100;
img.FixHeight = 100;
// تعيين نوع الصورة كـ SVG
img.FileType = Aspose.Pdf.ImageFileType.Unknown;
// المسار لملف المصدر
img.File = dataDir + "aspose-logo.jpg";
page.Paragraphs.Add(img);
//تعيين خصائص الصفحة
page.PageInfo.Width = 800;
page.PageInfo.Height = 800;
dataDir = dataDir + "SetImageSize_out.pdf";
// حفظ ملف PDF الناتج
doc.Save(dataDir);
Console.WriteLine("\nImage size added successfully.\nFile saved at " + dataDir);
```

## خاتمة

تهانينا! لقد نجحت في ضبط حجم الصورة في مستند PDF باستخدام Aspose.PDF for .NET. يمكنك الآن تطبيق هذه الطريقة على مشاريعك الخاصة لضبط حجم الصور في ملفات PDF.

### الأسئلة الشائعة حول تحديد حجم الصورة في ملف PDF

#### س: ما هو الغرض من تحديد حجم الصورة في مستند PDF باستخدام Aspose.PDF لـ .NET؟

ج: الغرض من ضبط حجم الصورة في مستند PDF هو التحكم في أبعاد الصورة عند إضافتها إلى ملف PDF. يتيح لك هذا ضبط مظهر وتخطيط الصور داخل ملفات PDF الخاصة بك.

#### س: كيف تتم عملية ضبط حجم الصورة في مستند PDF؟

 أ: تتضمن العملية إنشاء`Aspose.Pdf.Image` على سبيل المثال، تحديد عرضه وارتفاعه باستخدام`FixWidth` و`FixHeight` الخصائص، ثم إضافة الصورة إلى مستند PDF. بالإضافة إلى ذلك، يمكنك ضبط أبعاد الصفحة نفسها لاستيعاب الصورة.

#### س: هل يمكنني ضبط حجم الصورة إلى نسبة معينة من أبعاد الصفحة؟

ج: يحدد الكود المقدم العرض والارتفاع المطلقين للصورة بالنقاط. إذا كنت تريد تعيين حجم الصورة بناءً على نسبة مئوية من أبعاد الصفحة، فستحتاج إلى حساب الأبعاد وفقًا لذلك وضبط الكود وفقًا لذلك.

####  س: ما هي أهمية`FileType` property when adding an image to the PDF document?

 أ: ال`FileType`تحدد الخاصية نوع الصورة التي تتم إضافتها إلى مستند PDF. في الكود المقدم، القيمة`Unknown` يشير إلى أن نوع الصورة غير معروف، وسيحاول Aspose.PDF تحديد نوع الصورة استنادًا إلى امتداد الملف.

#### س: هل يمكنني إضافة صور متعددة إلى صفحة واحدة باستخدام هذه الطريقة؟

 ج: نعم، يمكنك إضافة صور متعددة إلى صفحة واحدة عن طريق إنشاء عدة`Aspose.Pdf.Image` قم بنسخ الأمثلة وإضافتها إلى مجموعة فقرات الصفحة. تأكد من ضبط موضع الصور وتخطيطها حسب الحاجة.

#### س: كيف يمكنني التحكم في وضع ومحاذاة الصورة المضافة على الصفحة؟

 أ: يمكن التحكم في وضع الصورة المضافة ومحاذاتها عن طريق ضبط إحداثيات وتخطيط الصورة باستخدام خصائص مثل`img.Left`, `img.Top`، وخصائص تنسيق الفقرة.

####  س: ما هو الغرض من ضبط خصائص الصفحة باستخدام`page.PageInfo.Width` and `page.PageInfo.Height`?

ج: يتيح لك ضبط خصائص الصفحة تحديد أبعاد الصفحة نفسها. وهذا يضمن أن أبعاد الصفحة تستوعب الصورة المضافة وأي محتوى آخر قد يكون موجودًا على الصفحة.

#### س: هل يمكنني تعيين أحجام مختلفة لصور مختلفة ضمن نفس مستند PDF؟

 ج: نعم، يمكنك تعيين أحجام مختلفة لصور مختلفة عن طريق إنشاء أحجام منفصلة`Aspose.Pdf.Image` الحالات وضبطها`FixWidth`, `FixHeight`، وخصائص الموضع لكل صورة.

#### س: كيف يمكنني دمج هذه الطريقة في مشاريعي الخاصة لتعيين أحجام الصور في ملفات PDF؟

ج: لدمج هذه الطريقة في مشاريعك، اتبع الخطوات الموضحة وقم بتعديل الكود حسب الحاجة. يمكنك استخدام هذه الطريقة لإضافة صور بأحجام معينة إلى مستندات PDF الخاصة بك بناءً على متطلبات تطبيقك.
