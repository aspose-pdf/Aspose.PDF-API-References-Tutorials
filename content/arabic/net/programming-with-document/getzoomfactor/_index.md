Aspose.PDF for .NET هي مكتبة معالجة PDF توفر العديد من الميزات لإجراء عمليات مختلفة على مستندات PDF. إحدى هذه الميزات هي القدرة على الحصول على عامل التكبير في ملف PDF. في هذا البرنامج التعليمي، سنشرح كيفية استخدام Aspose.PDF for .NET للحصول على عامل التكبير في ملف PDF باستخدام كود المصدر C#.


## الخطوة 1: إنشاء كائن مستند جديد

 الخطوة الأولى للحصول على عامل التكبير لملف PDF باستخدام Aspose.PDF لـ .NET هي إنشاء مثيل جديد`Document` الكائن.`Document` يمثل الكائن مستند PDF الذي يمكن تحميله من ملف أو مجرى.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// إنشاء كائن مستند جديد
Document doc = new Document(dataDir + "Zoomed_pdf.pdf");
```

 في الكود أعلاه، قمنا بإنشاء`Document` الكائن عن طريق تمرير مسار ملف PDF إلى منشئ`Document` يجب عليك استبدال "دليل المستندات الخاص بك" بالمسار الفعلي للدليل الذي يوجد به ملف PDF الخاص بك.

## الخطوة 2: إنشاء كائن GoToAction

 الخطوة التالية هي إنشاء`GoToAction` كائن. أ`GoToAction`يمثل الكائن إجراءً ينتقل إلى وجهة محددة في مستند PDF. في حالتنا، نريد الحصول على عامل التكبير لملف PDF، لذا سنستخدم`OpenAction` ممتلكات`Document` كائن للحصول على`GoToAction` هدف.

```csharp
// إنشاء كائن GoToAction
GoToAction action = doc.OpenAction as GoToAction;
```

 في الكود أعلاه، قمنا بإنشاء`GoToAction` كائن عن طريق الصب`OpenAction` ممتلكات`Document` الاعتراض على`GoToAction`.

## الخطوة 3: الحصول على عامل التكبير لملف PDF

 الخطوة الثالثة هي الحصول على عامل التكبير لملف PDF. يمكننا الحصول على عامل التكبير لملف PDF من خلال الوصول إلى`Destination` ممتلكات`GoToAction` الكائن ثم إرساله إلى`XYZExplicitDestination` . ال`XYZExplicitDestination` تمثل الفئة وجهة في مستند PDF تحدد الإحداثيات وعامل التكبير للانتقال إليها.

```csharp
// احصل على عامل التكبير لملف PDF
System.Console.WriteLine((action.Destination as XYZExplicitDestination).Zoom); // قيمة تكبير المستند؛
```

 في الكود أعلاه، قمنا بالوصول إلى`Destination` ممتلكات`GoToAction` الكائن ثم قم بإلقائه إلى`XYZExplicitDestination` . بعد ذلك، قمنا بالوصول إلى`Zoom` ممتلكات`XYZExplicitDestination` كائن للحصول على عامل التكبير لملف PDF.

## الخطوة 4: إخراج عامل التكبير

 الخطوة الأخيرة هي إخراج عامل التكبير لملف PDF. يمكننا استخدام`System.Console.WriteLine`

```csharp
// احصل على عامل التكبير لملف PDF
System.Console.WriteLine((action.Destination as XYZExplicitDestination).Zoom); // قيمة تكبير المستند؛
```        

### مثال على كود المصدر للحصول على عامل التكبير باستخدام Aspose.PDF لـ .NET

فيما يلي مثال كامل لمصدر الكود للحصول على عامل التكبير باستخدام Aspose.PDF لـ .NET:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// إنشاء كائن مستند جديد
Document doc = new Document(dataDir + "Zoomed_pdf.pdf");

// إنشاء كائن GoToAction
GoToAction action = doc.OpenAction as GoToAction;

// احصل على عامل التكبير لملف PDF
System.Console.WriteLine((action.Destination as XYZExplicitDestination).Zoom); // قيمة تكبير المستند؛
```
