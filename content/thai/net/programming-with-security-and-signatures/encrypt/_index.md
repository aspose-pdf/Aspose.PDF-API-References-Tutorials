การเข้ารหัสไฟล์ PDF เป็นมาตรการรักษาความปลอดภัยที่สำคัญเพื่อปกป้องข้อมูลที่เป็นความลับ ด้วย Aspose.PDF สำหรับ .NET คุณสามารถเข้ารหัสไฟล์ PDF ได้อย่างง่ายดายโดยใช้ซอร์สโค้ดต่อไปนี้:

## ขั้นตอนที่ 1: นำเข้าไลบรารีที่จำเป็น

ก่อนที่คุณจะเริ่มต้น คุณต้องนำเข้าไลบรารีที่จำเป็นสำหรับโครงการ C# ของคุณ ต่อไปนี้คือคำสั่งนำเข้าที่จำเป็น:

```csharp
using Aspose.Pdf;
```

## ขั้นตอนที่ 2: ตั้งค่าเส้นทางไปยังโฟลเดอร์เอกสาร

 ในขั้นตอนนี้ คุณต้องระบุเส้นทางไปยังโฟลเดอร์ที่มีไฟล์ PDF ที่จะเข้ารหัส แทนที่`"YOUR DOCUMENTS DIRECTORY"` ในโค้ดต่อไปนี้โดยมีเส้นทางจริงไปยังโฟลเดอร์เอกสารของคุณ:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 3: เปิดเอกสาร PDF

ขั้นตอนต่อไป คุณต้องเปิดเอกสาร PDF ที่ต้องการเข้ารหัส ใช้โค้ดต่อไปนี้เพื่อโหลดเอกสาร:

```csharp
Document document = new Document(dataDir + "Encrypt.pdf");
```

## ขั้นตอนที่ 4: เข้ารหัส PDF

ตอนนี้คุณสามารถเข้ารหัส PDF ได้โดยใช้รหัสต่อไปนี้:

```csharp
document. Encrypt("user", "owner", 0, CryptoAlgorithm.RC4x128);
```

ในตัวอย่างนี้ เราใช้อัลกอริทึมการเข้ารหัส RC4x128 กับรหัสผ่าน "ผู้ใช้" และ "เจ้าของ" คุณสามารถเปลี่ยนการตั้งค่าเหล่านี้ได้ตามต้องการ

## ขั้นตอนที่ 5: สำรอง PDF ที่เข้ารหัส

สุดท้ายคุณสามารถบันทึก PDF ที่เข้ารหัสไปยังตำแหน่งที่ระบุโดยใช้โค้ดดังต่อไปนี้:

```csharp
dataDir = dataDir + "Encrypt_out.pdf";
document. Save(dataDir);
```

อย่าลืมระบุเส้นทางและชื่อไฟล์ที่ต้องการสำหรับ PDF ที่เข้ารหัส

### ตัวอย่างโค้ดต้นฉบับสำหรับการเข้ารหัสโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// เปิดเอกสาร
Document document = new Document(dataDir+ "Encrypt.pdf");
// เข้ารหัส PDF
document.Encrypt("user", "owner", 0, CryptoAlgorithm.RC4x128);
dataDir = dataDir + "Encrypt_out.pdf";
// บันทึก PDF อัพเดต
document.Save(dataDir);
Console.WriteLine("\nPDF file encrypted successfully.\nFile saved at " + dataDir);
```
