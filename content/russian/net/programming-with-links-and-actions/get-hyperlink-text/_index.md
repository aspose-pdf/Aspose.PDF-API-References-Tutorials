## Шаг 1: Настройте путь к документу

В нашем коде нам сначала нужно указать путь, где находится наш PDF-документ. Это делается с помощью строковой переменной. 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Обязательно замените`"YOUR DOCUMENT DIRECTORY"` с фактическим путем к вашему PDF-файлу. Например, это может выглядеть примерно так`"C:\\Documents\\"`.

## Шаг 2: Загрузите PDF-документ

 Следующий шаг включает загрузку файла PDF, чтобы мы могли начать его обработку. Мы создадим экземпляр`Document` класс и передаем ему наш путь к файлу.

```csharp
Document document = new Document(dataDir + "input.pdf");
```

На этом этапе, если все настроено правильно, ваш PDF-файл будет загружен и готов к взаимодействию.

## Шаг 3: Повторите каждую страницу

PDF-файлы могут иметь несколько страниц, поэтому мы пройдемся по каждой странице, чтобы найти аннотации ссылок. Вот как этого можно добиться:

```csharp
foreach (Page page in document.Pages)
{
    // Показать аннотацию ссылки
    ShowLinkAnnotations(page);
}
```

 В этом цикле мы определим метод, называемый`ShowLinkAnnotations` который будет заниматься извлечением гиперссылок. 

## Шаг 4: Определите метод ShowLinkAnnotations

Вот где происходит волшебство! Вы создадите метод для извлечения текста гиперссылки на каждой странице. Вот упрощенная версия этого метода:

```csharp
private static void ShowLinkAnnotations(Page page)
{
    foreach (Annotation annotation in page.Annotations)
    {
        if (annotation is LinkAnnotation link)
        {
            Console.WriteLine("Link Text: " + link.Title);
            Console.WriteLine("Link URI: " + link.Action.URI);
        }
    }
}
```

-  Проверьте, является ли аннотация ссылкой: Здесь мы проверяем, является ли аннотация на странице ссылкой.`LinkAnnotation`. Если это так, мы приступаем к извлечению его заголовка и URI.
-  Отображение текста гиперссылки: Использование`Console.WriteLine`, мы выводим текст ссылки и соответствующий URI.
