ในบทช่วยสอนนี้ เราจะอธิบายทีละขั้นตอนเกี่ยวกับวิธีเพิ่มบัฟเฟอร์รูปภาพในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะแสดงวิธีใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเพิ่มบัฟเฟอร์รูปภาพแบบกำหนดเองลงในหน้าเฉพาะในไฟล์ PDF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 2: การโหลดเอกสาร PDF

ขั้นตอนแรกคือโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "AddImageStamp.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 3: การสร้างเฟรมบัฟเฟอร์

ตอนนี้คุณได้อัปโหลดเอกสาร PDF แล้ว คุณสามารถสร้างตราประทับรูปภาพเพื่อเพิ่มได้ วิธีการทำมีดังนี้:

```csharp
// สร้างเฟรมบัฟเฟอร์
ImageStamp imageStamp = new ImageStamp(dataDir + "aspose-logo.jpg");
```

โค้ดด้านบนจะสร้างบัฟเฟอร์รูปภาพใหม่โดยใช้ไฟล์ "aspose-logo.jpg" ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์รูปภาพถูกต้อง

## ขั้นตอนที่ 4: การกำหนดค่าคุณสมบัติบัฟเฟอร์ภาพ

ก่อนที่จะเพิ่มแสตมป์ภาพลงในเอกสาร PDF คุณสามารถกำหนดค่าคุณสมบัติต่างๆ ของแสตมป์ เช่น ความทึบ ขนาด ตำแหน่ง ฯลฯ ได้ ดังนี้:

```csharp
// กำหนดค่าคุณสมบัติบัฟเฟอร์ภาพ
imageStamp. Background = true;
imageStamp. XIndent = 100;
imageStamp. YIndent = 100;
imageStamp. Height = 300;
imageStamp. Width = 300;
imageStamp.Rotate = Rotate.on270;
imageStamp. Opacity = 0.5;
```

คุณสามารถปรับเปลี่ยนคุณสมบัติเหล่านี้ได้ตามความต้องการของคุณ

## ขั้นตอนที่ 5: เพิ่มตราประทับภาพลงใน PDF

เมื่อแสตมป์ภาพพร้อมแล้ว คุณสามารถเพิ่มแสตมป์ลงในหน้าเฉพาะของเอกสาร PDF ได้ ดังต่อไปนี้:

```csharp
// เพิ่มบัฟเฟอร์เฟรมลงในเพจเฉพาะ
pdfDocument.Pages[1].AddStamp(imageStamp);
```

โค้ดด้านบนจะเพิ่มบัฟเฟอร์รูปภาพลงในหน้าแรกของเอกสาร PDF คุณสามารถระบุหน้าอื่นได้หากจำเป็น

## ขั้นตอนที่ 6: บันทึกเอกสารผลลัพธ์

เมื่อคุณเพิ่มบัฟเฟอร์รูปภาพแล้ว คุณสามารถบันทึกเอกสาร PDF ที่แก้ไขแล้วได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
```

โค้ดด้านบนจะบันทึกเอกสาร PDF ที่แก้ไขแล้วไปยังไดเร็กทอรีที่ระบุ

### ตัวอย่างโค้ดต้นฉบับสำหรับ Add Image Stamp โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "AddImageStamp.pdf");

// สร้างภาพแสตมป์
ImageStamp imageStamp = new ImageStamp(dataDir + "aspose-logo.jpg");
imageStamp.Background = true;
imageStamp.XIndent = 100;
imageStamp.YIndent = 100;
imageStamp.Height = 300;
imageStamp.Width = 300;
imageStamp.Rotate = Rotation.on270;
imageStamp.Opacity = 0.5;

// เพิ่มตราประทับบนหน้าที่ต้องการ
pdfDocument.Pages[1].AddStamp(imageStamp);
dataDir = dataDir + "AddImageStamp_out.pdf";

// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
Console.WriteLine("\nImage stamp added successfully.\nFile saved at " + dataDir);
```

 A: ใช่ คุณสามารถหมุนบัฟเฟอร์รูปภาพก่อนเพิ่มลงในเอกสาร PDF ได้โดยการตั้งค่า`Rotate` ทรัพย์สินของ`ImageStamp` วัตถุ โค้ดในบทช่วยสอนแสดงวิธีการหมุนแสตมป์ภาพโดยใช้ค่าต่างๆ เช่น`Rotation.on270`แต่คุณสามารถปรับเปลี่ยนมุมการหมุนได้ตามต้องการ
