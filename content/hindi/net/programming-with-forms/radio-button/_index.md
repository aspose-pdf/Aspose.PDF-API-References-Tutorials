इस ट्यूटोरियल में, हम आपको दिखाएंगे कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में रेडियो बटन कैसे जोड़ा जाता है। हम इस प्रक्रिया में आपका मार्गदर्शन करने के लिए C# स्रोत कोड को चरण दर चरण समझाएंगे।

## चरण 1: तैयारी

सुनिश्चित करें कि आपने आवश्यक लाइब्रेरीज़ आयात कर ली हैं और अपने दस्तावेज़ निर्देशिका का पथ सेट कर लिया है:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## चरण 2: दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें

एक नया PDF दस्तावेज़ बनाने के लिए दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें:

```csharp
Document pdfDocument = new Document();
```

## चरण 3: पेज जोड़ें

पीडीएफ दस्तावेज़ में एक पृष्ठ जोड़ें:

```csharp
pdfDocument.Pages.Add();
```

## चरण 4: रेडियोबटनफील्ड ऑब्जेक्ट को इंस्टैंसिएट करें

पृष्ठ संख्या को तर्क के रूप में निर्दिष्ट करते हुए रेडियोबटनफील्ड ऑब्जेक्ट को तत्कालित करें:

```csharp
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

## चरण 5: रेडियो बटन विकल्प जोड़ें

Rectangle ऑब्जेक्ट के साथ प्रत्येक विकल्प के निर्देशांक निर्दिष्ट करके रेडियो बटन विकल्पों को RadioButtonField ऑब्जेक्ट में जोड़ें:

```csharp
radio.AddOption("Test", new Rectangle(0, 0, 20, 20));
radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
```

## चरण 6: फ़ॉर्म में रेडियो बटन जोड़ें

दस्तावेज़ के फ़ॉर्म ऑब्जेक्ट में रेडियो बटन जोड़ें:

```csharp
pdfDocument.Form.Add(radio);
```

## चरण 7: पीडीएफ दस्तावेज़ सहेजें

निर्मित PDF दस्तावेज़ को सहेजें:

```csharp
dataDir = dataDir + "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके रेडियो बटन के लिए नमूना स्रोत कोड 
```csharp
try
{
	// दस्तावेज़ निर्देशिका का पथ.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	// दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें
	Document pdfDocument = new Document();
	// किसी पृष्ठ को PDF फ़ाइल में जोड़ें
	pdfDocument.Pages.Add();
	// रेडियोबटनफील्ड ऑब्जेक्ट को पृष्ठ संख्या के साथ तर्क के रूप में प्रारंभ करें
	RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
	// पहला रेडियो बटन विकल्प जोड़ें और Rectangle ऑब्जेक्ट का उपयोग करके उसका मूल भी निर्दिष्ट करें
	radio.AddOption("Test", new Rectangle(0, 0, 20, 20));
	// दूसरा रेडियो बटन विकल्प जोड़ें
	radio.AddOption("Test1", new Rectangle(20, 20, 40, 40));
	// दस्तावेज़ ऑब्जेक्ट के फ़ॉर्म ऑब्जेक्ट में रेडियो बटन जोड़ें
	pdfDocument.Form.Add(radio);
	dataDir = dataDir + "RadioButton_out.pdf";
	// पीडीएफ फाइल को सेव करें
	pdfDocument.Save(dataDir);
	Console.WriteLine("\nRadio button field added successfully.\nFile saved at " + dataDir);
}
catch (Exception ex)
{
	Console.WriteLine(ex.Message);
}
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में रेडियो बटन कैसे जोड़ा जाता है। इन चरणों का पालन करके, आप आसानी से एक रेडियो बटन बना सकते हैं और इसे अपने PDF दस्तावेज़ में किसी विशिष्ट पृष्ठ पर रख सकते हैं।


### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं रेडियो बटन के स्वरूप को अनुकूलित कर सकता हूँ, जैसे उसका आकार और रंग?

 उत्तर: हां, आप रेडियो बटन के स्वरूप को अनुकूलित कर सकते हैं`Rectangle` ऑब्जेक्ट के निर्देशांक इसके आकार और स्थिति को परिभाषित करने के लिए। .NET के लिए Aspose.PDF आपको अपनी आवश्यकताओं के अनुरूप रेडियो बटन की उपस्थिति को समायोजित करने की अनुमति देता है।

#### प्रश्न: क्या मैं एक ही पृष्ठ पर विभिन्न समूहों के साथ एकाधिक रेडियो बटन जोड़ सकता हूँ?

उत्तर: हां, आप एक ही पेज पर अलग-अलग समूहों के साथ कई रेडियो बटन जोड़ सकते हैं। रेडियो बटन के प्रत्येक समूह का एक अनूठा नाम हो सकता है, और प्रत्येक समूह में से एक बार में केवल एक ही विकल्प चुना जा सकता है।

#### प्रश्न: मैं रेडियो बटन विकल्पों में लेबल या पाठ विवरण कैसे जोड़ सकता हूँ?

 उत्तर: रेडियो बटन विकल्पों में लेबल या पाठ विवरण जोड़ने के लिए, आप इसका उपयोग कर सकते हैं`TextStamp`.NET के लिए Aspose.PDF से क्लास का उपयोग करके PDF दस्तावेज़ पर विशिष्ट निर्देशांकों पर पाठ ओवरले करें।

#### प्रश्न: क्या .NET के लिए Aspose.PDF .NET फ्रेमवर्क के सभी संस्करणों के साथ संगत है?

उत्तर: हां, .NET के लिए Aspose.PDF .NET Core और .NET Standard सहित .NET Framework के सभी संस्करणों के साथ संगत है।

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ में रेडियो बटन विकल्प के चयन को प्रोग्रामेटिक रूप से नियंत्रित कर सकता हूं?

 उत्तर: हां, आप प्रोग्रामेटिक रूप से रेडियो बटन विकल्प के चयन को नियंत्रित कर सकते हैं।`IsSelected` की संपत्ति`RadioButtonOption` वर्ग. यह गुण आपको चयनित के रूप में एक विशिष्ट विकल्प सेट करने की अनुमति देता है.
