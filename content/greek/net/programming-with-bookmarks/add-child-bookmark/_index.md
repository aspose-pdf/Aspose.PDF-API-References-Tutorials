Έχετε βρεθεί ποτέ να κάνετε κύλιση ατελείωτα σε ένα μεγάλο PDF, προσπαθώντας να βρείτε μια συγκεκριμένη ενότητα; Απογοητευτικό, σωστά; Εκεί είναι χρήσιμοι οι σελιδοδείκτες! Λειτουργούν σαν πίνακας περιεχομένων, επιτρέποντας στους αναγνώστες να περιηγηθούν στο έγγραφο με ευκολία. Σε αυτό το σεμινάριο, θα διερευνήσουμε πώς να προσθέσετε θυγατρικούς σελιδοδείκτες σε ένα αρχείο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Μέχρι το τέλος αυτού του οδηγού, θα μπορείτε να βελτιώσετε τα έγγραφά σας PDF, καθιστώντας τα πιο φιλικά και οργανωμένα στον χρήστη.

### Εισαγάγετε τους απαιτούμενους χώρους ονομάτων

 Στην κορυφή σου`Program.cs` αρχείο, εισαγάγετε τους απαραίτητους χώρους ονομάτων:

using System;
using System.IO;
using Aspose.Pdf.Annotations;

Τώρα που έχετε ρυθμίσει τα πάντα, ας αναλύσουμε τη διαδικασία προσθήκης θυγατρικών σελιδοδεικτών βήμα προς βήμα.

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Για να μπορέσετε να χειριστείτε οποιοδήποτε PDF, πρέπει να καθορίσετε πού αποθηκεύονται τα έγγραφά σας. Αυτό είναι σημαντικό για τον κώδικα για τον εντοπισμό του αρχείου PDF σας.

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 Αντικαθιστώ`"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου βρίσκεται το αρχείο PDF σας. Αυτό είναι σαν να δίνετε στον κωδικό σας έναν χάρτη για να βρείτε τον θησαυρό!

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Τώρα που έχουμε ρυθμίσει τον κατάλογο, ήρθε η ώρα να ανοίξετε το έγγραφο PDF με το οποίο θέλετε να εργαστείτε.

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "AddChildBookmark.pdf");
```

 Εδώ, δημιουργούμε ένα νέο`Document` αντικείμενο που φορτώνει το αρχείο PDF σας. Σκεφτείτε αυτό σαν να ανοίγετε ένα βιβλίο για να αρχίσετε να διαβάζετε.

## Βήμα 3: Δημιουργήστε έναν σελιδοδείκτη γονέα

Στη συνέχεια, θα δημιουργήσουμε έναν γονικό σελιδοδείκτη. Αυτός ο σελιδοδείκτης θα χρησιμεύσει ως η κύρια επικεφαλίδα κάτω από την οποία θα προσθέσουμε θυγατρικούς σελιδοδείκτες.

```csharp
// Δημιουργήστε ένα γονικό αντικείμενο σελιδοδείκτη
OutlineItemCollection pdfOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfOutline.Title = "Parent Outline";
pdfOutline.Italic = true;
pdfOutline.Bold = true;
```

 Σε αυτό το απόσπασμα, δημιουργούμε ένα νέο`OutlineItemCollection` για τον γονικό σελιδοδείκτη. Ορίσαμε τον τίτλο και το στυλ του (πλάγια και έντονη γραφή) για να το κάνουμε να ξεχωρίζει. Είναι σαν να δίνετε στο κεφάλαιο σας έναν πιασάρικο τίτλο!

## Βήμα 4: Δημιουργήστε έναν παιδικό σελιδοδείκτη

Τώρα, ας προσθέσουμε έναν θυγατρικό σελιδοδείκτη κάτω από τον γονικό σελιδοδείκτη που μόλις δημιουργήσαμε.

```csharp
// Δημιουργήστε ένα αντικείμενο θυγατρικού σελιδοδείκτη
OutlineItemCollection pdfChildOutline = new OutlineItemCollection(pdfDocument.Outlines);
pdfChildOutline.Title = "Child Outline";
pdfChildOutline.Italic = true;
pdfChildOutline.Bold = true;
```

Παρόμοια με τον γονικό σελιδοδείκτη, δημιουργούμε έναν θυγατρικό σελιδοδείκτη με τον δικό του τίτλο και στυλ. Αυτός ο θυγατρικός σελιδοδείκτης θα είναι ένθετος κάτω από τον γονέα, δημιουργώντας μια ιεραρχία.

## Βήμα 5: Προσθέστε τον παιδικό σελιδοδείκτη στον Γονέα

Με τη δημιουργία και των δύο σελιδοδεικτών, ήρθε η ώρα να τους συνδέσετε μεταξύ τους.

```csharp
// Προσθήκη θυγατρικού σελιδοδείκτη στη συλλογή γονικών σελιδοδεικτών
pdfOutline.Add(pdfChildOutline);
```

Αυτή η γραμμή κώδικα προσθέτει τον θυγατρικό σελιδοδείκτη στη συλλογή του γονικού σελιδοδείκτη. Είναι σαν να βάζεις μια υπότιτλου κάτω από έναν τίτλο κεφαλαίου!

## Βήμα 6: Προσθέστε τον σελιδοδείκτη γονέα στο Έγγραφο

Τώρα που έχουμε ρυθμίσει τους γονικούς και θυγατρικούς σελιδοδείκτες μας, πρέπει να προσθέσουμε τον γονικό σελιδοδείκτη στη συλλογή περιλήψεων του εγγράφου.

```csharp
// Προσθήκη γονικού σελιδοδείκτη στη συλλογή περιλήψεων του εγγράφου.
pdfDocument.Outlines.Add(pdfOutline);
```

Αυτό το βήμα διασφαλίζει ότι ο γονικός σελιδοδείκτης, μαζί με το παιδί του, αποτελούν πλέον μέρος του εγγράφου PDF. Είναι σαν να δημοσιεύεις επίσημα το βιβλίο σου με όλα του τα κεφάλαια!

## Βήμα 7: Αποθηκεύστε το έγγραφο

Τέλος, ας αποθηκεύσουμε τις αλλαγές που κάναμε στο έγγραφο PDF.

```csharp
dataDir = dataDir + "AddChildBookmark_out.pdf";
// Αποθήκευση εξόδου
pdfDocument.Save(dataDir);
Console.WriteLine("\nChild bookmark added successfully.\nFile saved at " + dataDir);
```

Εδώ, καθορίζουμε το όνομα του αρχείου εξόδου και αποθηκεύουμε το έγγραφο. Θα δείτε ένα μήνυμα επιβεβαίωσης μόλις ολοκληρωθεί η διαδικασία. Είναι σαν να κλείνεις το βιβλίο αφού γράψεις το αριστούργημά σου!
