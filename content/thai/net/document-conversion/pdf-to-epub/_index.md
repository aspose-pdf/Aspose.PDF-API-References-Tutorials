## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ก่อนที่คุณจะแปลงไฟล์ PDF คุณต้องระบุไดเรกทอรีที่จัดเก็บเอกสารของคุณก่อน นี่คือที่ที่ไฟล์ PDF อินพุตและไฟล์ EPUB เอาท์พุตของคุณจะอยู่ในไดเรกทอรีนี้

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` ด้วยเส้นทางจริงบนเครื่องของคุณซึ่งไฟล์ PDF ของคุณอยู่

## ขั้นตอนที่ 2: โหลดเอกสาร PDF

 ตอนนี้คุณได้ตั้งค่าไดเรกทอรีเอกสารของคุณแล้ว ขั้นตอนต่อไปคือการโหลดเอกสาร PDF ที่คุณต้องการแปลง ซึ่งทำได้โดยใช้`Document` คลาสที่จัดทำโดย Aspose.PDF

```csharp
// โหลดเอกสาร PDF
Document pdfDocument = new Document(dataDir + "PDFToEPUB.pdf");
```

 อย่าลืมเปลี่ยน`"PDFToEPUB.pdf"` ด้วยชื่อไฟล์ PDF ของคุณ บรรทัดโค้ดนี้จะเริ่มต้นไฟล์ใหม่`Document` วัตถุที่มีไฟล์ PDF ที่ระบุ

## ขั้นตอนที่ 3: สร้างตัวเลือกการบันทึก EPUB

 ก่อนที่จะบันทึกเอกสารเป็น EPUB คุณต้องสร้างอินสแตนซ์ของ`EpubSaveOptions`คลาสนี้ช่วยให้คุณสามารถระบุตัวเลือกต่าง ๆ สำหรับเอาท์พุต EPUB

```csharp
// สร้างตัวเลือกการบันทึก Epub
EpubSaveOptions options = new EpubSaveOptions();
```

 บรรทัดนี้จะสร้างอินสแตนซ์ใหม่ของ`EpubSaveOptions`ซึ่งคุณจะกำหนดค่าในขั้นตอนถัดไป

## ขั้นตอนที่ 4: ระบุโหมดการจดจำเนื้อหา

คุณลักษณะสำคัญประการหนึ่งของ EPUB คือความสามารถในการไหลเนื้อหา คุณสามารถระบุได้ว่าเนื้อหาจะถูกจดจำอย่างไรในระหว่างกระบวนการแปลง สำหรับตัวอย่างนี้ เราจะตั้งค่าโหมดการจดจำเนื้อหาให้เป็นแบบไหล

```csharp
// ระบุเค้าโครงของเนื้อหา
options.ContentRecognitionMode = EpubSaveOptions.RecognitionMode.Flow;
```

การตั้งค่านี้จะช่วยให้แน่ใจว่าเนื้อหาใน PDF ของคุณจะได้รับการแปลงในลักษณะที่เหมาะสมสำหรับโปรแกรมอ่าน EPUB ช่วยให้ประสบการณ์การอ่านดีขึ้น

## ขั้นตอนที่ 5: บันทึกเอกสาร EPUB

 ในที่สุด ก็ถึงเวลาบันทึกเอกสารที่แปลงแล้ว คุณจะใช้`Save` วิธีการของ`Document` ชั้นเรียนที่จะทำสิ่งนี้

```csharp
// บันทึกเอกสาร ePUB
pdfDocument.Save(dataDir + "PDFToEPUB_out.epub", options);
```

 บรรทัดนี้จะบันทึกไฟล์ EPUB ที่แปลงแล้วไว้ในไดเร็กทอรีเดียวกับไฟล์ PDF ต้นฉบับของคุณ ไฟล์เอาต์พุตจะมีชื่อ`PDFToEPUB_out.epub`.
