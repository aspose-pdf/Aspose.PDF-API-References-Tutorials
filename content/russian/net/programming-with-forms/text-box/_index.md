 В верхней части вашего основного файла программы (обычно`Program.cs`), включите следующую строку для доступа к функциональным возможностям Aspose.PDF:

## Шаг 1: Определите каталог документов

 Во-первых, нам нужно указать, где находится наш PDF-документ. Обязательно замените`"YOUR DOCUMENT DIRECTORY"` с фактическим путем к вашим файлам.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Эта строка устанавливает наш рабочий каталог и сообщает программе, где искать PDF-файл, с которым мы хотим работать.

## Шаг 2: Откройте PDF-документ. 

Далее вам нужно будет открыть PDF-документ, в который вы планируете добавить текстовое поле. Вот как это сделать:

```csharp
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

 Эта строка загружает PDF-файл в экземпляр`Document` класс. Убедитесь, что`"TextField.pdf"` присутствует в указанном вами каталоге.

## Шаг 3: Создайте текстовое поле

А теперь самое интересное — создадим текстовое поле:

```csharp
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
```

Эта строка делает несколько вещей:
-  Он инициализирует новый`TextBoxField` объект, который будет добавлен на вторую страницу вашего PDF-файла (обратите внимание, что страницы индексируются, начиная с 1).
-  The`Rectangle` Параметр определяет положение и размер текстового поля, указываемые в виде координат (x1, y1, x2, y2).

## Шаг 4: Задайте свойства для поля текстового поля 

Вы можете настроить текстовое поле в соответствии с вашими потребностями. Вот как задать некоторые основные свойства:

```csharp
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```

В этом примере:
- `PartialName` задает уникальный идентификатор для текстового поля.
- `Value`определяет текст по умолчанию, который отображается внутри поля.

## Шаг 5: Настройте границу

Далее давайте придадим нашему текстовому полю немного изюминку, настроив его границу:

```csharp
Border border = new Border(textBoxField);
border.Width = 5; 
border.Dash = new Dash(1, 1);
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```

Этот фрагмент:
- Создает границу и задает ее ширину.
- Устанавливает пунктирный стиль границы.
- Присваивает текстовому полю зеленый цвет.

## Шаг 6: Добавьте текстовое поле в документ

Теперь, когда мы настроили текстовое поле, давайте добавим его в наш PDF-документ:

```csharp
pdfDocument.Form.Add(textBoxField, 1);
```

Эта строка сообщает PDF-файлу, что необходимо включить наше новое текстовое поле на первую страницу.

## Шаг 7: Сохраните измененный PDF-файл.

Наконец, пришло время сохранить изменения. Вот как это сделать:

```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

Этот код сохраняет измененный PDF под новым именем файла. Обязательно проверьте выходной путь для вашего свежесозданного PDF!
