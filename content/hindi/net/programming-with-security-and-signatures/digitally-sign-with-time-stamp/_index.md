इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके टाइम स्टैम्प के साथ PDF फ़ाइल में डिजिटल रूप से हस्ताक्षर करने की प्रक्रिया से अवगत कराएँगे। टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर, टाइमस्टैम्प के साथ इलेक्ट्रॉनिक फ़िंगरप्रिंट जोड़कर दस्तावेज़ की प्रामाणिकता और अखंडता की गारंटी देता है।

## चरण 1: पूर्वापेक्षाएँ

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान
- अपनी मशीन पर Visual Studio स्थापित करना
- .NET के लिए Aspose.PDF लाइब्रेरी स्थापित

## चरण 2: पर्यावरण सेटअप

आरंभ करने के लिए, अपना विकास वातावरण स्थापित करने हेतु इन चरणों का पालन करें:

1. Visual Studio खोलें और एक नया C# प्रोजेक्ट बनाएं।
2. आवश्यक नामस्थानों को अपनी कोड फ़ाइल में आयात करें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Forms;
```

## चरण 3: टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर

पहला कदम पीडीएफ फाइल पर टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर करना है। दिया गया कोड दिखाता है कि .NET के लिए Aspose.PDF के साथ इस हस्ताक्षर को कैसे प्राप्त किया जाए।

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
string pfxFile = "";
using (Document document = new Document(dataDir + @"DigitallySign.pdf"))
{
     using (PdfFileSignature signature = new PdfFileSignature(document))
     {
         PKCS7 pkcs = new PKCS7(pfxFile, "pfx_password");
         TimestampSettings timestampSettings = new TimestampSettings("https:\\your_timestamp_settings", "user:password");
         pkcs. TimestampSettings = timestampSettings;
         System.Drawing.Rectangle rect = new System.Drawing.Rectangle(100, 100, 200, 100);
         signature.Sign(1, "Reason for signing", "Contact", "Location", true, rect, pkcs);
         signature.Save(dataDir + "DigitallySignWithTimeStamp_out.pdf");
     }
}
```

यह कोड एक पीडीएफ फाइल लोड करता है, एक PFX फाइल (निजी कुंजी) और निर्दिष्ट टाइमस्टैम्प मापदंडों का उपयोग करके टाइमस्टैम्प के साथ एक डिजिटल हस्ताक्षर बनाता है। फिर हस्ताक्षर को पीडीएफ फाइल में जोड़ा जाता है और प्रत्यय "के साथ सहेजा जाता है_बाहर"।

### .NET के लिए Aspose.PDF का उपयोग करके टाइम स्टैम्प के साथ डिजिटल रूप से हस्ताक्षर करने के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
string pfxFile = "";
using (Document document = new Document(dataDir + @"DigitallySign.pdf"))
{
	using (PdfFileSignature signature = new PdfFileSignature(document))
	{
		PKCS7 pkcs = new PKCS7(pfxFile, "pfx_password");
		TimestampSettings timestampSettings = new TimestampSettings("https:\\your_timestamp_settings", "user:password"); // उपयोगकर्ता/पासवर्ड छोड़ा जा सकता है
		pkcs.TimestampSettings = timestampSettings;
		System.Drawing.Rectangle rect = new System.Drawing.Rectangle(100, 100, 200, 100);
		// तीन में से कोई भी हस्ताक्षर प्रकार बनाएँ
		signature.Sign(1, "Signature Reason", "Contact", "Location", true, rect, pkcs);
		// आउटपुट पीडीएफ फाइल सहेजें
		signature.Save(dataDir + "DigitallySignWithTimeStamp_out.pdf");
	}
}
```

## निष्कर्ष

बधाई हो! आपने .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल पर टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर सफलतापूर्वक किया है। इस ट्यूटोरियल में हस्ताक्षर बनाने से लेकर अपडेट की गई PDF फ़ाइल को सहेजने तक की चरण-दर-चरण प्रक्रिया को शामिल किया गया है। अब आप अपनी PDF फ़ाइलों में टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर जोड़ने के लिए इस सुविधा का उपयोग कर सकते हैं।

### पीडीएफ फाइल में टाइम स्टैम्प के साथ डिजिटल हस्ताक्षर के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर करने का उद्देश्य क्या है?

उत्तर: टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर करने से पीडीएफ फाइल में टाइमस्टैम्प के साथ एक इलेक्ट्रॉनिक फिंगरप्रिंट जुड़ जाता है, जिससे एक विशिष्ट समय पर दस्तावेज़ की प्रामाणिकता और अखंडता सुनिश्चित होती है।

#### प्रश्न: इस ट्यूटोरियल को शुरू करने के लिए क्या पूर्वापेक्षाएँ आवश्यक हैं?

उत्तर: आरंभ करने से पहले, सुनिश्चित करें कि आपको C# प्रोग्रामिंग भाषा की बुनियादी समझ है, Visual Studio स्थापित है, तथा .NET के लिए Aspose.PDF लाइब्रेरी स्थापित है।

#### प्रश्न: मैं अपना विकास वातावरण कैसे स्थापित कर सकता हूँ?

उत्तर: अपने विकास परिवेश को सेट करने के लिए दिए गए चरणों का पालन करें, जिसमें Visual Studio में एक नया C# प्रोजेक्ट बनाना और आवश्यक नामस्थानों को आयात करना शामिल है।

#### प्रश्न: मैं पीडीएफ में टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर कैसे जोड़ूं?

उत्तर: प्रदान किया गया नमूना कोड यह दर्शाता है कि पीडीएफ फाइल को कैसे लोड किया जाए, पीएफएक्स फाइल (निजी कुंजी) और निर्दिष्ट टाइमस्टैम्प सेटिंग्स का उपयोग करके टाइमस्टैम्प के साथ डिजिटल हस्ताक्षर कैसे बनाएं, हस्ताक्षर को पीडीएफ फाइल में कैसे जोड़ें, और अपडेट की गई फाइल को कैसे सेव करें।

#### प्रश्न: PFX फ़ाइल क्या है, और इसका उपयोग उदाहरण में क्यों किया गया है?

उत्तर: PFX (पर्सनल एक्सचेंज फ़ॉर्मेट) फ़ाइल में एक निजी कुंजी और प्रमाणपत्र होता है। इसका उपयोग डिजिटल हस्ताक्षरों के लिए क्रिप्टोग्राफ़िक कार्यक्षमता प्रदान करने के लिए किया जाता है। सुनिश्चित करें कि आप प्लेसहोल्डर को अपनी PFX फ़ाइल और पासवर्ड से बदल दें।

#### प्रश्न: टाइमस्टैम्प सेटिंग्स क्या हैं?

उत्तर: टाइमस्टैम्प सेटिंग्स इलेक्ट्रॉनिक टाइमस्टैम्प को हस्ताक्षर में जोड़ने के लिए उपयोग किए जाने वाले टाइमस्टैम्प सर्वर के लिए सेटिंग्स को परिभाषित करती हैं। इसमें टाइमस्टैम्प सर्वर URL और वैकल्पिक उपयोगकर्ता क्रेडेंशियल शामिल हैं।

#### प्रश्न: क्या मैं उदाहरण में दिए गए टाइमस्टैम्प सर्वर के अलावा किसी अन्य टाइमस्टैम्प सर्वर का उपयोग कर सकता हूँ?
 उत्तर: हां, आप किसी भी संगत टाइमस्टैम्प सर्वर का उपयोग कर सकते हैं। URL बदलें और, यदि आवश्यक हो, तो उचित उपयोगकर्ता क्रेडेंशियल प्रदान करें`TimestampSettings` वस्तु।

#### प्रश्न: हस्ताक्षर आयत निर्दिष्ट करने का उद्देश्य क्या है?

उत्तर: हस्ताक्षर आयत पीडीएफ पृष्ठ पर डिजिटल हस्ताक्षर की स्थिति और आयाम को परिभाषित करता है। हस्ताक्षर को इच्छानुसार स्थिति में रखने के लिए इन मानों को समायोजित करें।

#### प्रश्न: यदि हस्ताक्षर के दौरान टाइमस्टैम्प सर्वर अनुपलब्ध हो तो क्या होगा?

उत्तर: यदि हस्ताक्षर के दौरान टाइमस्टैम्प सर्वर अनुपलब्ध है, तो प्रक्रिया विफल हो सकती है या इसमें अधिक समय लग सकता है। सुनिश्चित करें कि आपका टाइमस्टैम्प सर्वर विश्वसनीय और सुलभ है।

#### प्रश्न: मैं हस्ताक्षरित पीडीएफ में टाइमस्टैम्प की उपस्थिति की पुष्टि कैसे कर सकता हूं?

 उत्तर: आप दिए गए नमूना कोड का उपयोग करके हस्ताक्षरित पीडीएफ की जांच कर सकते हैं।`TimestampSettings` हस्ताक्षर करते समय आपके द्वारा उपयोग किया गया हस्ताक्षर विवरण में उपलब्ध होना चाहिए।

#### प्रश्न: क्या टाइमस्टैम्प वाले डिजिटल हस्ताक्षर कानूनी रूप से बाध्यकारी हैं?

उत्तर: टाइमस्टैम्प वाले डिजिटल हस्ताक्षर कई अधिकार क्षेत्रों में कानूनी महत्व रखते हैं और अक्सर उन्हें साधारण डिजिटल हस्ताक्षरों की तुलना में अधिक विश्वसनीय माना जाता है। विशिष्ट विनियमों के लिए अपने अधिकार क्षेत्र के कानूनी विशेषज्ञों से परामर्श करें।

#### प्रश्न: क्या मैं किसी PDF में टाइमस्टैम्प के साथ एकाधिक डिजिटल हस्ताक्षर जोड़ सकता हूँ?

उत्तर: हां, आप हस्ताक्षर निर्माण प्रक्रिया को कई बार कॉल करके टाइमस्टैम्प के साथ एक पीडीएफ फाइल में कई डिजिटल हस्ताक्षर जोड़ सकते हैं। प्रत्येक हस्ताक्षर का अपना टाइमस्टैम्प होगा।
