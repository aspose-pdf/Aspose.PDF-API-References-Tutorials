Για να εξαγάγετε επισημασμένο κείμενο σε αρχείο PDF, μπορείτε να χρησιμοποιήσετε το Aspose.PDF για .NET API. Αυτό το API παρέχει έναν απλό τρόπο ανάκτησης όλου του κειμένου που έχει επισημανθεί σε ένα έγγραφο.

## Βήμα 1: Φορτώστε το έγγραφο PDF

 Το πρώτο βήμα για την εξαγωγή επισημασμένου κειμένου σε αρχείο PDF είναι να φορτώσετε το έγγραφο χρησιμοποιώντας το Aspose.PDF για .NET API. Μπορείτε να το κάνετε αυτό δημιουργώντας μια νέα παρουσία του`Document` κλάση και μεταβίβαση της διαδρομής στο έγγραφο PDF ως παράμετρος. 

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir ="YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "ExtractHighlightedText.pdf");
```

## Βήμα 2: Περιηγηθείτε σε όλους τους σχολιασμούς

 Το επόμενο βήμα είναι να πραγματοποιήσετε αναζήτηση σε όλους τους σχολιασμούς στο έγγραφο PDF. Μπορείτε να το κάνετε αυτό χρησιμοποιώντας ένα`foreach` βρόχος, όπως έτσι:

```csharp
foreach (Annotation annotation in doc.Pages[1].Annotations)
{
	// Ο κώδικας πηγαίνει εδώ
}
```

## Βήμα 3: Φιλτράρετε τους σχολιασμούς σήμανσης κειμένου

 μεσα στην`foreach` βρόχο, θα χρειαστεί να φιλτράρετε όλους τους σχολιασμούς που δεν είναι σχολιασμοί σήμανσης κειμένου. Μπορείτε να το κάνετε αυτό ελέγχοντας εάν ο σχολιασμός είναι ένα παράδειγμα του`TextMarkupAnnotation` τάξη.

```csharp
if (annotation is TextMarkupAnnotation)
{
	// Ο κώδικας πηγαίνει εδώ
}
```

## Βήμα 4: Ανάκτηση επισημασμένων τμημάτων κειμένου

 Αφού φιλτράρετε όλους τους σχολιασμούς σήμανσης κειμένου, μπορείτε να ανακτήσετε τα επισημασμένα τμήματα κειμένου για κάθε σχολιασμό. Μπορείτε να το κάνετε καλώντας το`GetMarkedTextFragments()` μέθοδος στο`TextMarkupAnnotation` αντικείμενο.

```csharp
TextMarkupAnnotation highlightedAnnotation = annotation as TextMarkupAnnotation;
TextFragmentCollection collection = highlightedAnnotation.GetMarkedTextFragments();
```

## Βήμα 5: Εμφανίστε το επισημασμένο κείμενο

 Τέλος, μπορείτε να εμφανίσετε το επισημασμένο κείμενο στον χρήστη. Μπορείτε να το κάνετε αυτό κάνοντας looping μέσα από το καθένα`TextFragment` αντικείμενο στο`TextFragmentCollection` και καλώντας το`Text` ιδιοκτησία.

```csharp
foreach (TextFragment tf in collection)
{
	Console.WriteLine(tf.Text);
}
```

### Παράδειγμα πηγαίου κώδικα για εξαγωγή επισημασμένου κειμένου χρησιμοποιώντας το Aspose.PDF για .NET

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir ="YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "ExtractHighlightedText.pdf");

foreach (Annotation annotation in doc.Pages[1].Annotations)
{
	if (annotation is TextMarkupAnnotation)
	{
		TextMarkupAnnotation highlightedAnnotation = annotation as TextMarkupAnnotation;
		TextFragmentCollection collection = highlightedAnnotation.GetMarkedTextFragments();
		foreach (TextFragment tf in collection)
		{
			Console.WriteLine(tf.Text);
		}
	}
}
```
