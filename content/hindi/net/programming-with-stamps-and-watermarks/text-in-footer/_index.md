इस ट्यूटोरियल में, हम सीखेंगे कि .NET के लिए Aspose.PDF का उपयोग करके पीडीएफ फ़ाइल के पाद लेख में टेक्स्ट कैसे जोड़ें। नीचे दिए गए चरणों का पालन करें:

## चरण 1: परियोजना की तैयारी

सुनिश्चित करें कि आपने .NET के लिए Aspose.PDF स्थापित किया है और एक C# प्रोजेक्ट बनाया है।

## चरण 2: नामस्थान आयात करना

अपनी C# स्रोत फ़ाइल में निम्नलिखित नामस्थान जोड़ें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## चरण 3: दस्तावेज़ खोलना

दिए गए पथ का उपयोग करके मौजूदा पीडीएफ दस्तावेज़ खोलें:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document pdfDocument = new Document(dataDir + "TextinFooter.pdf");
```

अपने दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ "आपकी दस्तावेज़ निर्देशिका" को बदलना सुनिश्चित करें।

## चरण 4: फ़ूटर टेक्स्ट बनाएं

जिस टेक्स्ट को आप पाद लेख में जोड़ना चाहते हैं, उसके साथ एक नया टेक्स्ट स्टैम्प बनाएं:

```csharp
TextStamp textStamp = new TextStamp("footer text");
```

आप टेक्स्ट के गुणों जैसे निचला मार्जिन, क्षैतिज संरेखण और ऊर्ध्वाधर संरेखण को बदलकर उसे अनुकूलित कर सकते हैं।

## चरण 5: सभी पृष्ठों पर पाद लेख जोड़ें

पीडीएफ दस्तावेज़ के सभी पृष्ठों पर जाएँ और पाद लेख में टेक्स्ट स्टैम्प जोड़ें:

```csharp
foreach(Page page in pdfDocument.Pages)
{
     page.AddStamp(textStamp);
}
```

## चरण 6: पीडीएफ दस्तावेज़ को सहेजना

एक बार जब सभी पृष्ठों पर पादलेख पाठ जोड़ दिया जाए, तो अद्यतन पीडीएफ दस्तावेज़ को सहेजें:

```csharp
dataDir = dataDir + "TextinFooter_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nText in footer added successfully.\nFile saved at: " + dataDir);
```

"आपकी दस्तावेज़ निर्देशिका" को उस निर्देशिका के वास्तविक पथ से बदलना सुनिश्चित करें जहाँ आप पीडीएफ दस्तावेज़ को सहेजना चाहते हैं।

### .NET के लिए Aspose.PDF का उपयोग करके टेक्स्टिन फूटर के लिए नमूना स्रोत कोड 
```csharp

// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ खोलें
Document pdfDocument = new Document(dataDir+ "TextinFooter.pdf");

// फ़ुटर बनाएँ
TextStamp textStamp = new TextStamp("Footer Text");

// स्टाम्प के गुण सेट करें
textStamp.BottomMargin = 10;
textStamp.HorizontalAlignment = HorizontalAlignment.Center;
textStamp.VerticalAlignment = VerticalAlignment.Bottom;

// सभी पृष्ठों पर पाद लेख जोड़ें
foreach (Page page in pdfDocument.Pages)
{
	page.AddStamp(textStamp);
}
dataDir = dataDir + "TextinFooter_out.pdf";

// अपडेट की गई पीडीएफ फाइल को सेव करें
pdfDocument.Save(dataDir);
Console.WriteLine("\nText in footer added successfully.\nFile saved at " + dataDir);

```

## निष्कर्ष

बधाई हो! आपने सीखा है कि .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के पाद लेख में टेक्स्ट कैसे जोड़ा जाता है। अब आप अपने पीडीएफ दस्तावेज़ों में अतिरिक्त टेक्स्ट जोड़कर अपने फ़ुटर को कस्टमाइज़ कर सकते हैं।

### पीडीएफ फ़ाइल के पादलेख में पाठ के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ दस्तावेज़ के पाद लेख में टेक्स्ट जोड़ने का उद्देश्य क्या है?

उ: पीडीएफ दस्तावेज़ के पाद लेख में पाठ जोड़ने से आप महत्वपूर्ण जानकारी, जैसे कॉपीराइट नोटिस, पृष्ठ संख्या, दस्तावेज़ संस्करण, या कोई अन्य पाठ शामिल कर सकते हैं जिसे आप प्रत्येक पृष्ठ के नीचे लगातार दिखाना चाहते हैं।

#### प्रश्न: प्रदान किया गया C# स्रोत कोड पीडीएफ दस्तावेज़ के पाद लेख में पाठ को कैसे जोड़ता है?

उ: कोड एक मौजूदा पीडीएफ दस्तावेज़ को खोलने, वांछित पादलेख पाठ के साथ एक टेक्स्ट स्टैम्प बनाने, टेक्स्ट गुणों को अनुकूलित करने, सभी पृष्ठों पर टेक्स्ट स्टैम्प जोड़ने और अंत में जोड़े गए पादलेख टेक्स्ट के साथ अद्यतन पीडीएफ दस्तावेज़ को सहेजने की प्रक्रिया को प्रदर्शित करता है।

#### प्रश्न: क्या मैं पाद लेख के स्वरूप, जैसे उसका फ़ॉन्ट, आकार, रंग और संरेखण को संशोधित कर सकता हूँ?

 उत्तर: हाँ, आप पाद लेख के गुणों को संशोधित करके उसके स्वरूप को अनुकूलित कर सकते हैं`TextStamp` वस्तु। कोड उदाहरण में निचला मार्जिन, क्षैतिज संरेखण और ऊर्ध्वाधर संरेखण जैसे सेटिंग गुण शामिल हैं। आप फ़ॉन्ट, आकार, रंग और अन्य पाठ-संबंधी गुणों को भी समायोजित कर सकते हैं।

#### प्रश्न: क्या प्रत्येक पृष्ठ के पादलेख में अलग-अलग पाठ जोड़ना संभव है?

 उत्तर: हाँ, आप प्रत्येक पृष्ठ के पादलेख में अलग-अलग टेक्स्ट बनाकर जोड़ सकते हैं`TextStamp` विभिन्न पाठ्य सामग्री या गुणों वाली वस्तुओं और फिर उन्हें आवश्यकतानुसार विशिष्ट पृष्ठों में जोड़ना।

#### प्रश्न: मैं यह कैसे सुनिश्चित करूं कि पादलेख पाठ पीडीएफ दस्तावेज़ के प्रत्येक पृष्ठ पर लगातार दिखाई दे?

उ: एक लूप का उपयोग करके जो पीडीएफ दस्तावेज़ के सभी पृष्ठों के माध्यम से पुनरावृत्त होता है और प्रत्येक पृष्ठ पर एक ही टेक्स्ट स्टैम्प जोड़कर, आप सुनिश्चित करते हैं कि पाद लेख प्रत्येक पृष्ठ पर लगातार दिखाई देता है।

#### प्रश्न: क्या मैं पाठ की एकाधिक पंक्तियाँ जोड़ सकता हूँ या पंक्ति विराम के साथ पादलेख पाठ को प्रारूपित कर सकता हूँ?

 उ: हां, आप टेक्स्ट स्ट्रिंग में लाइन ब्रेक शामिल करके पादलेख में टेक्स्ट की कई पंक्तियां जोड़ सकते हैं। उदाहरण के लिए, आप भागने के क्रम का उपयोग कर सकते हैं`\n` पाठ में एक पंक्ति विराम इंगित करने के लिए।

#### प्रश्न: यदि मैं एक ही पीडीएफ दस्तावेज़ के शीर्षलेख और पादलेख में अलग-अलग सामग्री जोड़ना चाहूं तो क्या होगा?

उ: शीर्ष लेख और पादलेख अनुभागों में अलग-अलग सामग्री जोड़ने के लिए, आपको दोनों अनुभागों के लिए समान चरणों का पालन करना होगा। कोड पाद लेख में पाठ जोड़ना दर्शाता है; आप हेडर में टेक्स्ट जोड़ने के लिए समान दृष्टिकोण का उपयोग कर सकते हैं।

#### प्रश्न: क्या इस दृष्टिकोण का उपयोग करके पादलेख पाठ के साथ चित्र या अन्य तत्व जोड़ना संभव है?

उ: जबकि प्रदान किया गया कोड विशेष रूप से पाद लेख में पाठ जोड़ने को दर्शाता है, आप Aspose.PDF लाइब्रेरी का उपयोग करके पाद लेख अनुभाग में छवियों, रेखाओं, आकृतियों या किसी अन्य सामग्री जैसे अन्य तत्वों को जोड़ने के दृष्टिकोण का विस्तार कर सकते हैं।
