ในบทช่วยสอนนี้ เราจะแนะนำคุณทีละขั้นตอนในการรับมิติของหน้าในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะอธิบายโค้ดต้นฉบับ C# ที่ให้มา และให้คำแนะนำที่ครอบคลุมแก่คุณเพื่อช่วยให้คุณเข้าใจและนำฟีเจอร์นี้ไปใช้ในโครงการของคุณเอง เมื่อจบบทช่วยสอนนี้ คุณจะทราบวิธีการรับมิติของหน้าในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET

## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- ความรู้พื้นฐานเกี่ยวกับภาษาการเขียนโปรแกรม C#
- Aspose.PDF สำหรับ .NET ติดตั้งอยู่ในสภาพแวดล้อมการพัฒนาของคุณ

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร
ขั้นแรก คุณต้องกำหนดเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือตำแหน่งที่ไฟล์ PDF ของคุณอยู่ แทนที่ "YOUR DOCUMENTS DIRECTORY" ด้วยเส้นทางที่เหมาะสม

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: เปิดเอกสาร PDF
 จากนั้นคุณสามารถเปิดไฟล์ PDF ได้โดยใช้`Document` คลาสของ Aspose.PDF โปรดระบุเส้นทางที่ถูกต้องไปยังไฟล์ PDF

```csharp
Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
```

## ขั้นตอนที่ 3: เพิ่มหน้าว่าง (ถ้าจำเป็น)
 หากเอกสาร PDF มีหลายหน้าอยู่แล้ว คุณสามารถข้ามไปยังหน้าที่มีอยู่แล้วโดยใช้ดัชนี`1` (หน้าแรกมีดัชนี 1) หรือจะเพิ่มหน้าใหม่ลงในเอกสารก็ได้

```csharp
Page page = pdfDocument.Pages.Count > 0? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
```

## ขั้นตอนที่ 4: รับขนาดหน้า
 ตอนนี้คุณสามารถรับขนาดหน้าได้โดยใช้`GetPageRect()` วิธีการของ`Page` วัตถุ วิธีการนี้ส่งคืน`Rectangle` วัตถุที่มีขนาดของหน้า คุณสามารถเข้าถึงความกว้างและความสูงได้โดยใช้`Width` และ`Height` คุณสมบัติ.

```csharp
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
```

## ขั้นตอนที่ 5: หมุนหน้า
 หากคุณต้องการหมุนหน้าคุณสามารถใช้`Rotate` ทรัพย์สินของ`Page`วัตถุ ในตัวอย่างนี้ หน้าจะหมุน 90 องศา

```csharp
page. Rotate = Rotate. on90;
```

## ขั้นตอนที่ 6: รับขนาดหน้าอีกครั้ง
 หลังจากหมุนหน้าแล้ว คุณสามารถรับขนาดหน้าได้อีกครั้งโดยใช้`GetPageRect()` วิธี.

```csharp
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
```

### ตัวอย่างโค้ดที่มาสำหรับรับมิติโดยใช้ Aspose.PDF สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "UpdateDimensions.pdf");
// เพิ่มหน้าว่างในเอกสาร PDF
Page page = pdfDocument.Pages.Count > 0 ? pdfDocument.Pages[1] : pdfDocument.Pages.Add();
// รับข้อมูลความสูงและความกว้างของหน้า
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);
// หมุนหน้ากระดาษเป็นมุม 90 องศา
page.Rotate = Rotation.on90;
// รับข้อมูลความสูงและความกว้างของหน้า
Console.WriteLine(page.GetPageRect(true).Width.ToString() + ":" + page.GetPageRect(true).Height);

```
