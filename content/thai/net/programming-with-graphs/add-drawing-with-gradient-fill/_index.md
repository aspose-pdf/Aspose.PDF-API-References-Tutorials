ในบทช่วยสอนนี้ เราจะพาคุณดูโค้ดต้นฉบับ C# ทีละขั้นตอนเพื่อเพิ่มรูปวาดด้วยการเติมแบบไล่ระดับให้กับการเขียนโปรแกรมด้วยกราฟิกโดยใช้ Aspose.PDF สำหรับ .NET

ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.PDF และตั้งค่าสภาพแวดล้อมการพัฒนาของคุณก่อนเริ่มต้น นอกจากนี้ ควรมีความรู้พื้นฐานเกี่ยวกับการเขียนโปรแกรม C# ด้วย

## ขั้นตอนที่ 1: การตั้งค่าไดเรกทอรีเอกสาร

ในซอร์สโค้ดที่ให้มา คุณต้องระบุไดเรกทอรีที่คุณต้องการบันทึกไฟล์ PDF ที่ได้ เปลี่ยนตัวแปร "dataDir" เป็นไดเรกทอรีที่ต้องการ

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: การสร้างวัตถุเอกสารและการเพิ่มหน้า

เราสร้างอินสแตนซ์ของคลาส Document และเพิ่มหน้าลงในเอกสารนี้

```csharp
Document doc = new Document();
Page page = doc.Pages.Add();
```

## ขั้นตอนที่ 3: การสร้างวัตถุกราฟและเพิ่มลงในหน้า

เราสร้างวัตถุ Graph ที่มีมิติที่ระบุและเพิ่มลงในคอลเล็กชันย่อหน้าของหน้า

```csharp
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(300, 300);
page.Paragraphs.Add(graph);
```

## ขั้นตอนที่ 4: สร้างวัตถุสี่เหลี่ยมผืนผ้าและเพิ่มลงในแผนภูมิ

เราสร้างวัตถุสี่เหลี่ยมผืนผ้าที่มีขนาดที่กำหนดและเพิ่มลงในคอลเล็กชันรูปร่างของแผนภูมิ

```csharp
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(0, 0, 300, 300);
graph.Shapes.Add(rect);
```

## ขั้นตอนที่ 5: การกำหนดค่าการเติมแบบไล่ระดับ

เราตั้งค่าการเติมไล่ระดับสำหรับสี่เหลี่ยมโดยใช้คลาส GradientAxialShading

```csharp
rect.GraphInfo.FillColor = new Aspose.Pdf.Color
{
PatternColorSpace = new GradientAxialShading(Color.Red, Color.Blue)
{
Start = new Point(0, 0),
End = new Point(300, 300)
}
};
```

ซึ่งจะสร้างการเติมแบบไล่ระดับจากสีแดงไปเป็นสีน้ำเงิน จากจุด (0, 0) ถึงจุด (300, 300)

## ขั้นตอนที่ 6: บันทึกไฟล์ PDF

ในที่สุด เราจะบันทึกไฟล์ PDF ผลลัพธ์ด้วยชื่อ "AddDrawingWithGradientFill_out.pdf" ในไดเร็กทอรีที่ระบุ

```csharp
doc.Save(dataDir + "AddDrawingWithGradientFill_out.pdf");
```

### ตัวอย่างโค้ดต้นฉบับสำหรับการเพิ่มรูปวาดด้วยการเติมแบบไล่ระดับโดยใช้ Aspose.PDF สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
Page page = doc.Pages.Add();
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(300, 300);
page.Paragraphs.Add(graph);
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(0, 0, 300, 300);
graph.Shapes.Add(rect);
rect.GraphInfo.FillColor = new Aspose.Pdf.Color
{
	PatternColorSpace = new GradientAxialShading(Color.Red, Color.Blue)
	{
		Start = new Point(0, 0),
		End = new Point(300, 300)
	}
};
doc.Save(dataDir + "AddDrawingWithGradientFill_out.pdf");

```

ตอบ: ใช่ คุณสามารถปรับแต่งสีและทิศทางของการเติมไล่ระดับได้โดยการปรับวัตถุสีและระบุจุดเริ่มต้นและจุดสิ้นสุดของ GradientAxialShading

#### ถาม: ฉันสามารถใช้การเติมแบบไล่ระดับกับรูปร่างอื่นนอกจากสี่เหลี่ยมผืนผ้าได้หรือไม่
