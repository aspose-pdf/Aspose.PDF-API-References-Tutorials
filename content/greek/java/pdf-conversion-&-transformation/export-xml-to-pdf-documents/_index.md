## Μετατροπή XML σε PDF

Τώρα που έχουμε ρυθμίσει το έργο μας και έχουμε προσθέσει το Aspose.PDF, ας βουτήξουμε στον κώδικα για τη μετατροπή XML σε PDF.

```java
// Φόρτωση δεδομένων XML
Document doc = new Document();
Page page = doc.getPages().add();

TextFragment textFragment = new TextFragment("Your XML Data Here");
page.getParagraphs().add(textFragment);

// Αποθήκευση ως PDF
doc.save("output.pdf");
```

 Σε αυτό το απόσπασμα κώδικα, δημιουργούμε πρώτα ένα νέο`Document` αντικείμενο και προσθέστε μια σελίδα σε αυτό. Στη συνέχεια δημιουργούμε ένα`TextFragment` περιέχει τα δεδομένα XML σας και προσθέστε τα στη σελίδα. Τέλος, αποθηκεύουμε το έγγραφο ως αρχείο PDF με το όνομα "output.pdf".
