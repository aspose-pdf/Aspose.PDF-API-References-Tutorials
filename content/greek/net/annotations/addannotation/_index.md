Η προσθήκη σχολιασμών σε έγγραφα PDF είναι μια ισχυρή δυνατότητα που μπορεί να βελτιώσει τις διαδικασίες συνεργασίας και αναθεώρησης. Το Aspose.PDF για .NET διευκολύνει την προσθήκη σχολιασμών μέσω προγραμματισμού σε έγγραφα PDF χρησιμοποιώντας C#. Σε αυτόν τον οδηγό, θα εξηγήσουμε βήμα προς βήμα πώς να χρησιμοποιήσετε το Aspose.PDF για .NET για να προσθέσετε σχολιασμούς σε ένα έγγραφο PDF.

## Βήμα 1: Δημιουργήστε ένα νέο έργο και εγκαταστήστε το Aspose.PDF για .NET

Πριν αρχίσουμε να γράφουμε τον κώδικα για την προσθήκη σχολιασμών, πρέπει να δημιουργήσουμε ένα νέο έργο και να εγκαταστήσουμε το Aspose.PDF για .NET. Για να εγκαταστήσετε το Aspose.PDF για .NET, ακολουθήστε τα εξής βήματα:

1. Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο C#.
2. Κάντε δεξί κλικ στο έργο στον Εξερεύνηση λύσεων και επιλέξτε "Διαχείριση πακέτων NuGet".
3. Αναζητήστε "Aspose.PDF" και επιλέξτε "Εγκατάσταση".

Μόλις ολοκληρωθεί η εγκατάσταση, μπορούμε να αρχίσουμε να γράφουμε τον κώδικα.

## Βήμα 2: Ανοίξτε το έγγραφο PDF

Το πρώτο βήμα για την προσθήκη σχολιασμών είναι να ανοίξετε το έγγραφο PDF. Μπορούμε να χρησιμοποιήσουμε τον ακόλουθο κώδικα για να ανοίξουμε το έγγραφο:

```csharp
string dataDir = "YOUR DATA DIRECTORY";
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

Σε αυτόν τον κώδικα, καθορίζουμε τη διαδρομή προς το έγγραφο PDF που θέλουμε να ανοίξουμε. Φροντίστε να αντικαταστήσετε το "YOUR DATA DECTORY" με την πραγματική διαδρομή προς τον κατάλογο δεδομένων σας.

## Βήμα 3: Δημιουργήστε τον σχολιασμό

 Για να προσθέσουμε έναν σχολιασμό, πρέπει να δημιουργήσουμε μια νέα παρουσία του`TextAnnotation` τάξη. Μπορούμε να χρησιμοποιήσουμε τον ακόλουθο κώδικα για να δημιουργήσουμε έναν νέο σχολιασμό κειμένου:

```csharp
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.State = AnnotationState.Accepted;
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;
```

Σε αυτόν τον κώδικα, δημιουργούμε έναν νέο σχολιασμό κειμένου στη δεύτερη σελίδα του εγγράφου PDF. Ορίζουμε επίσης τον τίτλο, το θέμα, την κατάσταση, τα περιεχόμενα, το άνοιγμα και τις ιδιότητες του εικονιδίου του σχολιασμού.

## Βήμα 4: Προσαρμόστε τον σχολιασμό

 Μπορούμε να προσαρμόσουμε την εμφάνιση του σχολιασμού χρησιμοποιώντας το`Border` τάξη. Μπορούμε να χρησιμοποιήσουμε τον ακόλουθο κώδικα για να προσαρμόσουμε το περίγραμμα του σχολιασμού:

```csharp
Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

 Σε αυτόν τον κώδικα, δημιουργούμε ένα νέο`Border`αντικείμενο και ορίστε τις ιδιότητες του πλάτους και της παύλας του. Στη συνέχεια ρυθμίσαμε το`Border` ιδιότητα του σχολιασμού στο νέο`Border` αντικείμενο. Τέλος, ορίσαμε το`Rect` ιδιότητα του σχολιασμού να προσδιορίζει τη θέση και το μέγεθός του.

## Βήμα 5: Προσθέστε τον σχολιασμό στο έγγραφο PDF

Αφού δημιουργήσουμε και προσαρμόσουμε τον σχολιασμό, πρέπει να τον προσθέσουμε στο έγγραφο PDF. Μπορούμε να χρησιμοποιήσουμε τον ακόλουθο κώδικα για να προσθέσουμε τον σχολιασμό στο έγγραφο PDF:

```csharp
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

Σε αυτόν τον κώδικα, προσθέτουμε τον σχολιασμό στη συλλογή σχολιασμών της δεύτερης σελίδας του εγγράφου PDF.

## Βήμα 6: Αποθηκεύστε το αρχείο εξόδου

Τέλος, πρέπει να αποθηκεύσουμε το έγγραφο PDF με τον προστιθέμενο σχολιασμό. Μπορούμε να χρησιμοποιήσουμε τον ακόλουθο κώδικα για να αποθηκεύσουμε το αρχείο εξόδου:

```csharp
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
```
### Παράδειγμα πηγαίου κώδικα για Προσθήκη σχολιασμού με χρήση Aspose.PDF για .NET


```csharp   
 // Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DATA DIRECTORY";

// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");

// Δημιουργία σχολιασμού
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600));
textAnnotation.Title = "Sample Annotation Title";
textAnnotation.Subject = "Sample Subject";
textAnnotation.State = AnnotationState.Accepted;
textAnnotation.Contents = "Sample contents for the annotation";
textAnnotation.Open = true;
textAnnotation.Icon = TextIcon.Key;

Border border = new Border(textAnnotation);
border.Width = 5;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);

// Προσθέστε σχολιασμούς στη συλλογή σχολιασμών της σελίδας
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
dataDir = dataDir + "AddAnnotation_out.pdf";
// Αποθήκευση αρχείου εξόδου
pdfDocument.Save(dataDir);
```
Αυτός ο κώδικας δείχνει πώς μπορείτε να προσθέσετε έναν σχολιασμό κειμένου με συγκεκριμένο τίτλο, θέμα, κατάσταση, περιεχόμενα και εικονίδιο σε μια σελίδα PDF χρησιμοποιώντας το Aspose.PDF για .NET. Μπορείτε να τροποποιήσετε αυτόν τον κώδικα σύμφωνα με τις απαιτήσεις σας για την προσθήκη σχολιασμών στα έγγραφά σας PDF. Απλώς θυμηθείτε να αντικαταστήσετε τον ΚΑΤΑΛΟΓΟ ΔΕΔΟΜΕΝΩΝ ΣΑΣ με την πραγματική διαδρομή καταλόγου όπου βρίσκεται το αρχείο PDF και όπου θέλετε να αποθηκεύσετε το αρχείο εξόδου.
