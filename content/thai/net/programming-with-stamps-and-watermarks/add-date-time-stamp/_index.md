ในบทความนี้ เราจะอธิบายทีละขั้นตอนเกี่ยวกับวิธีเพิ่มวันที่และเวลาในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะแสดงวิธีใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเพิ่มวันที่และเวลาในไฟล์ PDF ที่มีอยู่

## ความต้องการ

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเพิ่มวันที่และเวลาลงในเอกสาร PDF คุณต้องตั้งค่าสภาพแวดล้อมการพัฒนาของคุณเสียก่อน ขั้นตอนต่อไปนี้คือขั้นตอนที่ต้องปฏิบัติตาม:

1. เปิด IDE (Integrated Development Environment) ที่คุณชื่นชอบ
2. สร้างโครงการ C# ใหม่
3. ตรวจสอบให้แน่ใจว่าคุณได้เพิ่มการอ้างอิงไปยังไลบรารี Aspose.PDF สำหรับ .NET แล้ว

## ขั้นตอนที่ 2: การเพิ่มไลบรารี Aspose.PDF

จำเป็นต้องมีไลบรารี Aspose.PDF สำหรับ .NET เพื่อทำงานกับเอกสาร PDF ในโครงการของคุณ

## ขั้นตอนที่ 3: การโหลดเอกสาร PDF

ขั้นตอนแรกในการเพิ่มวันที่และเวลาคือการโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "AddTextStamp.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 4: การสร้างตราประทับวันที่และเวลา

ตอนนี้คุณได้อัพโหลดเอกสารแล้ว

  PDF คุณสามารถสร้างวันที่และเวลาที่จะเพิ่มได้ ดังต่อไปนี้:

```csharp
string annotationText = string.Empty;
annotationText = DateTime.Now.ToString("MM/dd/yy hh:mm:ss tt");

// สร้างบัฟเฟอร์ข้อความ
TextStamp textStamp = new TextStamp(annotationText);
```

โค้ดด้านบนจะสร้างบัฟเฟอร์ข้อความใหม่ซึ่งประกอบด้วยวันที่และเวลาปัจจุบัน

## ขั้นตอนที่ 5: การกำหนดค่าคุณสมบัติแสตมป์

ก่อนที่จะเพิ่มแสตมป์ลงในเอกสาร PDF คุณสามารถกำหนดค่าคุณสมบัติต่างๆ ของแสตมป์ได้ เช่น ระยะขอบ การจัดแนวแนวนอนและแนวตั้ง เป็นต้น ดังต่อไปนี้:

```csharp
// ตั้งค่าคุณสมบัติบัฟเฟอร์
textStamp.BottomMargin = 10;
textStamp. RightMargin = 20;
textStamp.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right;
textStamp.VerticalAlignment = VerticalAlignment.Bottom;
```

คุณสามารถปรับเปลี่ยนคุณสมบัติเหล่านี้ได้ตามความต้องการของคุณ

## ขั้นตอนที่ 6: เพิ่มแสตมป์ลงใน PDF

เมื่อวันที่และเวลาพร้อมแล้ว คุณสามารถเพิ่มลงในหน้าเฉพาะของเอกสาร PDF ได้ ดังต่อไปนี้:

```csharp
// เพิ่มแสตมป์เข้าในคอลเลกชันแสตมป์ของหน้า
pdfDocument.Pages[1].AddStamp(textStamp);
```

โค้ดด้านบนจะเพิ่มตราประทับลงในหน้าแรกของเอกสาร PDF คุณสามารถระบุหน้าอื่นได้หากจำเป็น

## ขั้นตอนที่ 7: บันทึกเอกสารผลลัพธ์

เมื่อคุณเพิ่มวันที่และเวลาแล้ว คุณสามารถบันทึกเอกสาร PDF ที่แก้ไขแล้วได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
```

โค้ดด้านบนจะบันทึกเอกสาร PDF ที่แก้ไขแล้วไปยังไดเร็กทอรีที่ระบุ

### ตัวอย่างโค้ดที่มาสำหรับการเพิ่มวันที่และเวลาโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "AddTextStamp.pdf");
string annotationText = string.Empty;
annotationText = DateTime.Now.ToString("MM/dd/yy hh:mm:ss tt ");

// สร้างแสตมป์ข้อความ
TextStamp textStamp = new TextStamp(annotationText);

// ตั้งค่าคุณสมบัติของแสตมป์
textStamp.BottomMargin = 10;
textStamp.RightMargin = 20;
textStamp.HorizontalAlignment = Aspose.Pdf.HorizontalAlignment.Right;
textStamp.VerticalAlignment = VerticalAlignment.Bottom;

// การเพิ่มแสตมป์ลงในคอลเลกชันแสตมป์
pdfDocument.Pages[1].AddStamp(textStamp);
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 6, System.Drawing.Color.Black);
FreeTextAnnotation textAnnotation = new FreeTextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(0, 0, 0, 0), default_appearance);
textAnnotation.Name = "Stamp";
textAnnotation.Accept(new AnnotationSelector(textAnnotation));
textAnnotation.Contents = textStamp.Value;

Border border = new Border(textAnnotation);
border.Width = 0;
border.Dash = new Dash(1, 1);
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(0, 0, 0, 0);
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
dataDir = dataDir + "AddDateTimeStamp_out.pdf";

// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
Console.WriteLine("\nDate time stamp added successfully.\nFile saved at " + dataDir);  
          
```
