## परिचय

प्रोग्रामेटिक रूप से पेशेवर दिखने वाले PDF बनाना एक कठिन काम लग सकता है, लेकिन .NET के लिए Aspose.PDF के साथ, यह बहुत आसान हो जाता है। चाहे आप रिपोर्ट, इनवॉइस या कोई अन्य दस्तावेज़ प्रकार बना रहे हों जिसके लिए सारणीबद्ध डेटा की आवश्यकता होती है, Aspose.PDF आपको आवश्यक उपकरण प्रदान करता है। इस ट्यूटोरियल में, हम चरण-दर-चरण PDF दस्तावेज़ में तालिकाओं को प्रस्तुत करने का तरीका जानने जा रहे हैं। अंत तक, आपको तालिकाओं में हेरफेर करने, पृष्ठ गुणों को प्रबंधित करने और PDF फ़ाइलों को आसानी से सहेजने के बारे में ठोस समझ हो जाएगी।

एक बार जब आप इन पूर्व-आवश्यकताओं को पूरा कर लेते हैं, तो आप अपने पीडीएफ दस्तावेज़ बनाना शुरू करने के लिए पूरी तरह तैयार हैं!

## पैकेज आयात करें

अपनी C# फ़ाइल की शुरुआत में, आपको आवश्यक Aspose.PDF नामस्थान आयात करने होंगे। इससे आप हमारे प्रोजेक्ट में लाइब्रेरी कार्यक्षमताओं का उपयोग कर सकेंगे।

```csharp
using System;
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

 सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.PDF लाइब्रेरी के लिए आवश्यक संदर्भ जोड़े हैं। यदि आप NuGet का उपयोग कर रहे हैं, तो आप इसे आसानी से खोज कर जोड़ सकते हैं`Aspose.PDF`.

अब जब हमने सब कुछ सेट कर लिया है, तो चलिए एक टेबल को PDF दस्तावेज़ में रेंडर करने के लिए प्रक्रिया को प्रबंधनीय चरणों में विभाजित करते हैं। चिंता न करें; मैं आपको स्पष्ट निर्देशों के साथ प्रत्येक चरण से गुज़रने में मदद करूँगा!

## चरण 1: दस्तावेज़ और पृष्ठ जानकारी सेट करें

सबसे पहले, हमें एक नया दस्तावेज़ बनाना होगा और उसकी पेज सेटिंग कॉन्फ़िगर करनी होगी। यह कैसे करना है, यहाँ बताया गया है:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
PageInfo pageInfo = doc.PageInfo;
Aspose.Pdf.MarginInfo marginInfo = pageInfo.Margin;

marginInfo.Left = 37;
marginInfo.Right = 37;
marginInfo.Top = 37;
marginInfo.Bottom = 37;

pageInfo.IsLandscape = true;
```

स्पष्टीकरण: 
- हम सबसे पहले यह परिभाषित करते हैं कि हमारा दस्तावेज़ कहाँ सहेजा जाएगा (`dataDir`). 
-  फिर, हम एक नया उदाहरण बनाते हैं`Document` कक्षा। 
- हम अपनी मेज के चारों ओर कुछ जगह बनाने के लिए पेज मार्जिन को कॉन्फ़िगर करते हैं।
- अंत में, हम दस्तावेज़ को लैंडस्केप ओरिएंटेशन पर सेट करते हैं, जो व्यापक तालिकाओं को प्रदर्शित करने में मदद करता है।

## चरण 2: पहली तालिका बनाएं

इसके बाद, आइए अपनी पहली तालिका बनाएं और उसमें कुछ नमूना डेटा भरें:

```csharp
Aspose.Pdf.Table table = new Aspose.Pdf.Table();
table.ColumnWidths = "50 100"; // स्तंभ की चौड़ाई निर्धारित करें
```

 स्पष्टीकरण: यहाँ, हम उदाहरण देते हैं`Table` क्लास पर जाएँ और कॉलम की चौड़ाई निर्धारित करें। पहला कॉलम 50 यूनिट चौड़ा होगा, और दूसरा कॉलम 100 यूनिट चौड़ा होगा।

## चरण 3: तालिका को पंक्तियों से भरें

अब, आइए अपनी तालिका में लूप में पंक्तियाँ जोड़ें:

```csharp
Page curPage = doc.Pages.Add(); // नया पृष्ठ जोड़ना
for (int i = 1; i <= 120; i++)
{
    Aspose.Pdf.Row row = table.Rows.Add();
    row.FixedRowHeight = 15; // पंक्तियों के लिए एक निश्चित ऊंचाई निर्धारित करें
    
    Aspose.Pdf.Cell cell1 = row.Cells.Add();
    cell1.Paragraphs.Add(new TextFragment("Content 1"));
    
    Aspose.Pdf.Cell cell2 = row.Cells.Add();
    cell2.Paragraphs.Add(new TextFragment("HHHHH"));
}
```

स्पष्टीकरण: 
- यहां हम अपनी तालिका जोड़ने के लिए एक नया पृष्ठ बनाते हैं।
-  हम एक का उपयोग करते हैं`for` हमारी तालिका में 120 पंक्तियाँ जोड़ने के लिए लूप। प्रत्येक पंक्ति की ऊँचाई 15 इकाई है।
- प्रत्येक पंक्ति के अंदर, हम दो कक्ष जोड़ते हैं और उन्हें पाठ से भर देते हैं।

## चरण 4: पृष्ठ पर पहली तालिका जोड़ें

एक बार जब हम तालिका भर देंगे, तो हम इसे वर्तमान पृष्ठ में जोड़ देंगे:

```csharp
Aspose.Pdf.Paragraphs paragraphs = curPage.Paragraphs;
paragraphs.Add(table);
```

स्पष्टीकरण: यह चरण हमारे द्वारा बनाई गई तालिका को वर्तमान पृष्ठ के पैराग्राफ में जोड़ता है, जिससे तालिका पीडीएफ दस्तावेज़ में दृश्यमान हो जाती है।

## चरण 5: दूसरी तालिका बनाएं

अब, आइए भिन्न सामग्री के साथ एक दूसरी तालिका बनाएं और इसे एक नए पृष्ठ में जोड़ें:

```csharp
Aspose.Pdf.Table table1 = new Aspose.Pdf.Table();
table1.ColumnWidths = "100 100";
for (int i = 1; i <= 10; i++)
{
    Aspose.Pdf.Row row = table1.Rows.Add();
    Aspose.Pdf.Cell cell1 = row.Cells.Add();
    cell1.Paragraphs.Add(new TextFragment("LAAAAAAA"));
    
    Aspose.Pdf.Cell cell2 = row.Cells.Add();
    cell2.Paragraphs.Add(new TextFragment("LAAGGGGGG"));
}
table1.IsInNewPage = true; // दूसरे टेबल को नए पेज पर प्रदर्शित करने के लिए सेट करना
paragraphs.Add(table1);
```

स्पष्टीकरण: 
- यह कोड स्निपेट दो कॉलमों वाली एक नई तालिका बनाता है, दोनों 100 इकाई चौड़े हैं।
-  ए`for` लूप नमूना सामग्री के साथ 10 पंक्तियाँ जोड़ता है।
-  सेटिंग करके`table1.IsInNewPage` यदि हम इसे true पर सेट करते हैं, तो हम यह सुनिश्चित करते हैं कि यह तालिका एक नए पृष्ठ पर दिखाई दे, जिससे चीजें व्यवस्थित और अव्यवस्थित न रहें।

## चरण 6: दस्तावेज़ सहेजें

अब चूंकि हमारी तालिकाएं तैयार हैं, तो आइए अपना दस्तावेज़ सेव करें:

```csharp
dataDir = dataDir + "IsNewPageProperty_Test_out.pdf";
doc.Save(dataDir);
```

 स्पष्टीकरण: हम फ़ाइल नाम निर्दिष्ट करते हैं और दस्तावेज़ को परिभाषित निर्देशिका में सहेजते हैं। जब आप इस कोड को चलाते हैं, तो एक पीडीएफ फ़ाइल जिसका शीर्षक होता है`IsNewPageProperty_Test_out.pdf` आपके निर्दिष्ट स्थान पर बनाया जाएगा.

## चरण 7: पुष्टिकरण संदेश

अंत में, उपयोगकर्ता को यह बताने के लिए कि सब कुछ सुचारू रूप से काम कर रहा है, हम एक दोस्ताना कंसोल संदेश जोड़ सकते हैं:

```csharp
Console.WriteLine("\nTable rendered successfully on a page.\nFile saved at " + dataDir);
```

स्पष्टीकरण: यह पुष्टि करने का एक सरल तरीका है कि ऑपरेशन सफल रहा और उपयोगकर्ता अपनी नई पीडीएफ फाइल कहां पा सकता है।
