في هذا البرنامج التعليمي، سنرشدك خطوة بخطوة حول كيفية إضافة صورة في قسم الرأس في مستند PDF باستخدام Aspose.PDF لـ .NET. سنستخدم كود المصدر C# المقدم لفتح مستند PDF موجود وإنشاء مخزن مؤقت للصور وتعيين خصائصه وإضافته إلى جميع صفحات مستند PDF.

## الخطوة 1: إعداد البيئة

قبل أن تبدأ، تأكد من أن لديك ما يلي:

- بيئة تطوير .NET مثبتة.
- تم تنزيل مكتبة Aspose.PDF لـ .NET والإشارة إليها في مشروعك.

## الخطوة 2: تحميل مستند PDF الموجود

الخطوة الأولى هي تحميل مستند PDF الموجود في مشروعك. وإليك الطريقة:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// افتح مستند PDF الموجود
Document pdfDocument = new Document(dataDir + "ImageinHeader.pdf");
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي إلى الدليل الذي يوجد به مستند PDF الخاص بك.

## الخطوة 3: إنشاء الصورة وإضافتها إلى قسم الرأس

الآن بعد تحميل مستند PDF، يمكننا إنشاء مخزن مؤقت للصور وإضافته إلى جميع صفحات المستند كقسم رأسي. وإليك الطريقة:

```csharp
// إنشاء إطار المخزن المؤقت
ImageStamp imageStamp = new ImageStamp(dataDir + "aspose-logo.jpg");

// تعيين خصائص مخزن الصور
imageStamp.TopMargin = 10;
imageStamp.HorizontalAlignment = HorizontalAlignment.Center;
imageStamp.VerticalAlignment = VerticalAlignment.Top;

//إضافة مخزن الصور إلى جميع الصفحات
foreach(Page page in pdfDocument.Pages)
{
     page.AddStamp(imageStamp);
}
```

يقوم الكود أعلاه بإنشاء مخزن مؤقت للصور من ملف "aspose-logo.jpg" وتعيين خصائصه، مثل الهامش العلوي والمحاذاة الأفقية والرأسية. ثم تتم إضافة ختم الصورة إلى جميع صفحات مستند PDF كقسم رأسي.

## الخطوة 4: حفظ مستند PDF المعدل

بمجرد إضافة الصورة إلى قسم الرأس، يمكننا حفظ مستند PDF المعدّل. إليك الطريقة:

```csharp
// حفظ مستند PDF المعدل
pdfDocument.Save(dataDir + "ImageinHeader_out.pdf");
```

يقوم الكود أعلاه بحفظ مستند PDF المحرر في الدليل المحدد.

### عينة من كود المصدر لـ Imagein Header باستخدام Aspose.PDF لـ .NET 

```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// فتح المستند
Document pdfDocument = new Document(dataDir+ "ImageinHeader.pdf");

// إنشاء رأس الصفحة
ImageStamp imageStamp = new ImageStamp(dataDir+ "aspose-logo.jpg");

// تعيين خصائص الطوابع
imageStamp.TopMargin = 10;
imageStamp.HorizontalAlignment = HorizontalAlignment.Center;
imageStamp.VerticalAlignment = VerticalAlignment.Top;

// إضافة رأس الصفحة إلى جميع الصفحات
foreach (Page page in pdfDocument.Pages)
{
	page.AddStamp(imageStamp);
}
dataDir = dataDir + "ImageinHeader_out.pdf";

// حفظ المستند المحدث
pdfDocument.Save(dataDir);
Console.WriteLine("\nImage in header added successfully.\nFile saved at " + dataDir);                        

```

## خاتمة

تهانينا! لقد تعلمت كيفية إضافة صورة في قسم الرأس في مستند PDF باستخدام Aspose.PDF لـ .NET. يمكنك الآن تخصيص رؤوس مستندات PDF الخاصة بك عن طريق إضافة الصور.

### الأسئلة الشائعة حول الصورة في العنوان

#### س: ما هو الغرض من إضافة صورة في قسم الرأس في مستند PDF؟

ج: إضافة صورة في قسم رأس الصفحة في مستند PDF يسمح لك بتضمين عناصر مرئية، مثل الشعار أو العلامة التجارية، في أعلى كل صفحة. ويمكن أن يعزز هذا المظهر العام لمحتوى PDF.

#### س: كيف يقوم كود المصدر C# المقدم بإضافة صورة إلى قسم الرأس في مستند PDF؟

 أ: يوضح الكود المقدم كيفية تحميل مستند PDF موجود وإنشاء`ImageStamp` كائن من ملف صورة، وتعيين خصائص مثل الهامش العلوي والمحاذاة، ثم إضافة طابع الصورة إلى رأس كل الصفحات.

#### س: هل يمكنني تعديل موضع ومحاذاة الصورة داخل قسم الرأس؟

 ج: نعم، يمكنك تعديل موضع ومحاذاة الصورة داخل قسم الرأس عن طريق تعديل خصائص`ImageStamp` الكائن. يحدد مقتطف التعليمات البرمجية خصائص مثل`TopMargin`, `HorizontalAlignment` ، و`VerticalAlignment`.

#### س: هل من الممكن إضافة صور مختلفة إلى قسم الرأس في صفحات مختلفة من مستند PDF؟

 ج: نعم، يمكنك إضافة صور مختلفة إلى قسم الرأس في صفحات مختلفة عن طريق إنشاء أقسام منفصلة`ImageStamp` الكائنات ذات ملفات الصور وخصائصها المختلفة، ثم إضافتها إلى صفحات محددة.

#### س: كيف يضمن الكود إضافة الصورة إلى جميع صفحات قسم رأس مستند PDF؟

 أ: يستخدم الكود المقدم`foreach` حلقة لتكرار جميع صفحات مستند PDF وإضافة نفس`ImageStamp` إلى قسم رأس كل صفحة.

#### س: هل يمكنني إضافة عناصر أخرى، مثل النص أو الأشكال، إلى قسم الرأس باستخدام نهج مماثل؟

 ج: نعم، يمكنك إضافة عناصر أخرى مثل النص أو الأشكال إلى قسم الرأس باستخدام نهج مماثل عن طريق إنشاء كائنات الطوابع المناسبة (على سبيل المثال،`TextStamp`) وضبط خصائصها وفقًا لذلك.

#### س: كيف أحدد المسار لملف الصورة الذي أريد إضافته إلى الرأس؟

 أ: يتم تحديد المسار إلى ملف الصورة عند إنشاء`ImageStamp` الكائن، كما هو موضح في الكود. تأكد من توفير المسار الصحيح لملف الصورة.

#### س: هل يمكنني تخصيص حجم الصورة داخل قسم الرأس؟

 ج: نعم، يمكنك تخصيص حجم الصورة داخل قسم الرأس عن طريق ضبط أبعاد`ImageStamp` استخدام خصائص مثل`Width` و`Height`.

#### س: هل من الممكن إزالة أو استبدال الصورة الموجودة في قسم الرأس بعد إضافتها؟

ج: نعم، يمكنك إزالة الصورة أو استبدالها في قسم الرأس عن طريق تعديل محتويات`ImageStamp` كائن أو إزالة الختم من صفحات معينة.

#### س: كيف يتعامل الكود مع السيناريوهات التي تتجاوز فيها أبعاد الصورة المساحة المتوفرة في العنوان؟

 أ: يقوم الكود بتعيين خصائص مثل`TopMargin`, `HorizontalAlignment` ، و`VerticalAlignment` للتحكم في وضع الصورة ومحاذاتها. تأكد من ضبط هذه الخصائص لمنع أي تداخل أو مشكلات في التخطيط.
