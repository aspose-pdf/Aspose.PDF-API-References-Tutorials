## Βήμα 1: Φόρτωση αρχείου Markdown
Σε αυτό το βήμα θα φορτώσουμε το αρχείο Markdown χρησιμοποιώντας το Aspose.PDF για .NET. Ακολουθήστε τον παρακάτω κώδικα:

```csharp

string dataDir = "YOUR DOCUMENTS DIRECTORY";

// Ανοίξτε το έγγραφο Markdown
Document doc = new Document(dataDir + "sample.md", new MdLoadOptions());
```

 Φροντίστε να αντικαταστήσετε`"YOUR DOCUMENTS DIRECTORY"` με τον πραγματικό κατάλογο όπου βρίσκεται το αρχείο Markdown.

## Βήμα 2: Μετατροπή Markdown σε PDF
Αφού φορτώσουμε το αρχείο Markdown, μπορούμε να προχωρήσουμε στη μετατροπή σε PDF. Χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
// Αποθηκεύστε το έγγραφο σε μορφή PDF

```

 Ο παραπάνω κώδικας μετατρέπει το αρχείο Markdown σε μορφή PDF και το αποθηκεύει ως όνομα αρχείου`"MarkdownToPDF.pdf"`.
