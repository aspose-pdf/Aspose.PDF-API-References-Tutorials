## مقدمة

في هذا الدليل التفصيلي، سنستكشف كيفية استخراج أسماء الصور المضمنة داخل ملف PDF باستخدام Java ومكتبة Aspose.PDF for Java. يمكن أن تحتوي الصور المضمنة على معلومات قيمة، وقد يكون من المفيد للغاية الوصول إلى أسمائها برمجيًا.

## إضافة Aspose.PDF لـ Java إلى مشروعك

Aspose.PDF for Java هي مكتبة قوية للعمل مع مستندات PDF. لاستخدامها في مشروعك، يمكنك إضافتها كتبعية. فيما يلي مقتطف من التبعية الخاصة بـ Maven:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-pdf</artifactId>
    <version>21.10</version>
</dependency>
```

## تحميل ملف PDF

 في هذه الخطوة، سنكتب كود Java لتحميل ملف PDF في تطبيقنا. يجب استبدال`"input.pdf"` مع المسار إلى ملف PDF الخاص بك.

```java
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document("input.pdf");
```

## استخراج الصور المضمنة

لاستخراج الصور المضمنة، نحتاج إلى الوصول إلى موارد مستند PDF. إليك كيفية القيام بذلك:

```java
com.aspose.pdf.Page page = pdfDocument.getPages().get_Item(1); // استبدل برقم الصفحة المطلوب
com.aspose.pdf.XImageCollection images = page.getResources().getImages();
```

## كتابة كود جافا لاستخراج الصور

الآن، دعنا ننتقل عبر الصور المضمنة ونستخرج أسمائها:

```java
for (com.aspose.pdf.XImage image : images) {
    System.out.println("Image Name: " + image.getName());
}
```
