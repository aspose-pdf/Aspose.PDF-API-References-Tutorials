यह ट्यूटोरियल बताता है कि PDF दस्तावेज़ में विशिष्ट टेक्स्ट खोजने, पाए गए टेक्स्ट के चारों ओर एक आयत बनाने और संशोधित दस्तावेज़ को सहेजने के लिए .NET के लिए Aspose.PDF का उपयोग कैसे करें। प्रदान किया गया C# स्रोत कोड चरण दर चरण प्रक्रिया को प्रदर्शित करता है।

## आवश्यक शर्तें

ट्यूटोरियल के साथ आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- C# प्रोग्रामिंग भाषा का बुनियादी ज्ञान।
- .NET लाइब्रेरी के लिए Aspose.PDF स्थापित है। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या अपने प्रोजेक्ट में इसे स्थापित करने के लिए NuGet का उपयोग कर सकते हैं।

## चरण 1: प्रोजेक्ट सेट अप करें

अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाकर आरंभ करें और .NET लाइब्रेरी के लिए Aspose.PDF का संदर्भ जोड़ें।

## चरण 2: आवश्यक नामस्थान आयात करें

आवश्यक नामस्थानों को आयात करने के लिए अपनी C# फ़ाइल के आरंभ में निम्नलिखित using निर्देश जोड़ें:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using Aspose.Pdf.Content;
using Aspose.Pdf.Facades;
```

## चरण 3: दस्तावेज़ निर्देशिका का पथ सेट करें

 का उपयोग करके अपने दस्तावेज़ निर्देशिका का पथ सेट करें`dataDir` चर:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 प्रतिस्थापित करें`"YOUR DOCUMENT DIRECTORY"` आपके दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ.

## चरण 4: पीडीएफ दस्तावेज़ लोड करें

 पीडीएफ दस्तावेज़ को लोड करें`Document` कक्षा:

```csharp
Document document = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
```

 प्रतिस्थापित करें`"SearchAndGetTextFromAll.pdf"` अपनी पीडीएफ फाइल के वास्तविक नाम के साथ।

## चरण 5: एक टेक्स्टफ़्रेगमेंटअब्ज़ॉर्बर बनाएँ

 एक बनाने के`TextFragmentAbsorber` इनपुट खोज वाक्यांश के सभी उदाहरण खोजने के लिए ऑब्जेक्ट:

```csharp
TextFragmentAbsorber textAbsorber = new TextFragmentAbsorber(@"[\S]+");
```

 प्रतिस्थापित करें`@"[\S]+"` अपने इच्छित नियमित अभिव्यक्ति पैटर्न के साथ।

## चरण 6: नियमित अभिव्यक्ति खोज सक्षम करें

 सेट करके नियमित अभिव्यक्ति खोज सक्षम करें`TextSearchOptions` अवशोषक की संपत्ति:

```csharp
TextSearchOptions textSearchOptions = new TextSearchOptions(true);
textAbsorber.TextSearchOptions = textSearchOptions;
```

## चरण 7: सभी पृष्ठों पर खोजें

दस्तावेज़ के सभी पृष्ठों के लिए अवशोषक स्वीकार करें:

```csharp
document.Pages.Accept(textAbsorber);
```

## चरण 8: पाए गए पाठ के चारों ओर एक आयत बनाएं

 एक बनाने के`PdfContentEditor` ऑब्जेक्ट और पुनर्प्राप्त पाठ टुकड़ों के माध्यम से लूप करें, प्रत्येक पाठ खंड के चारों ओर एक आयत बनाएं:

```csharp
var editor = new PdfContentEditor(document);
foreach (TextFragment textFragment in textAbsorber.TextFragments)
{
    foreach (TextSegment textSegment in textFragment.Segments)
    {
        DrawBox(editor, textFragment.Page.Number, textSegment, System.Drawing.Color.Red);
    }
}
```

## चरण 9: संशोधित दस्तावेज़ को सहेजें

संशोधित दस्तावेज़ सहेजें:

```csharp
dataDir = dataDir + "SearchTextAndDrawRectangle_out.pdf";
document.Save(dataDir);
```

 प्रतिस्थापित करना सुनिश्चित करें`"SearchTextAndDrawRectangle_out.pdf"` वांछित आउटपुट फ़ाइल नाम के साथ.

### .NET के लिए Aspose.PDF का उपयोग करके टेक्स्ट खोजें और आयत बनाएं के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ खोलें
Document document = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
// नियमित अभिव्यक्ति से मेल खाने वाले सभी वाक्यांशों को खोजने के लिए TextAbsorber ऑब्जेक्ट बनाएं
TextFragmentAbsorber textAbsorber = new TextFragmentAbsorber(@"[\S]+");
TextSearchOptions textSearchOptions = new TextSearchOptions(true);
textAbsorber.TextSearchOptions = textSearchOptions;
document.Pages.Accept(textAbsorber); 
var editor = new PdfContentEditor(document); 
foreach (TextFragment textFragment in textAbsorber.TextFragments)
{
	foreach (TextSegment textSegment in textFragment.Segments)
	{
			DrawBox(editor, textFragment.Page.Number, textSegment, System.Drawing.Color.Red);
	}
}
dataDir = dataDir + "SearchTextAndDrawRectangle_out.pdf";
document.Save(dataDir);
Console.WriteLine("\nRectangle drawn successfully on searched text.\nFile saved at " + dataDir);
```

## निष्कर्ष

बधाई हो! आपने सफलतापूर्वक सीख लिया है कि PDF दस्तावेज़ में विशिष्ट टेक्स्ट कैसे खोजें, पाए गए टेक्स्ट के चारों ओर एक आयत बनाएँ, और .NET के लिए Aspose.PDF का उपयोग करके संशोधित दस्तावेज़ को कैसे सेव करें। इस ट्यूटोरियल में प्रोजेक्ट को सेट अप करने से लेकर आवश्यक क्रियाएँ करने तक, चरण-दर-चरण मार्गदर्शन प्रदान किया गया है। अब आप इस कोड को अपने स्वयं के C# प्रोजेक्ट में शामिल कर सकते हैं ताकि PDF फ़ाइलों में टेक्स्ट में हेरफेर किया जा सके और आयतें बनाई जा सकें।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: "टेक्स्ट खोजें और आयत बनाएं" ट्यूटोरियल का उद्देश्य क्या है?

उत्तर: "टेक्स्ट खोजें और आयत बनाएं" ट्यूटोरियल का उद्देश्य उपयोगकर्ताओं को .NET के लिए Aspose.PDF लाइब्रेरी का उपयोग करने की प्रक्रिया के माध्यम से मार्गदर्शन करना है ताकि PDF दस्तावेज़ के भीतर विशिष्ट टेक्स्ट की खोज की जा सके, पाए गए टेक्स्ट सेगमेंट के चारों ओर आयत बनाएं और संशोधित दस्तावेज़ को सहेजें। ट्यूटोरियल प्रक्रिया के प्रत्येक चरण को स्पष्ट करने के लिए विस्तृत निर्देश और C# कोड नमूने प्रदान करता है।

#### प्रश्न: यह ट्यूटोरियल पीडीएफ दस्तावेज़ में विशिष्ट पाठ के चारों ओर आयत बनाने में कैसे मदद करता है?

उत्तर: यह ट्यूटोरियल PDF दस्तावेज़ में विशिष्ट टेक्स्ट खंडों के चारों ओर आयतों को खोजने और बनाने के तरीके पर एक व्यापक मार्गदर्शिका प्रदान करता है। यह एक प्रोजेक्ट सेट अप करने, PDF दस्तावेज़ लोड करने, रेगुलर एक्सप्रेशन खोज को सक्षम करने, पाए गए टेक्स्ट खंडों के चारों ओर आयतों को बनाने और संशोधित PDF को सहेजने की प्रक्रिया को प्रदर्शित करता है।

#### प्रश्न: इस ट्यूटोरियल का अनुसरण करने के लिए क्या पूर्वापेक्षाएँ आवश्यक हैं?

उत्तर: ट्यूटोरियल शुरू करने से पहले, आपको C# प्रोग्रामिंग भाषा की बुनियादी समझ होनी चाहिए। इसके अतिरिक्त, आपके पास .NET लाइब्रेरी के लिए Aspose.PDF स्थापित होना चाहिए। आप इसे Aspose वेबसाइट से प्राप्त कर सकते हैं या NuGet का उपयोग करके अपने प्रोजेक्ट में इंस्टॉल कर सकते हैं।

#### प्रश्न: मैं इस ट्यूटोरियल का अनुसरण करने के लिए अपना प्रोजेक्ट कैसे सेट करूँ?

उत्तर: अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक नया C# प्रोजेक्ट बनाकर शुरू करें। फिर, अपने प्रोजेक्ट में Aspose.PDF for .NET लाइब्रेरी का संदर्भ जोड़ें। यह आपको PDF दस्तावेज़ों में हेरफेर करने के लिए लाइब्रेरी की कार्यक्षमता का उपयोग करने में सक्षम करेगा।

#### प्रश्न: क्या मैं इस ट्यूटोरियल का उपयोग करके विशिष्ट पाठ के चारों ओर आयत बना सकता हूँ?

उत्तर: हां, यह ट्यूटोरियल पीडीएफ दस्तावेज़ के भीतर विशिष्ट टेक्स्ट खंडों के चारों ओर आयत बनाने पर केंद्रित है। यह दर्शाता है कि नियमित अभिव्यक्तियों का उपयोग करके वांछित टेक्स्ट का पता कैसे लगाया जाए, पहचाने गए टेक्स्ट खंडों के चारों ओर आयत कैसे बनाएं और संशोधित पीडीएफ को कैसे सेव करें।

#### प्रश्न: मैं वह पाठ कैसे निर्दिष्ट कर सकता हूँ जिसे मैं खोजना चाहता हूँ और उसके चारों ओर आयत कैसे बना सकता हूँ?

 उत्तर: जिस पाठ को आप खोजना चाहते हैं उसे निर्दिष्ट करने और उसके चारों ओर आयत बनाने के लिए, एक बनाएँ`TextFragmentAbsorber` ऑब्जेक्ट और इसका पैटर्न सेट करें`Text` पैरामीटर. डिफ़ॉल्ट पैटर्न बदलें`@"[\S]+"` ट्यूटोरियल के कोड में अपने इच्छित नियमित अभिव्यक्ति पैटर्न के साथ।

#### प्रश्न: मैं पाठ के लिए नियमित अभिव्यक्ति खोज कैसे सक्षम करूं?

 A: नियमित अभिव्यक्ति खोज एक बनाकर सक्षम की जाती है`TextSearchOptions` ऑब्जेक्ट और उसका मान सेट करना`true` . इस ऑब्जेक्ट को असाइन करें`TextSearchOptions` की संपत्ति`TextFragmentAbsorber` यह सुनिश्चित करता है कि पाठ खोज के दौरान नियमित अभिव्यक्ति पैटर्न का उपयोग किया जाता है।

#### प्रश्न: मैं पाए गए पाठ के चारों ओर आयत कैसे बनाऊं?

 उत्तर: पाठ खंडों की पहचान करने के बाद`TextFragmentAbsorber` , ट्यूटोरियल इन खंडों के माध्यम से पुनरावृति करने के लिए एक लूप प्रदान करता है। प्रत्येक पाठ खंड के लिए, ट्यूटोरियल दर्शाता है कि इसके चारों ओर एक आयत कैसे बनाया जाए`DrawBox` विधि का उपयोग करें और आयत का स्वरूप निर्दिष्ट करें।

#### प्रश्न: संशोधित पीडीएफ को खींचे गए आयतों के साथ सहेजने के चरण क्या हैं?

उत्तर: इच्छित पाठ खंडों के चारों ओर आयत बनाने के बाद, का उपयोग करें`Document` कक्षा का`Save` संशोधित दस्तावेज़ को सहेजने की विधि। ट्यूटोरियल का नमूना कोड दिखाता है कि संपादित पीडीएफ को कैसे सहेजा जाए और सफलता संदेश कैसे प्रदर्शित किया जाए।

#### प्रश्न: क्या मैं खींची गई आयतों के स्वरूप को अनुकूलित कर सकता हूँ?

 उत्तर: हां, आप खींचे गए आयतों की उपस्थिति को अनुकूलित कर सकते हैं। ट्यूटोरियल के नमूना कोड में,`DrawBox` आयत बनाने के लिए विधि का उपयोग किया जाता है। आप खींचे गए आयतों की उपस्थिति को अनुकूलित करने के लिए रंग, शैली और मोटाई जैसे गुणों को संशोधित कर सकते हैं।
