Aspose.PDF for .NET एक शक्तिशाली लाइब्रेरी है जो PDF फ़ाइलों के साथ काम करने के लिए विभिन्न सुविधाएँ प्रदान करती है। ऐसी ही एक विशेषता PDF दस्तावेज़ के लिए समाप्ति तिथि निर्धारित करने की क्षमता है। इस ट्यूटोरियल में, हम आपको Aspose.PDF for .NET का उपयोग करके PDF दस्तावेज़ के लिए समाप्ति तिथि निर्धारित करने की प्रक्रिया से परिचित कराएँगे। 

## चरण 1: दस्तावेज़ निर्देशिका का पथ सेट करें

शुरू करने से पहले, हमें उस निर्देशिका का पथ सेट करना होगा जहाँ हमारा PDF दस्तावेज़ स्थित है। हम इस पथ को "dataDir" नामक एक चर में संग्रहीत करेंगे।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: नया PDF दस्तावेज़ बनाना

 एक नया पीडीएफ दस्तावेज़ बनाने के लिए, हमें एक नया इंस्टैंसिएट करना होगा`Aspose.Pdf.Document` हम निम्नलिखित कोड का उपयोग करके ऐसा कर सकते हैं:

```csharp
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

## चरण 3: पीडीएफ दस्तावेज़ में नया पृष्ठ जोड़ना

एक बार जब हम PDF दस्तावेज़ बना लेते हैं, तो हम इसमें एक नया पेज जोड़ सकते हैं। हम निम्नलिखित कोड का उपयोग करके ऐसा कर सकते हैं:

```csharp
doc.Pages.Add();
```

## चरण 4: पीडीएफ दस्तावेज़ में पाठ जोड़ना

 पीडीएफ दस्तावेज़ में एक पृष्ठ जोड़ने के बाद, हम इसका उपयोग करके इसमें पाठ जोड़ सकते हैं`Paragraphs` संग्रह। हम निम्नलिखित कोड का उपयोग करके ऐसा कर सकते हैं:

```csharp
doc.Pages[1].Paragraphs.Add(new TextFragment("Hello World..."));
```

## चरण 5: जावास्क्रिप्ट का उपयोग करके पीडीएफ समाप्ति तिथि निर्धारित करना

पीडीएफ की समाप्ति तिथि निर्धारित करने के लिए, हमें एक जावास्क्रिप्ट ऑब्जेक्ट बनाना होगा। हम निम्नलिखित कोड का उपयोग करके ऐसा कर सकते हैं:

```csharp
JavascriptAction javaScript = new JavascriptAction(
"var year=2017;"
+ "var month=5;"
+ "today = new Date(); today = new Date(today.getFullYear(), today.getMonth());"
+ "expiry = new Date(year, month);"
+ "if (today.getTime() > expiry.getTime())"
+ "app.alert('The file is expired. You need a new one.');");

// जावास्क्रिप्ट को पीडीएफ खोलने की क्रिया के रूप में सेट करें
doc.OpenAction = javaScript;
```

इस कोड में, हम समाप्ति तिथि मई 2017 निर्धारित कर रहे हैं।

## चरण 6: पीडीएफ फाइल को सेव करें

 समाप्ति तिथि निर्धारित करने के बाद, आपको पीडीएफ फाइल को सहेजना होगा। ऐसा करने के लिए, आप इसका उपयोग कर सकते हैं`Save` की विधि`Document` ऑब्जेक्ट चुनें और उस पथ को पास करें जहां आप अपडेट की गई पीडीएफ फाइल को सहेजना चाहते हैं।

```csharp
dataDir = dataDir + "SetExpiryDate_out.pdf";
// पीडीएफ दस्तावेज़ सहेजें
doc.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके समाप्ति तिथि निर्धारित करने के लिए उदाहरण स्रोत कोड

.NET के लिए Aspose.PDF का उपयोग करके समाप्ति तिथि निर्धारित करने के लिए पूर्ण उदाहरण स्रोत कोड यहां दिया गया है:

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
// पीडीएफ फाइल के पेज टू पेज संग्रह को जोड़ें
doc.Pages.Add();
// पृष्ठ ऑब्जेक्ट के पैराग्राफ़ संग्रह में पाठ अंश जोड़ें
doc.Pages[1].Paragraphs.Add(new TextFragment("Hello World..."));
// पीडीएफ समाप्ति तिथि निर्धारित करने के लिए जावास्क्रिप्ट ऑब्जेक्ट बनाएं
JavascriptAction javaScript = new JavascriptAction(
"var year=2017;"
+ "var month=5;"
+ "today = new Date(); today = new Date(today.getFullYear(), today.getMonth());"
+ "expiry = new Date(year, month);"
+ "if (today.getTime() > expiry.getTime())"
+ "app.alert('The file is expired. You need a new one.');");
// जावास्क्रिप्ट को पीडीएफ खोलने की क्रिया के रूप में सेट करें
doc.OpenAction = javaScript;

dataDir = dataDir + "SetExpiryDate_out.pdf";
// पीडीएफ दस्तावेज़ सहेजें
doc.Save(dataDir);
```

## निष्कर्ष

.NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ के लिए समाप्ति तिथि निर्धारित करना यह सुनिश्चित करने के लिए एक उपयोगी सुविधा है कि दस्तावेज़ केवल निर्दिष्ट अवधि के लिए वैध है। चरण-दर-चरण मार्गदर्शिका का पालन करके और प्रदान किए गए C# स्रोत कोड का उपयोग करके, डेवलपर्स आसानी से समाप्ति तिथि निर्धारित कर सकते हैं और समय-सीमित वैधता वाले PDF बना सकते हैं। यह सुविधा उन दस्तावेज़ों के लिए विशेष रूप से सहायक हो सकती है जिन्हें सीमित अवधि के लिए एक्सेस या वितरित करने की आवश्यकता होती है।

### पीडीएफ फाइल में समाप्ति तिथि निर्धारित करने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं पीडीएफ दस्तावेज़ के लिए अलग समाप्ति तिथि निर्धारित कर सकता हूं?

 उत्तर: हां, आप चरण 5 में जावास्क्रिप्ट कोड को संशोधित करके पीडीएफ दस्तावेज़ के लिए एक अलग समाप्ति तिथि निर्धारित कर सकते हैं। दिए गए उदाहरण में, समाप्ति तिथि मई 2017 निर्धारित की गई है। एक अलग समाप्ति तिथि निर्धारित करने के लिए, आपको संशोधित करने की आवश्यकता है`year` और`month` जावास्क्रिप्ट कोड में चर को वांछित वर्ष और माह में बदलें।

#### प्रश्न: जब पीडीएफ दस्तावेज़ की समयसीमा समाप्त हो जाती है तो क्या होता है?

उत्तर: जब PDF दस्तावेज़ की समय-सीमा समाप्त हो जाती है, जैसा कि जावास्क्रिप्ट कोड में निर्दिष्ट है, तो दर्शक एक चेतावनी संदेश प्रदर्शित करेगा जो यह संकेत देगा कि फ़ाइल की समय-सीमा समाप्त हो गई है और उपयोगकर्ता को एक नई फ़ाइल की आवश्यकता है। यह चेतावनी संदेश तब दिखाया जाएगा जब PDF को खोला जाएगा।

#### प्रश्न: क्या मैं समाप्ति तिथि के लिए केवल तारीख के बजाय किसी विशिष्ट समय का उपयोग कर सकता हूँ?

 उत्तर: हां, आप जावास्क्रिप्ट कोड में समाप्ति तिथि के लिए एक विशिष्ट समय निर्धारित कर सकते हैं।`expiry` जावास्क्रिप्ट कोड में वांछित समय शामिल करने के लिए, आप समाप्ति तिथि के लिए एक विशिष्ट समय निर्धारित कर सकते हैं।
