ในบทช่วยสอนนี้ เราจะอธิบายทีละขั้นตอนเกี่ยวกับวิธีเพิ่มตราประทับหน้า PDF ในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะแสดงวิธีใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเพิ่มตราประทับแบบกำหนดเองลงในหน้าเฉพาะของไฟล์ PDF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 2: การโหลดเอกสาร PDF

ขั้นตอนแรกคือโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "PDFPageStamp.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 3: การสร้างบัฟเฟอร์หน้า

ตอนนี้คุณได้อัปโหลดเอกสาร PDF แล้ว คุณสามารถสร้างตราประทับหน้าเพื่อเพิ่มได้ วิธีการทำมีดังนี้:

```csharp
// สร้างบัฟเฟอร์หน้า
PdfPageStamp pageStamp = new PdfPageStamp(pdfDocument.Pages[1]);
```

โค้ดด้านบนจะสร้างบัฟเฟอร์หน้าใหม่โดยใช้หน้าแรกของเอกสาร PDF

## ขั้นตอนที่ 4: การกำหนดค่าคุณสมบัติบัฟเฟอร์หน้า

ก่อนที่จะเพิ่มตราประทับหน้าลงในเอกสาร PDF คุณสามารถกำหนดค่าคุณสมบัติต่างๆ ของตราประทับ เช่น พื้นหลัง ตำแหน่ง การหมุน ฯลฯ ได้ ดังนี้:

```csharp
// กำหนดค่าคุณสมบัติบัฟเฟอร์หน้า
pageStamp. Background = true;
pageStamp. XIndent = 100;
pageStamp. YIndent = 100;
pageStamp.Rotate = Rotate.on180;
```

คุณสามารถปรับเปลี่ยนคุณสมบัติเหล่านี้ได้ตามความต้องการของคุณ

## ขั้นตอนที่ 5: เพิ่มตราประทับหน้าลงใน PDF

เมื่อประทับตราหน้าเสร็จเรียบร้อยแล้ว คุณสามารถเพิ่มตราประทับหน้าลงในหน้าเฉพาะของเอกสาร PDF ได้ ดังต่อไปนี้:

```csharp
// เพิ่มบัฟเฟอร์หน้าเพจให้กับเพจที่ต้องการ
pdfDocument.Pages[1].AddStamp(pageStamp);
```

โค้ดด้านบนจะเพิ่มตราประทับหน้าลงในหน้าแรกของเอกสาร PDF คุณสามารถระบุหน้าอื่นได้หากจำเป็น

## ขั้นตอนที่ 6: บันทึกเอกสารผลลัพธ์

เมื่อคุณเพิ่มตราประทับหน้าแล้ว คุณสามารถบันทึกเอกสาร PDF ที่แก้ไขแล้วได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
```

### ตัวอย่างโค้ดที่มาสำหรับ Add PDFPage Stamp โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "PDFPageStamp.pdf");

// สร้างแสตมป์หน้า
PdfPageStamp pageStamp = new PdfPageStamp(pdfDocument.Pages[1]);
pageStamp.Background = true;
pageStamp.XIndent = 100;
pageStamp.YIndent = 100;
pageStamp.Rotate = Rotation.on180;

// เพิ่มตราประทับบนหน้าที่ต้องการ
pdfDocument.Pages[1].AddStamp(pageStamp);
dataDir = dataDir + "PDFPageStamp_out.pdf";

// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
Console.WriteLine("\nPdf page stamp added successfully.\nFile saved at " + dataDir);

```

โค้ดด้านบนจะบันทึกเอกสาร PDF ที่แก้ไขแล้วไปยังไดเร็กทอรีที่ระบุ

 A: คุณสามารถปรับแต่งตำแหน่งและการหมุนของแสตมป์หน้าได้โดยการแก้ไขคุณสมบัติของ`PdfPageStamp` วัตถุ โค้ดที่ให้ไว้ในบทช่วยสอนสาธิตวิธีการตั้งค่าคุณสมบัติ เช่น`XIndent`, `YIndent` , และ`Rotate` เพื่อควบคุมตำแหน่งและทิศทางของแสตมป์
