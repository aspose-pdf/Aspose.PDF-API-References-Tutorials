## الخطوة 1: تعيين الدليل

للبدء، تحتاج إلى تحديد المسار الذي سيتم تخزين ملف PDF فيه. هذا هو المكان الذي ستوجه فيه التعليمات البرمجية الخاصة بك لإنشاء مستند PDF.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يتغير`YOUR DOCUMENT DIRECTORY` إلى المسار الفعلي الذي تريد حفظ ملف PDF الخاص بك فيه. 

## الخطوة 2: تهيئة المستند

بعد تحديد الدليل، تكون الخطوة التالية هي إنشاء مستند PDF جديد. وهذا يشبه وضع أساس المنزل - فكل شيء آخر سيبنى فوق هذا الأساس.

```csharp
Document pdfDocument = new Document();
```

## الخطوة 3: إنشاء جدول

والآن حان الوقت لإنشاء نجم العرض: الطاولة.

```csharp
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

يقوم هذا الخط بإنشاء كائن جدول جديد جاهز لبعض السحر!

## الخطوة 4: إنشاء معلومات الحدود

 لإضفاء مظهر جميل على الجدول الخاص بك وتأثير الزوايا المستديرة، تحتاج إلى إنشاء مثيل لـ`BorderInfo`.

```csharp
GraphInfo graph = new GraphInfo();
graph.Color = Aspose.Pdf.Color.Red; // حدد اللون المفضل لديك
BorderInfo bInfo = new BorderInfo(BorderSide.All, graph);
```

هنا قمنا بتحديد الحدود وضبط لونها إلى الأحمر. يمكنك اختيار أي لون تريده!

## الخطوة 5: تعيين نصف قطر الحدود المستديرة

الآن، دعونا نعطي تلك الزوايا بعض الشخصية عن طريق تقريبها.

```csharp
bInfo.RoundedBorderRadius = 15; // ضبط نصف القطر حسب الحاجة
```

سيمنحك نصف القطر 15 تأثيرًا دائريًا ملحوظًا. لا تتردد في تعديل هذا الرقم ليناسب أسلوبك.

## الخطوة 6: تقريب زوايا الطاولة

بعد ذلك، أخبر الجدول باستخدام الزوايا المستديرة التي قمت بتعريفها للتو.

```csharp
tab1.CornerStyle = Aspose.Pdf.BorderCornerStyle.Round;
```

مع هذا الخط، طاولتك أصبحت رسميًا في نادي الزاوية المستديرة!

## الخطوة 7: تطبيق الحدود على الجدول

دعنا نجمع كل شيء معًا من خلال تطبيق معلومات الحدود على جدولك.

```csharp
tab1.Border = bInfo;
```

بوم! الآن أصبح لطاولتك حدود ذات زوايا مستديرة.

## الخطوة 8: إضافة الجدول إلى مستند PDF

حتى الآن، قمنا بإعداد كل شيء. الآن، دعنا نضيف الجدول إلى المستند.

```csharp
pdfDocument.Pages.Add().Paragraphs.Add(tab1);
```

يقوم هذا السطر بأخذ الجدول وإضافته إلى صفحة جديدة في ملف PDF الخاص بك. 

## الخطوة 9: حفظ المستند

الخطوة الأخيرة في رحلتنا هي حفظ مستند PDF. 

```csharp
pdfDocument.Save(dataDir + "RoundedCornerTable.pdf");
```

هنا، سوف تقوم بحفظه في الدليل المحدد باسم "RoundedCornerTable.pdf".
