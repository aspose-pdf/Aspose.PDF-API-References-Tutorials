เรียนรู้วิธีการตั้งค่าลิงก์ปลายทางในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET ด้วยคู่มือทีละขั้นตอนนี้

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่าสภาพแวดล้อมการพัฒนาของคุณด้วยโปรเจ็กต์ C# และการอ้างอิง Aspose.PDF ที่เหมาะสม

## ขั้นตอนที่ 2: การโหลดไฟล์ PDF

ตั้งค่าเส้นทางไดเร็กทอรีของเอกสารของคุณและอัปโหลดไฟล์ PDF โดยใช้โค้ดต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// โหลดไฟล์ PDF
Document doc = new Document(dataDir + "UpdateLinks.pdf");
```

## ขั้นตอนที่ 3: การแก้ไขลิงก์ปลายทาง

รับคำอธิบายลิงก์เพื่อแก้ไขโดยใช้โค้ดดังต่อไปนี้:

```csharp
LinkAnnotation linkAnnot = (LinkAnnotation)doc.Pages[1].Annotations[1];
```

 คุณสามารถปรับเปลี่ยนได้`[1]` ดัชนีเพื่อเลือกหน้าหรือคำอธิบายเฉพาะ

ขั้นตอนต่อไปแก้ไขลิงก์โดยเปลี่ยนการดำเนินการลิงก์และกำหนดเป้าหมายเป็นที่อยู่เว็บ:

```csharp
linkAnnot.Action = new GoToURIAction("www.aspose.com");
```

## ขั้นตอนที่ 4: บันทึกเอกสารด้วยลิงก์ที่อัปเดต

 บันทึกเอกสารด้วยลิงก์ที่อัปเดตโดยใช้`Save` วิธี:

```csharp
dataDir = dataDir + "SetDestinationLink_out.pdf";
doc.Save(dataDir);
```

## ขั้นตอนที่ 5: การแสดงผล

แสดงข้อความที่ระบุว่าลิงก์ปลายทางได้รับการกำหนดค่าสำเร็จแล้ว และระบุตำแหน่งของไฟล์ที่บันทึก:

```csharp
Console.WriteLine("\nDestination link configured successfully.\nFile saved to location: " + dataDir);
```

### ตัวอย่างโค้ดที่มาสำหรับ Set Destination Link โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
try
{
	// เส้นทางไปยังไดเร็กทอรีเอกสาร
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	// โหลดไฟล์ PDF
	Document doc = new Document(dataDir + "UpdateLinks.pdf");
	// รับคำอธิบายลิงก์แรกจากหน้าแรกของเอกสาร
	LinkAnnotation linkAnnot = (LinkAnnotation)doc.Pages[1].Annotations[1];
	// การแก้ไขลิงก์: เปลี่ยนการกระทำลิงก์และกำหนดเป้าหมายเป็นที่อยู่เว็บ
	linkAnnot.Action = new GoToURIAction("www.aspose.com");           
	dataDir = dataDir + "SetDestinationLink_out.pdf";
	// บันทึกเอกสารด้วยลิงค์อัปเดต
	doc.Save(dataDir);
	Console.WriteLine("\nDestination link setup successfully.\nFile saved at " + dataDir);
}
catch (Exception ex)
{
	Console.WriteLine(ex.Message);
}
```
