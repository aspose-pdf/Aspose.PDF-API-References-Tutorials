การสร้างลิงก์แอปพลิเคชันในไฟล์ PDF ช่วยให้คุณสร้างลิงก์ไปยังแอปพลิเคชันภายนอก เช่น ไฟล์ปฏิบัติการหรือ URL ด้วย Aspose.PDF สำหรับ .NET คุณสามารถสร้างลิงก์แอปได้อย่างง่ายดายโดยทำตามซอร์สโค้ดต่อไปนี้:

## ขั้นตอนที่ 1: นำเข้าไลบรารีที่จำเป็น

ก่อนที่คุณจะเริ่มต้น คุณต้องนำเข้าไลบรารีที่จำเป็นสำหรับโปรเจ็กต์ C# ของคุณ นี่คือคำสั่งการนำเข้าที่จำเป็น:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
using Aspose.Pdf.InteractiveFeatures;
```

## ขั้นตอนที่ 2: กำหนดเส้นทางไปยังโฟลเดอร์เอกสาร

 ในขั้นตอนนี้ คุณจะต้องระบุเส้นทางไปยังโฟลเดอร์ที่มีไฟล์ PDF ที่คุณต้องการเพิ่มลิงค์แอพ แทนที่`"YOUR DOCUMENT DIRECTORY"`ในรหัสต่อไปนี้พร้อมเส้นทางจริงไปยังโฟลเดอร์เอกสารของคุณ:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## ขั้นตอนที่ 3: เปิดเอกสาร PDF

ตอนนี้เราจะเปิดเอกสาร PDF ที่เราต้องการเพิ่มลิงค์แอปพลิเคชันโดยใช้รหัสต่อไปนี้:

```csharp
Document document = new Document(dataDir + "CreateApplicationLink.pdf");
```

## ขั้นตอนที่ 4: สร้างลิงค์แอปพลิเคชัน

 ในขั้นตอนนี้ เราจะสร้างลิงค์แอปพลิเคชันโดยใช้`LinkAnnotation`คำอธิบายประกอบ เราจะระบุพิกัดและพื้นที่ของลิงก์ตลอดจนการดำเนินการเปิดแอปพลิเคชัน นี่คือรหัสที่เกี่ยวข้อง:

```csharp
Page page = document.Pages[1];
LinkAnnotation link = new LinkAnnotation(page, new Aspose.Pdf.Rectangle(100, 100, 300, 300));
link.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
link. Action = new LaunchAction(document, dataDir + "CreateApplicationLink.pdf");
page.Annotations.Add(link);
```

## ขั้นตอนที่ 5: บันทึกไฟล์ที่อัพเดต

 ตอนนี้เรามาบันทึกไฟล์ PDF ที่อัปเดตโดยใช้ไฟล์`Save` วิธีการของ`document` วัตถุ. นี่คือรหัสที่เกี่ยวข้อง:

```csharp
dataDir = dataDir + "CreateApplicationLink_out.pdf";
document. Save(dataDir);
```

### ตัวอย่างซอร์สโค้ดสำหรับสร้างลิงก์แอปพลิเคชันโดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// เปิดเอกสาร
Document document = new Document( dataDir + "CreateApplicationLink.pdf");
// สร้างลิงค์
Page page = document.Pages[1];
LinkAnnotation link = new LinkAnnotation(page, new Aspose.Pdf.Rectangle(100, 100, 300, 300));
link.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
link.Action = new LaunchAction(document, dataDir + "CreateApplicationLink.pdf");
page.Annotations.Add(link);
dataDir = dataDir + "CreateApplicationLink_out.pdf";
// บันทึกเอกสารที่อัปเดต
document.Save(dataDir);
Console.WriteLine("\nApplication link created successfully.\nFile saved at " + dataDir);
```
