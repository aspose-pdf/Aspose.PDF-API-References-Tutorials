इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके PDF फ़ाइल में छवि का आकार सेट करने का तरीका बताएँगे। इस ऑपरेशन को आसानी से करने के लिए इन चरणों का पालन करें।

## आवश्यक शर्तें

आरंभ करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- विजुअल स्टूडियो या कोई अन्य विकास वातावरण स्थापित और कॉन्फ़िगर किया गया।
- C# प्रोग्रामिंग भाषा का मूलभूत ज्ञान।
- .NET के लिए Aspose.PDF लाइब्रेरी स्थापित है। आप इसे Aspose की आधिकारिक वेबसाइट से डाउनलोड कर सकते हैं।

## चरण 1: पीडीएफ दस्तावेज़ का निर्माण

आरंभ करने के लिए, एक नया PDF दस्तावेज़ बनाने के लिए निम्नलिखित कोड का उपयोग करें:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें
Document doc = new Document();

// पीडीएफ फाइल के पृष्ठों के संग्रह में एक पृष्ठ जोड़ें
Aspose.Pdf.Page page = doc.Pages.Add();
```

## चरण 2: चित्र जोड़ा गया

इसके बाद, हम PDF दस्तावेज़ के पृष्ठ पर एक छवि जोड़ेंगे। निम्नलिखित कोड का उपयोग करें:

```csharp
// एक छवि उदाहरण बनाएँ
Aspose.Pdf.Image img = new Aspose.Pdf.Image();

// छवि की चौड़ाई और ऊंचाई को बिंदुओं में सेट करें
img. FixWidth = 100;
img. FixHeight = 100;

//छवि प्रकार को अज्ञात पर सेट करें (अज्ञात)
img.FileType = Aspose.Pdf.ImageFileType.Unknown;

// छवि स्रोत फ़ाइल का पथ
img.File = dataDir + "aspose-logo.jpg";

// छवि को पृष्ठ के पैराग्राफ़ संग्रह में जोड़ें
page.Paragraphs.Add(img);
```

छवि स्रोत फ़ाइल का सही पथ प्रदान करना सुनिश्चित करें।

## चरण 3: पृष्ठ गुण सेट करना

अंत में, हम पेज की विशेषताएँ सेट करेंगे, जिसमें इसकी चौड़ाई और ऊँचाई भी शामिल है। निम्नलिखित कोड का उपयोग करें:

```csharp
// पृष्ठ गुण सेट करें
page.PageInfo.Width = 800;
page.PageInfo.Height = 800;
```

### .NET के लिए Aspose.PDF का उपयोग करके छवि का आकार निर्धारित करने के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ ऑब्जेक्ट को इंस्टैंशिएट करें
Document doc = new Document();
// पीडीएफ फाइल के संग्रह में पेज जोड़ें
Aspose.Pdf.Page page = doc.Pages.Add();
// एक छवि उदाहरण बनाएँ
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
// छवि की चौड़ाई और ऊंचाई को बिंदुओं में सेट करें
img.FixWidth = 100;
img.FixHeight = 100;
// छवि प्रकार को SVG के रूप में सेट करें
img.FileType = Aspose.Pdf.ImageFileType.Unknown;
// स्रोत फ़ाइल के लिए पथ
img.File = dataDir + "aspose-logo.jpg";
page.Paragraphs.Add(img);
//पृष्ठ गुण सेट करें
page.PageInfo.Width = 800;
page.PageInfo.Height = 800;
dataDir = dataDir + "SetImageSize_out.pdf";
// परिणामी पीडीएफ फाइल को सेव करें
doc.Save(dataDir);
Console.WriteLine("\nImage size added successfully.\nFile saved at " + dataDir);
```

## निष्कर्ष

बधाई हो! आपने .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में छवि का आकार सफलतापूर्वक सेट कर लिया है। अब आप PDF फ़ाइलों में छवियों के आकार को समायोजित करने के लिए इस विधि को अपने स्वयं के प्रोजेक्ट पर लागू कर सकते हैं।

### पीडीएफ फाइल में छवि का आकार निर्धारित करने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: .NET के लिए Aspose.PDF का उपयोग करके PDF दस्तावेज़ में छवि का आकार निर्धारित करने का उद्देश्य क्या है?

उत्तर: PDF दस्तावेज़ में छवि का आकार सेट करने का उद्देश्य PDF में जोड़े जाने पर छवि के आयामों को नियंत्रित करना है। यह आपको अपनी PDF फ़ाइलों में छवियों की उपस्थिति और लेआउट को समायोजित करने की अनुमति देता है।

#### प्रश्न: पीडीएफ दस्तावेज़ में छवि का आकार निर्धारित करने की प्रक्रिया कैसे काम करती है?

 उत्तर: इस प्रक्रिया में एक निर्माण शामिल है`Aspose.Pdf.Image` उदाहरण के लिए, इसकी चौड़ाई और ऊंचाई को निर्दिष्ट करना`FixWidth` और`FixHeight` प्रॉपर्टीज पर क्लिक करें और फिर पीडीएफ डॉक्यूमेंट में इमेज जोड़ें। इसके अलावा, आप इमेज को समायोजित करने के लिए पेज के आयाम भी सेट कर सकते हैं।

#### प्रश्न: क्या मैं किसी छवि का आकार पृष्ठ के आयामों के एक विशिष्ट प्रतिशत पर सेट कर सकता हूँ?

उत्तर: दिया गया कोड छवि की पूर्ण चौड़ाई और ऊंचाई को बिंदुओं में सेट करता है। यदि आप पृष्ठ आयामों के प्रतिशत के आधार पर छवि का आकार सेट करना चाहते हैं, तो आपको तदनुसार आयामों की गणना करनी होगी और कोड को तदनुसार समायोजित करना होगा।

####  प्रश्न: इसका क्या महत्व है?`FileType` property when adding an image to the PDF document?

 उत्तर:`FileType`प्रॉपर्टी पीडीएफ दस्तावेज़ में जोड़ी जा रही छवि के प्रकार को निर्दिष्ट करती है। दिए गए कोड में, मान`Unknown` यह इंगित करता है कि छवि का प्रकार अज्ञात है, और Aspose.PDF फ़ाइल एक्सटेंशन के आधार पर छवि प्रकार निर्धारित करने का प्रयास करेगा।

#### प्रश्न: क्या मैं इस विधि का उपयोग करके एक ही पृष्ठ पर एकाधिक छवियाँ जोड़ सकता हूँ?

 उत्तर: हां, आप एक से अधिक पेज बनाकर एक से अधिक छवियाँ जोड़ सकते हैं`Aspose.Pdf.Image` उदाहरणों को चुनें और उन्हें पृष्ठ के पैराग्राफ़ संग्रह में जोड़ें। आवश्यकतानुसार छवियों की स्थिति और लेआउट को समायोजित करना सुनिश्चित करें।

#### प्रश्न: मैं पृष्ठ पर जोड़ी गई छवि के स्थान और संरेखण को कैसे नियंत्रित कर सकता हूं?

 उत्तर: जोड़ी गई छवि के स्थान और संरेखण को निम्नलिखित गुणों का उपयोग करके छवि के निर्देशांक और लेआउट को समायोजित करके नियंत्रित किया जा सकता है:`img.Left`, `img.Top`, और पैराग्राफ़ फ़ॉर्मेटिंग गुण.

####  प्रश्न: पेज गुणधर्मों को सेट करने का उद्देश्य क्या है?`page.PageInfo.Width` and `page.PageInfo.Height`?

उत्तर: पेज प्रॉपर्टी सेट करने से आप पेज के आयाम निर्धारित कर सकते हैं। इससे यह सुनिश्चित होता है कि पेज के आयाम जोड़े गए चित्र और पेज पर मौजूद किसी भी अन्य सामग्री को समायोजित कर सकें।

#### प्रश्न: क्या मैं एक ही पीडीएफ दस्तावेज़ में अलग-अलग छवियों के लिए अलग-अलग आकार निर्धारित कर सकता हूं?

 उत्तर: हां, आप अलग-अलग छवियों के लिए अलग-अलग आकार निर्धारित कर सकते हैं`Aspose.Pdf.Image` उदाहरणों और समायोजन`FixWidth`, `FixHeight`, और प्रत्येक छवि के लिए प्लेसमेंट गुण।

#### प्रश्न: मैं पीडीएफ फाइलों में छवि आकार निर्धारित करने के लिए इस पद्धति को अपनी परियोजनाओं में कैसे एकीकृत कर सकता हूं?

उत्तर: इस विधि को अपनी परियोजनाओं में एकीकृत करने के लिए, उल्लिखित चरणों का पालन करें और आवश्यकतानुसार कोड को संशोधित करें। आप अपने आवेदन की आवश्यकताओं के आधार पर अपने पीडीएफ दस्तावेज़ों में विशिष्ट आकारों की छवियां जोड़ने के लिए इस विधि का उपयोग कर सकते हैं।
