इस ट्यूटोरियल में, हम आपको .NET के लिए Aspose.PDF का उपयोग करके टेबल सेल में इमेज जोड़ने की प्रक्रिया के बारे में बताएँगे। प्रदान किया गया C# स्रोत कोड दर्शाता है कि इस कार्यक्षमता को कैसे प्राप्त किया जाए। नीचे दिए गए चरणों का पालन करके, आप अपनी टेबल सेल में इमेज को प्रभावी ढंग से शामिल कर पाएँगे।

इससे पहले कि हम कोड में उतरें, सुनिश्चित करें कि आपके पास .NET लाइब्रेरी के लिए Aspose.PDF स्थापित है और आपके प्रोजेक्ट में संदर्भित है।

## चरण 1: दस्तावेज़ सेट करना

 आरंभ करने के लिए, हमें एक नया उदाहरण बनाना होगा`Document` Aspose.Pdf नामस्थान से क्लास। यह क्लास एक PDF दस्तावेज़ का प्रतिनिधित्व करता है।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें
Document pdfDocument = new Document();
```

## चरण 2: पेज बनाना

इसके बाद, हमें पीडीएफ दस्तावेज़ में एक पेज जोड़ना होगा। एक पेज टेबल और अन्य तत्वों के लिए एक कंटेनर के रूप में कार्य करता है।

```csharp
// पीडीएफ दस्तावेज़ में एक पृष्ठ बनाएँ
Page sec1 = pdfDocument.Pages.Add();
```

## चरण 3: तालिका जोड़ना

 इस चरण में, हम इन्स्टेन्शियेट करके एक तालिका बनाएंगे।`Table` Aspose.Pdf नामस्थान से क्लास.

```csharp
// तालिका ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
```

## चरण 4: डिफ़ॉल्ट सेल बॉर्डर सेट करना

 एकरूपता सुनिश्चित करने के लिए, हम इसका उपयोग करके एक डिफ़ॉल्ट सेल बॉर्डर सेट कर सकते हैं`DefaultCellBorder` तालिका की संपत्ति`BorderInfo` वस्तु।

```csharp
// BorderInfo ऑब्जेक्ट का उपयोग करके डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
```

## चरण 5: कॉलम की चौड़ाई निर्धारित करना

 तालिका में प्रत्येक कॉलम की चौड़ाई निर्धारित करने के लिए, हम सेट कर सकते हैं`ColumnWidths` प्रॉपर्टी। चौड़ाई को स्पेस से अलग किए गए मानों के साथ स्ट्रिंग के रूप में निर्दिष्ट करें।

```csharp
// तालिका की स्तंभ चौड़ाई के साथ सेट करें
tab1.ColumnWidths = "100 100 120";
```

## चरण 6: तालिका सेल में छवि जोड़ना

अब रोमांचक हिस्सा आता है, टेबल सेल में एक छवि जोड़ना। ऐसा करने के लिए, हम इन उप-चरणों का पालन करेंगे:

## चरण 6.1: एक छवि ऑब्जेक्ट बनाना

 इसका एक उदाहरण बनाएं`Image` Aspose.Pdf नामस्थान से क्लास सेट करें।`File` प्रॉपर्टी को उस छवि फ़ाइल के पथ में जोड़ें जिसे आप जोड़ना चाहते हैं।

```csharp
// एक छवि ऑब्जेक्ट बनाएँ
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose.jpg";
```

## चरण 6.2: पंक्ति और कक्ष बनाना

तालिका में छवि जोड़ने के लिए, हमें पहले एक पंक्ति और आवश्यक कक्ष बनाने होंगे।

```csharp
// तालिका में एक पंक्ति बनाएँ
Aspose.Pdf.Row row1 = tab1.Rows.Add();

// पंक्ति में टेक्स्ट सेल जोड़ें
row1.Cells.Add("Sample text in cell");

// वह सेल जोड़ें जिसमें छवि है
Aspose.Pdf.Cell cell2 = row1.Cells.Add();
```

## चरण 6.3: तालिका सेल में छवि जोड़ना

अंत में, हम छवि को सेल के भीतर पैराग्राफ के रूप में जोड़कर तालिका सेल में जोड़ सकते हैं।

```csharp
//तालिका सेल में छवि जोड़ें
cell2.Paragraphs.Add(img);
```

## चरण 6.4: अतिरिक्त कक्ष जोड़ना

छवि सेल जोड़ने के बाद, यदि आवश्यक हो तो हम पंक्ति में और अधिक सेल जोड़ सकते हैं।

```csharp
// पंक्ति में एक और सेल जोड़ें
row1.Cells.Add("Previous cell with image");

// तीसरे सेल के ऊर्ध्वाधर संरेखण को समायोजित करें
row1.Cells[2].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
```

## चरण 7: दस्तावेज़ को सहेजना

 अंत में, हम संशोधित दस्तावेज़ को निर्दिष्ट स्थान पर सहेज सकते हैं`Save` तरीका।

```csharp
//दस्तावेज़ सहेजें
pdfDocument.Save(dataDir + "AddImageInTableCell_out.pdf");
```

बधाई हो! आपने सफलतापूर्वक सीख लिया है कि .NET के लिए Aspose.PDF का उपयोग करके टेबल सेल में छवि कैसे जोड़ें। आगे के अनुकूलन विकल्पों का पता लगाने और अपनी परियोजनाओं में इस कार्यक्षमता को एकीकृत करने के लिए स्वतंत्र महसूस करें।

### .NET के लिए Aspose.PDF का उपयोग करके तालिका कक्ष में छवि जोड़ने के लिए उदाहरण स्रोत कोड

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// दस्तावेज़ ऑब्जेक्ट को इंस्टैंसिएट करें
Document pdfDocument = new Document();
// पीडीएफ दस्तावेज़ में एक पृष्ठ बनाएँ
Page sec1 = pdfDocument.Pages.Add();
// तालिका ऑब्जेक्ट को इंस्टैंसिएट करें
Aspose.Pdf.Table tab1 = new Aspose.Pdf.Table();
// इच्छित पृष्ठ के पैराग्राफ संग्रह में तालिका जोड़ें
sec1.Paragraphs.Add(tab1);
// BorderInfo ऑब्जेक्ट का उपयोग करके डिफ़ॉल्ट सेल बॉर्डर सेट करें
tab1.DefaultCellBorder = new Aspose.Pdf.BorderInfo(Aspose.Pdf.BorderSide.All, 0.1F);
// तालिका की स्तंभ चौड़ाई के साथ सेट करें
tab1.ColumnWidths = "100 100 120";
Aspose.Pdf.Image img = new Aspose.Pdf.Image();
img.File = dataDir + "aspose.jpg";
//तालिका में पंक्तियाँ बनाएँ और फिर पंक्तियों में कक्ष बनाएँ
Aspose.Pdf.Row row1 = tab1.Rows.Add();
row1.Cells.Add("Sample text in cell");
// वह सेल जोड़ें जिसमें छवि है
Aspose.Pdf.Cell cell2 = row1.Cells.Add();
//तालिका सेल में छवि जोड़ें
cell2.Paragraphs.Add(img);
row1.Cells.Add("Previous cell with image");
row1.Cells[2].VerticalAlignment = Aspose.Pdf.VerticalAlignment.Center;
//दस्तावेज़ सहेजें
pdfDocument.Save(dataDir + "AddImageInTableCell_out.pdf");
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने .NET के लिए Aspose.PDF का उपयोग करके टेबल सेल में छवि जोड़ने के तरीके पर चरण-दर-चरण मार्गदर्शिका को कवर किया है। हमने दस्तावेज़ सेट अप करके, एक पेज बनाकर और एक टेबल जोड़कर शुरुआत की। फिर, हमने डिफ़ॉल्ट सेल बॉर्डर और कॉलम की चौड़ाई सेट की। हमने दिखाया कि टेबल सेल में छवि कैसे जोड़ें और सेल के वर्टिकल अलाइनमेंट को कैसे एडजस्ट करें। अंत में, हमने संशोधित दस्तावेज़ को सहेजा। इन चरणों का पालन करके, आप अपने PDF दस्तावेज़ों को टेबल सेल में छवियों के साथ कुशलतापूर्वक बढ़ा सकते हैं।

### अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: क्या मैं .NET के लिए Aspose.PDF का उपयोग करके एक ही तालिका के विभिन्न कक्षों में एकाधिक छवियाँ जोड़ सकता हूँ?

उत्तर: हां, आप .NET के लिए Aspose.PDF का उपयोग करके एक ही टेबल के भीतर अलग-अलग सेल में कई इमेज जोड़ सकते हैं। बस ट्यूटोरियल में बताई गई उसी प्रक्रिया का पालन करें, जिस पर आप टेबल में जोड़ना चाहते हैं।

#### प्रश्न: क्या मैं तालिका सेल के भीतर छवि का आकार और स्थिति अनुकूलित कर सकता हूँ?

 उत्तर: हां, आप तालिका सेल के भीतर छवि के आकार और स्थिति को गुणों को समायोजित करके अनुकूलित कर सकते हैं।`Image` ऑब्जेक्ट। आप छवि की चौड़ाई और ऊंचाई, साथ ही सेल के भीतर संरेखण सेट कर सकते हैं।

#### प्रश्न: क्या मैं पंक्तियों और स्तंभों की गतिशील संख्या वाली तालिका में छवियाँ जोड़ सकता हूँ?

उत्तर: हाँ, आप पंक्तियों और स्तंभों की गतिशील संख्या वाली तालिका में छवियाँ जोड़ सकते हैं। .NET के लिए Aspose.PDF अलग-अलग आयामों वाली तालिकाएँ बनाने में लचीलापन प्रदान करता है। आप आवश्यकतानुसार पंक्तियाँ और कक्ष जोड़ सकते हैं, और फिर उसके अनुसार विशिष्ट कक्षों में छवियाँ जोड़ सकते हैं।

#### प्रश्न: तालिका कक्षों में चित्र जोड़ने के लिए Aspose.PDF for .NET द्वारा कौन से चित्र प्रारूप समर्थित हैं?

उत्तर: .NET के लिए Aspose.PDF कई तरह के इमेज फ़ॉर्मेट को सपोर्ट करता है, जिसमें JPEG, PNG, GIF, BMP और TIFF शामिल हैं। आप इनमें से किसी भी फ़ॉर्मेट की इमेज को टेबल सेल में जोड़ने के लिए इस्तेमाल कर सकते हैं।

#### प्रश्न: क्या मैं किसी मौजूदा पीडीएफ दस्तावेज़ में तालिकाओं में छवियाँ जोड़ सकता हूँ?

उत्तर: हाँ, आप .NET के लिए Aspose.PDF का उपयोग करके किसी मौजूदा PDF दस्तावेज़ में तालिकाओं में छवियाँ जोड़ सकते हैं। बस मौजूदा दस्तावेज़ को लोड करें और तालिका में छवियाँ जोड़ने के लिए ट्यूटोरियल में दिखाए गए चरणों का पालन करें।
