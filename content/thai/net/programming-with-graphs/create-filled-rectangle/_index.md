ในบทช่วยสอนนี้ เราจะพาคุณดูโค้ดต้นฉบับ C# ทีละขั้นตอนเพื่อสร้างสี่เหลี่ยมผืนผ้าที่เติมเต็มโดยใช้ Aspose.PDF สำหรับ .NET

ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.PDF และตั้งค่าสภาพแวดล้อมการพัฒนาของคุณก่อนเริ่มต้น นอกจากนี้ ควรมีความรู้พื้นฐานเกี่ยวกับการเขียนโปรแกรม C# ด้วย

## ขั้นตอนที่ 1: การตั้งค่าไดเรกทอรีเอกสาร

ในซอร์สโค้ดที่ให้มา คุณต้องระบุไดเรกทอรีที่คุณต้องการบันทึกไฟล์ PDF ที่ได้ เปลี่ยนตัวแปร "dataDir" เป็นไดเรกทอรีที่ต้องการ

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

## ขั้นตอนที่ 2: การสร้างอินสแตนซ์เอกสารและเพิ่มหน้า

เราสร้างอินสแตนซ์ของคลาส Document และเพิ่มหน้าลงในเอกสารนี้

```csharp
Document doc = new Document();
Page page = doc.Pages.Add();
```

## ขั้นตอนที่ 3: การสร้างวัตถุกราฟและเพิ่มลงในหน้า

เราสร้างวัตถุ Graph ที่มีมิติที่ระบุและเพิ่มลงในคอลเล็กชันย่อหน้าของหน้า

```csharp
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100, 400);
page.Paragraphs.Add(graph);
```

## ขั้นตอนที่ 4: สร้างวัตถุสี่เหลี่ยมผืนผ้าและเพิ่มลงในแผนภูมิ

เราสร้างวัตถุสี่เหลี่ยมผืนผ้าที่มีขนาดที่กำหนดและเพิ่มลงในคอลเล็กชันรูปร่างของแผนภูมิ

```csharp
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
graph.Shapes.Add(rect);
```

## ขั้นตอนที่ 5: การตั้งค่าสีเติม

เราสามารถระบุสีเติมให้กับสี่เหลี่ยมได้โดยใช้คุณสมบัติ FillColor ของอ็อบเจ็กต์ GraphInfo

```csharp
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
```

## ขั้นตอนที่ 6: บันทึกไฟล์ PDF ที่ได้

ในที่สุด เราจะบันทึกไฟล์ PDF ผลลัพธ์ด้วยชื่อ "CreateFilledRectangle_out.pdf" ในไดเร็กทอรีที่ระบุ

```csharp
doc.Save(dataDir + "CreateFilledRectangle_out.pdf");
```

### ตัวอย่างโค้ดต้นฉบับสำหรับการสร้างรูปสี่เหลี่ยมผืนผ้าแบบเติมสีโดยใช้ Aspose.PDF สำหรับ .NET 

```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// สร้างอินสแตนซ์เอกสาร
Document doc = new Document();
// เพิ่มหน้าเข้าไปยังคอลเลคชันไฟล์ PDF
Page page = doc.Pages.Add();
// สร้างอินสแตนซ์กราฟ
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100, 400);
// เพิ่มวัตถุกราฟลงในคอลเล็กชั่นย่อหน้าของอินสแตนซ์หน้า
page.Paragraphs.Add(graph);
// สร้างอินสแตนซ์สี่เหลี่ยมผืนผ้า
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
// ระบุสีเติมสำหรับวัตถุกราฟ
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
// เพิ่มวัตถุสี่เหลี่ยมผืนผ้าลงในคอลเล็กชั่นรูปร่างของวัตถุกราฟ
graph.Shapes.Add(rect);
dataDir = dataDir + "CreateFilledRectangle_out.pdf";
// บันทึกไฟล์ PDF
doc.Save(dataDir);
Console.WriteLine("\nFilled rectangle object created successfully.\nFile saved at " + dataDir);            

```

## บทสรุป

ในบทช่วยสอนนี้ เราได้อธิบายวิธีการสร้างรูปสี่เหลี่ยมผืนผ้าที่เติมสีโดยใช้ Aspose.PDF สำหรับ .NET ตอนนี้คุณสามารถใช้ความรู้เหล่านี้เพื่อสร้างรูปทรงเรขาคณิตด้วยสีเติมที่กำหนดเองในไฟล์ PDF ของคุณ

## คำถามที่พบบ่อย

#### ถาม: จุดประสงค์ของบทช่วยสอนนี้คืออะไร?

ตอบ: วัตถุประสงค์ของบทช่วยสอนนี้คือเพื่อแนะนำคุณตลอดกระบวนการสร้างสี่เหลี่ยมผืนผ้าที่เติมสีโดยใช้ Aspose.PDF สำหรับ .NET ซึ่งทำให้คุณสามารถเพิ่มรูปทรงเรขาคณิตแบบกำหนดเองพร้อมสีเติมลงในไฟล์ PDF ของคุณได้

#### ถาม: มีข้อกำหนดเบื้องต้นอะไรบ้างก่อนเริ่มต้น?

A: ก่อนเริ่มต้น ให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.PDF และตั้งค่าสภาพแวดล้อมการพัฒนาของคุณแล้ว นอกจากนี้ ขอแนะนำให้มีความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรม C#

#### ถาม: ฉันจะระบุไดเร็กทอรีสำหรับบันทึกไฟล์ PDF ได้อย่างไร

A: ในโค้ดต้นฉบับที่ให้มา คุณสามารถปรับเปลี่ยนตัวแปร "dataDir" เพื่อระบุไดเร็กทอรีที่คุณต้องการบันทึกไฟล์ PDF ผลลัพธ์

#### ถาม: วัตถุประสงค์ของวัตถุ Graph คืออะไร

A: วัตถุ Graph ทำหน้าที่เป็นคอนเทนเนอร์สำหรับองค์ประกอบการวาดภาพ โดยจะสร้างขึ้นด้วยขนาดที่กำหนดและเพิ่มลงในคอลเล็กชันย่อหน้าของหน้า

#### ถาม: ฉันจะเพิ่มสี่เหลี่ยมผืนผ้าที่เติมสีลงในเอกสาร PDF ได้อย่างไร

ตอบ: หากต้องการเพิ่มสี่เหลี่ยมผืนผ้าที่เติม ให้สร้างอินสแตนซ์ของคลาส Rectangle ที่มีขนาดและสีเติมที่ระบุ แล้วเพิ่มลงในคอลเล็กชันรูปร่างของกราฟ

#### ถาม: ฉันสามารถกำหนดขนาดและสีเติมของสี่เหลี่ยมผืนผ้าได้ไหม

 A: ใช่ คุณสามารถปรับแต่งขนาดและสีเติมของรูปสี่เหลี่ยมผืนผ้าได้โดยการแก้ไขพารามิเตอร์ที่ส่งไปยัง`Aspose.Pdf.Drawing.Rectangle` ตัวสร้างและการตั้งค่าคุณสมบัติ FillColor

#### ถาม: ฉันจะบันทึกไฟล์ PDF ผลลัพธ์หลังจากสร้างสี่เหลี่ยมผืนผ้าที่เติมเต็มแล้วได้อย่างไร

 A: หลังจากสร้างสี่เหลี่ยมผืนผ้าที่เติมแล้ว คุณสามารถบันทึกไฟล์ PDF ที่ได้โดยใช้`doc.Save(dataDir + "CreateFilledRectangle_out.pdf");` บรรทัดในโค้ดต้นฉบับที่ให้ไว้
