Если вы работаете с PDF-документами в .NET, вы можете столкнуться с проблемами, когда шрифт, используемый в PDF, недоступен в системе, где он просматривается или печатается. Это может привести к тому, что текст будет отображаться неправильно или не отображаться вообще. Aspose.PDF для .NET предоставляет решение этой проблемы, позволяя вам установить шрифт по умолчанию для документа. В этом примере показано, как установить шрифт по умолчанию с помощью Aspose.PDF для .NET.

## Шаг 1: Укажите путь к каталогу документов.

нам нужно задать путь к каталогу, где находится наш PDF-документ. Мы сохраним этот путь в переменной с именем "dataDir".

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Шаг 2: Загрузите PDF-документ.

 Начнем с загрузки существующего PDF-документа, в котором отсутствуют шрифты. В этом примере мы предположим, что PDF-документ находится в каталоге, указанном`dataDir` переменная.

```csharp
string documentName = dataDir + "input.pdf";
using (System.IO.FileStream fs = new System.IO.FileStream(documentName, System.IO.FileMode.Open))
using (Document document = new Document(fs))
{
    // код идет здесь
}
```

## Шаг 3: Установите шрифт по умолчанию

 Далее мы установим шрифт по умолчанию для документа PDF с помощью`PdfSaveOptions`класс. В этом примере мы установим шрифт по умолчанию на «Arial».

```csharp
PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
pdfSaveOptions.DefaultFontName = "Arial";
```

## Шаг 4: Сохраните обновленный документ.

Наконец, мы сохраним обновленный документ в новый файл. В этом примере мы сохраним обновленный документ в файл с именем "output_out.pdf" в том же каталоге, что и входной файл.

```csharp
document.Save(dataDir + "output_out.pdf", pdfSaveOptions);
```

### Пример исходного кода для установки шрифта по умолчанию с помощью Aspose.PDF для .NET

```csharp
// Путь к каталогу документов.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// Загрузите существующий PDF-документ с отсутствующим шрифтом
string documentName = dataDir + "input.pdf";
string newName = "Arial";
using (System.IO.FileStream fs = new System.IO.FileStream(documentName, System.IO.FileMode.Open))
using (Document document = new Document(fs))
{
	PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();
	// Укажите имя шрифта по умолчанию
	pdfSaveOptions.DefaultFontName = newName;
	document.Save(dataDir + "output_out.pdf", pdfSaveOptions);
}
```

Следуя пошаговому руководству и используя предоставленный исходный код C#, разработчики могут легко установить шрифт по умолчанию и создавать PDF-файлы, которые предлагают единообразный и надежный просмотр в различных средах. Эта функция особенно полезна в сценариях, где PDF-файлы будут просматриваться или печататься на различных системах, в которых могут быть установлены разные наборы шрифтов.

### Часто задаваемые вопросы по установке шрифта по умолчанию в PDF-файле

#### В: Почему так важна настройка шрифта по умолчанию в PDF-документах?

A: Установка шрифта по умолчанию в документах PDF важна, поскольку она гарантирует, что текст будет отображаться правильно, даже если исходные шрифты недоступны в системе, где просматривается или печатается PDF. Это помогает предотвратить такие проблемы, как отсутствие или искажение текста, обеспечивая последовательный и надежный просмотр.

#### В: Могу ли я выбрать любой шрифт в качестве шрифта по умолчанию, используя Aspose.PDF для .NET?

 A: Да, вы можете выбрать любой шрифт, который доступен в системе, в качестве шрифта по умолчанию с помощью Aspose.PDF for .NET. Просто укажите имя шрифта в`DefaultFontName` собственность`PdfSaveOptions` сорт.

#### В: Что произойдет, если указанный шрифт по умолчанию недоступен в системе?

A: Если указанный шрифт по умолчанию недоступен в системе, средство просмотра PDF будет использовать резервный шрифт для отображения текста. Рекомендуется выбрать общедоступный шрифт, например Arial или Times New Roman, чтобы обеспечить совместимость с различными системами.
