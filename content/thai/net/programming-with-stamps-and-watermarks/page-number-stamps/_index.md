## การแนะนำ

คุณเคยพบว่าตัวเองต้องดิ้นรนกับเอกสาร PDF และอยากให้มีหมายเลขหน้าเพื่อให้ใช้งานได้ง่ายขึ้นหรือไม่ ไม่ว่าคุณจะเป็นนักเรียนที่กำลังแชร์บันทึก มืออาชีพในการนำเสนอรายงาน หรือใครก็ตามที่จัดการเอกสารหลายหน้า การเพิ่มหมายเลขหน้าสามารถช่วยให้ไฟล์ PDF ของคุณชัดเจนขึ้นได้อย่างแท้จริง โชคดีที่ไลบรารี Aspose.PDF สำหรับ .NET ที่ทรงพลังช่วยให้คุณเพิ่มตราประทับหมายเลขหน้าในเอกสาร PDF ได้อย่างง่ายดาย ในคู่มือนี้ เราจะแนะนำคุณตลอดกระบวนการทั้งหมดทีละขั้นตอน เพื่อให้แน่ใจว่าคุณได้รับความรู้ทั้งหมดที่จำเป็น มาเริ่มกันเลย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มเพิ่มตราประทับหมายเลขหน้าลงในเอกสาร PDF ของคุณ ตรวจสอบให้แน่ใจว่าคุณมีข้อกำหนดเบื้องต้นต่อไปนี้:

1. Visual Studio: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Visual Studio ไว้ในระบบของคุณแล้ว คุณจะเขียนและรันโค้ดได้ที่นี่
2. .NET Framework: ความคุ้นเคยกับการเขียนโปรแกรม C# และ .NET framework เป็นสิ่งจำเป็น เนื่องจาก Aspose.PDF ได้รับการออกแบบมาสำหรับแอปพลิเคชัน .NET
3.  ไลบรารี Aspose.PDF: คุณสามารถดาวน์โหลดไลบรารี Aspose.PDF ได้จาก[เผยแพร่ PDF ของ Aspose](https://releases.aspose.com/pdf/net/). 
4. ความเข้าใจพื้นฐานเกี่ยวกับ PDF: แม้ว่าคุณไม่จำเป็นต้องเป็นผู้เชี่ยวชาญ ความเข้าใจพื้นฐานเกี่ยวกับการทำงานของไฟล์ PDF จะช่วยให้คุณเข้าใจบทช่วยสอนได้ดีขึ้น

เมื่อคุณตั้งค่าข้อกำหนดเบื้องต้นเหล่านี้เสร็จเรียบร้อยแล้ว คุณก็พร้อมเริ่มประทับหมายเลขหน้าได้แล้ว!

## แพ็คเกจนำเข้า

ก่อนจะเริ่มเขียนโค้ด คุณต้องแน่ใจว่าได้นำแพ็คเกจ Aspose.PDF ที่จำเป็นเข้าไปในโปรเจ็กต์ของคุณแล้ว ซึ่งถือเป็นสิ่งสำคัญสำหรับการใช้ประโยชน์จากฟังก์ชันไลบรารีอย่างราบรื่น วิธีดำเนินการมีดังต่อไปนี้:

### สร้างโครงการใหม่

1. เปิด Visual Studio
2.  คลิ๊กที่นี่`File` -`New` -`Project`.
3.  เลือกเทมเพลตที่เหมาะกับ C# (เช่น แอปพลิเคชันคอนโซล) ตั้งชื่อ และคลิก`Create`.

### เพิ่มการอ้างอิง Aspose.PDF

1. คลิกขวาที่ชื่อโครงการใน Solution Explorer
2.  คลิ๊กที่นี่`Manage NuGet Packages`.
3.  ค้นหา`Aspose.PDF` และติดตั้งเวอร์ชั่นล่าสุด

```csharp
using System.IO;
using System;
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

เมื่อไลบรารีพร้อมแล้ว มาเริ่มการเขียนโค้ดกันเลย!

ตอนนี้เราได้ตั้งค่าสภาพแวดล้อมเรียบร้อยแล้ว ถึงเวลาเพิ่มตราประทับหมายเลขหน้าลงในไฟล์ PDF เราจะแบ่งกระบวนการนี้ออกเป็นขั้นตอนที่ชัดเจนเพื่อให้เข้าใจได้ดีขึ้น

## ขั้นตอนที่ 1: ระบุไดเรกทอรีเอกสาร

ในการเริ่มต้น คุณต้องระบุไดเรกทอรีที่ไฟล์ PDF ของคุณตั้งอยู่ นี่คือจุดเริ่มต้นของโครงการของคุณ

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // อัพเดตเส้นทางนี้
```

 คำอธิบาย : แทนที่`"YOUR DOCUMENT DIRECTORY"` โดยมีเส้นทางไปยังไดเร็กทอรีที่มีไฟล์ PDF ของคุณ ซึ่งถือเป็นเรื่องสำคัญ เนื่องจากจะแจ้งให้โค้ดของคุณทราบว่าจะค้นหาไฟล์ที่คุณต้องการจัดการได้จากที่ใด

## ขั้นตอนที่ 2: เปิดเอกสาร

ต่อไปเราจะเปิดเอกสาร PDF ที่มีอยู่ซึ่งเราต้องการเพิ่มแสตมป์หมายเลขหน้า

```csharp
Document pdfDocument = new Document(dataDir + "PageNumberStamp.pdf");
```

 คำอธิบาย: ที่นี่เราใช้`Document` คลาสที่ Aspose.PDF จัดเตรียมไว้เพื่อเปิดไฟล์ PDF เฉพาะของเรา ตรวจสอบให้แน่ใจว่าชื่อไฟล์ตรงกับไฟล์จริงที่คุณมีในไดเร็กทอรีของคุณ

## ขั้นตอนที่ 3: สร้างตราประทับหมายเลขหน้า

ตอนนี้มาถึงส่วนสนุก ๆ แล้ว มาสร้างแสตมป์หมายเลขหน้าเพื่อเพิ่มลงใน PDF ของเรากัน

```csharp
PageNumberStamp pageNumberStamp = new PageNumberStamp();
```

 คำอธิบาย :`PageNumberStamp`คลาสนี้จะอนุญาตให้เราสร้างสแตมป์ที่จะแสดงหมายเลขหน้าปัจจุบันเทียบกับจำนวนหน้าทั้งหมดในเอกสาร

## ขั้นตอนที่ 4: กำหนดค่าแสตมป์

ตอนนี้คุณต้องกำหนดค่าการตั้งค่าแสตมป์ของคุณ นี่คือขั้นตอนการออกแบบว่าแสตมป์จะมีลักษณะและการทำงานอย่างไร

```csharp
pageNumberStamp.Background = false;
pageNumberStamp.Format = "Page # of " + pdfDocument.Pages.Count;
pageNumberStamp.BottomMargin = 10;
pageNumberStamp.HorizontalAlignment = HorizontalAlignment.Center;
pageNumberStamp.StartingNumber = 1;
```

คำอธิบาย:
- `Background = false`:นั่นหมายความว่าแสตมป์จะปรากฏในเบื้องหน้า
- `Format`:ที่นี่ คุณกำลังตั้งค่ารูปแบบให้แสดง "หน้า X จาก Y" ซึ่งคุณดึงจำนวนหน้าทั้งหมดในเอกสารแบบไดนามิก
- `BottomMargin`: ปรับระยะห่างจากด้านล่างของหน้า
- `HorizontalAlignment`:จัดแสตมป์ให้อยู่กึ่งกลางแนวนอน
- `StartingNumber`: กำหนดหมายเลขหน้าเริ่มต้น โดยทั่วไปจะเริ่มจาก 1

## ขั้นตอนที่ 5: ตั้งค่าคุณสมบัติข้อความ

จากนั้น คุณสามารถปรับแต่งลักษณะของข้อความในแสตมป์ได้

```csharp
pageNumberStamp.TextState.Font = FontRepository.FindFont("Arial");
pageNumberStamp.TextState.FontSize = 14.0F;
pageNumberStamp.TextState.FontStyle = FontStyles.Bold;
pageNumberStamp.TextState.FontStyle = FontStyles.Italic;
pageNumberStamp.TextState.ForegroundColor = Color.Aqua;
```

คำอธิบาย: คุณลักษณะเหล่านี้จะกำหนดค่าชนิดของแบบอักษร ขนาดแบบอักษร สไตล์ (ทั้งแบบหนาและแบบเอียง) และสีของข้อความภายในแสตมป์เพื่อให้ดูน่าสนใจ

## ขั้นตอนที่ 6: เพิ่มตราประทับลงในหน้าเฉพาะ

เมื่อคุณกำหนดค่าตราประทับของคุณแล้ว ก็ถึงเวลาที่จะเพิ่มตราประทับนั้นลงในหน้าเฉพาะในเอกสารของคุณ

```csharp
pdfDocument.Pages[1].AddStamp(pageNumberStamp);
```

 คำอธิบาย: บรรทัดนี้จะเพิ่มตราประทับลงในหน้าแรกของไฟล์ PDF คุณสามารถปรับเปลี่ยนได้`Pages[1]` ดัชนีสำหรับหน้าอื่น ๆ ตามความจำเป็น

## ขั้นตอนที่ 7: บันทึกเอกสารผลลัพธ์

สุดท้ายให้บันทึกเอกสาร PDF ที่แก้ไขแล้วเพื่อให้การเปลี่ยนแปลงของคุณเป็นแบบถาวร

```csharp
dataDir = dataDir + "PageNumberStamp_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nPage number stamp added successfully.\nFile saved at " + dataDir);
```

คำอธิบาย: คุณกำลังกำหนดเส้นทางไฟล์เอาต์พุตและบันทึกเอกสาร คอนโซลจะแจ้งให้คุณทราบว่าได้เพิ่มตราประทับเรียบร้อยแล้วและบันทึกไฟล์ไว้ที่ใด
