## مقدمة

في هذا البرنامج التعليمي الشامل، سنتعمق في عملية إضافة نص مُدار إلى مستند PDF باستخدام Java. سواء كنت بحاجة إلى تسمية المخططات أو إنشاء علامات مائية أو إضافة تأثيرات خاصة إلى ملفات PDF الخاصة بك، فسيرشدك هذا الدليل خلال الخطوات. سنستخدم Aspose.PDF for Java، وهي مكتبة قوية لمعالجة ملفات PDF، لتوضيح العملية.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من توفر المتطلبات الأساسية التالية:

1. بيئة تطوير Java: تأكد من تثبيت Java على نظامك.

2.  Aspose.PDF for Java: قم بتنزيل مكتبة Aspose.PDF وتضمينها في مشروع Java الخاص بك. يمكنك العثور على رابط التنزيل[هنا](https://releases.aspose.com/pdf/java/).

## الخطوة 1: إنشاء مستند PDF جديد

لنبدأ بإنشاء مستند PDF جديد باستخدام Aspose.PDF. سيعمل هذا المستند كلوحة للنص الذي قمنا بتدويره.

```java
// تهيئة مستند PDF
com.aspose.pdf.Document pdfDocument = new com.aspose.pdf.Document();
```

## الخطوة 2: إضافة صفحة

بعد ذلك، أضف صفحة إلى مستند PDF حيث تريد إدراج النص المُدار:

```java
//إضافة صفحة جديدة إلى المستند
com.aspose.pdf.Page page = pdfDocument.getPages().add();
```

## الخطوة 3: تحديد النص المُدار

الآن، دعنا نحدد النص الذي تريد إدراجه وتدويره. يمكنك تخصيص النص والخط وزاوية التدوير وفقًا لمتطلباتك:

```java
// تحديد محتوى النص
String text = "Rotated Text Example";

// إنشاء كائن TextFragment
com.aspose.pdf.TextFragment textFragment = new com.aspose.pdf.TextFragment(text);

// ضبط حجم الخط ونمطه
textFragment.getTextState().setFontSize(12);
textFragment.getTextState().setFont(com.aspose.pdf.FontRepository.findFont("Arial"));

// تحديد زاوية الدوران (بالدرجات)
textFragment.setTextRotation(45);
```

في هذا المثال، قمنا بتعيين النص على "مثال النص المُدار"، واخترنا الخط Arial، وضبطنا حجم الخط على 12، وقمنا بتدوير النص بمقدار 45 درجة. اضبط هذه المعلمات لتتوافق مع متطلباتك المحددة.

## الخطوة 4: تحديد موضع النص المُدار

حدد الموضع على الصفحة حيث تريد وضع النص المُدار:

```java
// تعيين موضع النص
textFragment.setPosition(new com.aspose.pdf.Position(100, 200));
```

هنا، قمنا بوضع النص عند الإحداثيات (100، 200) على الصفحة. قم بتعديل هذه الإحداثيات لوضع النص بدقة حيث تريد.

## الخطوة 5: إضافة نص مُدار إلى الصفحة

الآن، أضف النص المُدار إلى الصفحة:

```java
// أضف النص المُدار إلى الصفحة
page.getParagraphs().add(textFragment);
```

## الخطوة 6: احفظ ملف PDF

وأخيرًا، احفظ مستند PDF بالنص المُدار:

```java
// حفظ مستند PDF
pdfDocument.save("output.pdf");
```

## خاتمة

في هذا البرنامج التعليمي، استكشفنا عملية إضافة نص مُدار إلى مستند PDF باستخدام Java وAspose.PDF for Java. لقد تعلمت كيفية إنشاء ملف PDF جديد، وتحديد النص المُدار باستخدام أنماط مخصصة، ووضعه على الصفحة، وحفظ ملف PDF المُعدل.

يمكن أن يكون النص المدور إضافة قيمة إلى ملفات PDF الخاصة بك لأغراض مختلفة، مثل تسمية المخططات، أو وضع العلامات المائية، أو إضافة عناصر إبداعية إلى مستنداتك.

قم بتعزيز مستندات PDF الخاصة بك عن طريق دمج النص المدور بكل سهولة، وذلك بفضل إمكانيات Aspose.PDF لـ Java.

## الأسئلة الشائعة

### 1. هل يمكنني تدوير النص بزوايا مختلفة في نفس ملف PDF؟
   نعم، يمكنك إضافة عدة نسخ من النص المدور بزوايا مختلفة إلى نفس مستند PDF. ما عليك سوى تكرار العملية الموضحة في هذا البرنامج التعليمي لكل جزء من النص المدور.

### 2. كيف يمكنني تغيير لون النص المُدار؟
    لتغيير لون النص، استخدم`textFragment.getTextState().setForegroundColor` الطريقة وتحديد اللون بتنسيق RGB. على سبيل المثال، لتعيين لون النص إلى اللون الأحمر، استخدم`textFragment.getTextState().setForegroundColor(com.aspose.pdf.Color.getRed());`.

### 3. هل Aspose.PDF for Java مكتبة مجانية؟
   Aspose.PDF for Java هي مكتبة تجارية قوية، ولكنها تقدم نسخة تجريبية مجانية للاختبار والتقييم. بناءً على متطلبات مشروعك، يمكنك اختيار خيار الترخيص المناسب.

### 4. هل يمكنني تدوير النص بمقدار 90 درجة لإنشاء نص عمودي؟
   نعم، يمكنك تدوير النص بمقدار 90 درجة لإنشاء نص رأسي. ما عليك سوى ضبط زاوية التدوير على 90 درجة، وسيظهر النص رأسيًا على الصفحة.

### 5. هل هناك مكتبات أخرى للعمل مع ملفات PDF في Java؟
   نعم، تتوفر العديد من المكتبات، مثل iText وPDFBox، لمعالجة ملفات PDF في Java. تتمتع كل مكتبة بميزاتها وقدراتها الفريدة، لذا اختر المكتبة التي تناسب احتياجات مشروعك على أفضل وجه.
