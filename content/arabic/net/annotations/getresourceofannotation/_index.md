## استيراد الحزم

قبل أن نبدأ العمل مع ملفات PDF، دعنا نستورد الحزم اللازمة. إذا كنت تستخدم Visual Studio، فما عليك سوى إضافة المرجع إلى مكتبة Aspose.PDF for .NET في مشروعك. قد يشبه الكود الخاص بك ما يلي:

```csharp
using Aspose.Pdf.Annotations;
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text;
```

من خلال استيراد هذه المساحات، يمكنك فتح الوظائف اللازمة للتعامل مع مستندات PDF بفعالية. تأكد من التحقق من أن تكوين المشروع الخاص بك يستهدف إصدار الإطار المناسب المتوافق مع Aspose.PDF.

الآن، دعنا نقسم العملية إلى خطوات يمكن إدارتها. سيكون لكل خطوة عنوان وشرح واضح. هل أنت مستعد؟ لنبدأ!

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً وقبل كل شيء، عليك تحديد مكان وجود المستند. وهذا أمر بالغ الأهمية لأن مكتبة Aspose ستحتاج إلى معرفة المسار للوصول إلى ملف PDF. 

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: افتح مستند PDF الخاص بك

 الآن بعد أن قمت بتعيين دليل المستندات الخاص بك، حان الوقت لفتح ملف PDF الذي تريد العمل به. يجعل Aspose.PDF هذا الأمر سهلاً من خلال`Document` فصل.

```csharp
// فتح المستند
Document doc = new Document(dataDir + "AddAnnotation.pdf");
```

في هذا المثال، نقوم بفتح ملف PDF باسم AddAnnotation.pdf. تأكد من وجود هذا الملف في الدليل المحدد، وإلا فسوف تواجه أخطاء!

## الخطوة 3: إنشاء تعليق توضيحي

 بعد ذلك، سنقوم بإنشاء تعليق توضيحي جديد. هنا، نستخدم`ScreenAnnotation`، والذي يسمح لك بإرفاق ملفات الوسائط المتعددة مثل SWF داخل ملف PDF.

```csharp
// إنشاء التعليقات التوضيحية
ScreenAnnotation sa = new ScreenAnnotation(doc.Pages[1], new Rectangle(100, 400, 300, 600), dataDir + "AddSwfFileAsAnnotation.swf");
```

ستلاحظ أننا نحدد رقم الصفحة وحجم التعليقات التوضيحية. وهذا يشبه إنشاء نافذة تملأها بمحتوى الوسائط المتعددة الخاص بك. 

## الخطوة 4: إضافة التعليقات التوضيحية إلى ملف PDF

الآن بعد أن قمت بإنشاء التعليق التوضيحي، حان الوقت لإضافته إلى المستند.

```csharp
doc.Pages[1].Annotations.Add(sa);
```

من خلال إضافة التعليقات التوضيحية إلى مجموعة التعليقات التوضيحية للصفحة المختارة، يمكنك دمج الوسائط المتعددة الخاصة بك بسلاسة في ملف PDF.

## الخطوة 5: احفظ مستندك

بعد إضافة التعليقات التوضيحية، لا تنس حفظ التغييرات. هذه هي اللحظة التي تؤتي فيها كل جهودك ثمارها!

```csharp
// حفظ المستند
doc.Save(dataDir + "GetResourceOfAnnotation_Out.pdf");
```

تضمن هذه الخطوة الحفاظ على ملف PDF المحدث الخاص بك، بما في ذلك التعليقات التوضيحية.

## الخطوة 6: افتح المستند المحفوظ

بعد ذلك، افتح المستند الذي قمت بحفظه للتو للوصول إلى التعليقات التوضيحية التي قمت بإنشائها.

```csharp
// فتح المستند
Document doc1 = new Document(dataDir + "GetResourceOfAnnotation_Out.pdf");
```

تعتبر هذه الخطوة بالغة الأهمية لاسترجاع المعلومات الموجودة في التعليق التوضيحي، لذا تأكد من صحة مسارات ملفاتك!

## الخطوة 7: استرداد إجراء التعليق التوضيحي

الآن حان الوقت للانتقال إلى لب الموضوع الخاص بالتعليق التوضيحي. هنا، نقوم باسترداد الإجراء المرتبط بالتعليق التوضيحي.

```csharp
RenditionAction action = (doc.Pages[1].Annotations[1] as ScreenAnnotation).Action as RenditionAction;
```

هذا الخط مهم للغاية؛ فأنت تحدد نوع الإجراء المرتبط بتعليقك التوضيحي على الشاشة. الأمر أشبه بقراءة الملصق الموجود على علبة لمعرفة ما بداخلها!

## الخطوة 8: الوصول إلى عرض الإجراء

بمجرد حصولك على الإجراء، فإن الخطوة التالية هي سحب الإصدار المرتبط به.

```csharp
Rendition rendition = ((doc.Pages[1].Annotations[1] as ScreenAnnotation).Action as RenditionAction).Rendition;
```

تُعد الصيغ هي التنسيقات التي يتم تخزين ملفات الوسائط المتعددة بها. يساعدك هذا على فهم كيفية تمثيل المحتوى الأصلي في ملف PDF.

## الخطوة 9: الحصول على مقطع الوسائط

الآن، دعونا نذهب خطوة أبعد ونصل إلى مقطع الوسائط من العرض.

```csharp
MediaClip clip = (rendition as MediaRendition).MediaClip;
```

فكر في مقطع الوسائط كمقطع فيديو أو صوت مضمن داخل ملف PDF الخاص بك - وهو مفاجأة صغيرة لأي شخص يفتحه!

## الخطوة 10: الوصول إلى بيانات الوسائط

أخيرًا، قد ترغب في الوصول إلى البيانات الفعلية المرتبطة بمقطع الوسائط. إليك كيفية القيام بذلك:

```csharp
FileSpecification data = (clip as MediaClipData).Data;
MemoryStream ms = new MemoryStream();
byte[] buffer = new byte[1024];
int read = 0;
//بيانات الوسائط المتاحة في FileSpecification.Contents
Stream source = data.Contents;
while ((read = source.Read(buffer, 0, buffer.Length)) > 0)
{
    ms.Write(buffer, 0, read);
}
```

هذه الخطوة تشبه سكب محتويات صندوق لمعرفة ما لديك بالفعل. لقد جمعت للتو كل البيانات المرتبطة بمقطع الوسائط الخاص بك!

## الخطوة 11: إخراج تفاصيل العرض

وأخيرًا، قم بطباعة تفاصيل التسليم على وحدة التحكم، مما يسمح لك بالتحقق من عملك.

```csharp
Console.WriteLine(rendition.Name);
Console.WriteLine(action.RenditionOperation);
```

هذه هي النهاية الكبرى حيث تكشف عن محتوى تعليقاتك التوضيحية وكيفية عملها.
