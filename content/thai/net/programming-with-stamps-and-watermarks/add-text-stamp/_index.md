ในบทช่วยสอนนี้ เราจะอธิบายทีละขั้นตอนเกี่ยวกับวิธีเพิ่มตราประทับข้อความในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เราจะแสดงวิธีใช้โค้ดต้นฉบับ C# ที่ให้มาเพื่อเพิ่มตราประทับข้อความแบบกำหนดเองลงในหน้าเฉพาะของไฟล์ PDF

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนที่คุณจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- สภาพแวดล้อมการพัฒนา .NET ที่ติดตั้งแล้ว
- ไลบรารี Aspose.PDF สำหรับ .NET ดาวน์โหลดและอ้างอิงในโครงการของคุณ

## ขั้นตอนที่ 2: การโหลดเอกสาร PDF

ขั้นตอนแรกคือโหลดเอกสาร PDF ที่มีอยู่ลงในโครงการของคุณ ดังต่อไปนี้:

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENTS DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir + "AddTextStamp.pdf");
```

อย่าลืมแทนที่ “ไดเรกทอรีเอกสารของคุณ” ด้วยเส้นทางจริงไปยังไดเรกทอรีที่เอกสาร PDF ของคุณอยู่

## ขั้นตอนที่ 3: การสร้างบัฟเฟอร์ข้อความ

ตอนนี้คุณได้อัปโหลดเอกสาร PDF แล้ว คุณสามารถสร้างตราประทับข้อความเพื่อเพิ่มได้ วิธีการทำมีดังนี้:

```csharp
// สร้างบัฟเฟอร์ข้อความ
TextStamp textStamp = new TextStamp("Example Stamp");
```

โค้ดด้านบนจะสร้างบัฟเฟอร์ข้อความใหม่ที่มีข้อความที่ระบุ

## ขั้นตอนที่ 4: การกำหนดค่าคุณสมบัติแสตมป์ข้อความ

ก่อนที่จะเพิ่มตราประทับข้อความลงในเอกสาร PDF คุณสามารถกำหนดค่าคุณสมบัติต่างๆ ของตราประทับได้ เช่น พื้นหลัง ตำแหน่ง การหมุน แบบอักษร ขนาด ฯลฯ ดังต่อไปนี้:

```csharp
// กำหนดค่าคุณสมบัติบัฟเฟอร์ข้อความ
textStamp. Background = true;
textStamp. XIndent = 100;
textStamp. YIndent = 100;
textStamp.Rotate = Rotate.on90;
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold | FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);
```

คุณสามารถปรับเปลี่ยนคุณสมบัติเหล่านี้ได้ตามความต้องการของคุณ

## ขั้นตอนที่ 5: เพิ่มแสตมป์ข้อความลงใน PDF

เมื่อแสตมป์ข้อความพร้อมแล้ว คุณสามารถเพิ่มข้อความลงในหน้าเฉพาะของเอกสาร PDF ได้ ดังต่อไปนี้:

```csharp
//เพิ่มบัฟเฟอร์ข้อความลงในหน้าเฉพาะ
pdfDocument.Pages[1].AddStamp(textStamp);
```

โค้ดด้านบนจะเพิ่มข้อความประทับลงในหน้าแรกของเอกสาร PDF คุณสามารถระบุหน้าอื่นได้หากจำเป็น

## ขั้นตอนที่ 6: บันทึกเอกสารผลลัพธ์

เมื่อคุณเพิ่มตราประทับข้อความแล้ว คุณสามารถบันทึกเอกสาร PDF ที่แก้ไขได้ ดังต่อไปนี้:

```csharp
// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
```

โค้ดด้านบนจะบันทึกเอกสาร PDF ที่แก้ไขแล้วในไดเร็กทอรีที่ระบุ

### ตัวอย่างโค้ดที่มาสำหรับ Add Text Stamp โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp

// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";

// เปิดเอกสาร
Document pdfDocument = new Document(dataDir+ "AddTextStamp.pdf");

// สร้างแสตมป์ข้อความ
TextStamp textStamp = new TextStamp("Sample Stamp");

// ตั้งค่าว่าแสตมป์จะเป็นพื้นหลังหรือไม่
textStamp.Background = true;

// ตั้งค่าที่มา
textStamp.XIndent = 100;
textStamp.YIndent = 100;

// หมุนแสตมป์
textStamp.Rotate = Rotation.on90;

// ตั้งค่าคุณสมบัติข้อความ
textStamp.TextState.Font = FontRepository.FindFont("Arial");
textStamp.TextState.FontSize = 14.0F;
textStamp.TextState.FontStyle = FontStyles.Bold;
textStamp.TextState.FontStyle = FontStyles.Italic;
textStamp.TextState.ForegroundColor = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Aqua);

// เพิ่มตราประทับบนหน้าที่ต้องการ
pdfDocument.Pages[1].AddStamp(textStamp);
dataDir = dataDir + "AddTextStamp_out.pdf";

// บันทึกเอกสารผลลัพธ์
pdfDocument.Save(dataDir);
Console.WriteLine("\nText stamp added successfully.\nFile saved at " + dataDir);            

```
