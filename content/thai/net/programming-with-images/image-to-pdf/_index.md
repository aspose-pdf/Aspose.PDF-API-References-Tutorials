Aspose.PDF สำหรับ .NET เป็นไลบรารีอันทรงพลังที่ช่วยให้นักพัฒนาสามารถสร้าง จัดการ และแปลงเอกสาร PDF โดยใช้ C# หรือภาษา .NET ใดๆ ก็ได้ ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับขั้นตอนการแปลงรูปภาพเป็น PDF โดยใช้ Aspose.PDF สำหรับ .NET

## ขั้นตอนที่ 1: การตั้งค่าสภาพแวดล้อม

ก่อนเริ่มต้น ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Aspose.PDF สำหรับ .NET ไว้ในระบบของคุณแล้ว คุณสามารถดาวน์โหลดและติดตั้งได้จากเว็บไซต์อย่างเป็นทางการของ Aspose เมื่อติดตั้งแล้ว ให้สร้างโปรเจ็กต์ C# ใหม่ในสภาพแวดล้อมการพัฒนาที่คุณต้องการ

## ขั้นตอนที่ 2: นำเข้าไลบรารีที่จำเป็น

หากต้องการใช้ Aspose.PDF สำหรับ .NET ในโปรเจ็กต์ของคุณ คุณต้องนำเข้าไลบรารีที่จำเป็น เพิ่มคำสั่ง using ต่อไปนี้ที่จุดเริ่มต้นของไฟล์ C#:

```csharp
using Aspose.Pdf;
using System.IO;
using System.Drawing;
```

## ขั้นตอนที่ 3: การเริ่มต้นวัตถุเอกสาร

 ในโค้ด C# ขั้นตอนแรกคือการเริ่มต้น`Document` วัตถุ วัตถุนี้แสดงถึงเอกสาร PDF ที่เราจะสร้าง เพิ่มโค้ดต่อไปนี้ลงในโปรเจ็กต์ของคุณ:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document();
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"`ด้วยเส้นทางจริงที่คุณต้องการบันทึกไฟล์ PDF

## ขั้นตอนที่ 4: เพิ่มหน้าลงในเอกสาร

 ต่อไปเราต้องเพิ่มหน้าลงในเอกสาร หน้าหนึ่งจะแสดงโดย`Page` คลาส ใช้โค้ดต่อไปนี้เพื่อเพิ่มหน้าลงในเอกสาร:

```csharp
Page page = doc.Pages.Add();
```

 โค้ดนี้จะสร้างหน้าใหม่และเพิ่มเข้าไปใน`Pages` การเก็บรวบรวมเอกสาร

## ขั้นตอนที่ 5: การโหลดไฟล์ภาพ

 ในการแปลงรูปภาพเป็น PDF ก่อนอื่นเราต้องโหลดไฟล์รูปภาพต้นฉบับ ในตัวอย่างนี้ เราถือว่าไฟล์รูปภาพมีชื่อว่า`aspose-logo.jpg` และอยู่ในไดเร็กทอรีเดียวกับไฟล์ C# ของคุณ ใช้โค้ดต่อไปนี้เพื่อโหลดไฟล์อิมเมจ:

```csharp
FileStream fs = new FileStream(dataDir + "aspose-logo.jpg", FileMode.Open, FileAccess.Read);
byte[] tmpBytes = new byte[fs.Length];
fs.Read(tmpBytes, 0, int.Parse(fs.Length.ToString()));
MemoryStream mystream = new MemoryStream(tmpBytes);
```

 อย่าลืมเปลี่ยน`"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไฟล์ภาพ

## ขั้นตอนที่ 6: การตั้งค่าระยะขอบและกล่องครอบตัด

ก่อนที่จะเพิ่มรูปภาพลงในหน้า PDF เราสามารถปรับแต่งเค้าโครงหน้าได้ ตัวอย่างเช่น เราสามารถตั้งค่าระยะขอบและช่องครอบตัดให้พอดีกับขนาดของรูปภาพ ใช้โค้ดต่อไปนี้เพื่อปรับการตั้งค่าหน้า:

```csharp
Bitmap b = new Bitmap(mystream);
page.PageInfo.Margin.Bottom = 0;
page.PageInfo.Margin.Top = 0;
page.PageInfo.Margin.Left = 0;
page

.PageInfo.Margin.Right = 0;
page.CropBox = new Aspose.Pdf.Rectangle(0, 0, b.Width, b.Height);
```

การตั้งค่าเหล่านี้จะช่วยให้แน่ใจว่ารูปภาพจะพอดีกับหน้าโดยไม่ต้องมีระยะขอบเพิ่มเติมใดๆ

## ขั้นตอนที่ 7: การสร้างวัตถุรูปภาพ

 ตอนนี้เรามาสร้างกัน`Aspose.Pdf.Image` วัตถุที่จะเก็บข้อมูลภาพ เพิ่มโค้ดต่อไปนี้ลงในโปรเจ็กต์ของคุณ:

```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
```

วัตถุนี้จะแสดงถึงรูปภาพที่เราต้องการเพิ่มลงในหน้า PDF

## ขั้นตอนที่ 8: การเพิ่มรูปภาพลงในหน้า

 ในการเพิ่มรูปภาพลงในหน้า PDF เราจำเป็นต้องกำหนดข้อมูลรูปภาพให้กับ`ImageStream` ทรัพย์สินของ`Aspose.Pdf.Image` วัตถุ ใช้โค้ดต่อไปนี้เพื่อเพิ่มรูปภาพ:

```csharp
image1.ImageStream = mystream;
page.Paragraphs.Add(image1);
```

 ที่นี่เราจะกำหนดสตรีมภาพให้กับ`ImageStream` คุณสมบัติแล้วเพิ่มวัตถุภาพลงใน`Paragraphs` การรวบรวมหน้า

## ขั้นตอนที่ 9: บันทึกไฟล์ PDF

เมื่อเราเพิ่มรูปภาพลงในหน้า PDF แล้ว เราสามารถบันทึกไฟล์ PDF ที่ได้ โดยใช้โค้ดต่อไปนี้เพื่อบันทึกไฟล์:

```csharp
dataDir = dataDir + "ImageToPDF_out.pdf";
doc.Save(dataDir);
```

 แทนที่`"YOUR DOCUMENT DIRECTORY"` พร้อมด้วยไดเร็กทอรีเอาท์พุตและชื่อไฟล์ที่ต้องการ

## ขั้นตอนที่ 10: การปิดสตรีมหน่วยความจำ

หลังจากบันทึกไฟล์ PDF แล้ว สิ่งสำคัญคือต้องปิดสตรีมหน่วยความจำเพื่อปลดปล่อยทรัพยากรระบบ เพิ่มโค้ดต่อไปนี้เพื่อปิดสตรีมหน่วยความจำ:

```csharp
mystream. Close();
```

## การรันโค้ดและการตรวจสอบผลลัพธ์

ตอนนี้คุณได้ดำเนินการใช้งานโค้ดเสร็จเรียบร้อยแล้ว รันโค้ดและตรวจสอบว่ารูปภาพได้รับการแปลงเป็น PDF เรียบร้อยแล้ว ไฟล์เอาต์พุตควรบันทึกไว้ในไดเร็กทอรีที่ระบุ


### ตัวอย่างโค้ดต้นฉบับสำหรับการแปลงภาพเป็น PDF โดยใช้ Aspose.PDF สำหรับ .NET 
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
// สร้างอินสแตนซ์ของวัตถุเอกสาร
Document doc = new Document();
// เพิ่มหน้าเข้าในคอลเลคชันหน้าเอกสาร
Page page = doc.Pages.Add();
// โหลดไฟล์ภาพต้นฉบับไปยังวัตถุ Stream
FileStream fs = new FileStream(dataDir + "aspose-logo.jpg", FileMode.Open, FileAccess.Read);
byte[] tmpBytes = new byte[fs.Length];
fs.Read(tmpBytes, 0, int.Parse(fs.Length.ToString()));
MemoryStream mystream = new MemoryStream(tmpBytes);
// สร้างอินสแตนซ์ของวัตถุ BitMap ด้วยสตรีมภาพที่โหลด
Bitmap b = new Bitmap(mystream);
// กำหนดระยะขอบให้รูปภาพพอดีเป็นต้น
page.PageInfo.Margin.Bottom = 0;
page.PageInfo.Margin.Top = 0;
page.PageInfo.Margin.Left = 0;
page.PageInfo.Margin.Right = 0;
page.CropBox = new Aspose.Pdf.Rectangle(0, 0, b.Width, b.Height);
// สร้างวัตถุภาพ
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
// เพิ่มรูปภาพลงในคอลเลกชันย่อหน้าของส่วน
page.Paragraphs.Add(image1);
// ตั้งค่าสตรีมไฟล์ภาพ
image1.ImageStream = mystream;
dataDir = dataDir + "ImageToPDF_out.pdf";
// บันทึกไฟล์ PDF ที่ได้
doc.Save(dataDir);
// ปิดวัตถุ memoryStream
mystream.Close();
Console.WriteLine("\nImage converted to pdf successfully.\nFile saved at " + dataDir); 
```
