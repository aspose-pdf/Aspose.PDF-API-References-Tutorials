## การแนะนำ

คุณเคยเปิดไฟล์ PDF แล้วพบว่าบุ๊กมาร์กทั้งหมดยุบลงหรือไม่ การทำเช่นนี้จะสร้างความหงุดหงิด โดยเฉพาะเมื่อคุณพยายามค้นหาเอกสารยาวๆ โชคดีที่ Aspose.PDF สำหรับ .NET ช่วยให้คุณสามารถขยายบุ๊กมาร์กในไฟล์ PDF ของคุณได้อย่างง่ายดายด้วยโปรแกรม คู่มือนี้จะแนะนำคุณทีละขั้นตอน เพื่อให้คุณเข้าใจแต่ละส่วนของโค้ดและวิธีการทำงานของโค้ด ดังนั้น หยิบเครื่องดื่มที่คุณชอบ แล้วมาดำดิ่งสู่โลกแห่งการจัดการ PDF กันเถอะ!

### นำเข้าเนมสเปซ

ที่ด้านบนของไฟล์ C# ของคุณ นำเข้าเนมสเปซ Aspose.PDF:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
```

ตอนนี้คุณได้ตั้งค่าทุกอย่างเรียบร้อยแล้ว มาไปที่โค้ดจริงกันเลย!

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ขั้นแรก คุณต้องระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ นี่คือตำแหน่งที่ไฟล์ PDF อินพุตของคุณตั้งอยู่และตำแหน่งที่ไฟล์เอาต์พุตจะถูกบันทึก

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## ขั้นตอนที่ 2: เปิดเอกสาร PDF

 ต่อไปคุณจะเปิดเอกสาร PDF ที่มีบุ๊กมาร์กที่คุณต้องการขยาย ซึ่งทำได้โดยใช้`Document` คลาสจากไลบรารี Aspose.PDF

```csharp
// เปิดเอกสาร
Document doc = new Document(dataDir + "input-bookmark.pdf");
```

## ขั้นตอนที่ 3: ตั้งค่าโหมดการดูหน้า

ตอนนี้ คุณต้องตั้งค่าโหมดการดูหน้าเอกสาร ซึ่งจะกำหนดว่าไฟล์ PDF จะแสดงอย่างไรเมื่อเปิดขึ้นมา ในกรณีนี้ เราต้องการใช้โครงร่าง (บุ๊กมาร์ก)

```csharp
//ตั้งค่าโหมดการดูหน้า เช่น แสดงภาพขนาดย่อ เต็มจอ แสดงแผงแนบ
doc.PageMode = PageMode.UseOutlines;
```

## ขั้นตอนที่ 4: พิจารณาโครงร่างรายการ

มาถึงส่วนสนุกแล้ว! คุณจะดูรายการโครงร่างแต่ละรายการในคอลเลกชันโครงร่างของ PDF และตั้งสถานะการเปิดเป็น true วิธีนี้จะช่วยขยายบุ๊กมาร์ก

```csharp
// เดินผ่านรายการโครงร่างแต่ละรายการในคอลเล็กชันโครงร่างของไฟล์ PDF
foreach (OutlineItemCollection item in doc.Outlines)
{
    // กำหนดสถานะเปิดสำหรับรายการโครงร่าง
    item.Open = true;
}
```

## ขั้นตอนที่ 5: บันทึกเอกสารผลลัพธ์

หลังจากขยายบุ๊กมาร์กแล้ว ก็ถึงเวลาบันทึกเอกสารที่แก้ไขแล้ว คุณจะต้องระบุชื่อไฟล์ใหม่สำหรับไฟล์ PDF ที่ส่งออก

```csharp
dataDir = dataDir + "ExpandBookmarks_out.pdf";
// บันทึกเอาท์พุต
doc.Save(dataDir);
```

## ขั้นตอนที่ 6: ข้อความยืนยัน

ในที่สุดคุณสามารถพิมพ์ข้อความยืนยันไปยังคอนโซลเพื่อแจ้งให้คุณทราบว่าการขยายบุ๊กมาร์กสำเร็จแล้ว

```csharp
Console.WriteLine("\nBookmarks expanded successfully.\nFile saved at " + dataDir);
```
