## परिचय

क्या आपने कभी PDF फ़ाइल खोली है और पाया है कि सभी बुकमार्क संकुचित हैं? यह निराशाजनक हो सकता है, खासकर जब आप किसी लंबे दस्तावेज़ में नेविगेट करने का प्रयास कर रहे हों। सौभाग्य से, .NET के लिए Aspose.PDF के साथ, आप आसानी से प्रोग्रामेटिक रूप से अपनी PDF फ़ाइलों में बुकमार्क का विस्तार कर सकते हैं। यह मार्गदर्शिका आपको चरण-दर-चरण प्रक्रिया से गुजारेगी, यह सुनिश्चित करते हुए कि आप कोड के प्रत्येक भाग को समझते हैं और यह कैसे काम करता है। तो, अपना पसंदीदा पेय लें, और PDF हेरफेर की दुनिया में गोता लगाएँ!

### नामस्थान आयात करें


अब जब आपने सब कुछ सेट कर लिया है, तो चलिए वास्तविक कोड पर चलते हैं!

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

सबसे पहले, आपको अपने दस्तावेज़ निर्देशिका का पथ निर्दिष्ट करना होगा। यह वह जगह है जहाँ आपकी इनपुट पीडीएफ फाइल स्थित है और जहाँ आउटपुट फ़ाइल सहेजी जाएगी।

```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 2: पीडीएफ दस्तावेज़ खोलें

 इसके बाद, आप पीडीएफ दस्तावेज़ खोलेंगे जिसमें वे बुकमार्क हैं जिन्हें आप विस्तारित करना चाहते हैं।`Document` Aspose.PDF लाइब्रेरी से क्लास.

```csharp
// दस्तावेज़ खोलें
Document doc = new Document(dataDir + "input-bookmark.pdf");
```

## चरण 3: पेज व्यू मोड सेट करें

अब, आपको दस्तावेज़ का पेज व्यू मोड सेट करना होगा। यह निर्धारित करता है कि पीडीएफ खुलने पर कैसे प्रदर्शित होगा। इस मामले में, हम आउटलाइन (बुकमार्क) का उपयोग करना चाहते हैं।

```csharp
//पृष्ठ दृश्य मोड सेट करें अर्थात थंबनेल दिखाएं, पूर्ण-स्क्रीन दिखाएं, अनुलग्नक पैनल दिखाएं
doc.PageMode = PageMode.UseOutlines;
```

## चरण 4: आउटलाइन आइटम के माध्यम से आगे बढ़ें

अब आता है मज़ेदार हिस्सा! आप PDF के आउटलाइन संग्रह में प्रत्येक आउटलाइन आइटम को लूप करेंगे और उनकी खुली स्थिति को true पर सेट करेंगे। इससे बुकमार्क विस्तृत हो जाएँगे।

```csharp
// पीडीएफ फाइल के आउटलाइन संग्रह में प्रत्येक आउटलाइन आइटम के माध्यम से यात्रा करें
foreach (OutlineItemCollection item in doc.Outlines)
{
    // आउटलाइन आइटम के लिए खुली स्थिति सेट करें
    item.Open = true;
}
```

## चरण 5: आउटपुट दस्तावेज़ सहेजें

बुकमार्क का विस्तार करने के बाद, संशोधित दस्तावेज़ को सहेजने का समय आ गया है। आप आउटपुट पीडीएफ के लिए एक नया फ़ाइल नाम निर्दिष्ट करेंगे।

```csharp
dataDir = dataDir + "ExpandBookmarks_out.pdf";
// आउटपुट सहेजें
doc.Save(dataDir);
```

## चरण 6: पुष्टिकरण संदेश

अंत में, आप कंसोल पर एक पुष्टिकरण संदेश प्रिंट कर सकते हैं ताकि आपको पता चल सके कि बुकमार्क सफलतापूर्वक विस्तारित हो गए हैं।

```csharp
Console.WriteLine("\nBookmarks expanded successfully.\nFile saved at " + dataDir);
```
