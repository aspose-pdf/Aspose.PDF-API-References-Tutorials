## مقدمة

في العصر الرقمي، أصبحت ملفات PDF منتشرة على نطاق واسع. سواء كنت تشارك التقارير أو العروض التقديمية أو الكتب الإلكترونية، فإن تنسيق المستندات المحمولة (PDF) هو الخيار الأمثل للحفاظ على سلامة مستنداتك. ومع ذلك، مع إنشاء ومشاركة المزيد من ملفات PDF، يمكن أن تتضخم أحجام الملفات، مما يجعل إرسالها أو تخزينها أمرًا مرهقًا. وهنا يأتي دور Aspose.PDF for .NET، حيث يوفر أدوات قوية لتحسين ملفات PDF الخاصة بك. في هذا البرنامج التعليمي، سنتعمق في كيفية إلغاء تضمين الخطوط وتحسين ملفات PDF باستخدام Aspose.PDF for .NET.

## المتطلبات الأساسية

قبل أن ننتقل إلى التفاصيل الدقيقة، دعنا نتأكد من أن لديك كل ما تحتاجه للبدء:

1. Visual Studio: تأكد من تثبيت Visual Studio على جهازك. فهو بيئة التطوير المتكاملة التي سنستخدمها لكتابة وتشغيل كود .NET الخاص بنا.
2.  Aspose.PDF لـ .NET: ستحتاج إلى تنزيل مكتبة Aspose.PDF وتثبيتها. يمكنك الحصول عليها من[رابط التحميل](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة ببرمجة C# على فهم مقتطفات التعليمات البرمجية التي سنستخدمها.
4.  ملف PDF: قم بإعداد ملف PDF الذي تريد تحسينه. يمكنك استخدام أي ملف PDF، ولكن للتوضيح، سنشير إليه باسم`OptimizeDocument.pdf`.

## استيراد الحزم

للبدء، تحتاج إلى استيراد الحزم اللازمة في مشروع C# الخاص بك. إليك كيفية القيام بذلك:

1. افتح مشروعك في Visual Studio.
2. أضف مرجعًا إلى Aspose.PDF: انقر بزر الماوس الأيمن على مشروعك في مستكشف الحلول، وحدد "إدارة حزم NuGet"، وابحث عن`Aspose.PDF`. تثبيت الحزمة.

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

الآن بعد أن قمنا بإعداد كل شيء، دعنا نقسم عملية التحسين إلى خطوات قابلة للإدارة.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً وقبل كل شيء، عليك تحديد المسار إلى دليل المستندات. هذا هو المكان الذي سيتم تخزين ملفات PDF فيه. وإليك كيفية القيام بذلك:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

 يستبدل`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يوجد به ملف PDF الخاص بك. وهذا أمر بالغ الأهمية لأن البرنامج يحتاج إلى معرفة مكان العثور على ملف PDF الذي تريد تحسينه.

## الخطوة 2: افتح مستند PDF

الآن بعد أن قمنا بإعداد الدليل، حان الوقت لفتح مستند PDF الذي نريد تحسينه. إليك الكود اللازم للقيام بذلك:

```csharp
// فتح المستند
Document pdfDocument = new Document(dataDir + "OptimizeDocument.pdf");
```

 يؤدي هذا السطر من التعليمات البرمجية إلى إنشاء سطر جديد`Document` الكائن الذي يمثل ملف PDF الخاص بك. تأكد من أن اسم الملف يتطابق مع الاسم الموجود في الدليل الخاص بك.

## الخطوة 3: تعيين خيارات التحسين

بعد ذلك، نحتاج إلى تحديد خيارات التحسين. في هذه الحالة، نريد إلغاء تضمين الخطوط. وإليك كيفية إعداد ذلك:

```csharp
// تعيين خيار UnembedFonts
var optimizeOptions = new Pdf.Optimization.OptimizationOptions
{
    UnembedFonts = true
};
```

 عن طريق الإعداد`UnembedFonts` ل`true`نحن نطلب من Aspose.PDF تحسين ملف PDF عن طريق إزالة الخطوط المضمنة. يمكن أن يؤدي هذا إلى تقليل حجم الملف بشكل كبير، خاصة إذا كان ملف PDF يحتوي على العديد من الخطوط المضمنة.

## الخطوة 4: تحسين مستند PDF

بعد ضبط الخيارات، حان الوقت لتحسين مستند PDF. إليك الكود الذي سيساعدك على القيام بذلك:

```csharp
Console.WriteLine("Start");
// تحسين مستند PDF باستخدام OptimizationOptions
pdfDocument.OptimizeResources(optimizeOptions);
```

هذا المقطع من التعليمات البرمجية يستدعي`OptimizeResources` الطريقة على`pdfDocument` الكائن، وتطبيق خيارات التحسين التي حددناها سابقًا. سترى رسالة في وحدة التحكم تشير إلى أن عملية التحسين قد بدأت.

## الخطوة 5: احفظ المستند المحدث

بعد تحسين ملف PDF، نحتاج إلى حفظ المستند المحدث. وإليك كيفية القيام بذلك:

```csharp
// حفظ المستند المحدث
pdfDocument.Save(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Finished");
```

 يحفظ هذا الكود ملف PDF المحسن كـ`OptimizeDocument_out.pdf` في نفس الدليل. يمكنك اختيار اسم مختلف إذا كنت تفضل ذلك، ولكن الحفاظ على الاسم مشابهًا يساعد في تحديد الإصدارات الأصلية والمحسّنة.

## الخطوة 6: مقارنة أحجام الملفات

أخيرًا، من الجيد دائمًا التحقق من مقدار المساحة التي وفرتها. إليك كيفية مقارنة أحجام الملفات الأصلية والمحسّنة:

```csharp
var fi1 = new System.IO.FileInfo(dataDir + "OptimizeDocument.pdf");
var fi2 = new System.IO.FileInfo(dataDir + "OptimizeDocument_out.pdf");
Console.WriteLine("Original file size: {0}. Reduced file size: {1}", fi1.Length, fi2.Length);
```

يقوم هذا الكود باسترداد أحجام الملفات لكل من ملفات PDF الأصلية والمحسّنة ويطبعها على وحدة التحكم. إنها لحظة مرضية لرؤية مقدار ما قمت بتقليصه من حجم الملف!

## خاتمة
