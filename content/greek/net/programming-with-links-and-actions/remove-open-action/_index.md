Ορίστε τη διαδρομή καταλόγου των εγγράφων σας και μεταφορτώστε το αρχείο PDF χρησιμοποιώντας τον ακόλουθο κώδικα:


 Καταργήστε την ανοιχτή ενέργεια από το έγγραφο ορίζοντας το`OpenAction` μηδενική ιδιότητα:

```csharp

```

 Αποθηκεύστε το ενημερωμένο έγγραφο χρησιμοποιώντας το`Save` μέθοδος:


Εμφανίστε ένα μήνυμα που υποδεικνύει ότι η ανοιχτή ενέργεια καταργήθηκε με επιτυχία και καθορίστε τη θέση του αποθηκευμένου αρχείου:


### Δείγμα πηγαίου κώδικα για Κατάργηση ανοιχτής ενέργειας χρησιμοποιώντας το Aspose.PDF για .NET 
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.

// Άνοιγμα εγγράφου
```
