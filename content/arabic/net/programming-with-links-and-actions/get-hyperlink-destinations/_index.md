## استيراد الحزم

للبدء في استخدام كود C#، ستحتاج إلى التأكد من استيراد مساحات أسماء Aspose الضرورية. إليك الشكل الذي تبدو عليه:

```csharp
using System;
using System.IO;
using Aspose.Pdf;
using System.Collections;
using Aspose.Pdf.Annotations;
using System.Collections.Generic;
```

تخبر هذه الأسطر برنامجنا بشكل أساسي بالمكتبات التي نريد استخدامها، حتى يتمكن من التعرف على الفئات والطرق التي نخطط للعمل بها.

## الخطوة 3: قم بتحميل مستند PDF الخاص بك

الآن، لنبدأ في تحميل ملف PDF الذي تريد العمل عليه. ستحتاج إلى تعيين المسار إلى ملف PDF بشكل صحيح.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document document = new Document(dataDir + "input.pdf");
```

 شرح: استبدال`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي الذي يوجد به ملف PDF الخاص بك. يقوم هذا الجزء من التعليمات البرمجية بتهيئة`Document` دورة تدريبية مقدمة من Aspose، والتي تفتح حرفيًا ملف PDF الخاص بك للأعمال!

## الخطوة 4: تصفح صفحات ملف PDF

بعد ذلك، سننتقل عبر كل صفحة في مستند PDF. الأمر أشبه بتقليب صفحات كتاب.

```csharp
foreach (Aspose.Pdf.Page page in document.Pages)
{
    //... الكود لاستخراج الروابط يذهب هنا
}
```

لماذا التكرار؟: يمكن أن تحتوي كل صفحة على روابط مختلفة، ونريد التحقق من كل رابط على حدة. بهذه الطريقة، لن تفوتك أي قطع ذهبية مخفية بداخلها.

## الخطوة 5: استرداد تعليقات الروابط

 بمجرد وصولنا إلى الصفحة، فلنبدأ في البحث عن تعليقات الروابط. سنستخدم`AnnotationSelector`.

```csharp
AnnotationSelector selector = new AnnotationSelector(new LinkAnnotation(page, Rectangle.Trivial));
page.Accept(selector);
```

 فهم`AnnotationSelector` :تساعد هذه الفئة في تحديد الروابط الموجودة على الصفحة من خلال تحديد التعليقات التوضيحية من النوع`LinkAnnotation`. 

## الخطوة 6: تخزين التعليقات التوضيحية للروابط وتكرارها

سنقوم الآن بجمع الروابط في قائمة وتكرارها لعرض وجهاتها.

```csharp
IList<Annotation> list = selector.Selected;
foreach (LinkAnnotation a in list)
{
    Console.WriteLine("\nDestination: " + (a.Action as GoToURIAction).URI + "\n");
}
```

التفصيل التفصيلي:
- `IList<Annotation> list`:هذا يخزن كافة تعليقات الرابط المحددة.
- التكرار خلال القائمة: بالنسبة لكل رابط، نقوم بالتقاط عنوان URI الخاص به وطباعته.
