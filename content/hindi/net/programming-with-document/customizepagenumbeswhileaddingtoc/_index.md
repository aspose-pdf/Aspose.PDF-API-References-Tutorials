## परिचय

दस्तावेज़ प्रबंधन की दुनिया में, PDF सर्वोच्च स्थान पर है। वे विभिन्न प्लेटफ़ॉर्म पर दस्तावेज़ों को साझा करने और संरक्षित करने के लिए जाने-माने फ़ॉर्मेट हैं। लेकिन क्या होगा जब आप अपने PDF दस्तावेज़ों को टेबल ऑफ़ कॉन्टेंट (TOC) जैसी सुविधाओं के साथ बढ़ाना चाहते हैं? यहीं पर Aspose.PDF for .NET काम आता है! यह शक्तिशाली लाइब्रेरी डेवलपर्स को आसानी से PDF फ़ाइलों में हेरफेर करने की अनुमति देती है, जिससे वे आसानी से सामग्री जोड़, संशोधित और अनुकूलित कर सकते हैं। इस ट्यूटोरियल में, हम Aspose.PDF for .NET का उपयोग करके अपने PDF दस्तावेज़ों में TOC जोड़ते समय पेज नंबर को कस्टमाइज़ करने के तरीके के बारे में जानेंगे। तो, अपनी कोडिंग टोपी पकड़ो, और चलो शुरू करते हैं!

## आवश्यक शर्तें

इससे पहले कि हम कोड में प्रवेश करें, कुछ चीजें हैं जिन्हें आपको ध्यान में रखना होगा:

1. विज़ुअल स्टूडियो: सुनिश्चित करें कि आपके मशीन पर विज़ुअल स्टूडियो स्थापित है। यह हमारा विकास वातावरण होगा।
2. .NET के लिए Aspose.PDF: आपको Aspose.PDF लाइब्रेरी डाउनलोड और इंस्टॉल करनी होगी। आप इसे पा सकते हैं[यहाँ](https://releases.aspose.com/pdf/net/).
3. C# का बुनियादी ज्ञान: C# प्रोग्रामिंग से परिचित होने से आपको कोड स्निपेट को बेहतर ढंग से समझने में मदद मिलेगी।
4. एक नमूना पीडीएफ फाइल: एक नमूना पीडीएफ फाइल तैयार रखें जिस पर हम काम कर सकें। आप एक सरल फाइल बना सकते हैं या मौजूदा पीडीएफ डाउनलोड कर सकते हैं।

## पैकेज आयात करें

आरंभ करने के लिए, हमें आवश्यक पैकेज आयात करने की आवश्यकता है। अपना Visual Studio प्रोजेक्ट खोलें और Aspose.PDF लाइब्रेरी में संदर्भ जोड़ें। आप NuGet पैकेज मैनेजर का उपयोग करके ऐसा कर सकते हैं:

1. समाधान एक्सप्लोरर में अपने प्रोजेक्ट पर राइट-क्लिक करें।
2. "NuGet पैकेज प्रबंधित करें" चुनें.
3. "Aspose.PDF" खोजें और इसे इंस्टॉल करें।

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

एक बार जब आप लाइब्रेरी स्थापित कर लेते हैं, तो आप कोडिंग शुरू कर सकते हैं!

## चरण 1: अपनी दस्तावेज़ निर्देशिका सेट करें

सबसे पहले, हमें अपनी डॉक्यूमेंट डायरेक्टरी सेट अप करनी होगी। यहीं पर हम अपनी इनपुट और आउटपुट पीडीएफ फाइलें स्टोर करेंगे। 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string inFile = dataDir + "42824.pdf";
string outFile = dataDir + "42824_out.pdf";
```

इस स्निपेट में, प्रतिस्थापित करें`YOUR DOCUMENT DIRECTORY` वास्तविक पथ के साथ जहाँ आपकी PDF फ़ाइलें स्थित हैं। इससे हमें मौजूदा PDF को लोड करने और संशोधित संस्करण को सहेजने में मदद मिलेगी।

## चरण 2: मौजूदा पीडीएफ फाइल लोड करें

अब जबकि हमारी दस्तावेज़ निर्देशिका स्थापित हो गई है, तो चलिए मौजूदा पीडीएफ फाइल को लोड करते हैं। 

```csharp
Document doc = new Document(inFile);
```

 यहाँ, हम एक नया निर्माण करते हैं`Document` इनपुट फ़ाइल पथ को पास करके ऑब्जेक्ट। यह हमें प्रोग्रामेटिक रूप से पीडीएफ सामग्री में हेरफेर करने की अनुमति देता है।

## चरण 3: TOC के लिए एक नया पृष्ठ डालें

इसके बाद, हमें अपने पीडीएफ में एक नया पेज बनाना होगा जहां विषय-सूची होगी।

```csharp
Aspose.Pdf.Page tocPage = doc.Pages.Insert(1);
```

यह पंक्ति दस्तावेज़ की शुरुआत में एक नया पृष्ठ सम्मिलित करती है। TOC इस पृष्ठ पर प्रदर्शित किया जाएगा।

## चरण 4: TOC जानकारी बनाएँ

अब, TOC जानकारी को दर्शाने के लिए एक ऑब्जेक्ट बनाएं।

```csharp
TocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocInfo.PageNumbersPrefix = "P";
tocPage.TocInfo = tocInfo;
```

 इस चरण में, हम एक बनाते हैं`TocInfo` ऑब्जेक्ट को चुनें और उसका शीर्षक "विषय-सूची" पर सेट करें। हम फ़ॉन्ट आकार और शैली को भी अनुकूलित करते हैं।`PageNumbersPrefix` "P" पर सेट है, जो TOC में पृष्ठ संख्या के आगे जोड़ देगा।

## चरण 5: TOC में शीर्षक जोड़ें

अब आता है मज़ेदार हिस्सा! हम दस्तावेज़ के पन्नों को देखेंगे और अपने TOC में शीर्षक जोड़ेंगे।

```csharp
for (int i = 1; i < doc.Pages.Count; i++)
{
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
    TextSegment segment2 = new TextSegment();
    heading2.TocPage = tocPage;
    heading2.Segments.Add(segment2);
    heading2.DestinationPage = doc.Pages[i + 1];
    heading2.Top = doc.Pages[i + 1].Rect.Height;
    segment2.Text = "Page " + i.ToString();
    tocPage.Paragraphs.Add(heading2);
}
```

 इस लूप में, हम एक नया लूप बनाते हैं`Heading` प्रत्येक पृष्ठ के लिए ऑब्जेक्ट। हम प्रत्येक शीर्षक के लिए गंतव्य पृष्ठ सेट करते हैं और प्रदर्शित करने के लिए पाठ निर्दिष्ट करते हैं, जो "पृष्ठ X" है जहाँ X पृष्ठ संख्या है। अंत में, हम शीर्षक को TOC पृष्ठ में जोड़ते हैं।

## चरण 6: अपडेट किए गए दस्तावेज़ को सहेजें

सभी आवश्यक शीर्षक जोड़ने के बाद, अब हमारे अद्यतन दस्तावेज़ को सहेजने का समय है।

```csharp
doc.Save(outFile);
```

यह लाइन संशोधित पीडीएफ को TOC के साथ सहेजती है। अब आप आउटपुट फ़ाइल खोल सकते हैं और अपनी अनुकूलित सामग्री तालिका देख सकते हैं!
