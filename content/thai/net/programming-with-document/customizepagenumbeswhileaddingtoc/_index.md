## การแนะนำ

ในโลกของการจัดการเอกสาร PDF ถือเป็นรูปแบบที่ได้รับความนิยมสูงสุด เนื่องจากเป็นรูปแบบที่ทุกคนเลือกใช้เพื่อแชร์และรักษาเอกสารบนแพลตฟอร์มต่างๆ แต่จะเกิดอะไรขึ้นเมื่อคุณต้องการเพิ่มคุณลักษณะต่างๆ เช่น สารบัญ (TOC) ให้กับเอกสาร PDF ของคุณ? นั่นคือจุดที่ Aspose.PDF สำหรับ .NET เข้ามามีบทบาท! ไลบรารีอันทรงพลังนี้ช่วยให้ผู้พัฒนาสามารถจัดการไฟล์ PDF ได้อย่างง่ายดาย ทำให้พวกเขาสามารถเพิ่ม แก้ไข และปรับแต่งเนื้อหาได้อย่างง่ายดาย ในบทช่วยสอนนี้ เราจะเจาะลึกถึงวิธีการปรับแต่งหมายเลขหน้าพร้อมกับเพิ่ม TOC ลงในเอกสาร PDF ของคุณโดยใช้ Aspose.PDF สำหรับ .NET ดังนั้น คว้าหมวกแห่งการเขียนโค้ดของคุณแล้วเริ่มกันเลย!

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มต้นเขียนโค้ด มีบางสิ่งที่คุณต้องมี:

1. Visual Studio: ตรวจสอบว่าคุณได้ติดตั้ง Visual Studio ไว้ในเครื่องของคุณแล้ว ซึ่งจะเป็นสภาพแวดล้อมการพัฒนาของเรา
2. Aspose.PDF สำหรับ .NET: คุณต้องดาวน์โหลดและติดตั้งไลบรารี Aspose.PDF คุณสามารถค้นหาได้[ที่นี่](https://releases.aspose.com/pdf/net/).
3. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับการเขียนโปรแกรม C# จะช่วยให้คุณเข้าใจชิ้นส่วนโค้ดได้ดีขึ้น
4. ไฟล์ PDF ตัวอย่าง: เตรียมไฟล์ PDF ตัวอย่างไว้ให้เราใช้งาน คุณสามารถสร้างไฟล์ง่ายๆ หรือดาวน์โหลดไฟล์ PDF ที่มีอยู่แล้วก็ได้

## แพ็คเกจนำเข้า

ในการเริ่มต้น เราจำเป็นต้องนำเข้าแพ็คเกจที่จำเป็น เปิดโปรเจ็กต์ Visual Studio ของคุณและเพิ่มการอ้างอิงไปยังไลบรารี Aspose.PDF คุณสามารถทำได้โดยใช้ตัวจัดการแพ็คเกจ NuGet:

1. คลิกขวาที่โครงการของคุณใน Solution Explorer
2. เลือก "จัดการแพ็คเกจ NuGet"
3. ค้นหา "Aspose.PDF" และติดตั้ง

```csharp
using Aspose.Pdf.Text;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

เมื่อคุณติดตั้งไลบรารีแล้ว คุณสามารถเริ่มเขียนโค้ดได้!

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

สิ่งแรกที่เราต้องทำคือตั้งค่าไดเร็กทอรีเอกสารของเรา นี่คือที่ที่เราจะเก็บไฟล์ PDF อินพุตและเอาท์พุต 

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string inFile = dataDir + "42824.pdf";
string outFile = dataDir + "42824_out.pdf";
```

ในสคริปท์นี้ ให้แทนที่`YOUR DOCUMENT DIRECTORY` ด้วยเส้นทางจริงที่ไฟล์ PDF ของคุณตั้งอยู่ ซึ่งจะช่วยให้เราโหลด PDF ที่มีอยู่และบันทึกเวอร์ชันที่แก้ไขแล้ว

## ขั้นตอนที่ 2: โหลดไฟล์ PDF ที่มีอยู่

ตอนนี้เราได้ตั้งค่าไดเร็กทอรีเอกสารแล้ว มาโหลดไฟล์ PDF ที่มีอยู่กัน 

```csharp
Document doc = new Document(inFile);
```

 ที่นี่เราสร้างใหม่`Document` วัตถุโดยส่งเส้นทางไฟล์อินพุต วิธีนี้ช่วยให้เราจัดการเนื้อหา PDF ได้ด้วยโปรแกรม

## ขั้นตอนที่ 3: แทรกหน้าใหม่สำหรับ TOC

ต่อไปเราต้องสร้างหน้าใหม่ใน PDF ซึ่งจะมีสารบัญอยู่ในนั้น

```csharp
Aspose.Pdf.Page tocPage = doc.Pages.Insert(1);
```

บรรทัดนี้จะแทรกหน้าใหม่ที่จุดเริ่มต้นของเอกสาร สารบัญจะแสดงบนหน้านี้

## ขั้นตอนที่ 4: สร้างข้อมูล TOC

ตอนนี้มาสร้างอ็อบเจ็กต์เพื่อแสดงข้อมูล TOC กัน

```csharp
TocInfo tocInfo = new TocInfo();
TextFragment title = new TextFragment("Table Of Contents");
title.TextState.FontSize = 20;
title.TextState.FontStyle = FontStyles.Bold;
tocInfo.Title = title;
tocInfo.PageNumbersPrefix = "P";
tocPage.TocInfo = tocInfo;
```

 ในขั้นตอนนี้เราจะสร้าง`TocInfo` วัตถุและตั้งชื่อเป็น "สารบัญ" เรายังปรับขนาดและรูปแบบของตัวอักษรได้อีกด้วย`PageNumbersPrefix` ตั้งค่าเป็น "P" ซึ่งจะใส่หมายเลขหน้าไว้ข้างหน้าใน TOC

## ขั้นตอนที่ 5: เพิ่มหัวข้อใน TOC

ตอนนี้มาถึงส่วนสนุก ๆ แล้ว! เราจะวนซ้ำหน้าเอกสารและเพิ่มหัวข้อลงใน TOC ของเรา

```csharp
for (int i = 1; i < doc.Pages.Count; i++)
{
    Aspose.Pdf.Heading heading2 = new Aspose.Pdf.Heading(1);
    TextSegment segment2 = new TextSegment();
    heading2.TocPage = tocPage;
    heading2.Segments.Add(segment2);
    heading2.DestinationPage = doc.Pages[i + 1];
    heading2.Top = doc.Pages[i + 1].Rect.Height;
    segment2.Text = "Page " + i.ToString();
    tocPage.Paragraphs.Add(heading2);
}
```

 ในลูปนี้เราสร้างใหม่`Heading` วัตถุสำหรับแต่ละหน้า เราตั้งค่าหน้าปลายทางสำหรับแต่ละหัวข้อและระบุข้อความที่จะแสดง ซึ่งก็คือ "หน้า X" โดยที่ X คือหมายเลขหน้า สุดท้าย เราเพิ่มหัวข้อลงในหน้า TOC

## ขั้นตอนที่ 6: บันทึกเอกสารที่อัปเดต

หลังจากเพิ่มหัวเรื่องที่จำเป็นทั้งหมดแล้ว ก็ถึงเวลาบันทึกเอกสารที่อัปเดตของเรา

```csharp
doc.Save(outFile);
```

บรรทัดนี้จะบันทึก PDF ที่แก้ไขแล้วพร้อม TOC ในตัว ตอนนี้คุณสามารถเปิดไฟล์เอาต์พุตและดูสารบัญที่คุณปรับแต่งแล้ว!
