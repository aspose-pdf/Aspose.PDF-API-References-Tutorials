PDF फ़ाइल में स्थानीय हाइपरलिंक बनाने से आप क्लिक करने योग्य लिंक बना सकते हैं जो उपयोगकर्ताओं को उसी PDF दस्तावेज़ में अन्य पृष्ठों पर ले जाते हैं। .NET के लिए Aspose.PDF के साथ, आप निम्न स्रोत कोड का पालन करके आसानी से ऐसे लिंक बना सकते हैं:

## चरण 1: आवश्यक लाइब्रेरीज़ आयात करें

आरंभ करने से पहले, आपको अपने C# प्रोजेक्ट के लिए आवश्यक लाइब्रेरीज़ आयात करनी होंगी। यहाँ आवश्यक आयात निर्देश दिया गया है:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using Aspose.Pdf.InteractiveFeatures;
```

## चरण 2: दस्तावेज़ फ़ोल्डर का पथ सेट करें

 इस चरण में, आपको उस फ़ोल्डर का पथ निर्दिष्ट करना होगा जहाँ आप परिणामी PDF फ़ाइल को सहेजना चाहते हैं।`"YOUR DOCUMENT DIRECTORY"` अपने दस्तावेज़ फ़ोल्डर के वास्तविक पथ के साथ निम्नलिखित कोड में:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## चरण 3: दस्तावेज़ का एक उदाहरण बनाएँ

 हम इसका एक उदाहरण बनाएंगे`Document` क्लास का उपयोग करके अपने PDF दस्तावेज़ का प्रतिनिधित्व करें। यहाँ संबंधित कोड है:

```csharp
Document doc = new Document();
```

## चरण 4: हाइपरलिंक के साथ पृष्ठ और पाठ जोड़ें

इस चरण में, हम अपने PDF दस्तावेज़ में एक पृष्ठ जोड़ने जा रहे हैं और स्थानीय हाइपरलिंक युक्त कुछ पाठ जोड़ेंगे। हम प्रत्येक लिंक के लिए लक्ष्य पृष्ठ परिभाषित करेंगे। यहाँ संबंधित कोड है:

```csharp
Page page = doc.Pages.Add();

TextFragment text = new TextFragment("Link to page 7");
LocalHyperlink link = new LocalHyperlink();
link.TargetPageNumber = 7;
text. Hyperlink = link;
page.Paragraphs.Add(text);

text = new TextFragment("Link to page 1");
text. IsInNewPage = true;
link = new LocalHyperlink();
link.TargetPageNumber = 1;
text. Hyperlink = link;
page.Paragraphs.Add(text);
```

## चरण 5: अपडेट किए गए दस्तावेज़ को सहेजें

अब आइए अपडेट की गई पीडीएफ फाइल को सेव करें`Save` की विधि`doc` ऑब्जेक्ट. यहाँ संबंधित कोड है:

```csharp
dataDir = dataDir + "CreateLocalHyperlink_out.pdf";
doc.Save(dataDir);
```

### .NET के लिए Aspose.PDF का उपयोग करके स्थानीय हाइपरलिंक बनाने के लिए नमूना स्रोत कोड 
```csharp
// दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// दस्तावेज़ उदाहरण बनाएँ
Document doc = new Document();
// पीडीएफ फाइल के पेज टू पेज संग्रह को जोड़ें
Page page = doc.Pages.Add();
// टेक्स्ट फ़्रैगमेंट इंस्टेंस बनाएँ
Aspose.Pdf.Text.TextFragment text = new Aspose.Pdf.Text.TextFragment("link page number test to page 7");
// स्थानीय हाइपरलिंक इंस्टेंस बनाएँ
Aspose.Pdf.LocalHyperlink link = new Aspose.Pdf.LocalHyperlink();
// लिंक इंस्टेंस के लिए लक्ष्य पृष्ठ सेट करें
link.TargetPageNumber = 7;
// टेक्स्टफ़्रेगमेंट हाइपरलिंक सेट करें
text.Hyperlink = link;
// पेज के पैराग्राफ़ संग्रह में पाठ जोड़ें
page.Paragraphs.Add(text);
// नया TextFragment इंस्टेंस बनाएँ
text = new TextFragment("link page number test to page 1");
// TextFragment को नए पृष्ठ पर जोड़ा जाना चाहिए
text.IsInNewPage = true;
// एक अन्य स्थानीय हाइपरलिंक इंस्टैंस बनाएँ
link = new LocalHyperlink();
// दूसरे हाइपरलिंक के लिए लक्ष्य पृष्ठ सेट करें
link.TargetPageNumber = 1;
// दूसरे TextFragment के लिए लिंक सेट करें
text.Hyperlink = link;
// पेज ऑब्जेक्ट के पैराग्राफ़ संग्रह में टेक्स्ट जोड़ें
page.Paragraphs.Add(text);    
dataDir = dataDir + "CreateLocalHyperlink_out.pdf";
// अद्यतन दस्तावेज़ सहेजें
doc.Save(dataDir);
Console.WriteLine("\nLocal hyperlink created successfully.\nFile saved at " + dataDir);            
```

## निष्कर्ष

बधाई हो! अब आपके पास .NET के लिए Aspose.PDF का उपयोग करके PDF में स्थानीय हाइपरलिंक बनाने के लिए चरण-दर-चरण मार्गदर्शिका है। आप इस कोड का उपयोग क्लिक करने योग्य लिंक बनाने के लिए कर सकते हैं जो उपयोगकर्ताओं को उसी दस्तावेज़ में अन्य पृष्ठों पर ले जाते हैं।

उन्नत हाइपरलिंकिंग सुविधाओं के बारे में अधिक जानकारी के लिए आधिकारिक Aspose.PDF दस्तावेज़ अवश्य देखें।

### पीडीएफ फाइल में स्थानीय हाइपरलिंक बनाने के लिए अक्सर पूछे जाने वाले प्रश्न

#### प्रश्न: पीडीएफ फाइल में स्थानीय हाइपरलिंक क्या हैं?

उत्तर: पीडीएफ फाइल में स्थानीय हाइपरलिंक क्लिक करने योग्य लिंक होते हैं जो उपयोगकर्ताओं को उसी दस्तावेज़ के भीतर विभिन्न पृष्ठों पर ले जाते हैं। ये लिंक नेविगेशन को बेहतर बनाते हैं और पाठकों को प्रासंगिक अनुभागों तक जल्दी से पहुँचने की अनुमति देते हैं।

#### प्रश्न: स्थानीय हाइपरलिंक मेरे पीडीएफ दस्तावेज़ को कैसे लाभ पहुंचा सकते हैं?

उत्तर: स्थानीय हाइपरलिंक एक ही पीडीएफ दस्तावेज़ में संबंधित सामग्री को जोड़ने का एक कुशल तरीका प्रदान करते हैं। वे पाठकों को पूरे दस्तावेज़ को स्क्रॉल किए बिना विशिष्ट अनुभागों पर जल्दी से जाने में सक्षम बनाकर उपयोगकर्ता अनुभव को बेहतर बनाते हैं।

#### प्रश्न: .NET के लिए Aspose.PDF स्थानीय हाइपरलिंक के निर्माण का समर्थन कैसे करता है?
उत्तर: .NET के लिए Aspose.PDF स्थानीय हाइपरलिंक बनाने के लिए व्यापक सहायता प्रदान करता है। इस गाइड में दिए गए चरण-दर-चरण ट्यूटोरियल में दिखाया गया है कि C# का उपयोग करके अपने PDF दस्तावेज़ में स्थानीय हाइपरलिंक कैसे जोड़ें।

#### प्रश्न: क्या मैं स्थानीय हाइपरलिंक्स के स्वरूप को अनुकूलित कर सकता हूँ?

उत्तर: हां, आप स्थानीय हाइपरलिंक्स के स्वरूप को अनुकूलित कर सकते हैं, जिसमें पाठ का रंग और शैली भी शामिल है, ताकि यह सुनिश्चित हो सके कि वे आपके दस्तावेज़ के डिज़ाइन से मेल खाते हों और एक सुसंगत दृश्य अनुभव प्रदान करें।

#### प्रश्न: क्या एक ही पीडीएफ पृष्ठ में अनेक स्थानीय हाइपरलिंक बनाना संभव है?

उत्तर: बिल्कुल! आप एक ही पीडीएफ पेज में कई स्थानीय हाइपरलिंक बना सकते हैं, जिससे पाठकों को आवश्यकतानुसार विभिन्न अनुभागों या पेजों पर जाने की सुविधा मिलती है। प्रत्येक स्थानीय हाइपरलिंक को उसके संबंधित लक्ष्य के अनुसार तैयार किया जा सकता है।

#### प्रश्न: क्या मैं स्थानीय हाइपरलिंक का उपयोग करके किसी पृष्ठ के विशिष्ट अनुभागों से लिंक कर सकता हूँ?

उत्तर: जबकि स्थानीय हाइपरलिंक आमतौर पर पूरे पृष्ठों पर नेविगेट करते हैं, आप लक्षित लिंकिंग प्राप्त करने के लिए अपने पीडीएफ दस्तावेज़ के भीतर एंकर या बुकमार्क बना सकते हैं। .NET के लिए Aspose.PDF विभिन्न हाइपरलिंकिंग विकल्पों का समर्थन करता है।

#### प्रश्न: मैं कैसे सत्यापित कर सकता हूं कि मेरे स्थानीय हाइपरलिंक सही ढंग से काम कर रहे हैं?

उत्तर: दिए गए ट्यूटोरियल और सैंपल कोड का पालन करके, आप आत्मविश्वास से कार्यात्मक स्थानीय हाइपरलिंक बना सकते हैं। आप जेनरेट किए गए पीडीएफ दस्तावेज़ को खोलकर और हाइपरलिंक किए गए टेक्स्ट पर क्लिक करके लिंक का परीक्षण कर सकते हैं।

#### प्रश्न: स्थानीय हाइपरलिंक का उपयोग करते समय क्या कोई सीमाएं हैं?

उत्तर: स्थानीय हाइपरलिंक दस्तावेज़ नेविगेशन को बेहतर बनाने का एक प्रभावी तरीका है, लेकिन यह सुनिश्चित करना महत्वपूर्ण है कि दस्तावेज़ की संरचना स्पष्ट और सहज बनी रहे। उचित रूप से लेबल किए गए हाइपरलिंक और एंकर सकारात्मक उपयोगकर्ता अनुभव में योगदान करते हैं।

#### प्रश्न: क्या मैं तालिकाओं या छवियों के भीतर स्थानीय हाइपरलिंक बना सकता हूँ?

उत्तर: हाँ, आप अपने PDF दस्तावेज़ के विभिन्न तत्वों, जैसे कि टेबल, छवियाँ और टेक्स्ट के भीतर स्थानीय हाइपरलिंक बना सकते हैं। .NET के लिए Aspose.PDF विभिन्न प्रकार की सामग्री में हाइपरलिंक जोड़ने में लचीलापन प्रदान करता है।
