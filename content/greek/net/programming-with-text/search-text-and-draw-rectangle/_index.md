## Εισαγωγή χώρων ονομάτων

Στην κορυφή του αρχείου του προγράμματός σας, θα θέλετε να εισαγάγετε τους σχετικούς χώρους ονομάτων από τη βιβλιοθήκη Aspose:

```csharp
using System.IO;
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System;
using Aspose.Pdf.Facades;
```

Αυτό διευκολύνει την πρόσβαση σε κλάσεις και μεθόδους στη βιβλιοθήκη Aspose.PDF για τις εργασίες σας.


Τώρα που έχετε ρυθμίσει τα πάντα, ας αναλύσουμε τη διαδικασία αναζήτησης κειμένου σε PDF και σχεδίασης ενός ορθογωνίου γύρω από αυτό σε διαχειρίσιμα βήματα.

## Βήμα 1: Ορίστε τη διαδρομή για το έγγραφό σας

 Πρώτα, ορίστε τη διαδρομή προς το αρχείο PDF. Φροντίστε να αντικαταστήσετε`YOUR DOCUMENT DIRECTORY` με την πραγματική διαδρομή όπου σας`SearchAndGetTextFromAll.pdf` αποθηκεύεται.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Ανοίξτε το έγγραφο PDF

 Στη συνέχεια, δημιουργήστε μια παρουσία του`Document` τάξη για να φορτώσετε το PDF σας:

```csharp
Document document = new Document(dataDir + "SearchAndGetTextFromAll.pdf");
```

Αυτή η γραμμή κώδικα ανοίγει το καθορισμένο αρχείο PDF, επιτρέποντάς σας να το χειριστείτε περαιτέρω.

## Βήμα 3: Δημιουργήστε έναν απορροφητή κειμένου

 Τώρα, θα χρειαστείτε έναν τρόπο για να αναζητήσετε κείμενο σε αυτό το έγγραφο. Για αυτό, χρησιμοποιούμε το`TextFragmentAbsorber`:

```csharp
TextFragmentAbsorber textAbsorber = new TextFragmentAbsorber(@"[\S]+");
```

 Η κανονική έκφραση`@"[\S]+"` έχει σχεδιαστεί για να ταιριάζει με οποιαδήποτε συμβολοσειρά χωρίς κενά στο PDF. 

## Βήμα 4: Διαμορφώστε τις επιλογές αναζήτησης κειμένου

Στη συνέχεια, θα πρέπει να ρυθμίσετε τις επιλογές αναζήτησης κειμένου:

```csharp
TextSearchOptions textSearchOptions = new TextSearchOptions(true);
textAbsorber.TextSearchOptions = textSearchOptions;
```

 Εδώ, το`true` παράμετρος σημαίνει ότι η αναζήτηση θα γίνεται με διάκριση πεζών-κεφαλαίων. Μπορείτε να το ρυθμίσετε σε`false` εάν θέλετε μια αναζήτηση χωρίς διάκριση πεζών-κεφαλαίων.

## Βήμα 5: Αποδεχτείτε τον απορροφητή κειμένου στο έγγραφο

 Με το δικό σου`TextFragmentAbsorber` και έτοιμες οι επιλογές αναζήτησης, ήρθε η ώρα να απορροφήσετε κείμενο από το έγγραφο:

```csharp
document.Pages.Accept(textAbsorber);
```

Αυτή η μέθοδος εξετάζει κάθε σελίδα στο PDF σας για να βρει τμήματα κειμένου που ταιριάζουν με το καθορισμένο μοτίβο.

## Βήμα 6: Δημιουργήστε ένα PdfContentEditor

 Για να σχεδιάσετε σχήματα στο έγγραφο, θα χρειαστείτε το`PdfContentEditor`:

```csharp
var editor = new PdfContentEditor(document);
```

Αυτό το πρόγραμμα επεξεργασίας σάς επιτρέπει να χειρίζεστε και να επεξεργάζεστε εύκολα το περιεχόμενο PDF.

## Βήμα 7: Κάντε βρόχο μέσω των τεμαχίων κειμένου που βρέθηκαν

Τώρα, θα θελήσετε να κάνετε κύκλο στα τεμάχια κειμένου που βρέθηκαν για να σχεδιάσετε ορθογώνια γύρω τους:

```csharp
foreach (TextFragment textFragment in textAbsorber.TextFragments)
{
    foreach (TextSegment textSegment in textFragment.Segments)
    {
        DrawBox(editor, textFragment.Page.Number, textSegment, System.Drawing.Color.Red);
    }
}
```

 Αυτός ο βρόχος επαναλαμβάνεται πάνω από κάθε τμήμα κειμένου και τα τμήματα του, καλώντας a`DrawBox` μέθοδος σχεδίασης ορθογωνίου.

## Βήμα 8: Καθορίστε τη μέθοδο DrawBox

 Πρέπει να ορίσετε το`DrawBox` μέθοδο, η οποία θα χειριστεί τη λογική σχεδίασης του ορθογωνίου. Εδώ είναι μια απλή υλοποίηση:

```csharp
private static void DrawBox(PdfContentEditor editor, int pageNumber, TextSegment textSegment, System.Drawing.Color color)
{
    // Υπολογίστε τις διαστάσεις του ορθογωνίου με βάση το τμήμα κειμένου
    float x = textSegment.Rectangle.LLX;
    float y = textSegment.Rectangle.LLY;
    float width = textSegment.Rectangle.Width;
    float height = textSegment.Rectangle.Height;

    // Σχεδιάστε ένα ορθογώνιο χρησιμοποιώντας τις υπολογισμένες τιμές
    editor.DrawRectangle(pageNumber, x, y, width, height, color, 1);
}
```

Αυτή η μέθοδος καθορίζει τη θέση και το μέγεθος του ορθογωνίου με βάση το οριοθετημένο ορθογώνιο του τμήματος και χρησιμοποιεί τον επεξεργαστή για να το σχεδιάσει.

## Βήμα 9: Αποθηκεύστε το τροποποιημένο έγγραφο

Αφού σχεδιάσετε τα ορθογώνια γύρω από το κείμενο που βρέθηκε, μπορείτε να αποθηκεύσετε το τροποποιημένο έγγραφο:

```csharp
dataDir = dataDir + "SearchTextAndDrawRectangle_out.pdf";
document.Save(dataDir);
```

Βεβαιωθείτε ότι το νέο σας αρχείο έχει αποθηκευτεί με ένα ξεχωριστό όνομα για να αποφύγετε την αντικατάσταση του αρχικού σας εγγράφου.

## Βήμα 10: Μήνυμα επιβεβαίωσης

Τέλος, εκτυπώστε ένα μήνυμα επιβεβαίωσης στην κονσόλα για να σας ενημερώσει ότι η λειτουργία ήταν επιτυχής:
