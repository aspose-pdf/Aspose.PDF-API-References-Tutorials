## Βήμα 4: Πλοήγηση στις σελίδες του εγγράφου

Τώρα που φορτώθηκε το αρχείο PDF, πρέπει να περάσετε από όλες τις σελίδες του εγγράφου. Αυτό θα σας επιτρέψει να αποκτήσετε

ir τους σχολιασμούς υπερ-συνδέσμων που υπάρχουν σε κάθε σελίδα. Χρησιμοποιήστε τον ακόλουθο κώδικα για να επαναλάβετε τις σελίδες του εγγράφου:

```csharp
foreach(Aspose.Pdf.Page page in document.Pages)
{
     // Λάβετε τους σχολιασμούς συνδέσμων μιας συγκεκριμένης σελίδας
     AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
     page. Accept(selector);
     // Δημιουργήστε μια λίστα για να αποθηκεύσετε όλους τους συνδέσμους
     IList<Annotation> list = selector. Selected;
     // Περιηγηθείτε σε κάθε στοιχείο της λίστας
     foreach(LinkAnnotation a in list)
     {
         // Εκτύπωση διεύθυνσης URL προορισμού
         Console.WriteLine("\nDestination: " + (a.Action as Aspose.Pdf.Annotations.GoToURIAction).URI + "\n");
     }
}
```

Αυτός ο κώδικας περιστρέφεται σε κάθε σελίδα του εγγράφου και επιλέγει τους σχολιασμούς υπερ-συνδέσμων που υπάρχουν σε κάθε σελίδα. Στη συνέχεια, αποθηκεύει αυτούς τους σχολιασμούς σε μια λίστα και εκτυπώνει τη διεύθυνση URL προορισμού για κάθε σύνδεσμο.

### Δείγμα πηγαίου κώδικα για Λήψη προορισμών υπερσύνδεσης χρησιμοποιώντας το Aspose.PDF για .NET 
```csharp
try
{
	// Η διαδρομή προς τον κατάλογο εγγράφων.
	string dataDir = "YOUR DOCUMENT DIRECTORY";
	// Φορτώστε το αρχείο PDF
	Document document = new Document(dataDir + "input.pdf");
	// Διασχίστε όλη τη σελίδα του PDF
	foreach (Aspose.Pdf.Page page in document.Pages)
	{
		// Λάβετε τους σχολιασμούς του συνδέσμου από μια συγκεκριμένη σελίδα
		AnnotationSelector selector = new AnnotationSelector(new Aspose.Pdf.Annotations.LinkAnnotation(page, Aspose.Pdf.Rectangle.Trivial));
		page.Accept(selector);
		// Δημιουργήστε λίστα με όλους τους συνδέσμους
		IList<Annotation> list = selector.Selected;
		// Επανάληψη μέσω της εσωτερικής λίστας ενός στοιχείου
		foreach (LinkAnnotation a in list)
		{
			// Εκτυπώστε τη διεύθυνση URL προορισμού
			Console.WriteLine("\nDestination: " + (a.Action as Aspose.Pdf.Annotations.GoToURIAction).URI + "\n");
		}
	}
}
catch (Exception ex)
{
	Console.WriteLine(ex.Message);
}
```
