في هذا البرنامج التعليمي، سنستعرض خطوة بخطوة كيفية تحويل صورة CGM كبيرة إلى ملف PDF باستخدام مكتبة Aspose.PDF في .NET. يوضح كود المصدر C# المقدم عملية تحويل ملف CGM إلى تنسيق PDF، وتحديد حجم الصفحة، وحفظ ملف الإخراج. سنشرح كل خطوة بالتفصيل لمساعدتك على فهم العملية بشكل كامل.

## متطلبات
قبل أن نبدأ، تأكد من أن لديك ما يلي:
- المعرفة الأساسية للغة البرمجة C#.
- تم تثبيت مكتبة Aspose.PDF لـ .NET في مشروعك.
- ملف صورة CGM الذي تريد تحويله إلى PDF.

## الخطوة 1: إعداد المشروع
1. قم بإنشاء مشروع C# جديد في بيئة التطوير المفضلة لديك.
2. أضف مرجعًا إلى مكتبة Aspose.PDF في مشروعك.

## الخطوة 2: استيراد المساحات الأساسية الضرورية
في بداية ملف C#، قم باستيراد مساحات الأسماء المطلوبة للوصول إلى فئات وطرق Aspose.PDF. فيما يلي مثال:
```csharp
using System;
using Aspose.Pdf;
using System.Drawing;
```

## الخطوة 3: تهيئة المتغيرات والمسارات
قبل إجراء التحويل، نحتاج إلى إعداد المتغيرات والمسارات اللازمة.
```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
string inputFile = dataDir + "corvette.cgm";
dataDir = dataDir + "LargeCGMImageToPDF_out.pdf";
```
 تأكد من الاستبدال`"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي إلى دليل المستند الخاص بك.

## الخطوة 4: تحويل CGM إلى PDF
لتحويل صورة CGM إلى تنسيق PDF، اتبع الخطوات التالية:
1.  إنشاء مثيل لـ`CgmImportOptions` فصل.
```csharp
CgmImportOptions options = new CgmImportOptions();
```
2. حدد أبعاد حجم الصفحة المستوردة.
```csharp
options. PageSize = new SizeF(1000, 1000);
```
هنا، قمنا بتعيين حجم الصفحة إلى 1000×1000 بكسل. يمكنك تعديل الأبعاد وفقًا لمتطلباتك.
 3. استخدم`PdfProducer.Produce` طريقة تحويل ملف CGM إلى صيغة PDF.
```csharp
PdfProducer.Produce(inputFile, ImportFormat.Cgm, dataDir);
```
4. عرض رسالة نجاح مع المسار الذي تم حفظ ملف PDF فيه.
```csharp
Console.WriteLine("\nLarge CGM file converted to PDF successfully.\nFile saved at " + dataDir);
```

### عينة من كود المصدر لـ Large CGMImage إلى PDF باستخدام Aspose.PDF لـ .NET 
```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string inputFile = dataDir + "corvette.cgm";
dataDir = dataDir + "LargeCGMImageToPDF_out.pdf";
//إنشاء مثيل لـ CgmImportOptions
CgmImportOptions options = new CgmImportOptions();
// تحديد أبعاد استيراد حجم الصفحة
options.PageSize = new SizeF(1000, 1000);
// حفظ CGM بصيغة PDF
PdfProducer.Produce(inputFile, ImportFormat.Cgm, dataDir);
Console.WriteLine("\nLarge CGM file converted to pdf successfully.\nFile saved at " + dataDir); 
```
