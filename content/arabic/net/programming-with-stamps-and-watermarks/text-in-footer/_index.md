في هذا البرنامج التعليمي، سنتعلم كيفية إضافة نص في تذييل ملف PDF باستخدام Aspose.PDF لـ .NET. اتبع الخطوات التالية:

## الخطوة 1: إعداد المشروع

تأكد من أنك قمت بتثبيت Aspose.PDF لـ .NET وقمت بإنشاء مشروع C#.

## الخطوة 2: استيراد المساحات الاسمية

أضف المساحات الأسماء التالية إلى ملف المصدر C# الخاص بك:

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
```

## الخطوة 3: فتح المستند

افتح مستند PDF الموجود باستخدام المسار المقدم:

```csharp
string dataDir = "YOUR DOCUMENTS DIRECTORY";
Document pdfDocument = new Document(dataDir + "TextinFooter.pdf");
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي إلى دليل المستندات الخاص بك.

## الخطوة 4: إنشاء نص التذييل

قم بإنشاء طابع نصي جديد بالنص الذي تريد إضافته في التذييل:

```csharp
TextStamp textStamp = new TextStamp("footer text");
```

يمكنك تخصيص النص عن طريق تغيير خصائصه مثل الهامش السفلي والمحاذاة الأفقية والمحاذاة الرأسية.

## الخطوة 5: إضافة نص التذييل إلى جميع الصفحات

قم بالانتقال إلى جميع صفحات مستند PDF وأضف ختم النص في التذييل:

```csharp
foreach(Page page in pdfDocument.Pages)
{
     page.AddStamp(textStamp);
}
```

## الخطوة 6: حفظ مستند PDF

بمجرد إضافة نص التذييل إلى جميع الصفحات، احفظ مستند PDF المحدث:

```csharp
dataDir = dataDir + "TextinFooter_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nText in footer added successfully.\nFile saved at: " + dataDir);
```

تأكد من استبدال "دليل المستندات الخاص بك" بالمسار الفعلي للدليل الذي تريد حفظ مستند PDF فيه.

### عينة من كود المصدر لـ Textin Footer باستخدام Aspose.PDF لـ .NET 
```csharp

// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// فتح المستند
Document pdfDocument = new Document(dataDir+ "TextinFooter.pdf");

// إنشاء تذييل
TextStamp textStamp = new TextStamp("Footer Text");

// تعيين خصائص الطوابع
textStamp.BottomMargin = 10;
textStamp.HorizontalAlignment = HorizontalAlignment.Center;
textStamp.VerticalAlignment = VerticalAlignment.Bottom;

// إضافة تذييل لجميع الصفحات
foreach (Page page in pdfDocument.Pages)
{
	page.AddStamp(textStamp);
}
dataDir = dataDir + "TextinFooter_out.pdf";

// حفظ ملف PDF المحدث
pdfDocument.Save(dataDir);
Console.WriteLine("\nText in footer added successfully.\nFile saved at " + dataDir);

```

## خاتمة

تهانينا! لقد تعلمت كيفية إضافة نص في تذييل مستند PDF باستخدام Aspose.PDF لـ .NET. يمكنك الآن تخصيص تذييلاتك عن طريق إضافة نص إضافي إلى مستندات PDF الخاصة بك.

### الأسئلة الشائعة حول النص الموجود في تذييل ملف PDF

#### س: ما هو الغرض من إضافة نص في تذييل مستند PDF؟

أ: إضافة نص في تذييل مستند PDF يسمح لك بتضمين معلومات مهمة، مثل إشعارات حقوق النشر، أو أرقام الصفحات، أو إصدار المستند، أو أي نص آخر تريد ظهوره بشكل متسق في أسفل كل صفحة.

#### س: كيف يقوم كود المصدر C# المقدم بإضافة نص في تذييل مستند PDF؟

ج: يوضح الكود عملية فتح مستند PDF موجود، وإنشاء ختم نصي بنص التذييل المطلوب، وتخصيص خصائص النص، وإضافة ختم النص إلى جميع الصفحات، وأخيرًا حفظ مستند PDF المحدث بنص التذييل المضاف.

#### س: هل يمكنني تعديل مظهر نص التذييل، مثل الخط والحجم واللون والمحاذاة؟

 ج: نعم، يمكنك تخصيص مظهر نص التذييل عن طريق تعديل خصائص`TextStamp` يتضمن مثال التعليمات البرمجية ضبط خصائص مثل الهامش السفلي والمحاذاة الأفقية والمحاذاة الرأسية. يمكنك أيضًا ضبط الخط والحجم واللون وغير ذلك من الخصائص المتعلقة بالنص.

#### س: هل من الممكن إضافة نص مختلف إلى تذييل كل صفحة؟

 ج: نعم، يمكنك إضافة نص مختلف إلى تذييل كل صفحة عن طريق إنشاء نص منفصل`TextStamp` الكائنات ذات محتوى نصي أو خصائص مختلفة ثم إضافتها إلى صفحات محددة حسب الحاجة.

#### س: كيف يمكنني التأكد من ظهور نص التذييل بشكل متسق في كل صفحة من مستند PDF؟

ج: من خلال استخدام حلقة تتكرر عبر جميع صفحات مستند PDF وإضافة نفس طابع النص إلى كل صفحة، يمكنك التأكد من ظهور نص التذييل بشكل متسق في كل صفحة.

#### س: هل يمكنني إضافة أسطر متعددة من النص أو تنسيق نص التذييل باستخدام فواصل الأسطر؟

 ج: نعم، يمكنك إضافة عدة أسطر من النص إلى التذييل عن طريق تضمين فواصل الأسطر في سلسلة النص. على سبيل المثال، يمكنك استخدام تسلسل الإفلات`\n` للإشارة إلى كسر السطر في النص.

#### س: ماذا يحدث إذا أردت إضافة محتوى مختلف إلى رأس وتذييل نفس مستند PDF؟

ج: لإضافة محتوى مختلف إلى أقسام الرأس والتذييل، عليك اتباع خطوات مماثلة لكلا القسمين. يوضح الكود إضافة نص إلى التذييل؛ ويمكنك استخدام نهج مماثل لإضافة نص إلى الرأس.

#### س: هل من الممكن إضافة صور أو عناصر أخرى إلى جانب نص التذييل باستخدام هذا النهج؟

ج: على الرغم من أن الكود المقدم يوضح على وجه التحديد إضافة نص إلى التذييل، يمكنك توسيع النهج لإضافة عناصر أخرى مثل الصور أو الخطوط أو الأشكال أو أي محتوى آخر إلى قسم التذييل باستخدام مكتبة Aspose.PDF.
