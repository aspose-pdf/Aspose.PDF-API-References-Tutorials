Το Aspose.PDF για .NET είναι ένα ισχυρό και εύχρηστο API χειρισμού εγγράφων PDF που σας επιτρέπει να εργάζεστε με αρχεία PDF μέσω προγραμματισμού στις εφαρμογές σας .NET. Μία από τις δυνατότητες που παρέχει το Aspose.PDF για .NET είναι η δυνατότητα ρύθμισης μορφοποίησης σχολιασμού ελεύθερου κειμένου σε έγγραφα PDF. Σε αυτό το άρθρο, θα σας καθοδηγήσουμε στη διαδικασία βήμα προς βήμα ρύθμισης μορφοποίησης σχολιασμού ελεύθερου κειμένου χρησιμοποιώντας το Aspose.PDF για .NET.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

- Microsoft Visual Studio 2010 ή νεότερη έκδοση
- Aspose.PDF για .NET
- Βασικές γνώσεις C#



## Βήμα 1: Δημιουργήστε μια νέα εφαρμογή κονσόλας C#

Αρχικά, δημιουργήστε μια νέα εφαρμογή κονσόλας C# στο Microsoft Visual Studio. Για να δημιουργήσετε μια νέα εφαρμογή κονσόλας, επιλέξτε «Αρχείο» > «Νέο» > «Έργο» > «Visual C#» > «Εφαρμογή Κονσόλας» από το κύριο μενού.

## Βήμα 2: Προσθήκη αναφοράς στο Aspose.PDF για .NET

Στη συνέχεια, προσθέστε μια αναφορά στο Aspose.PDF για .NET στο έργο σας. Για να το κάνετε αυτό, κάντε δεξί κλικ στο έργο σας στο παράθυρο "Εξερεύνηση λύσεων", επιλέξτε "Προσθήκη" > "Αναφορά" και, στη συνέχεια, περιηγηθείτε στη θέση όπου αποθηκεύσατε το αρχείο Aspose.PDF για .NET DLL. Επιλέξτε το αρχείο DLL και κάντε κλικ στο "OK" για να προσθέσετε την αναφορά στο έργο σας.

## Βήμα 3: Ρυθμίστε το περιβάλλον

Αφού προσθέσετε την αναφορά στο Aspose.PDF για .NET, πρέπει να ρυθμίσετε το περιβάλλον. Για να το κάνετε αυτό, δημιουργήστε μια νέα μεταβλητή συμβολοσειράς που ονομάζεται "dataDir" και ορίστε τη στη διαδρομή του καταλόγου όπου βρίσκεται το έγγραφο PDF σας. Αντικαταστήστε τον "ΚΑΤΑΛΟΓΟ ΕΓΓΡΑΦΩΝ ΣΑΣ" στον παρακάτω κωδικό με την πραγματική διαδρομή του καταλόγου εγγράφων σας:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 4: Ανοίξτε το έγγραφο PDF

Αφού ρυθμίσετε το περιβάλλον, μπορείτε να ανοίξετε το έγγραφο PDF χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "SetFreeTextAnnotationFormatting.pdf");
```

Αντικαταστήστε το "SetFreeTextAnnotationFormatting.pdf" με το πραγματικό όνομα του εγγράφου PDF σας.

## Βήμα 5: Ρυθμίστε την προεπιλεγμένη εμφάνιση

Για να ρυθμίσετε την προεπιλεγμένη εμφάνιση του σχολιασμού ελεύθερου κειμένου, πρέπει να δημιουργήσετε το αντικείμενο DefaultAppearance με την επιθυμητή γραμματοσειρά, μέγεθος γραμματοσειράς και χρώμα. Σε αυτό το σεμινάριο, ορίζουμε τη γραμματοσειρά σε "Arial", το μέγεθος γραμματοσειράς σε 28 και το χρώμα σε κόκκινο.

```csharp
// Instantiate DefaultAppearance αντικείμενο
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 28, System.Drawing.Color.Red);
```

## Βήμα 6: Δημιουργήστε έναν σχολιασμό ελεύθερου κειμένου

Τώρα που έχετε ρυθμίσει την προεπιλεγμένη εμφάνιση, μπορείτε να δημιουργήσετε έναν σχολιασμό ελεύθερου κειμένου χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
// Δημιουργία σχολιασμού
FreeTextAnnotation freetext = new FreeTextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600), default_appearance);
```

Ο παραπάνω κώδικας δημιουργεί έναν νέο σχολιασμό ελεύθερου κειμένου στη δεύτερη σελίδα του εγγράφου PDF. Ο σχολιασμός θα τοποθετηθεί στο (200, 400) και θα έχει πλάτος 400 και ύψος 600.

## Βήμα 7: Καθορίστε τα περιεχόμενα του σχολιασμού

Αφού δημιουργήσετε τον σχολιασμό ελεύθερου κειμένου, μπορείτε να καθορίσετε τα περιεχόμενα του σχολιασμού χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
// Καθορίστε τα περιεχόμενα του σχολιασμού
freetext.Contents = "Free Text
```

### Παράδειγμα πηγαίου κώδικα για Ορισμός μορφοποίησης σχολίων ελεύθερου κειμένου χρησιμοποιώντας Aspose.PDF για .NET
```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";

// Άνοιγμα εγγράφου
Document pdfDocument = new Document(dataDir + "SetFreeTextAnnotationFormatting.pdf");

// Instantiate DefaultAppearance αντικείμενο
DefaultAppearance default_appearance = new DefaultAppearance("Arial", 28, System.Drawing.Color.Red);
// Δημιουργία σχολιασμού
FreeTextAnnotation freetext = new FreeTextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600), default_appearance);
// Καθορίστε τα περιεχόμενα του σχολιασμού
freetext.Contents = "Free Text";
// Προσθήκη σχολιασμού στη συλλογή σχολιασμών της σελίδας
pdfDocument.Pages[1].Annotations.Add(freetext);
dataDir = dataDir + "SetFreeTextAnnotationFormatting_out.pdf";
// Αποθηκεύστε το ενημερωμένο έγγραφο
pdfDocument.Save(dataDir);            
```
