## การแนะนำ

ในโลกปัจจุบันที่มีเอกสารดิจิทัล ความสามารถในการสร้าง PDF แบบเลเยอร์อย่างมืออาชีพนั้นมีค่าอย่างยิ่ง ไม่ว่าคุณจะเพิ่มลายน้ำ แทรกข้อความบนรูปภาพ หรือสร้างเลย์เอาต์ที่ซับซ้อน คุณจะต้องมีโซลูชันที่มีประสิทธิภาพซึ่งช่วยให้คุณควบคุมเลเยอร์ PDF ได้อย่างเต็มที่ Aspose.PDF สำหรับ .NET เป็นเครื่องมืออันทรงพลังที่ทำให้กระบวนการนี้ราบรื่นและตรงไปตรงมา

## การนำเข้าแพ็คเกจที่จำเป็น

 หากต้องการเริ่มต้นสร้าง PDF หลายชั้น คุณต้องนำเข้าเนมสเปซที่เหมาะสม แพ็คเกจเหล่านี้ช่วยให้สามารถใช้คลาสที่จำเป็นทั้งหมดได้ เช่น`Document`, `Page`, `TextFragment` , และ`FloatingBox`.

```csharp
using Aspose.Pdf;
using Aspose.Pdf.Text;
using System.Drawing;
```

ตอนนี้ เมื่อผ่านขั้นตอนเบื้องต้นไปแล้ว เรามาเริ่มส่วนหลักกันเลย: การสร้างไฟล์ PDF หลายชั้น

คู่มือนี้ได้รับการออกแบบมาเพื่อนำคุณผ่านแต่ละขั้นตอนอย่างละเอียดและเหมาะสำหรับผู้เริ่มต้น ดังนั้น มาเริ่มกันเลยดีกว่า!

## ขั้นตอนที่ 1: เริ่มต้นเอกสารและตั้งค่าเส้นทาง

สิ่งแรกที่เราต้องการคือวัตถุเอกสาร PDF และวิธีการอ้างอิงตำแหน่งที่เราจะบันทึก PDF ขั้นสุดท้ายของเรา

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR DOCUMENT DIRECTORY";
Aspose.Pdf.Document doc = new Aspose.Pdf.Document();
```

 ในสไนปเป็ตนี้ เราได้สร้าง`Document` วัตถุที่แสดงถึง PDF ของเรา`dataDir` ตัวแปรควรตั้งค่าไปยังไดเร็กทอรีที่คุณต้องการบันทึกไฟล์ PDF ที่คุณสร้างขึ้น

## ขั้นตอนที่ 2: เพิ่มหน้าลงในเอกสาร PDF ของคุณ

เอกสาร PDF ทุกฉบับประกอบด้วยหน้าหนึ่งหน้าขึ้นไป มาเพิ่มหน้าในเอกสารของเรากัน

```csharp
Aspose.Pdf.Page page = doc.Pages.Add();
```

โค้ดนี้จะเพิ่มหน้าว่างลงในเอกสาร ง่ายใช่ไหม? ต่อไปเราจะเพิ่มเลเยอร์ลงในหน้านี้

## ขั้นตอนที่ 3: สร้างและปรับแต่งส่วนข้อความ

ต่อไปเราจะสร้างข้อความส่วนย่อย ซึ่งเป็นบล็อกข้อความที่เราสามารถปรับเปลี่ยนสี ขนาด และตำแหน่งได้

```csharp
Aspose.Pdf.Text.TextFragment t1 = new Aspose.Pdf.Text.TextFragment("paragraph 3 segment");
t1.TextState.ForegroundColor = Color.Red;
t1.IsInLineParagraph = true;
t1.TextState.FontSize = 12;
```

นี่คือสิ่งที่เกิดขึ้น:
-  การ`TextFragment` วัตถุ`t1` จะมีการเริ่มต้นด้วยข้อความว่า "ส่วนย่อหน้าที่ 3"
-  เราเปลี่ยนสีข้อความเป็นสีแดงโดยใช้`ForegroundColor` คุณสมบัติ.
-  ขนาดของข้อความถูกกำหนดเป็น 12 พอยต์ และจัดวางให้เรียงตามย่อหน้าโดยใช้`IsInLineParagraph`.

## ขั้นตอนที่ 4: เพิ่มส่วนข้อความลงใน FloatingBox

 ตอนนี้เรามีข้อความส่วนแล้ว เราจำเป็นต้องวางไว้ใน PDF แทนที่จะเพิ่มลงในหน้าโดยตรง เราจะใช้`FloatingBox` เพื่อกำหนดตำแหน่งให้เจาะจง

```csharp
Aspose.Pdf.FloatingBox TextFloatingBox1 = new Aspose.Pdf.FloatingBox(117, 21);
TextFloatingBox1.ZIndex = 1;
TextFloatingBox1.Left = -4;
TextFloatingBox1.Top = -4;
page.Paragraphs.Add(TextFloatingBox1);
TextFloatingBox1.Paragraphs.Add(t1);
```

มาแยกส่วนนี้ออก:
-  เราสร้าง`FloatingBox` และกำหนดขนาด (117x21)
-  การ`ZIndex` คุณสมบัติถูกตั้งเป็น 1 ซึ่งหมายความว่าจะอยู่ที่เลเยอร์ล่างสุด
-  การ`Left` และ`Top` คุณสมบัติจะกำหนดตำแหน่งที่แน่นอนของกล่องบนหน้า
-  สุดท้ายนี้ ส่วนที่ตัดตอนมา`t1`จะถูกเพิ่มเข้าไปในกล่องลอยซึ่งจะถูกเพิ่มเข้าไปในหน้าต่อไป

## ขั้นตอนที่ 5: แทรกภาพลงใน FloatingBox อื่น

 ต่อไปเราจะเพิ่มรูปภาพลงใน PDF เช่นเดียวกับข้อความ เราจะวางไว้ภายใน`FloatingBox`.

```csharp
Aspose.Pdf.Image image1 = new Aspose.Pdf.Image();
image1.File = dataDir + "test_image.png";
Aspose.Pdf.FloatingBox ImageFloatingBox = new Aspose.Pdf.FloatingBox(117, 21);
ImageFloatingBox.Left = -4;
ImageFloatingBox.Top = -4;
ImageFloatingBox.ZIndex = 2;
ImageFloatingBox.Paragraphs.Add(image1);
page.Paragraphs.Add(ImageFloatingBox);
```

รายละเอียดมีดังนี้:
-  เราสร้าง`Image` วัตถุและกำหนดเส้นทางไปยังไฟล์รูปภาพ
-  ใหม่`FloatingBox` ถูกสร้างขึ้นมาสำหรับรูปภาพโดยให้มีขนาดเท่ากับกล่องข้อความแบบลอย
-  กล่องลอยภาพจะวางเลเยอร์เหนือกล่องลอยข้อความโดยการตั้งค่า`ZIndex` ถึง 2.
-  การ`Left` และ`Top` คุณสมบัติจะวางตำแหน่งรูปภาพให้ตรงจุดที่เราต้องการ
- รูปภาพจะถูกเพิ่มลงในกล่องลอย จากนั้นจึงเพิ่มลงในหน้า

## ขั้นตอนที่ 6: บันทึกเอกสาร PDF

ในที่สุดเราจะบันทึกไฟล์ PDF หลายชั้นที่เพิ่งสร้างไปยังไดเร็กทอรีที่ระบุ

```csharp
doc.Save(dataDir + @"Multilayer-2ndApproach_out.pdf");
```

บรรทัดนี้จะบันทึกไฟล์ PDF ของคุณด้วยชื่อ "Multilayer-2ndApproach_out.pdf" ในไดเร็กทอรีที่คุณระบุ ขอแสดงความยินดี คุณได้สร้าง PDF หลายชั้นโดยใช้ Aspose.PDF สำหรับ .NET สำเร็จแล้ว!
