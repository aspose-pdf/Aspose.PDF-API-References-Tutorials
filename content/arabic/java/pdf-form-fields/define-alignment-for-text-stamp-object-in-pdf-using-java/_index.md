## مقدمة

تُعد طوابع النص أداة متعددة الاستخدامات عندما يتعلق الأمر بالتعليق وإضافة المعلومات إلى ملفات PDF. ومع ذلك، لكي تكون فعّالة حقًا، فإن المحاذاة الصحيحة ضرورية. في هذه المقالة، سنستكشف كيفية تحديد المحاذاة لكائنات ختم النص في ملفات PDF باستخدام Java، مع الاستفادة بشكل خاص من قوة Aspose.PDF لـ Java.

## ابدء

قبل أن نتعمق في تفاصيل محاذاة ختم النص، من الضروري إعداد بيئة التطوير الخاصة بنا. تأكد من تثبيت Aspose.PDF for Java وتكوينه في مشروع Java الخاص بك. يمكنك الوصول إلى الموارد الضرورية هنا:

-  التوثيق:[مرجع Aspose.PDF لواجهة برمجة تطبيقات Java](https://reference.aspose.com/pdf/java/)
-  تحميل:[Aspose.PDF لجافا](https://releases.aspose.com/pdf/java/)

بمجرد أن يكون كل شيء في مكانه، دعنا ننتقل إلى جزء الترميز.

## إنشاء مستند PDF

للبدء، نحتاج إلى مستند PDF للعمل عليه. فيما يلي مخطط أساسي لكيفية إنشاء مستند PDF باستخدام Aspose.PDF لـ Java:

```java
// تهيئة مستند PDF
Document pdfDocument = new Document();

// إضافة صفحة إلى المستند
Page page = pdfDocument.getPages().add();

// تعيين خصائص الصفحة (على سبيل المثال، الأبعاد، الهوامش)
page.setPageSize(new PageSize(600, 400));
```

الآن بعد أن أصبح مستند PDF جاهزًا، دعنا ننتقل إلى تحديد طابع النص.

## تعريف طابع النص

ختم النص هو في الأساس جزء من النص الذي تريد إضافته إلى مستند PDF الخاص بك. يمكن أن يحتوي على معلومات متنوعة، مثل التواريخ أو العلامات المائية أو التعليقات التوضيحية. فيما يلي كيفية تعريف ختم النص الأساسي:

```java
// إنشاء طابع نصي
TextStamp textStamp = new TextStamp("Confidential");

// تكوين محتوى النص ومظهره
textStamp.getTextState().setFont(FontRepository.findFont("Arial"));
textStamp.getTextState().setFontSize(12);
textStamp.getTextState().setForegroundColor(Color.getRed());
```

الآن بعد أن أصبح لدينا ختم النص الخاص بنا، دعنا نستكشف خيارات المحاذاة.

## خيارات المحاذاة

يمكن أن يكون محاذاة طوابع النص في مستند PDF أمرًا بالغ الأهمية لتحقيق المظهر والشعور المطلوبين. يوفر Aspose.PDF for Java خيارات محاذاة مختلفة، بما في ذلك:

- المحاذاة إلى أعلى اليسار، أعلى الوسط، أعلى اليمين
- محاذاة إلى المنتصف الأيسر، المنتصف الأوسط، المنتصف الأيمن
- المحاذاة إلى أسفل اليسار، أسفل الوسط، أسفل اليمين

بالإضافة إلى ذلك، يمكنك أيضًا تحديد إحداثيات مخصصة للمحاذاة الدقيقة.

## إضافة طوابع نصية إلى PDF

بمجرد تكوين طابع النص وتحديد المحاذاة، حان الوقت لإضافته إلى مستند PDF. يمكنك تحديد الصفحة التي تريد وضع طابع النص عليها وتطبيق المحاذاة المطلوبة:

```java
// إضافة ختم النص إلى صفحة معينة
page.addStamp(textStamp);

// قم بمحاذاة ختم النص إلى الجزء العلوي الأوسط من الصفحة
textStamp.setVerticalAlignment(VerticalAlignment.Top);
textStamp.setHorizontalAlignment(HorizontalAlignment.Center);
```

## تطبيق المحاذاة

الآن بعد أن قمنا بتنفيذ المحاذاة في الكود الخاص بنا، حان الوقت لاختبارها على مستند PDF نموذجي. تأكد من أن لديك ملف PDF جاهزًا للاختبار، وقم بتشغيل تطبيق Java الخاص بك. سترى كيف يتم محاذاة ختم النص بشكل مثالي وفقًا لمواصفاتك.
